## 1. 总体框架：四大核心原则与完整交易系统

### 1.1 海龟培训的四大核心原则

整个海龟培训计划，乃至所有成功交易的基础，都可以归结为四句话：

| 原则 | 内涵 | 失败后果 |
|------|------|----------|
| **掌握优势** | 找到一个期望值为正的交易策略，长期看能创造正回报 | 赌徒式的负和游戏，注定亏损 |
| **管理风险** | 控制风险、守住阵地，确保等得到系统发挥作用的那一天 | 在正期望兑现前就被破产风险淘汰 |
| **坚定不移** | 唯有坚定不移地执行策略，才能真正获得系统的正期望值 | 错过大趋势，全年利润化为泡影 |
| **简单明了** | 海龟方法精髓很简单——抓住每一个趋势，大部分利润来自两三次成功交易 | 复杂系统脆弱，难以坚持 |

**关键认知**：理查德·丹尼斯打赢了与比尔·埃克哈特的赌——交易才能可以传授。但首届13名海龟中仍有1/3到1/2表现逊色甚至失败。差别不在方法（所有人都学同样的法则），而在**心理特征**——能否坚定不移地执行。作者柯蒂斯·费思是唯一在第一个月满仓抓住民用燃料油大趋势的海龟，回报是其他人的3倍。

### 1.2 完整交易系统的六要素

海龟系统是一个**完整的机械性交易系统**，囊括交易中的每一个环节，不给交易者留下任何主观思考的余地。一个完整系统必须回答以下六个问题：

| 要素 | 问题 | 海龟的回答 |
|------|------|------------|
| **市场** | 买卖什么？ | 流动性最高的美国期货市场 |
| **头寸规模** | 买卖多少？ | 基于波动性（N/ATR）标准化，1N=账户1% |
| **入市** | 什么时候买卖？ | 20日突破（系统1）/ 55日突破（系统2） |
| **止损** | 什么时候放弃亏损头寸？ | 入市价±2N，单笔风险≤2% |
| **退出** | 什么时候退出赢利头寸？ | 10日反向突破（系统1）/ 20日反向突破（系统2） |
| **战术** | 怎么买卖？ | 限价单、买强卖弱、合约滚动 |

**为什么用机械系统**：
- 系统在闭市冷静时设计规则，成为市场波动中的理性行为之锚
- 知道系统能在长期内赚钱，就更容易在不利时期坚持信号
- 自主判断往往让你"在应该大胆的时候畏首畏尾，在应该谨慎的时候胆大包天"
- 一个千锤百炼的机械系统能赋予你**信心、统一性和纪律性**

**理查德·丹尼斯的名言**：
> "你可以把我的交易法则登在报纸上，但没人会遵守它们。关键是统一性和纪律性。几乎每一个人都可以列出一串法则，而且不比我们的那些法则差多少。但他们不能给别人信心，而唯有对法则充满信心，你才会坚持这些法则，即使遭遇逆境。"

## 2. 海龟思维：认知偏差与心理控制

> 人类的情绪既是机会所在，也是最大的挑战。掌控了它，你就能成功；忽视了它，你就危险了。

### 2.1 交易的四种情绪陷阱

| 情绪 | 表现 | 危害 |
|------|------|------|
| **希望** | "我当然希望买了之后马上就涨" | 促使在不该入市时建仓 |
| **恐惧** | "我再也赔不起了，这次躲远点" | 错过正期望的系统性机会 |
| **贪心** | "我赚翻了，要把头寸扩大一倍" | 在盈利后过度加仓，毁灭性衰落 |
| **绝望** | "这个系统不管用，我一直赔钱" | 在系统即将发挥作用时放弃它 |

### 2.2 八大认知偏差——市场机会的根源

人类的系统性、重复性非理性导致市场波动，海龟方式之所以有效，正因为它以这种非理性波动为基础。每一种认知偏差都意味着赚钱的机会。

| 认知偏差 | 定义 | 对交易的影响 |
|----------|------|--------------|
| **损失厌恶** | 不赔钱远比赚钱重要；损失的心理影响比赢利大一倍 | 无法执行机械系统的止损，因对损失感受过强 |
| **沉没成本效应** | 更重视已花掉的钱，而非未来要花的钱 | 不愿止损，"白白浪费那1000美元"，直到赔光30%~50% |
| **处置效应** | 早早兑现利润，却让损失持续 | 大损失无法弥补，潜力交易早早终结 |
| **结果偏好** | 根据决策结果而非决策质量来判断好坏 | 一连串亏损后怀疑正确的方法 |
| **近期偏好** | 更重视近期数据或经验 | 因近期赔钱就放弃长期有效的系统 |
| **锚定效应** | 过度依赖容易获得的信息（如近期高点） | "等价格回到X美元我就卖"，结果一路跌到1/100 |
| **潮流效应** | 盲目相信一件事只因其他人都相信 | 泡沫末期追高 |
| **小数定律** | 从太少信息得出结论 | 系统6次测试4次成功就当"好系统"，过早建立/失去信心 |

**核心机制（以沉没成本+损失厌恶为例）**：
新手设止损点亏损10%退出 → 价格跌破止损 → 损失厌恶让他不愿"把损失变成现实" → 沉没成本效应让他想"挽回那1000美元" → 继续持有 → 跌得越多越不愿退出 → 最终恐慌性逃出，账户赔掉30%~50%。

### 2.3 行为金融学的启示

- 市场参与者**不是理性的**——这是交易优势的来源
- 在紧张情绪中，人对风险和事件概率会作出糟糕判断
- 成功的交易者理解这些现象，并从中获利
- 其他人的错误判断就是你的机会，这些错误终将在市场价格变化中现出原形

## 3. 市场状态与交易策略类型

### 3.1 投机市场的四种状态

海龟们**从不预测市场动向，而是寻找市场处于某种特定状态的指示信号**。优秀的交易者不预测下一步会怎么样，而是观察信号，判断市场现在处于什么状态。

| 市场状态 | 特征 | 适合的策略 |
|----------|------|------------|
| **稳定平静** | 价格在较小范围内上下波动，很少超出 | 反趋势交易 |
| **稳定波动** | 有大的日间/周间变化，但无重大月际变化 | 反趋势交易、波段交易 |
| **平静的趋势** | 几个月中缓慢运动，无剧烈回调 | **趋势跟踪（最佳）** |
| **波动的趋势** | 有大变化，偶尔伴剧烈短期逆转 | 趋势跟踪（较棘手） |

**关键认知**：
- 市场经常同时呈现两种状态特征，某个属性会由低变高或由高变低
- 一个市场可能从"平静的趋势"逐渐转变为"波动的趋势"
- 判断市场状态是选择策略的前提——如果市场状态对系统不利，必须远离

### 3.2 四种交易策略类型

| 策略 | 时间跨度 | 原理 | 适合市场状态 |
|------|----------|------|--------------|
| **趋势跟踪** | 几个月大趋势 | 历史高位/低位入市，逆转维持几周即退出 | 平静的趋势 |
| **反趋势交易** | 短期 | 新高突破大多不会引发趋势，在接近新高时卖空 | 稳定波动 |
| **波段交易** | 3-4天 | 短期价格模式判断方向 | 任何波动性市场 |
| **当日交易** | 当日内 | 头寸交易/抢帽子/套利 | — |

### 3.3 趋势跟踪策略的三大特征

趋势跟踪是海龟的核心策略，但它有三个让人难以坚持的特点：

1. **大趋势很少出现**——典型趋势跟踪系统65%~70%的交易是赔钱的
2. **衰落期难熬**——趋势结束后会有衰落期，可能持续几个月；衰落程度大致接近回报水平（年回报30%的系统，衰落期可能下跌30%）
3. **需要较大资金量**——入市价与止损价差距大，资金太少会提高破产概率

## 4. 系统优势方法论

> 能否掌握优势是专家与业余者之间的区别。如果你没有优势，交易的成本会让你赔钱。

### 4.1 优势的定义

**优势**（借鉴自赌博理论）是指一种可以利用的统计学优势，以会重复发生的市场行为为基础。**最好的优势来自于人类认知偏差所酿成的市场行为**。

**期望值**：量化优势的方法，来自赌博理论。
- 期望值 = 每笔交易的平均赢利 ÷ 每笔交易的平均风险投入
- 期望值为0.2意味着每1美元风险投入能获20美分利润
- 风险投入 = |入场价 − 止损价| × 合约数量 × 合约规模

### 4.2 系统优势的三大要素

| 要素 | 含义 |
|------|------|
| **资产组合的选择** | 决定应该进入哪些市场的运算系统 |
| **入市信号** | 决定什么时候开始一笔交易的运算系统 |
| **退出信号** | 决定什么时候退出一笔交易的运算系统 |

**关键认知**：入市信号和退出信号必须配对。趋势跟踪入市配趋势跟踪退出，反趋势入市配反趋势退出。某个入市信号可能短期有显著优势但中期没有——优势只在对应的时间跨度内体现。

### 4.3 优势比率（E-比率）

衡量入市信号优势的科学方法。核心思路：比较入市后"好的变动"（MFE，最大有利变动幅度）与"坏的变动"（MAE，最大不利变动幅度）。

**E-比率计算四步**：
1. 为每一个入市信号计算指定时间段内的MFE和MAE
2. 将各MFE和MAE分别除以入市时的ATR（波动性标准化）
3. 调整后的MFE和MAE分别求和，除以入市信号总次数，得调整后平均MFE和MAE
4. **E-比率 = 调整后平均MFE ÷ 调整后平均MAE**

**解读**：
- E-比率 = 1.0 → 随机性入市（无优势），好变动与坏变动相等
- E-比率 > 1.0 → 存在正优势
- E-比率 < 1.0 → 存在负优势

**EN-比率**：N代表考察天数，如E10-比率用10天的MFE和MAE。

**重要发现**（唐奇安通道突破系统，1996-2006年28个美国期货市场）：
- E5-比率 = 0.99，E10-比率 = 1.0（短期无优势）
- E70-比率 = 1.20（中期有20%优势）
- **教训**：突破法的短期不利变动一般大于有利变动（这是突破法交易者难以克服的心理障碍），优势要在中期才体现

### 4.4 趋势组合过滤器

**规则**：只在50日均线高于300日均线的市场中做多，在50日均线低于300日均线的市场中做空。

**效果**（与突破法结合）：
- E70-比率从1.20提升到1.33
- E120-比率达到1.6左右
- 优势比率曲线变得更高、更平滑
- 剔除了与大趋势方向相反的突破交易（这些交易最易造成重大不利变动）

**核心原理**：过滤器把处于不利状态的市场剔除出去，让系统只在有利状态下交易，从而提高稳健性。

### 4.5 退出策略的优势

退出策略的优势更难单独衡量，因为退出策略与入市信号有复杂的交互作用。

**关键原则**：
- 退出之后所发生的事对你的成果毫无影响，有影响的只是退出之前所发生的事
- 应根据退出策略对**系统整体表现**的影响来评判，而非观察退出后的变化
- 赢利头寸的退出时机完全可以决定一笔交易的成败

## 5. 寻找交易时机：支撑、阻力与突破

> 优势是在买者和卖者之间的战场上发现的。你的任务就是找到这些战场，静观谁胜谁败。

### 5.1 支撑和阻力的心理根源

支撑和阻力来源于三种认知偏差：

| 认知偏差 | 对支撑/阻力的作用 |
|----------|-------------------|
| **锚定效应** | 近期新高/新低成为"锚"，之后每个价位都据此衡量 |
| **近期偏好** | 近期的高点/低点比早期更重要，影响更大 |
| **处置效应** | 早早锁定利润的倾向，使交易者在前期高点处卖出，形成阻力 |

**自我实现机制**：大多数交易者相信支撑/阻力现象的存在 → 这进一步强化了支撑/阻力。如果很多人相信价格跌到某水平会有大批买入者接盘，他们就更愿意在该价位等待，于是支撑位真的形成了反弹。

### 5.2 支撑位与阻力位的突破

**支撑/阻力的局限**：只是笼统概念，不是金科玉律。价格不一定在前期高/低点反弹，有时提前，有时延迟，有时根本不反弹。

| 交易者类型 | 利用方式 |
|------------|----------|
| 反趋势交易者 | 价格在前期高/低点反弹的倾向就是优势；若支撑守住就赚钱 |
| 趋势跟踪交易者 | 支撑/阻力的**突破**才是要害；突破后惯性前行比其他时候更显著 |

**突破后的认知滞后**（趋势跟踪者的优势来源）：
- 支撑位被打破时，人们仍固守以往信念，迟迟不肯转变
- 市场变化速度不够快，不足以反映新的现实
- 统计证据显示：市场在支撑/阻力被打破后的惯性前行趋势比其他时候更显著

**突破后的连锁反应**（以支撑位跌破为例）：
1. 所有在近期做多的交易者都赔了钱
2. 交易者看不到有希望提供支撑的近期价位
3. 想买的人会等待市场止跌（不急于买）
4. 持仓者恐慌抛售，把价格压得更低
5. 趋势延续，直到卖方耗尽能量、买方相信价格已低得不能再低

### 5.3 价格不稳定点——最佳交易机会

**定义**：接近支撑位和阻力位边缘的价位。在这些点上，价格不太容易保持稳定，更容易走高或走低。

| 情况 | 价格走向 |
|------|----------|
| 支撑位能守住 | 价格走高 |
| 阻力位能守住 | 价格在阻力位反弹走低 |
| 支撑位失守 | 价格沿突破方向继续前行相当长距离 |
| 阻力位失守 | 同上 |

**为什么价格不稳定点是绝好机会**：
- 好交易和坏交易之间的价格差异相对较小
- 犯错误的代价降低了
- 价格突破战"不可能既成功又失败"——只有两种可能

**战争比喻**：当价格处于支撑/阻力之间时，多空双方都不会真的投入战斗，难以判断谁胜谁败。随着价格逼近支撑/阻力，双方越来越投入，总有一方会失败。**最容易判断战争胜败的时候就是战争即将结束的时候**。

**止损设置示例**：
- 在2.10美元买入做多的反趋势交易者 → 止损2.04美元（支撑位突破）
- 在2.10美元突破位卖空的趋势跟踪者 → 止损2.15~2.16美元（支撑位守住则退出）

## 6. 风险衡量方法论

> 理解风险，尊重风险，这就是顶尖交易者们的标志。如果你不去注意风险，风险就会找上门来。

### 6.1 四大交易风险

| 风险 | 定义 | 特征 |
|------|------|------|
| **衰落** | 一连串损失使账户缩水 | 交易者放弃或失败的主要原因；新手高估自己承受衰落的能力 |
| **低回报** | 回报太低，赚到的钱微不足道 | 稳定创造高回报的系统在未来更容易创造理想回报 |
| **价格动荡** | 一个或多个市场中价格骤然变动，导致无可挽回的重大损失 | 由自然灾害、政治事件、经济灾难造成；历史检验会低估一倍 |
| **系统死亡** | 市场状态改变，致使曾经有效的系统突然失效 | 蹩脚测试方式造成，短期最优化系统风险更大 |

**关于衰落的关键认知**：
- 新手约翰看到历史测试中38%的衰落，心想"我能承受" → 实际遭遇42%衰落时开始怀疑系统
- 大多数人承受不住30%~40%的衰落，会放弃系统或有选择地交易，错失赚钱机会
- 作者估计自己最大衰落幅度约70%，能承受这种波折的人很少

**关于价格动荡的真实教训**：
- 1987年黑色星期一，作者为理查德·丹尼斯管理2000万美元账户，一夜损失1100万美元（全年利润烟消云散）
- 美联储一夜之间突然降息，欧洲美元跳高两个多点开盘
- 这次日内跌幅比系统任何一次历史检验结果都大一倍——历史检验会低估衰落风险一倍

### 6.2 风险的量化指标

| 指标 | 含义 |
|------|------|
| **最大衰落** | 测试期中从最高点到随后最低点的下跌百分比 |
| **最长衰落期** | 从一个顶峰到下一个新顶峰的最长周期，衡量恢复速度 |
| **回报标准差** | 回报率分散状况；低=接近平均值，高=差异大 |
| **R平方值** | 实际回报率与平均复合增长率的吻合程度；1.0=完全吻合 |

### 6.3 回报的量化指标

| 指标 | 含义 |
|------|------|
| **平均复合增长率（CAGR）** | 特定投资期间内的平均化复利率 |
| **滚动平均一年期回报率** | 连续滚动计算的一年期平均回报率，更反映典型一年回报 |
| **平均月度回报率** | 测试期内各月份回报率平均值 |

### 6.4 综合指标：夏普比率与MAR比率

**夏普比率**（1966年威廉·夏普发明）：
- = (平均复合增长率 − 无风险回报率) ÷ 回报率标准差
- 原为比较共同基金（非杠杆股票组合长期持有）而设计
- **局限**：对期货对冲基金不够好，因为期货有管理策略风险、分散化不足、潜在杠杆风险、信心风险

**关键警示——稳定性≠低风险**：
> 在很多情况下，回报的稳定性越高，实际风险水平就越大。

- LTCM（长期资本管理公司）策略曾几年内年均40%回报且相当稳定（高夏普比率），但1998年俄罗斯债券违约后几乎赔光47亿美元
- Amaranth对冲基金90亿美元在两个月内损失65%，此前夏普比率也不错
- 许多卖出期权的基金一边创造稳定回报，一边严重暴露在价格动荡风险中

**MAR比率**：
- = 年均回报率 ÷ 最大衰落幅度（月末数据计算）
- 快捷直接的风险回报衡量
- **注意**：用月末数据会低估实际衰落；作者用最高点到最低点跌幅计算，更准确

### 6.5 模仿效应与系统死亡

**模仿效应**：在风险回报比率上拥有傲人记录的策略最容易被整个行业群起模仿，数十亿美元竞相追随，结果自毁长城，超出市场承受能力，早早成为系统死亡的牺牲品。

**反例——趋势跟踪的生命力**：
- 大多数大投资者很难忍受趋势跟踪的大衰落和价值波动
- 正因如此，趋势跟踪策略在长期内始终有效
- 每当大量资金在稳定回报期后涌入 → 通常出现几个艰苦年头
- 当投资者在艰苦期后纷纷撤资 → 好时期再度来临
- 作者至少三四次听到"趋势跟踪已失效"的论调——这往往意味着遍地黄金的市场已不远

## 7. 资金管理：生存第一与头寸规模

> 破产风险是最需要警惕的风险。如果你对它掉以轻心，它随时可能会降临，像个夜贼一样把你洗劫一空。

### 7.1 破产风险

**定义**：在一连串失败中损失过大，不得不退出交易。

**重要特征**：破产风险随赌注增加而**不成比例地迅速增大**。每次赌金翻一番，破产风险可能翻两番、三番甚至四番。

**交易者破产的三大主因**（并非随机性不利变化）：
1. **没有计划**：依据直觉、传闻、猜测和对自己预测能力的自信
2. **风险过大**：承受比谨慎水平大5~10倍的风险
3. **不切实际的期望**：对赢利能力和回报水平抱有不切实际的期望（如医生相信自己聪明就能在交易中同样成功）

### 7.2 生存第一法则

> 交易的首要目标应该是生存。时间站在你这边。一个期望值为正的系统或方法早晚会给你带来财富。但前提是——你必须留在游戏场中。

**死亡的两种形式**：
- **痛苦的慢性死亡**：没完没了的衰落让交易者在苦闷中放弃
- **戏剧性的快速死亡**：崩溃，如价格动荡中一夜破产

**关键认知**：
- 大多数新手高估自己承受痛苦的能力（自以为能承受30%~40%衰落，实际不能）
- 想获得100%以上回报率，必须做好承受70%大衰落的准备
- **复利的力量**：5万美元本金 + 30%年均回报 + 20年 ≈ 1000万美元——前提是没有中途破产
- 强烈建议新手头几年采用保守方式，每年20%~30%稳定回报就能在短期内带来巨大财富

### 7.3 N值——波动性标准化头寸规模

这是海龟系统**最核心的创新**：根据市场波动性调整每个市场的头寸规模，使所有市场的绝对波动幅度大致相等。

**N的含义**：真实波动幅度的20日指数移动平均值，即ATR（Average True Range）。

**真实波动幅度（TR）的计算**：
```
TR = Max(H − L, H − PDC, PDC − L)
```
- H = 当日最高价，L = 当日最低价，PDC = 前一日收盘价

**N的计算**（指数移动平均）：
```
N = (PDN × 19 + TR) / 20
```
- PDN = 前一日的N值
- 首次计算N时，用真实波动幅度的20日简单平均值

**美元波动幅度**：
```
绝对波动幅度 = N × 每一点数所代表的美元
```

**头寸单位规模公式**（让1N = 账户净值的1%）：
```
头寸单位规模 = (账户净值 × 1%) ÷ (N × 每点数美元价值)
```

**示例**（2003年3月民用燃料油，2002年12月4日）：
- N = 0.0141
- 账户规模 = 1,000,000美元
- 每点数美元 = 42,000（每份合约42,000加仑）
- 头寸单位规模 = (1,000,000 × 0.01) ÷ (0.0141 × 42,000) = 10,000 ÷ 592.2 ≈ **16.88份合约**
- 因合约不可拆分，舍去小数 = **16份合约**

**波动性标准化的价值**：
- 不同市场上的不同交易在盈亏概率上相同（都有同样机会赚/赔1美元）
- 提高了多重市场分散化的效果
- 波动性弱的市场，合约持有量更多（一次大趋势能打大胜仗）
- 波动性强的市场，合约持有量更少

**小账户的局限**：10万美元账户的头寸单位可能仅1份合约，调整精确性太低，大大降低分散化效果。合理分散化至少需要10万美元。

### 7.4 四大层面头寸单位限制法则

这是海龟系统**不可或缺的组成部分**，用于控制总体风险水平，特别是在价格动荡中把损失最小化。1987年股崩次日，这些限制为里奇挽回了超过1亿美元的损失。

| 层面 | 限制范围 | 头寸单位上限 |
|------|----------|--------------|
| 1 | 单个市场 | 4 |
| 2 | 高度关联的多个市场 | 6 |
| 3 | 松散关联的多个市场 | 10 |
| 4 | 单个方向（多头或空头） | 12 |

**高度关联市场组合**：民用燃料油与原油、黄金和白银、所有外汇、利率类期货（短期国债与欧洲美元）等。

**松散关联市场组合**：黄金与铜、白银与铜等。

**单个方向**：一个海龟理论上可同时持有12个空头单位和12个多头单位。

**头寸单位限制的额外价值——滤除滞后市场**：
- 利率期货4个市场（欧洲美元、长期国债、短期国债、两年期中期国债）常同时发出信号
- 海龟一般只持有前两个发出信号市场的头寸
- 最迟发出信号的市场趋势往往不会延续太久，更可能以亏损收场
- **警告**：很多人测试海龟系统时去掉了头寸单位限制，导致结论错误

### 7.5 账户规模调整

- 海龟账户是名义账户，每年年初根据上一年度盈亏状况调整
- **亏损10% → 账户规模缩减20%**
- 例：100万美元账户亏损10万（10%）→ 账户缩至80万美元；再亏8万（80万的10%，共亏18万）→ 账户缩至64万美元
- 这种机制在亏损时强制降低风险，保护生存

### 7.6 风险衡量法则

- 回顾过去30~50年的严重价格动荡，思考某些类型头寸可能遭受的损失
- 用电脑模拟软件，可以看出在那些灾难期会持有什么头寸、遭遇什么衰落
- 设想更可怕的事情（如核弹袭击主要金融中心）会发生什么
- 任何鼓吹100%以上回报率的豪言壮语，都不要忘记价格动荡的可能

## 8. 海龟式积木：构建系统的五大工具

> 不要整天去膜拜杂志上的那些梦幻工具，学会如何用好最基础的工具才是第一位的。重要的不是你的工具有多大威力，而是你能不能用好这些工具。

积木是用来指示市场状态变化的工具：什么时候从稳定状态转变为有趋势状态，什么时候又回归稳定。没有任何积木是万能的，它们最多只能帮你判断趋势开始或结束的概率。

| 积木 | 原理 | 应用 |
|------|------|------|
| **突破** | 价格冲破过去特定时期内的最高/最低水平 | 原版海龟系统的首要工具；20日突破指示短期趋势，55日突破指示长期趋势 |
| **移动平均价** | 连续计算的特定时段内平均价格 | 短期均线穿越长期均线则顺势入市（如20日EMA穿越70日EMA） |
| **波幅通道** | 移动平均价 + 基于标准差或ATR的特定数值 | 价格超越通道意味着趋势启动 |
| **定时退出** | 在事先确定的特定时间退出（如10天或80天后） | 避开趋势衰竭导致的衰落，常在衰落之前退出 |
| **简单回顾** | 当前价格与早些时候的历史价位比较 | 如：价格 > 100天前价格 + 2ATR → 买入 |

**移动平均价的注意点**：
- 简单移动平均价（SMA） vs 指数移动平均价（EMA）
- 越来越复杂的移动平均指标在实际应用中大多并不有效，更可能造成曲线拟合

**一个重要忠告**：
- 如果一个市场开始上扬，无论用什么趋势跟踪系统，迟早都会引发入市信号
- 所有积木都可以调整反应速度，用任何积木搭建的系统不会有太大区别
- **与其整天寻找新颖、完美的超级指标，不如把时间花在更有意义的地方——用简单的系统**

## 9. 原版海龟交易系统完整规则

这是附录中给出的完整海龟交易系统全套法则。

### 9.1 市场选择

**首要标准：流动性**。海龟只选流动性最高的市场（理查德·丹尼斯的选择标准），因为大交易会导致小市场剧烈动荡。

**海龟交易的市场**（1980年代）：
- 芝加哥期货交易所：30年期美国长期国债、10年期美国中期国债
- 纽约咖啡糖与可可交易所：咖啡、可可、糖、棉花
- 芝加哥商业交易所：瑞士法郎、德国马克、英镑、法国法郎、日元、加拿大元、标普500指数、欧元、90天美国短期国债
- 纽约商品交易所：黄金、白银、铜
- 纽约商业期货交易所：原油、民用燃料油、无铅汽油

**排除谷物**（丹尼斯账户已达法定头寸上限）和**肉类**（场内交易者腐败问题）。

**规则**：海龟有权不选择某产品，但一旦排除某市场，就再也不能进入该市场（保持行动统一性）。

### 9.2 头寸规模

详见[7.3 N值](#73-n值波动性标准化头寸规模)。核心要点：
- 1N的价格变动 = 账户净值的1%
- 每周一拿到头寸单位规模表，列明每种合约的N值和头寸单位规模
- 头寸单位 = 单个头寸和整个头寸组合的风险衡量指标

### 9.3 入市策略——两个突破系统

| 系统 | 突破周期 | 特点 |
|------|----------|------|
| **系统1**（短期） | 20日突破 | 中期系统，4个交易周 |
| **系统2**（长期） | 55日突破 | 长期系统，12个交易周 |

**突破定义**：价格超越过去特定时期内的最高点或最低点。海龟总是在突破发生时**立即入市**，不等当日收盘或次日开盘。跳空开盘时，若开盘价已跳过突破价，则在开盘时入市。

#### 系统1入市法则
- 价格超越20日最高点1个最小单位 → 买入1个头寸单位（做多）
- 价格跌破20日最低点1个最小单位 → 卖出1个头寸单位（做空）
- **过滤法则**：若上一次突破是**赢利性突破**，则当前系统1信号被忽略
  - 亏损性突破定义：突破后价格在头寸有机会退出获利（10日突破退出）前发生2N不利变动
  - 无论上次突破方向如何，只要是亏损型，新突破（无论方向）即视为有效信号
- **保障性突破点**：若信号因过滤法则被忽略，交易者将在55日突破点入市（避免错过重大趋势）

#### 系统2入市法则
- 价格超越55日最高/最低点1个最小单位 → 入市
- **所有突破都视为有效信号**，无论上次突破是亏损性还是赢利性

#### 逐步建仓（金字塔加仓）
- 首先在突破点建立1个单位头寸
- 按 **1/2N** 的价格间隔逐步扩大头寸
- 间隔以上一份定单的实际成交价格为基础
- 若最初突破交易发生1/2N成交价偏差，新定单价格将与突破点相差1N
- 持续到头寸规模达到上限（市场活跃时一天内可加满4个单位）

**示例**（黄金，N=2.50，55日突破价=310.00）：
| 单位 | 入市价 |
|------|--------|
| 第1个 | 310.00 |
| 第2个 | 310.00 + 1/2×2.5 = 311.25 |
| 第3个 | 311.25 + 1/2×2.5 = 312.50 |
| 第4个 | 312.50 + 1/2×2.5 = 313.75 |

**统一性的重要性**：一整年大部分利润可能就来自两三次大赢利交易。哪怕只忽略或错过一个信号，全年回报率也可能大打折扣。表现最好的海龟都是入市法则的坚定信徒。

### 9.4 止损——2N法则

**铁律**：任何一笔交易的风险程度不得超过2%。

**标准止损**：
- 1N = 账户1%，因此2%风险限制下，价格变动上限 = 2N
- 多头：止损价 = 入市价 − 2N
- 空头：止损价 = 入市价 + 2N

**加仓后的止损调整**：若按1/2N间隔补充头寸单位，之前头寸单位的止损点相应调整1/2N。整个头寸的止损点与最新添加的头寸单位相距2N。

**示例**（原油，N=1.20，55日突破价=28.30）：
| 阶段 | 单位 | 入市价 | 止损价 |
|------|------|--------|--------|
| 第1单位 | 1 | 28.30 | 25.90 |
| 加第2单位 | 2 | 28.30 / 28.90 | 26.50（统一调整） |
| 加第3单位 | 3 | 28.30 / 28.90 / 29.50 | 27.10 |
| 加第4单位 | 4 | 28.30 / 28.90 / 29.50 / 30.10 | 27.70 |

**跳空高开情况**：若第4单位跳空高开在30.80成交，止损调整不同，第4单位止损28.40，前3单位止损27.70。

**执行方式**：海龟不真的下止损单（避免暴露头寸状况），而是事先确定止损价位，市场到达时通过限价单或市价单退出。**只要市场到达止损价位，毫不迟疑地坚决退出。**

**备选止损策略——双重损失（Whipsaw）**：
- 每笔交易风险上限 = 0.5%（即1/2N止损）
- 头寸单位止损退出后，价格恢复到最初入市价时重新建立该单位
- 优点：更高利润率；不需要随新单位补充调整之前止损点（最多4单位，总风险≤2%）
- 缺点：更难执行，损失次数更多，胜负比率大大降低

**海龟止损优势**：止损点以N为基础，与市场波动性息息相关。更具波动性的市场有更大止损范围，但每单位合约数更少 → 统一所有交易的风险水平。

### 9.5 退出——10日/20日突破退出

> 落袋为安的人永远也不会破产——但海龟们不会同意这种论调。趋势跟踪者最常犯的错误之一就是过早退出赢利头寸。

| 系统 | 退出法则 |
|------|----------|
| **系统1** | 10日突破退出：多头在价格跌破过去10日最低点时退出；空头在价格超过10日最高点时退出 |
| **系统2** | 20日突破退出：多头在20日向下突破时退出；空头在20日向上突破时退出 |

- 只要价格发生不利于头寸的10日/20日突破，**所有头寸单位**都退出
- 海龟一般不用止损单退出，而是紧密观察，在价格达到退出突破点时马上下单

**退出为何最难**：
- 必须等待市场创下10日/20日新低才能退出
- 往往意味着目睹20%、40%甚至100%的巨额利润瞬间蒸发
- 交易者有提早退出的强烈冲动
- 需要极强纪律性才能坚守阵地，忍受利润蒸发，抓住真正的大趋势
- **在大的赢利交易中遵守纪律和法则的能力就是经验丰富成功交易者的标志**

**为何不能过早退出**：
- 海龟系统在突破点入市，但大多数突破不会引发趋势 → 大多数交易是亏损的
- 如果少数赢利交易赚不到足够多的钱弥补损失，海龟会是输家
- 若总在1N利润退出、2N止损，赢利次数必须是亏损次数两倍才能盈亏平衡

### 9.6 战术执行

#### 下单原则
- **限价定单优于市价定单**：更容易以有利价格成交，成交价偏差更小
- 限价单把价格定在反弹区域底端，而非简单接受当前市价
- 小限价单不会触动市场，大限价单对市场的触动也小于市价单

#### 急变市场处理
- 市场变化太快时，限价单根本没机会成交
- **海龟原则：保持冷静，等市场稳定下来再做打算**
- 大多数新手惊慌失措发市价单，成交价是一天内最差的价格（最高点或最低点）
- 急变市场中流动性会暂时固定，卖出价急剧上升，买卖价差扩大
- 若稳定后价格已超过止损价，会退出但不惊慌

#### 同步入市信号
- 相互关联的市场常同时发出信号（如原油、民用燃料油、无铅汽油同时跳空开盘）
- 同一市场不同月份合约也可能同时出现信号
- 必须高效、快速行动，不能惊慌，不能乱下市价单

#### 买强卖弱
- 多个信号同时出现时，**在同一类市场中选择最强势的市场买入，选择最弱势的市场卖空**
- 同一时间只选择一个月份的合约，只建立一个头寸单位
- 最好的多头交易存在于最强势的市场中，最好的空头交易存在于最弱势的市场中

**衡量市场强弱的方法**：
1. 察看图表，视觉判断"看起来"更强/弱的市场
2. 比较各市场自突破后已变化了多少个N，在变动幅度最大的市场买入
3. 标准化比较：(当前价 − 3个月前价格) ÷ N值，最高=最强，最低=最弱

#### 合约滚动
- 转向新合约前注意：近期合约有好趋势但远期合约未必有同样势头（除非新合约趋势符合要求，否则不滚动）
- 赶在现有合约成交量和未平仓量大幅萎缩前滚向新合约
- 一般在合约到期前几个星期滚动，除非现有合约表现远好于后期新合约

### 9.7 调整交易规模
- 连续多月无趋势 → 账户净值可能遭受巨大损失
- 大捷之后 → 可能想扩大账户规模进而扩大头寸规模
- 损失初始账户10% → 账户规模缩减20%

## 10. 历史测试的四大谎言

> 骗子和无赖潜伏在黑暗的角落里，等待着不设防的猎物。不要成为他们的盘中餐。

历史测试结果与实际交易结果的差异主要由四大因素造成：

### 10.1 交易者效应（Trader Effect）

**定义**：交易行为本身可能改变交易赖以成功的潜在市场状态。一种近期表现突出的策略很可能被诸多交易者注意到，太多人利用同一种策略，策略将不再像从前那样有效。

**突破策略被利用的例子**：
- 知道ACME公司会在410.5美元大举买入1000份黄金合约
- 抢在前面买入，把价格推高到411美元，引发ACME买单
- 在411美元把1000份合约卖给ACME，赚20万美元
- ACME的突破法交易者手里有巨大亏损头寸，进入原因与历史检验原理完全不同

**应对方法**：
- 开发自己的系统，优势不容易被其他交易者毁掉
- 海龟会故意发出相反方向的假定单迷惑市场（如想买入先放假卖单）
- 像扑克牌的"虚张声势"，偶尔为之让对手摸不着头脑

### 10.2 随机效应（Random Effect）

**核心**：纯随机性因素可能造成交易结果的巨大差异，历史测试结果可能夸大了系统的内在优势。

**测试数据**（100次随机入市+定时退出策略）：
- 最好一次：年均回报16.9%，10.5年把100万变成550万
- 最差一次：年均亏损20%
- 加入趋势过滤器后：平均回报32.46%，平均衰落43.74%；最好53.3%回报/MAR 1.58，最差17.5%回报/衰落62.7%

**均值回归**：高个子父母的孩子更可能接近平均身高（基因可遗传，运气不可遗传）。同理，运气好的平庸交易者多于运气不佳的优秀交易者——拥有良好记录的人中，可能只有1/4是真正的优秀交易者。

**3年短期记录更不可靠**：100次随机测试中有17次MAR比率>1.54，7次回报率>52.2%，最好一次达71.4%回报率/MAR 2.07。

**结论**：好的投资者投资于人，而不是历史记录。透过表面记录深入研究记录背后的人。

### 10.3 最优化矛盾（Optimization Paradox）

**定义**：参数最优化过程有两种相互矛盾的效果：
- **正面**：提高系统在未来表现良好的概率
- **负面**：降低系统未来表现符合模拟测试结果的概率

**最优化是好事**（反驳"最优化有害"论）：
- 了解参数变化的影响总比忽略好
- 不做最优化 = 糊里糊涂被运气因素完全左右
- 示例：布林格突破系统2.4倍标准差 vs 3倍标准差，前者利润是后者8倍

**矛盾的图示解释**（山峰形曲线）：
- A点（非最优化参数）：左右移动系统表现有升有降，预测价值高
- B点（最优化参数，峰顶）：左右移动系统表现都下降，预测价值低，但实际表现仍高于A点
- **应采用B点**：即使未来有变化，B点左右的系统表现仍高于A点左右

**骗局温床**：不道德的系统兜售者炫耀特定市场最优化（特别是短期）的超高利润，但知道这不可能在实际交易中实现。

### 10.4 过度拟合（Overfitting）/曲线拟合

**定义**：系统变得过于复杂，与历史数据吻合度太高，以至于失去预测价值。市场行为轻微变化就会造成效果明显恶化。

**与最优化矛盾的区别**：过度拟合通常发生在系统变得过于复杂时；最优化是参数选择过程，过度拟合是添加法则导致复杂化。

**典型手法**：
- 添加一条法则"衰落达38%时头寸减小90%"→ 回报从41.4%升到45.7%，MAR从0.74升到1.17
- 但这条法则在整个测试期只发挥一次作用（发生在测试期快结束时）
- 衰落限制下调1%（38%→37%）→ 系统从年赢利45.7%变为年亏损0.4%

**峭壁（Cliff）现象**：参数值的微小变化引发交易结果的剧烈变化。峭壁出现是过度拟合的强烈信号。

**季节性过度拟合**：
- 10年数据只有10个例子，样本规模无统计学价值
- 加入"9月头两天+7月头25天削减头寸96%"法则 → 回报从45.7%升到58.2%，MAR从1.17升到1.48
- 实际回报只有41.4%，衰落56%，MAR 0.74——注定令人失望

**避免过度拟合**：
- 样本规模要有统计学价值（小数定律的陷阱）
- 检查参数变化是否产生峭壁
- 警惕只在少数关键交易中发挥作用的法则

## 11. 统计基础与稳健指标

### 11.1 测试样本的有效性

- 历史测试的预测价值来源于"过去是对未来的模拟"——市场反映人类永恒行为特征
- 未来永远不会等于过去，但好的系统能利用这些永恒特征赢利
- **样本规模**：越多越好；季节性因素10年只有10个例子，无统计学价值

### 11.2 稳健的衡量指标

针对夏普比率和MAR比率的缺陷，作者设计了更稳健的指标：

| 指标 | 计算方法 | 优势 |
|------|----------|------|
| **RAR（回归年度回报率）** | 用线性回归线斜率计算回报率，而非CAGR | 对单个异常高回报期不敏感，更稳健 |
| **R立方（稳健风险回报比率）** | RAR ÷ 平均衰落幅度（而非最大衰落） | 不依赖单一最大衰落事件，更稳健 |
| **稳健夏普比率** | RAR ÷ 回报率标准差 | 比传统夏普比率更稳健 |

**示例**：单独使用布林格突破多头系统RAR 38.2%/R立方1.19，空头系统RAR 14.5%/R立方0.41；两者结合后RAR 61.2%/R立方5.20——远优于单独使用任一系统。

### 11.3 样本代表性与蒙特卡洛检验

**样本代表性**：
- 测试样本应代表未来可能遇到的各种市场状态
- 测试期应包含多种市场状态（平静趋势、波动趋势、稳定平静、稳定波动）

**蒙特卡洛检验**：
- 通过随机调整交易顺序或参数，生成多种可能的结果分布
- 检验系统在不同情境下的稳健性
- 如果结果对参数或顺序高度敏感，系统可能不稳健

### 11.4 从虚拟测试到实战交易

- 历史测试结果是"最乐观的历史回顾"，未来实际结果很可能逊于最优化参数下的测试结果
- 把真金白银投入市场与练习或模拟完全不是一回事
- 必须亲自用历史数据检验系统——别人说的不算数
- 研究交易记录、每日净值报告，把系统的交易方式和损失频率摸熟

## 12. 防卫系统：稳健交易策略

> 交易不是赛跑，而是拳击。市场会挥动铁拳痛击你。要想获胜，你必须在12个回合的结束钟声敲响时仍然安然无恙地站立在拳击台上。

### 12.1 稳健交易策略的两大特征

**核心理念**：一旦你的交易策略考虑到未来的不可知性，你的表现反而会变得更加容易预测。因为未来的任何市场状态都已在你的预料之中，你不需要预测什么。

| 特征 | 原理 | 生态类比 |
|------|------|----------|
| **分散化** | 不依赖一两个市场/系统/物种完成一个任务 | 大自然不会只有一种食肉动物、一种食物源 |
| **简化** | 简单法则在各种境况下都能发挥作用，不依赖特殊环境 | 变化时期，病毒和细菌等简单生命体更坚强 |

**简化的重要性**：
- 复杂系统依赖特殊市场条件，未来不具备这些条件的可能性大
- 以基础概念为根基的简单法则在实际交易中适用性更强
- **长久来看，简单的系统更有生命力**

### 12.2 加强系统稳健性的方法

1. **确保系统法则能适应各种不同市场状况**
2. **让系统保持简明**，不容易受市场变化影响
3. **使用过滤器**剔除不利状态下的市场（如趋势组合过滤器）
4. **减少对特殊市场条件的依赖性法则**

### 12.3 选择多个不同的市场

**核心原则**：
- 市场越多，越可能在至少某一个市场中碰到有利状态
- 对趋势跟踪系统：参与的市场越多，其中某个市场存在趋势的可能性越大
- 市场间关联性不能太高（几种美国短期利率产品几乎总是同涨同跌，选择多个并不能提高分散化）

**排除市场的主要原因**：
1. **流动性问题**：低流动性市场在特殊情况下无人回应，更容易受价格动荡冲击
2. **市场类型不适合系统**

**不要因短期测试表现差而排除市场**：
- 有些市场每隔几年才出现一次趋势，5~10年测试不能充分反映潜力
- 有的市场利润水平不理想，但分散化价值大，权衡利大于弊
- 海龟时代错失的咖啡交易：本可赚1400万美元，相当于500万美元账户的280%回报

### 12.4 三大类市场

| 市场类型 | 例子 | 特征 | 趋势跟踪难度 |
|----------|------|------|--------------|
| **基本面市场** | 外汇、利率产品 | 价格变动主要动因是宏观经济事件，非交易行为 | 最容易把握（流动性最高、趋势最清晰） |
| **投机者市场** | 股票、咖啡、黄金、白银、原油 | 投机者影响力大于政府/大对冲者，价格由市场态度决定 | 较难把握 |
| **综合衍生市场** | e-mini标普500指数期货 | 投机是主要动力但有所缓和，动力平均化和中和 | 最难把握 |

**关键认知**：同一类别中所有市场都大同小异，偶然差异源自交易者记忆效应和大趋势动因的罕见性。只需根据市场类型和流动性作决策。

### 12.5 交易者记忆效应

**定义**：市场参与者对过去大趋势的记忆会影响当前市场行为。

**金银市场例子**：
- 1978年黄金涨到900美元/盎司，白银50美元/盎司
- 之后多年，每当市场稍涨，所有人争先抢购，价格忽上忽下，趋势跟踪者难有作为
- 20年后大多数人忘记1978年，2006年春季行情远比以前容易把握
- 同一市场，走势图对比像"变了样"

**启示**：一个市场过去20年没出现大趋势，不代表它不是好市场。只要容量足够大且不同于组合中其他市场，就应选择。

### 12.6 使用多个不同的系统

**系统分散化**：同时使用多个交易系统（特别是彼此差别极大的系统），交易稳健性大大提高。

**负相关系统的威力**：
- 布林格突破多头系统RAR 38.2%/R立方1.19
- 布林格突破空头系统RAR 14.5%/R立方0.41
- 两者结合后RAR 61.2%/R立方5.20（远优于单独使用任一系统）

**适合不同市场状态的系统结合**：
- 一个在有趋势市场表现良好，另一个在没有趋势市场更出众
- 一个遭遇衰落时另一个可能财源滚滚，反之亦然
- 大大提高交易稳健性

**资金限制**：
- 适度分散化长期趋势跟踪系统至少需10万美元
- 同时使用四五个不同系统可能需100万美元
- 这是许多人选择专业期货基金/对冲基金经理的原因

## 13. 像海龟一样思考：五大思维准则

> 好的交易无关乎对与错，只代表正确的交易。如果你想成功，必须把眼光放长远，忽视个别交易的个别结果。

### 13.1 避免结果偏好

**原则**：好的交易者考虑的是现在，而避免对未来考虑过多。

- 海龟不在乎对错，只在乎赚钱不赚钱
- 海龟不会扮演能预见未来的先知，从不会瞄一眼市场就说"金价就要上涨"
- 对未来，**细节不可知，但特征可以预测**——你不知道市场会涨会跌，但可以知道趋势即将出现、价格变动规律不会变（因为人类情感和认知特征不会变）
- 如果你大多数时间都是错的，赚钱反而容易得多（说明你没有试图预测未来）
- **个别交易的结果丝毫不能代表你的智力水平**

### 13.2 避免近期偏好

**原则**：海龟从整体角度观察过去，不会特别重视近期的事件。

**反面教训**（作者朋友做可可交易）：
- 1998年4月到11月，突破法做可可连续17笔赔钱
- 第18笔（11月6日）抓住大趋势，单笔赚24,940美元
- 之后又有几笔大赢利，最终总利润55,903美元
- 朋友因近期赔太多，认为可可风险大，完全错过这波行情

**启示**：
- 在某些市场可能要等上几年才能等到一个好趋势
- 过于看重近期情况，会错误认为这些市场不能碰
- 似乎正是在其他所有人都放弃之后，趋势才会出现

### 13.3 避免预测未来

**原则**：从概率角度考虑未来，而不是摆出一副预测的架势。

- 交易与保险非常相似：充满不确定性，你不知道一笔交易会不会赚钱
- 充其量只能相信，从长期看，回报将大于风险水平
- 不要试图作出正确预测，唯有使用概率对你有利的方法，才能在长期内成功

### 13.4 从概率角度考虑问题

**R乘数概念**（查克·布兰斯科姆发明）：
- R乘数 = 一笔交易的利润 ÷ 这笔交易的风险投入
- 示例：黄金450美元买入，止损440美元，风险投入1000美元；赚5000美元 = 5R交易

**海龟的概率思维**：
- 永远不会知道一笔交易最后会赚钱还是赔钱
- 只知道交易结果的大致分布形状
- 每一笔交易都有可能赚钱，但**可能性最大的是赔钱**
- 有些交易会得到4R~5R的中等回报，有些则是12R、20R甚至30R的大捷
- 胜利的成果足以弥补失败的损失，总会赢利
- **从概率角度考虑问题，正因为如此，在巨大风险和不确定性下依然信心十足**

### 13.5 对自己的交易结果负责

**原则**：不要把错误和失败归咎于其他人、市场、经纪人。

- 交易只是你和市场之间的事，你在市场面前无所隐瞒
- 喜欢推卸责任的人必败无疑
- 做了愚蠢的事要从错误中学习，不要装作没犯错
- 想办法避免再犯同样的错误

**海龟中的反面例子**：有海龟怀疑里奇向作者单独密授秘诀——这只不过是为自己不佳表现找借口。秘而不宣的诀窍并不存在，作者实际使用的方法比其他大多数海龟都简单得多。

### 13.6 像海龟一样思考的三大要点

1. **重要的是现在**：不要对过去念念不忘，也不要去预测未来。前者对你无益，后者是徒劳的。
2. **从概率角度考虑问题，不要预测未来**：不要试图作出正确预测，唯有使用概率对你有利的方法，你才能在长期内获得成功。
3. **对你自己的交易结果负责**：不要把你的错误和失败归咎于其他人、市场、你的经纪人等。要对自己的错误负责，从错误中学习。

## 14. 掌控心魔：心理建设

> 市场不会在乎你的感受。它既不会在你得意洋洋时吹捧你，也不会在你灰心丧气时安慰你。如果你不愿意接受市场的现实，不愿意承认自己的缺陷、恐惧和失败，你就不会成功。

### 14.1 克服自负心理

**自负的表现**：
- 新手喜欢自主交易（依赖主观判断），而非系统性交易（机械性法则）
- 自主交易成功时自尊感得到满足，可向朋友吹嘘
- 论坛上常见吹嘘"恰好上涨前买入""圣杯系统""3个月200%回报"——往往几个月后输光
- 自负者希望事事正确、预见到未来、知晓秘诀——很难避开认知偏差

**自负的危害**（乒乓球大战的启示）：
- 海龟中的乒乓球第一高手在锦标赛决赛输给作者
- 原因：获胜对他太过重要，压力影响发挥
- 这位高手在海龟计划中也不成功，把失败归咎于"没得到秘诀"
- 不愿面对现实：糟糕表现源于自身，是自负心理作怪

### 14.2 谦虚为上

**谦虚的作用**：
- 使你接受未来不可知的事实
- 让你放弃预测未来的企图
- 避免你从个人化角度看待失败的交易
- 帮助你接纳简简单单的交易法则，不去寻找无人知晓的秘密来证明自己与众不同

**切莫妄自尊大**（作者自省）：
- 作者曾在满仓获利时无意炫耀，被其他海龟称为"自高自大的蠢货"
- 那一天发誓再也不做妄自尊大的傻瓜
- 每次行动前花时间想想言行对其他人的影响

### 14.3 坚定不移

> 贵在坚持这条最重要的生活箴言总是说起来容易做起来难。

**建立信心的最佳方法**：
- 用交易模拟软件检验系统，看到过去恰似真实历史重现
- 测试各种交易系统，根据实际市场数据检验假设
- 亲自用历史交易数据检验系统——别人说的不算数
- 研究交易记录、每日净值报告，把系统的交易方式和损失程度、频率摸熟
- **如果你知道过去20年中曾有很多旷日持久的亏损期，你就更容易忍受一个历时8个月的亏损期**

**作者的特殊性**：由于生理和教育上的原因，坚持法则对作者不是难事。如果需要克服心理问题，可参考范·撒普、布雷特·斯蒂恩博格、阿里·基夫、马克·道格拉斯的著作。

### 14.4 成功海龟的交易心得（四大核心原则重申）

1. **掌握优势**：找到一个期望值为正的交易策略
2. **管理风险**：控制风险，守住阵地
3. **坚定不移**：唯有坚定不移地执行策略，才能真正获得系统的正期望值
4. **简单明了**：长久来看，简单的系统比复杂的系统更有生命力

**记住**：如果计划无法执行，计划就毫无意义。

### 14.5 尾声——人生哲学

- **没有冒险就没有收获**：风险是你的朋友，不要怕它，要理解它、控制它、与它共舞
- **失败不是坏事**：失败是学习的前提，如果不愿冒失败的风险，就什么也学不到
- **面对现实，果断改变**：不要在乎沉没成本，发现走错路就果断退出
- **金钱不是一切**：不在乎钱的人更容易发大财；在乎交易本身而非每天的资金流动
- **勇敢向前，大胆尝试**：结果也许不如期望好，但只要坚持不懈，最后可能比期望还好

## 15. 核心检查清单

### 15.1 系统设计检查清单

- [ ] 系统是否回答了完整交易系统的六个问题（市场、头寸规模、入市、止损、退出、战术）？
- [ ] 系统是否有正期望值？是否用历史数据检验过？
- [ ] 是否用E-比率衡量过入市信号的优势？
- [ ] 是否用趋势组合过滤器剔除了不利市场状态？
- [ ] 系统是否简单？是否避免了过度拟合？
- [ ] 是否检查过参数变化是否产生峭壁（过度拟合信号）？
- [ ] 是否用蒙特卡洛检验过稳健性？

### 15.2 资金管理检查清单

- [ ] 是否用N（ATR）标准化了所有市场的头寸规模？
- [ ] 1N是否等于账户净值的1%？
- [ ] 单笔交易风险是否≤2%（即2N止损）？
- [ ] 是否遵守了四大层面头寸单位限制（4/6/10/12）？
- [ ] 账户亏损10%时是否缩减规模20%？
- [ ] 历史测试的最大衰落是否在心理承受能力的一半以内？
- [ ] 是否考虑过30~50年价格动荡的最坏情况？

### 15.3 入市前检查清单

- [ ] 当前市场处于哪种状态（稳定平静/稳定波动/平静趋势/波动趋势）？
- [ ] 趋势组合过滤器是否允许此方向交易（50日均线 vs 300日均线）？
- [ ] 系统1：上一次突破是亏损性还是赢利性？（赢利性则忽略当前20日信号，等55日保障性突破）
- [ ] 系统2：所有55日突破都视为有效
- [ ] 是否准备好按1/2N间隔逐步建仓？
- [ ] 止损价是否已确定（入市价±2N）？
- [ ] 是否用限价单而非市价单？

### 15.4 持仓中检查清单

- [ ] 是否按1/2N间隔加仓至头寸上限？
- [ ] 加仓后是否调整了之前单位的止损点（上移1/2N）？
- [ ] 是否避免过早"落袋为安"？
- [ ] 是否忍受了利润蒸发（可能10%~30%甚至更多）？
- [ ] 市场急变时是否保持冷静，等市场稳定再做打算？
- [ ] 多个信号同时出现时是否买强卖弱？

### 15.5 退出检查清单

- [ ] 系统1：是否等待10日反向突破才退出？
- [ ] 系统2：是否等待20日反向突破才退出？
- [ ] 是否所有头寸单位一起退出？
- [ ] 是否避免用止损单退出（避免暴露头寸）？
- [ ] 合约到期前几个星期是否考虑滚动到新合约？

### 15.6 心理检查清单

- [ ] 是否避免结果偏好（不因个别交易结果评判决策质量）？
- [ ] 是否避免近期偏好（不因近期赔钱就放弃系统）？
- [ ] 是否从概率角度而非预测角度思考？
- [ ] 是否对自己的交易结果负责？
- [ ] 是否克服了自负心理，保持谦虚？
- [ ] 是否坚定不移地执行系统法则？

## 16. 关键公式与参数速查

### 16.1 波动性与头寸规模公式

| 公式 | 计算 |
|------|------|
| 真实波动幅度TR | Max(H − L, H − PDC, PDC − L) |
| N（ATR） | N = (PDN × 19 + TR) / 20（20日指数移动平均） |
| 首次N | 真实波动幅度的20日简单平均值 |
| 绝对波动幅度 | N × 每点数所代表的美元 |
| 头寸单位规模 | (账户净值 × 1%) ÷ (N × 每点数美元价值) |
| 美元波动幅度 | N × 每点数美元 × 合约数量 |

### 16.2 入市与突破公式

| 公式 | 说明 |
|------|------|
| 系统1入市 | 价格超越20日最高/最低点1个最小单位 |
| 系统2入市 | 价格超越55日最高/最低点1个最小单位 |
| 保障性突破点 | 55日突破（系统1信号被过滤时使用） |
| 逐步建仓间隔 | 1/2N（以上一份定单实际成交价为基础） |
| 亏损性突破定义 | 突破后价格在10日退出前发生2N不利变动 |
| 趋势组合过滤器 | 50日均线 > 300日均线 → 只做多；50日均线 < 300日均线 → 只做空 |

### 16.3 止损与退出公式

| 公式 | 计算 |
|------|------|
| 标准止损 | 入市价 ± 2N（单笔风险 = 2%） |
| 加仓后止损调整 | 之前单位止损点相应调整1/2N |
| 整体头寸止损 | 与最新添加单位相距2N |
| 双重损失止损 | 入市价 ± 1/2N（单笔风险 = 0.5%） |
| 系统1退出 | 10日反向突破（所有单位退出） |
| 系统2退出 | 20日反向突破（所有单位退出） |

### 16.4 头寸单位限制速查

| 层面 | 限制范围 | 头寸单位上限 |
|------|----------|--------------|
| 1 | 单个市场 | 4 |
| 2 | 高度关联的多个市场 | 6 |
| 3 | 松散关联的多个市场 | 10 |
| 4 | 单个方向（多头或空头） | 12 |

### 16.5 优势与风险衡量公式

| 公式 | 计算 |
|------|------|
| 期望值 | 每笔交易平均赢利 ÷ 每笔交易平均风险投入 |
| 风险投入 | \|入场价 − 止损价\| × 合约数量 × 合约规模 |
| E-比率 | 调整后平均MFE ÷ 调整后平均MAE |
| MFE/MAE标准化 | MFE（或MAE）÷ 入市时ATR |
| R乘数 | 一笔交易利润 ÷ 风险投入 |
| 夏普比率 | (CAGR − 无风险回报率) ÷ 回报率标准差 |
| MAR比率 | 年均回报率 ÷ 最大衰落幅度 |
| RAR | 线性回归线斜率计算的年度回报率 |
| R立方 | RAR ÷ 平均衰落幅度 |

### 16.6 关键参数总览

| 参数 | 数值 | 用途 |
|------|------|------|
| N的计算周期 | 20日 | ATR指数移动平均 |
| 账户风险/1N | 1% | 头寸单位规模计算 |
| 单笔最大风险 | 2%（2N） | 止损设定 |
| 系统1突破周期 | 20日 | 中期入市 |
| 系统2突破周期 | 55日 | 长期入市 |
| 系统1退出周期 | 10日 | 短期退出 |
| 系统2退出周期 | 20日 | 长期退出 |
| 加仓间隔 | 1/2N | 逐步建仓 |
| 趋势过滤器 | 50日均线 vs 300日均线 | 判断大趋势方向 |
| 账户亏损缩减阈值 | 10%亏损 → 缩减20% | 账户规模调整 |

## 结语：海龟方法论的整合

柯蒂斯·费思方法论的精髓在于**简单而坚定**：

- **没有优势**，交易只是负和游戏，佣金和滑点会吞噬本金
- **没有风险管理**，一次价格动荡或一连串衰落就能终结交易生涯
- **没有坚定不移**，即使有正期望系统也等不到它发挥作用
- **没有简单明了**，复杂系统脆弱，难以在市场变化中存活

海龟方法论的核心链条：
1. **掌握优势** → 找到期望值为正的系统（突破法+趋势组合过滤器）
2. **管理风险** → 用N值标准化头寸 + 四大层面限制 + 2N止损
3. **坚定不移** → 克服认知偏差，机械执行，不错过任何趋势
4. **简单明了** → 用基础积木搭建系统，避免过度拟合

**最终目标**：像作者柯蒂斯·费思一样——在19岁时成为海龟实验中表现最出色的学员，4年为期里奇赚得超过1亿美元。这不是因为他掌握了什么秘诀（他的方法比其他海龟更简单），而是因为他**能坚定不移地执行系统法则，杜绝任何自负因素的影响**。

> "你可以把我的交易法则登在报纸上，但没人会遵守它们。关键是统一性和纪律性。"——理查德·丹尼斯

> "如果计划无法执行，计划就毫无意义。如果你真的很想成为一个成功的交易者，那就首先做到第一条（掌握优势）。我当年就是这么做的，而且我从未后悔过。"——柯蒂斯·费思
