## 1. 总体框架：威科夫操盘法的核心理念

威科夫操盘法是"市场本质交易法"——抛弃技术指标表象，从价格和成交量的细节变化中解读供求关系，从而实时掌握市场脉搏。

### 1.1 理论根基

| 层级 | 内容 | 作用 |
|------|------|------|
| **表象层** | K线形态、技术指标、几何图形 | 大众交易者的判断依据（易被操纵） |
| **行为层** | 价格、成交量、走势速度 | 聪明钱的观察对象 |
| **本质层** | 供求关系的不平衡 | 价格涨跌的根本动力 |
| **博弈层** | 主力资金（CM）与大众交易者的博弈 | 趋势形成与反转的幕后逻辑 |

### 1.2 市场运行的底层循环

```
吸筹（底部）→ 上涨趋势 → 派发（顶部）→ 下跌趋势 → 吸筹……
   ↑                                              ↓
   └──────────── 供应耗尽 → 需求扩大 ←───────────┘
```

- **吸筹**：主力资金在底部区间被动收购低价股票，清除浮动供应
- **派发**：主力资金在顶部区间隐蔽出货，将风险转移给大众交易者
- **牛熊转换的枢纽**：供求秩序的改变（供不应求↔供需平衡↔供过于求）

### 1.3 核心认知

1. **市场是负和游戏**：佣金+滑点持续抽走资金，仅"好于平均"不够，必须出类拔萃
2. **技术指标是滞后的**：指标数据来自历史，无法预测未来，是主力利润的源泉
3. **趋势反转需要准备过程**：没有"因"（吸筹/派发过程）就不会有持续的"果"（趋势）
4. **右手边进场原则**：在趋势准备过程结束、新趋势出现强势特征后才动手
5. **好中最好的时机**：在最低风险位置介入，而非最早可能的位置

## 2. 聪明钱思维：认知转变方法论

### 2.1 大众交易者 vs 聪明钱

| 维度 | 大众交易者（羊） | 聪明钱（狼） |
|------|------------------|--------------|
| **判断依据** | 消息、专家、技术指标、K线形态 | 市场自身行为：盘口和量价 |
| **分析核心** | 表象（几何图形、参数） | 本质（供求关系、资金博弈） |
| **思维方式** | 表象思维，视觉驱动 | 逻辑思维，内涵驱动 |
| **操作习惯** | 急中生智，凭视觉决定 | 提前计划，凭耐心等待 |
| **进场时机** | 左手边抢跑，第一个反弹就抄底 | 右手边进场，等准备过程结束 |
| **情绪管理** | 贪婪与恐慌交替 | 客观评估供求，不受K线形状左右 |

### 2.2 大众交易者的认知误区

1. **听信别人**：依赖小道消息、专家、媒体，从他人看法中找确认
2. **迷信基本面/技术分析**：把经济形势硬套到操作上，或盲目叠加技术指标
3. **盲目依赖技术指标曲线**：认为指标可提供可靠买卖信号，殊不知指标只分析过去
4. **认为市场很简单**：不愿每天花时间钻研，热衷寻找捷径和现成系统
5. **以小博大幻想**：本金不足导致冒进操作，急于翻倍发财

### 2.3 聪明钱的七大特征

机构资深交易员的共同点：

1. **分析供求关系**，并随着行情发展持续分析买压和卖压的力量
2. **把分析重点放在成交量上**，因为成交量代表资金量——价格变化的核心因素
3. **分析主力机构（CM）的操作意图**，保证自己总是站在对的一方
4. **长线遵循吸筹和派发的循环走势**，短线关注双方压力的博弈
5. **在趋势开始之前锁定最好时机**，并付诸行动
6. **关注被大众忽视的细节**，这些细节是探知趋势方向的关键
7. **动手前做好全流程计划**：从建仓到离场所有可能发生的事都提前应对

### 2.4 思维转变路径

```
表象思维（看K线形状/指标信号）
    ↓
行为思维（看量价变化，识别谁在控盘）
    ↓
内涵思维（解读供求关系，判断市场背景）
    ↓
博弈思维（理解主力意图，站在对的一方）
```

**关键转变**：从"看到大涨就追、看到大跌就抄"转变为"先问供求秩序是否改变，再决定是否行动"。

## 3. 三大要素与看盘顺序

### 3.1 解读市场的三大基本要素

| 要素 | 作用 | 可读出的信息 |
|------|------|--------------|
| **价格** | 识别支撑阻力、走势速度、超买超卖 | 趋势方向、突破幅度 |
| **成交量** | 识别资金流入流出、供求强弱 | 主力意图、供应是否耗尽 |
| **走势速度** | 趋势线角度、垂直程度 | 趋势健康度、是否进入高潮 |

### 3.2 聪明钱的看盘顺序（五步法）

这是区别于大众交易者的核心方法论——不是等待技术指标信号，而是按固定顺序解读市场：

| 步骤 | 内容 | 关键问题 |
|------|------|----------|
| **① 背景** | 确定当前处于牛熊还是震荡区 | 大背景是什么？供求秩序如何？ |
| **② 价量形态** | K线长度、成交量高度、变化速度 | 量价在告诉我们什么？ |
| **③ 形态性质** | 走势强弱、原因和可能后果 | 谁在控盘？供求谁占上风？ |
| **④ 结论/预测** | 判断处于上涨还是下跌秩序中 | 趋势秩序是否完整？ |
| **⑤ 措施/行动** | 决定进场、离场、收紧止损 | 风险最低的行动是什么？ |

### 3.3 从量价中读出的九类信息

聪明钱能从价格和成交量中读出：

1. 识别支撑和阻力（用价格判断）
2. 走势涨跌的速度——趋势线角度（用价格判断）
3. 识别超买或超卖行情（用成交量和价格判断）
4. 识别某阶段哪方力量在控盘（用价格和成交量判断）
5. 顶部识别主力是否制造狂热吸引公众入场（用成交量判断）
6. 底部识别主力是否制造恐慌迫使套牢盘割肉（用成交量判断）
7. 利用供求关系判断支撑阻力的质量（用成交量判断）
8. 识别主力吸筹或派发的起点到终点（用成交量判断）
9. 识别主力是否在吸收阻力上的卖盘并判断是否会突破（用成交量判断）

## 4. 四大核心原则

这四条原则是主力资金观察走势所遵循的准则，也是所有量价分析的底层逻辑。

### 4.1 供求关系原则

> 趋势的动力来自供求关系的不平衡。

| 供求状态 | 价格表现 | 市场背景 |
|----------|----------|----------|
| 需求 > 供应 | 上涨趋势 | 牛市秩序 |
| 供应 > 需求 | 下降趋势 | 熊市秩序 |
| 供需平衡 | 震荡区间 | 危险区（易亏钱） |

**核心应用**：
- 不断衡量供求关系变化，决定持有（秩序支持判断）或离场（秩序否定判断）
- **上涨趋势中**：价格上涨因供不应求；若高成交量但价格无进展，说明供应开始增加（主力出货），秩序被打破
- **下降趋势中**：价格下跌因供应大于需求；真正的买家只有主力资金，他们要等价格压到满意区才出手
- **回调的本质**：需求减弱导致的回调不会反转趋势；供应压力导致的回调需警惕

### 4.2 因果关系原则

> 趋势形成之前需要准备过程。没有"因"（准备过程）就不会有持续的"果"（趋势）。

| 概念 | 含义 |
|------|------|
| **因** | 趋势出现前的准备过程（吸筹/派发阶段） |
| **果** | 趋势形成 |

**核心应用**：
- 避免盲目抄底——大众交易者在熊市第一个反弹就买入，结果被套或被震仓出局
- 抄底前必须等待吸筹过程结束（供应耗尽 + 需求扩大）
- 做顶前必须等待派发过程结束（需求耗尽 + 供应扩大）
- **右手边进场**：给趋势反转留一段准备过程，消耗原趋势动力

### 4.3 努力和结果原则

> 成交量的增长没有使价格大幅增长，这是走势停止行为。

| 概念 | 含义 |
|------|------|
| **努力** | 成交量（市场流入订单总量） |
| **结果** | 价格变动幅度 |

**停止行为的两种表现**：

| 类型 | 表现 | 含义 |
|------|------|------|
| **放量滞跌**（熊市中） | 大成交量 + 短K线（价格没跌） | 卖盘大增但价格未跌→需求接盘→停止行为 |
| **放量滞涨**（牛市中） | 大成交量 + 短K线（价格没涨） | 买盘大增但价格未涨→供应出货→停止行为 |

**核心应用**：在支撑/阻力价位，利用努力和结果提前判断后市方向——大量卖单流入但价格不跌，说明有需求截住了下跌。

### 4.4 秩序原则

> 市场要保持一种趋势，必须遵循现有的秩序。只要秩序不被打破，趋势就存在，可以安心持仓。

**上涨趋势的秩序**：
- 上坡：成交量递增、价格幅度递增（需求持续有力）
- 下坡：成交量递减、价格幅度递减（供应不足）

**下跌趋势的秩序**：
- 下坡：供应持续有力，创新低
- 上坡：反弹零散、不持久（需求不足）

**秩序被打破的信号**：供求关系的变化首先出现在很小的价格行为中，这些小行为隐藏着趋势改变的端倪。

## 5. 关键概念体系

### 5.1 支撑与阻力的本质

> 均线、黄金分割线等都不能单独代表支撑阻力。**真正起到支撑阻力作用的是成交量**。

| 概念 | 本质 | 判断方法 |
|------|------|----------|
| **支撑** | 需求超过供应（买单流入量 >> 卖单流入量） | 观察是否有支撑动作（Spring、量增SOT） |
| **阻力** | 供应超过需求（卖单流入量 >> 买单流入量） | 观察供应压力是否扩大 |

**支撑有效的判断**：
- 触底后出现强烈反弹 → 买家愿意追高
- 测试支撑的K线必须是**短K线** → 供应枯竭
- 进入支撑区时**成交量低** → 供应严重短缺

**支撑失效的判断**：
- 触底后无力反弹 → 买家等待更低价格
- 低量小幅波动 → 没有大量愿意追高的买单
- 立刻离场等候

### 5.2 核心市场行为术语表

| 术语 | 英文/缩写 | 含义 | 位置 |
|------|-----------|------|------|
| 恐慌抛售 | SC (Selling Climax) | 下降趋势中潮水般的恐慌抛售，标志下降趋势结束 | 底部 |
| 自动反弹 | AR | 恐慌抛售后供应消耗导致的自然反弹 | 底部 |
| 二次测试 | ST | 测试恐慌抛售后供应是否恢复 | 底部/顶部 |
| 强势出现 | SOS (Sign of Strength) | 需求控制市场的信号，超长阳线+超大成交量 | 底部/上涨中 |
| 跳离震荡区 | JOC (Jump Over Creek) | SOS突破震荡区顶部，越过供应区（小溪） | 底部/上涨中 |
| 弱势出现 | SOW (Sign of Weakness) | 供应控制市场，价格下降+宽幅+巨量 | 顶部/下跌中 |
| 上冲回落 | UT (Upthrust) | 突破阻力后回落，主力欺骗行为 | 顶部 |
| 上冲回落派发 | UTAD | 派发阶段的上冲回落，确认派发结束 | 顶部 |
| 弹簧效应 | Spring | 价格小幅突破支撑后立刻返回，测试供应量 | 底部 |
| 终极震仓 | TSO | 深度跌破吸筹区间底部，扫清最后供应 | 底部末期 |
| 普通震仓 | Shake out | 上升趋势中受消息影响的临时下跌 | 上涨中 |
| 突破幅度递减 | SOT (Shortening of Thrust) | 上升波顶点距离递减，需求减弱 | 顶部预警 |
| 垂直需求柱 | VDB | 放量长阳，需求完全控制市场，底部是支撑 | 上涨中 |
| 垂直供应柱 | VSB | 放量长阴，供应完全控制市场，顶部是阻力 | 下跌中 |
| 下冲反弹 | Spring | 同弹簧效应，突破支撑后返回 | 底部 |
| 抢购高潮 | BC (Buying Climax) | 上升趋势末期大众疯狂买入，成交量陡增 | 顶部 |
| 初次供应 | PSY | 牛市中首次出现大量供应（放量） | 顶部预警 |
| 冰线 | Creek | 上升趋势中最后一道防线，破冰后跌幅大 | 顶部 |
| 破冰 | ICE | 价格以极大供应力度突破主要支撑 | 顶部后 |
| 测冰 | — | 破冰后价格反弹测试前支撑（低量小柱=做空时机） | 顶部后 |
| 死角 | — | 支撑/阻力位K线高点降低、低点升高，波动收窄 | 突破前 |
| 最后支撑点 | LPS | 上涨中回落时供应耗尽的进场点 | 上涨中 |

### 5.3 三高与三低

| 形态 | 定义 | 含义 |
|------|------|------|
| **三高** | 更高的高点、更高的低点、更高的收盘 | 牛市特征，需求占上风 |
| **三低** | 更低的低点、更低的高点、更低的收盘 | 熊市特征，供应占上风 |

### 5.4 左手边与右手边

| 位置 | 含义 | 操作原则 |
|------|------|----------|
| **左手边** | 趋势准备过程的开始阶段 | 不动手，只观察 |
| **右手边** | 准备过程结束，新趋势出现强势特征 | 进场位置 |

> 很多交易者在左手边看到停止行为就急于反转，没有耐心等到右手边。强调等到右手边是为了给趋势反转留一段准备过程（因果关系）。

## 6. 底部研究：吸筹四阶段方法论

底部形成的主线是**供应耗尽**——从供应大于需求，到需求吸收供应，再到供应彻底耗尽、需求占绝对优势。

### 6.1 吸筹四阶段总览

| 阶段 | 名称 | 核心任务 | 关键行为 |
|------|------|----------|----------|
| **第一阶段** | 熊市终止 | 需求扩大后供应逐渐耗尽 | 初次支撑→恐慌抛售→自动反弹→二次测试 |
| **第二阶段** | 震荡区吸筹 | 主力控制价格大量收购低价股票 | 浮动供应逐渐减少，波动和成交量递减 |
| **第三阶段** | 测试确认 | 确认市场是吸筹而非派发 | 终极震仓（TSO）/ 弹簧效应（Spring） |
| **第四阶段** | 进入牛市 | 主力允许价格上涨 | SOS + 测试 / JOC + 测试 |

### 6.2 第一阶段：熊市终止的四步行为

这四个行为按顺序出现，确认熊市终止：

| 步骤 | 行为 | 含义 | 关键观察点 |
|------|------|------|------------|
| ① | **初次支撑** | 主力资金需求进场，阻止价格下跌 | 成交量明显扩大（有时与恐慌抛售一起发生） |
| ② | **恐慌抛售** | 大众交易者恐惧抛售，主力接盘 | 放量超长阴线，振幅极宽，供应迅速消耗 |
| ③ | **自动反弹** | 供应消耗后的自然反弹 | 多为空头平仓，不会持久，**不能抄底** |
| ④ | **二次测试** | 测试供应是否恢复 | 成交量和K线都小=成功；大=需再等测试 |

**熊市终止的理论根据**：
1. 市场供应出现枯竭（卖单流入量已很少）
2. 需求必须扩大并保持力度

**判断底部的两个核心观察点**：
- 不论怎样，**不再下跌**（主力持续收购的结果）
- 横盘相持中，**供应越来越少**

### 6.3 第二阶段：震荡区吸筹

**主力资金的操盘手法**：

| 手法 | 目的 | 盘面表现 |
|------|------|----------|
| 被动等卖单 | 低价收购 | 底部有支撑但不主动竞价 |
| 偶尔竞价后拦截 | 阻止跟风 | 急涨后立刻失去动力 |
| 疲劳战术 | 逼迫死扛者割肉 | 枯燥的低量短柱缓慢下滑 |
| 震仓 | 扫清最后供应 | 急速跌破所有支撑 |

**观察主线**：
- 早期：价格波动和成交量较大
- 中期：随着主力收购，供应越来越少，波动和成交量逐渐减小
- 末期：主力测试浮动供应是否耗尽

**带目的看图的四个观察点**：
1. 供应是否在逐渐耗尽（每次回到底部都是低量）
2. 主力是否在用疲劳战术（枯燥的低量短柱滑落）
3. 吸筹是否接近结束（空中出现停止行为/支撑动作）
4. 主力是否在提高收购价（高支撑）

### 6.4 第三阶段：测试确认

主力资金测试市场上的股票供应，确认震荡区是吸筹而非派发。

| 测试方法 | 表现 | 作用 |
|----------|------|------|
| **终极震仓（TSO）** | 深度跌破吸筹区间底部 | 吸收剩余股票、迫使弱者卖出、制造下跌假象 |
| **弹簧效应（Spring）** | 小幅突破支撑后立刻返回 | 评估突破时的供应量 |

**终极震仓的三大任务**：
1. 让价格迅速冲破所有支撑，吸收市场上剩下的股票
2. 迫使弱者卖掉手里的股票（迅速消耗供应）
3. 制造继续下跌的假象，让大众交易者不敢进场

**震仓后的确认**：终极震仓或弹簧效应后必须出现**二次测试**，判断浮动筹码是否全部被吸收。一旦全部吸收，主力只能抬高价格吸筹——这是**风险最低的抄底点**。

### 6.5 第四阶段：进入牛市

两个重要市场行为标志牛市开始：

| 行为 | 特点 | 确认条件 |
|------|------|----------|
| **SOS + 测试** | 震荡区内持续上涨+明显增高成交量 | 回测必须是短K线+低成交量 |
| **JOC + 测试** | 放量长阳突破震荡区顶部（越过小溪） | 突破后成交量必须有跟随 |

**JOC（跳离震荡区）的比喻**：
- 小溪（阻力区）充满卖单（供应）
- 男孩要找水流缓、窄的地方跳过小溪
- 跳之前要退几步助跑（回调准备）
- 跳过后需要短暂休息（测试确认）

**最佳进场点**：JOC突破点回测时，缩量回落（成交量几乎为零）= 供应耗尽 = 进场信号。

### 6.6 吸筹进场的两个位置

| 位置 | 条件 | 安全度 |
|------|------|--------|
| **位置一** | SOS/JOC后的回测（短K线+低量） | 最安全 |
| **位置二** | 终极震仓后的二次测试（供应耗尽） | 风险最低的抄底点 |

**进场后的确认**：必须看到三高和量增，然后低量小幅回落。若进场后看到放量阴线（遇到大量供应），立刻离场。

## 7. 震仓方法论

### 7.1 震仓的本质

震仓是主力资金在震荡区中故意使价格下降的行为，目的是激起大众交易者平仓，主力趁机接盘。

### 7.2 两种震仓对比

| 维度 | 终极震仓（TSO） | 普通震仓 |
|------|------------------|----------|
| **发生位置** | 吸筹阶段末期 / 交易区间末期 | 上升趋势中 |
| **发生原因** | 二次测试后反弹无力，需清除浮动供应 | 受不利消息影响的临时波动 |
| **表现形态** | 急速大幅度深跌，冲破所有支撑 | 临时性下跌 |
| **目的** | 扫清最后供应，确认吸筹结束 | 清洗意志不坚定者 |
| **后续** | 快速反弹进入牛市 | 价格快速恢复上涨 |
| **买卖双方** | 卖方=恐慌大众，买方=主力 | — |

### 7.3 震仓的识别要点

1. **前提**：之前有震荡区或主要支撑区
2. **表现**：急速深跌冲破所有支撑，但随后快速反弹
3. **确认**：反弹必须带有牛市特点（持续的三高）
4. **二次测试**：震仓后必须等待二次测试确认供应耗尽再进场

### 7.4 震仓前的预警信号

如果在震荡区内：
- 反弹后价格总是到不了前高点
- 每个高点呈台阶式下降
- 主力最低收购价没有逐渐抬高

→ 说明浮动供应还在，容易出现震仓。

## 8. 顶部研究：派发四阶段方法论

顶部充满谎言——繁荣表象让大众交易者疯狂接盘，主力资金趁机出货。派发的主线是**需求耗尽**。

### 8.1 派发四阶段总览

| 阶段 | 名称 | 核心任务 | 关键行为 |
|------|------|----------|----------|
| **第一阶段** | 牛市停止 | 供应开始大量进入，需求耗尽 | SOT→初次供应→抢购高潮→自然回落→二次测试 |
| **第二阶段** | 派发确认 | 主力设立派发区，引诱公众接盘 | 震荡区交易，每次回到支撑反弹，顶部巨量滞涨 |
| **第三阶段** | 测试需求 | 确认需求已耗尽，准备降价 | 上冲回落（UTAD）测试需求 |
| **第四阶段** | 熊市启动 | SOW出现，市场进入熊市 | 破冰→测冰→下跌秩序形成 |

### 8.2 第一阶段：牛市停止的五步行为

| 步骤 | 行为 | 含义 | 警告级别 |
|------|------|------|----------|
| ① | **SOT（突破幅度递减）** | 上升波顶点距离递减，需求减弱 | 预警 |
| ② | **初次供应（PSY）** | 牛市中首次放量（供应扩大） | 预警 |
| ③ | **抢购高潮（BC）** | 大众疯狂买入，主力暗自出货 | 警告 |
| ④ | **自然回落** | 抢购高潮后的自然下跌 | — |
| ⑤ | **二次测试** | 测试需求是否耗尽 | 成功=牛市终止 |

**抢购高潮的识别**：
- 成交量陡然剧增 + 上涨速度骤然加快（趋势线趋于垂直）
- 必须经过迅速回落确认
- 上涨趋势停止必须经过二次测试确认
- 必须在上涨秩序被打破后才考虑反转

**主力引诱公众接盘的手法**：
1. 买通媒体营造利好气氛
2. 伙同上市公司发布产品和业绩利好
3. 制造价格急涨（大阳线/长下影线）吸引眼球
4. 利用大众贪婪情绪出逃（转移风险）

### 8.3 第二阶段：派发确认

**主力派发的明显行为**：
- 每次价格回到支撑就反弹（主力用买单保护支撑）
- 价格反弹到顶部时出现**巨量滞涨**（出货）
- 当支撑被长阴线放量突破 → 主力货已出完，撤掉夹板 → 熊市开始

**震荡区内的演变**：
- 初期：波动猛烈（大众还在抢购）
- 中期：主力不断投放股票，上涨总是无功而返
- 后期：需求越消耗越少，波动和成交量逐渐减弱

> 尽量避免在震荡区内交易，耐心等待右手边出现明显强势或弱势特征。

### 8.4 第三阶段：测试需求

| 测试行为 | 表现 | 含义 |
|----------|------|------|
| **上冲回落（UTAD）** | 突破派发区顶部后迅速回落 | 测试需求是否耗尽，确认派发结束 |

UTAD是主力的欺骗行为：向上突破吸引买家投入，同时扫清空头止损，迅速回落达到出货目的。

### 8.5 第四阶段：熊市启动

| 行为 | 含义 |
|------|------|
| **SOW（弱势出现）** | 供应控制市场，价格下降+宽幅+巨量 |
| **破冰** | 价格以极大供应力度突破主要支撑（冰线） |
| **测冰** | 破冰后反弹测试前支撑，低量小柱=做空时机 |

**破冰的含义**：冰层之上套牢大量大众交易者，主力不会帮他们解套（那需要更高价吃进）。破冰标志着熊市秩序的启动。

### 8.6 顶部做空的时机

1. **震荡区右手边出现三低**（顶部右手边三低=市场行为改变的信号）
2. **破冰后的测冰**（低量小柱无力反弹）
3. **双顶 + 停止行为量价配合**
4. 必须看到下跌趋势的秩序形成（量增三低）

> 做空前必须确认：上涨趋势秩序已被打破 + 下跌趋势秩序已形成。

## 9. 持仓与趋势跟踪方法论

### 9.1 趋势的本质

> 趋势是价格所遵循的一种秩序，秩序的核心是供求关系。

**从三个角度理解趋势**：

| 角度 | 上涨趋势的表现 |
|------|----------------|
| **资金活动** | 资金流入力度 >> 资金流出力度 |
| **市场行为原则** | 上坡：需求有努力有结果；下坡：供应无增加或有努力无结果 |
| **量价关系** | 上坡：量增价增；下坡：量减价减 |

**摸准趋势脉搏的三个因素**：
1. 量价关系（表象信息）
2. 资金活动情况
3. 供求关系（核心）

→ 从表象信息解读出后两者，才能读懂趋势。

### 9.2 持仓的核心原则

**持仓条件**：只要供求秩序不被打破，趋势就存在，安心持仓。

**上涨趋势持仓的量价确认**：
- 上涨中：成交量递增、价格幅度递增（需求有力）
- 回调中：成交量递减、价格幅度递减（供应不足）
- 更应关心**阴线的成交量**（特别是三低），因为阴线量代表供应力度

### 9.3 移动止损的触发条件

**上涨趋势中离场/收紧止损的信号**：

| 信号 | 含义 | 行动 |
|------|------|------|
| 回调成交量扩大 + 价格振荡幅度扩大 | 供应在扩大 | 不做多，等供应枯竭 |
| 巨量不涨 | 供应出货 | 缩紧止损/离场 |
| 突破幅度递减（SOT） | 需求减弱 | 准备出逃计划 |
| 上冲回落（UT）后有阴线跟随+量增 | 出货持续 | 离场 |
| 趋势线被突破 | 供应远超需求 | 启动保护措施 |
| 破冰（光头阴线大幅突破前低） | 熊市秩序启动 | 立即离场，不在乎账面盈亏 |

**下降趋势中持仓/做空的确认**：
- 下跌：量增三低（供应持续有力）
- 反弹：低量、零散、不持久（需求不足）
- 测冰：低量小柱无力反弹 = 做空好时机

### 9.4 关键价位的观察重点

不要每根K线都分析，而是聚焦关键价位的供求关系：

| 关键价位 | 观察重点 | 决策依据 |
|----------|----------|----------|
| 支撑位 | 是否有支撑动作（Spring/量增SOT） | 支撑有效→持仓；失效→离场 |
| 阻力位 | 供应是否扩大 / 需求是否耗尽 | 突破→进场；反转→离场 |
| 冰线 | 破冰后的测冰力度 | 低量小柱→做空 |
| 震荡区边界 | 右手边出现强势或弱势特征 | SOS→做多；SOW→做空 |
| 死角 | 波动收窄后的突破方向 | 三高突破→进场 |

## 10. 综合分析：进场、持仓、离场

### 10.1 市场本质交易法

> 每次分析走势时，先考虑理论基础，然后在图中找能够确认理论基础的价量。

| 步骤 | 内容 |
|------|------|
| ① 理论基础 | 突破阻力的理论：阻力上任何卖盘都被需求全部吸收，且需求占上风 |
| ② 图上确认 | 从价量关系发现吸收行为 → 理论基础成立 → 判断阻力会被突破 |
| ③ 行动决策 | 在风险最低的位置进场 |

### 10.2 进场原则

**总原则**：右手边进场 + 好中最好的时机 + 最低风险位置。

| 场景 | 进场位置 | 确认条件 |
|------|----------|----------|
| 底部做多 | SOS/JOC后的回测 | 短K线+低成交量 |
| 底部做多 | 终极震仓后的二次测试 | 供应耗尽 |
| 上涨中加仓 | LPS（最后支撑点） | 需求控制市场背景下，回落时供应耗尽 |
| 顶部做空 | 破冰后的测冰 | 低量小柱无力反弹 |
| 顶部做空 | 震荡区右手边三低 | 量增+三低+下跌秩序形成 |

**进场后的风控**：
- 进场后必须看到三高和量增（需求持续大于供应）
- 若进场后看到放量阴线（遇到大量供应）→ 立刻离场
- 等上涨秩序恢复后再进场

### 10.3 持仓原则

- 只要供求秩序不被打破，安心持仓
- 不在持仓时计算浮盈/浮亏，不因单次成败而情绪波动
- 关注回调的成交量（供应力度），而非K线形状
- 连胜后警惕自大

### 10.4 离场原则

| 离场场景 | 触发条件 |
|----------|----------|
| 多头离场 | 供求秩序被打破（巨量不涨、SOT、破冰） |
| 空头离场 | 下跌秩序被打破（低量回落+上涨秩序恢复） |
| 止损离场 | 进场逻辑被市场否定 |
| 计划外离场 | 看到不完整条件就进场后，市场证明错误 |

**核心准则**：一旦市场自身行为与判断相悖，首先放弃自己的想法，采取风控措施。

## 11. 交易计划与危机管理

### 11.1 动手前的全流程计划

主力资金在动手前，把从建仓到离场过程中所有可能发生的事都提前做好应对计划。交易者也应如此：

| 计划环节 | 内容 |
|----------|------|
| **进场计划** | 在哪个位置进场？确认条件是什么？ |
| **风控计划** | 初始止损放在哪里？什么条件触发离场？ |
| **持仓计划** | 进场成功后，什么条件下继续持有？ |
| **危机管理计划** | 进场失败后，如何应对？ |

### 11.2 危机管理的核心原则

1. **守住原则不随意改变**：看到不完整的条件不着急进场，忘记交易计划中的原则是危险的
2. **市场证明错误时立即止损**：不要用情绪和希望否定市场本质行为
3. **不与市场主控力量作对**：好中最好的进场必须等到供应严重短缺
4. **分批应对**：缩紧止损 → 扫掉就认 → 不抱侥幸心理

### 11.3 风控的量价依据

风控不是机械的固定百分比，而是基于供求关系的量价行为：

| 风控信号 | 量价依据 | 行动 |
|----------|----------|------|
| 供应扩大 | 回调量增+幅度扩大 | 缩紧止损 |
| 需求耗尽 | 巨量不涨、反弹无力 | 准备离场 |
| 秩序改变 | 趋势线突破+破冰 | 立即离场 |
| 进场失败 | 放量阴线跟随 | 立刻离场 |

### 11.4 交易者成长路径

1. **承认自己是羊**：不丢脸，只是还没找到正确打开方式
2. **从表象思维过渡到逻辑思维**：以市场本质行为为核心交易依据
3. **每天花时间钻研**：市场是艺术，需毕生时间钻研
4. **杜绝拷贝别人**：一个人的交易方式对另一个人没用
5. **不通吃**：选择适合自己的交易方式（长线极端行情 / 中短线波段）
6. **持续反思**：吸收的最好方式是给别人讲述所思所得

## 12. 核心检查清单

### 12.1 看盘前自检

- [ ] 当前大背景是什么（牛市/熊市/震荡区）？
- [ ] 供求秩序目前是什么状态？
- [ ] 我应该观察什么（而非被K线牵着走）？

### 12.2 进场前自检

- [ ] 是否看到了准备过程（因果关系）？
- [ ] 是否在右手边（而非左手边抢跑）？
- [ ] 供应是否已经耗尽（低量触底）？
- [ ] 需求是否保持力度（三高+量增）？
- [ ] 进场后看到放量阴线是否准备好立刻离场？
- [ ] 初始止损位置是否明确？
- [ ] 危机管理计划是否制定？

### 12.3 持仓中自检

- [ ] 供求秩序是否仍然支持我的判断？
- [ ] 回调的成交量是否递减（供应不足）？
- [ ] 是否看到供应扩大的信号（巨量不涨、SOT）？
- [ ] 是否因浮盈/浮亏干扰了决策？

### 12.4 离场自检

- [ ] 供求秩序是否被打破？
- [ ] 市场自身行为是否与我的判断相悖？
- [ ] 是否在用情绪和希望否定市场本质行为？

### 12.5 顶部做空前自检

- [ ] 上涨趋势的秩序是否已被打破？
- [ ] 下跌趋势的秩序是否已形成？

## 附：威科夫操盘法核心逻辑链

```
供求关系不平衡 → 产生趋势
         ↓
趋势反转需要准备过程（因果关系）
         ↓
底部：吸筹四阶段（供应耗尽→需求扩大）
  · 第一阶段：初次支撑→恐慌抛售→自动反弹→二次测试
  · 第二阶段：震荡区吸筹（浮动供应递减）
  · 第三阶段：终极震仓/弹簧效应（测试供应耗尽）
  · 第四阶段：SOS/JOC（进入牛市）
         ↓
顶部：派发四阶段（需求耗尽→供应扩大）
  · 第一阶段：SOT→初次供应→抢购高潮→自然回落→二次测试
  · 第二阶段：震荡区派发（巨量滞涨）
  · 第三阶段：UTAD（测试需求耗尽）
  · 第四阶段：SOW→破冰→测冰（熊市启动）
         ↓
持仓：跟踪供求秩序
  · 秩序完整→安心持仓
  · 秩序打破→立即离场
         ↓
核心准则：
  · 右手边进场，好中最好时机
  · 先看理论基础，再找量价确认
  · 市场否定判断时，先放弃想法，再风控
```
