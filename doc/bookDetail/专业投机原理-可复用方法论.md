## 1. 总体框架：投机的知识体系

### 1.1 "赌博例"的奥秘——投机的本质

斯波朗迪以"赌博例"（Gamboni）的故事开篇，揭示金融市场生存的核心：

> **如果你希望获胜，你必须了解规则；另外，你还必须愿意坐上赌桌，才有获胜的机会。**

这意味着投机的两大前提：
- **界定交易所需的知识与规则**
- **将知识付诸实践的计划，并按规则组织计划**

### 1.2 三种市场角色的区分

任何市场同时存在三种价格趋势，对应三种参与者：

| 角色 | 关注趋势 | 持仓周期 | 核心能力 |
|------|----------|----------|----------|
| **交易者** | 短期趋势（数天至数周） | 数分钟至数周 | 盘中趋势把握 |
| **投机者** | 中期趋势（数周至数月） | 数周至数月 | 趋势反转判断 |
| **投资者** | 长期趋势（数月至数年） | 数月至数年 | 基本面判断 |

**关键认知**：
- 投机≠赌博。**赌博是承担不利胜算的风险；投机是仅在掌握有利胜算时承担风险**
- 投机者为市场提供流动性和资产配置功能
- 本书以**中期趋势投机**为核心，但原理经调整后适用于交易和投资

### 1.3 七大知识领域的整合

斯波朗迪的方法论整合了七大领域知识，**不可单一依赖**：

| 知识领域 | 作用 |
|----------|------|
| 胜算（几率） | 衡量风险/回报关系 |
| 市场与交易工具 | 选择交易标的 |
| 技术分析 | 判断趋势与时机 |
| 统计概率 | 评估走势幅度与期限 |
| 经济学 | 理解市场驱动因素 |
| 政治学 | 预判政策干预影响 |
| 人类心理学 | 控制情绪、理解群体行为 |

**核心原则**：将复杂体系简化为相对单纯、易于操作的基本思想。大多数市场玩家仅依赖1~3个领域，而作者结合所有领域从各角度评估风险与回报。

## 2. 鳄鱼原则：重点思考方法论

### 2.1 鳄鱼原则——认赔的绝对性

**来源**：鳄鱼的吞噬方式——猎物越挣扎，鳄鱼收获越多。咬住你的脚时，越用手臂挣脱，便陷得越深。

**市场语言的表达**：

> **当你知道自己犯错时，立即了结出场！不可再找借口、期待、祷告或采取其他任何动作，赶紧离场！**

**常见违背场景**：
- 不接受小额亏损（如500美元），反而加码或对冲 → 扩大为巨额亏损（如6000美元）
- 根源：**谬误的自尊（false pride）与虚荣**，认为承认错误等于承认无能

### 2.2 根据重点思考（Thinking in Essentials）

这是全书方法论的思维基础：

**核心定义**：
- **重点未必在于你知道多少，而在于你知道内容的真实性与相关性**
- 处理信息的关键：从每天无数资料中提取**重点（essentials）**
- 以**原则（principle）**统辖信息——原则是界定金融市场性质的基本概念

**"根据原则思考"的操作程序**：
1. 将明确事件转化为抽象概念
2. 将抽象原则套用在实际事件上
3. 根据对现在事件的分析，预测长期的未来
4. 以适用近期与过去历史的同一概念，了解现在发生的事件

**示例——"增长需要依赖储蓄"原则的推理链**：
- 原则成立 → 信用扩张的企业兼并不可能永远持续
- 经济衰退期，高负债公司遭销售量减退+信用成本提高双重打击
- 政府预算赤字=以明天生产融通今天支出 → 最终造成金融灾难

**界定名词的重要性**：多头市场、空头市场、趋势、风险/回报……金融业内大多知道这些名词，但很少有人能明确界定。**界定名词是了解市场的基本步骤**。

## 3. 一致性成功的事业经营哲学

### 3.1 三大原则的层级关系

斯波朗迪将交易视为事业经营，基于三个按重要性排列的原则，**三者相互衔接、不可颠倒**：

| 优先级 | 原则 | 核心含义 |
|--------|------|----------|
| **第一** | 保障资本 | 风险是最重要因素；追求绝对回报而非相对回报 |
| **第二** | 一致性的获利能力 | 资本保障的辅助原则；稳定增长而非大起大落 |
| **第三** | 追求卓越的回报 | 仅在已获利时，以更大风险追求更高回报 |

**关键认知**：资本保障→一致性获利→追求卓越，三者构成正向循环。资本保障造就一致获利，一致获利使追求卓越成为可能。

### 3.2 保障资本

**操作准则**：
- 问"可以实现的潜在利润如何"之前，先问"可能遭受的潜在亏损如何"
- **最大可接受风险/回报比率为1:3**
- 风险/回报情况差时，**持有现金**，不论普遍看法如何
- 追求**绝对回报**，不关心是否"超越大盘指数"

**黑色与白色的哲学**：
- 钞票是黑色或白色——对应"真理与谬误"、"正确与错误"
- 拒绝基金管理圈推销的"灰色观念"（大盘跌20%，基金仅跌10%就该高兴）
- **唯一的问题："我有没有赚钱？"**

**避免的高风险活动**：
- 多头市场末期投资兼并股票与垃圾债券
- 在顶部附近介入杠杆收购（LBO）
- 空头市场中持有满仓多头

### 3.3 一致性的获利能力

**核心方法——季度操作示例**：

| 阶段 | 操作 |
|------|------|
| 季度开始 | 新头寸规模很小（相对风险资本），预先设定出场点 |
| 第一笔亏损 | 新头寸按比例缩小 |
| 有获利 | 一部分获利运用在新头寸，其余存入银行 |
| 持续获利 | 增加持仓规模（但保持风险/回报≥1:3） |

**5万美元商品期货账户示例**：
- 最初头寸：不超过总资本10%（5000美元）
- 止损限制潜在损失在10%~20%（500~1000美元）
- 总亏损绝不超过风险资本的1%~2%
- 第一笔亏损1000美元 → 次笔头寸减至4000美元

**胜算管理的核心逻辑**：
- 不预期一半以上交易获利（棒球最佳打击率仅30%~40%）
- 回报应总是大于风险（1:3比率下，每3笔1次获利即可维持一致性）

### 3.4 追求卓越的回报

**前提条件**：已在获利状态下，风险/回报存在合理关系时，**仅增加头寸规模**（不改变风险/回报准则）。

**1974年经典案例**：
- 7月已赚10.4万美元（季回报率超40%）
- 强烈看空 → 以半数获利（5万美元）建立合成看跌期权空头头寸
- 8月尼克松宣布辞职 → 当月获利超26.9万美元
- **即使判断完全错误，也仅损失先前半数获利**

## 4. 道氏理论方法论

> 道氏理论是现代技术分析的鼻祖。斯波朗迪研究发现：道氏理论可精确掌握74.5%的经济扩张走势与62%的经济衰退走势。

### 4.1 三大假设

| 假设 | 内容 | 现代修正 |
|------|------|----------|
| **假设1** | 人为操纵：日波动可能被操纵，次级折返有限影响，**主要趋势绝不会被操纵** | 电脑程式交易是重要操纵形式，但不影响主要趋势 |
| **假设2** | 市场指数会反映每一条信息 | 应补充"上帝的旨意与**政府的行动**除外"，尤其美联储行动 |
| **假设3** | 这项理论并非不会错误 | 需深入研究+客观综合判断，绝不可一厢情愿 |

**对有效市场理论的批判**：即使所有人同时取得相同信息，反应也会不同。市场存在是为促进交易，交易产生于参与者对价值判断的不同。

### 4.2 五大定理

#### 定理1：道的三种走势

市场指数有三种走势，可同时出现：

| 走势类型 | 期间 | 重要性 | 关注者 |
|----------|------|--------|--------|
| 主要趋势 | 数月至数年 | 最重要，最易辨认 | 投资者 |
| 次级折返走势 | 三周至数月 | 最难捉摸 | 投机者 |
| 每日波动 | 数天 | 较不重要 | 交易者 |

**现代调整**：自80年代初，中期趋势波动加大，以**中期趋势作为准则**较为明智。

#### 定理2：主要走势

- 即多头或空头市场，持续一年以内至数年
- **正确判断主要走势方向是投机成功最重要的因素**
- 无已知方法预测持续期限，但可用历史价格资料统计归纳

#### 定理3：主要的空头市场

**三个阶段**：
1. 市场参与者不再期待股票维持过度膨胀的价格
2. 卖压反映经济状况与企业盈余衰退
3. 健全股票的失望性卖压（不论价值如何急于求现）

**空头市场的统计特质**（1896年至今）：
- 跌幅平均数：**29.4%**，75%跌幅介于20.4%~47.1%
- 持续期限平均数：**1.1年**，75%介于0.8~2.8年
- 开始时通常以偏低成交量"试探"前一个多头高点
- 确认日：两种指数都向下突破多头市场最近修正低点
- 中期反弹通常呈颠倒"V"形，低价量偏高、高价量偏低

#### 定理4：主要的多头市场

**三个阶段**：
1. 人们对未来景气恢复信心
2. 股票对已知的公司盈余改善产生反应
3. 投机热潮转炽，股价明显膨胀（基于期待与希望）

**多头市场的统计特质**：
- 涨幅平均数：**77.5%**
- 期限平均数：**2年零4个月**（2.33年），75%超过1.8年
- 次级折返走势跌势通常比前后涨势都剧烈
- 确认日：两种指数都向上突破空头市场前一个修正走势高点

#### 定理5：次级折返走势

- 多头市场重要下跌或空头市场重要上涨，持续**三周至数月**
- 折返幅度为前一个主要走势波段的**33%~66%**
- 61%的折返幅度约为前一波段的30%~70%
- 65%的折返期限介于三周至三个月，98%介于两周至八个月
- 价格变动速度有**暴涨暴跌倾向**
- 常被误认为主要趋势改变（最微妙困难的一环）

**锅炉比喻**：多头市场次级折返=安全阀（释放超买压力）；空头市场次级折返=添加燃料（补充超卖流失压力）。

## 5. 趋势定义与行情阶段

### 5.1 趋势的精确定义

> 这是全书最基础的定义，务必牢记——它们普遍适用于任何市场与任何时间结构。

| 趋势类型 | 定义 |
|----------|------|
| **上升趋势** | 由一系列连续涨势构成，每段涨势向上穿越先前高点，中间夹杂的跌势不会跌破前一波跌势低点。即**高点与低点都不断垫高**的价格走势 |
| **下降趋势** | 由一系列连续跌势构成，每段跌势向下穿越先前低点，中间夹杂的涨势不会突破前一波涨势高点。即**高点与低点都不断下滑**的价格走势 |

**关键原则**：唯有真正了解趋势意义，才可能判断趋势何时变化。唯有精确判断变化，才能提升利润或降低损失。

### 5.2 验证的重要性——指数相互确认

**雷亚原则**：两种市场指数（工业指数与运输指数）必须相互验证，仅根据一种指数判断几乎必然错误。

**现代扩展**：除了道琼斯工业与运输指数，还应观察S&P500、价值线指数等。"**所有相关指数都必须相互确认**"。

**背离现象**：某指数反转数周至数月，另一指数呈相反走势 → 仅有负面用途，提示趋势可能变化。

### 5.3 行情的四个阶段

任何市场必然处于以下四种技术阶段之一：

| 阶段 | 含义 | 特征 |
|------|------|------|
| **承接** | 长期投资者买进 | 窄幅盘整后向上突破 |
| **出货** | 长期投资者卖出 | 窄幅盘整后向下突破 |
| **向上/向下趋势** | 价格持续运行 | 高低点不断垫高/下滑 |
| **整理** | 获利了结后的调整 | 趋势中暂时的停顿 |

#### 窄幅盘整（Line）

**雷亚定义**：长达两三周及以上的价格走势，两种指数约在5%价格区间内波动。

| 突破方向 | 含义 | 操作 |
|----------|------|------|
| 同时向上穿越上限 | 承接盘整 | 买进 |
| 同时向下穿越下限 | 出货盘整 | 卖空 |

**突破是唯一对每种市场参与者都重要的走势**。

### 5.4 重要的成交量关系

| 市场状态 | 价量关系 |
|----------|----------|
| 超买行情 | 价涨量缩、价跌量增 |
| 超卖行情 | 价跌量缩、价涨量增 |
| 多头市场末期 | 交易活跃 |
| 多头市场初期 | 交易相对冷清 |

**成交量是辅助工具而非主要考虑因素**（商品市场成交量公布滞后）。

## 6. 技术分析核心准则

> 本章是全书最具操作性的部分，斯波朗迪将其技术方法维持在最简单而基本的范围内。

### 6.1 趋势线的正确绘制方法

**大多数人绘制趋势线缺乏一致性，导致趋势变动误判。以下方法简单、精确且具有一致性：**

#### 上升趋势线绘制
1. 选择考虑期间（长期/中期/短期）
2. 以**最低的低点**为起点
3. 向右上方绘制直线，连接**最高之高点前的某一个低点**
4. 使这条直线在两个低点之间**未穿越任何价位**
5. 延伸经过最高点

#### 下降趋势线绘制
1. 以**最高的高点**为起点
2. 向右下方绘制直线，连接**最低之低点前的某一个高点**
3. 使这条直线在两个高点之间**未穿越任何价位**
4. 延伸经过最低点

**优点**：与线性回归分析的最佳套入直线斜率相当近似，避免主观看法，提供趋势变动的判断基准。

### 6.2 1-2-3趋势变动准则

**价格走势的三种基本变动，相互配合可界定任何市场的趋势变化：**

| 条件 | 内容 |
|------|------|
| **①** | 趋势线被突破（价格穿越绘制的趋势线） |
| **②** | 上升趋势不再创新高/下降趋势不再创新低（试探高点或低点失败） |
| **③** | 下降趋势中价格向上穿越先前短期反弹高点/上升趋势中价格向下穿越先前短期回档低点 |

**判断规则**：
- 三个条件**全部出现** = 趋势已经发生变化（相当于道氏理论确认）
- 出现**两个** = 增加趋势变动概率
- 仅出现**一个** = 代表可能的趋势变动

**图形操作步骤**：
1. 绘制趋势线
2. 下降趋势中：绘制水平线穿越目前最低价 + 前一波反弹高点
3. 上升趋势中：绘制水平线穿越目前最高价 + 前一波回档低点
4. 在穿越位置标示①②③

### 6.3 2B准则——斯波朗迪最偏爱的方法

> "2B是否可以成立往往是输赢的分野"——这是作者最重视的准则，可让你精确掌握顶部与底部。

**2B准则定义**：
- **上升趋势中**：价格创新高后未能持续上升，稍后又跌破先前高点 → 趋势很可能反转
- **下降趋势中**：价格创新低后未能持续下降，稍后又向上穿越先前低点 → 趋势很可能反转

**不同时间周期的2B时间窗口**：

| 趋势类型 | 2B成立时间 |
|----------|------------|
| 短期趋势 | 一天之内或更短 |
| 中期趋势 | 3~5天之内 |
| 长期转折点 | 7~10天之内 |

**2B发生的市场机制——清除止损**：
- 场内交易员了解止损点通常设在先前高点/低点
- 推动价格至支撑/压力水平稍上方/下方，迫使止损单成交
- 止损清除后，市场重新调整 → 2B模式形成

**2B操作要点**：
- 当价格重新跌破/突破先前高/低点时，以止损单进场
- 回补/止损单设定在2B的高/低点
- 中期趋势2B的回报风险比**几乎总是高于5:1**
- 即使连续震荡出场三四次，成功一次的获利将多于先前所有损失
- **股票市场创新高后成交量通常低于正常水平，但反转确认时爆发大额成交量**

### 6.4 次级修正走势的ABC三波浪

**模式特征**（不论多头还是空头市场）：
- 次级修正走势几乎都呈ABC三波浪模式
- BC浪的成交量会萎缩
- 多头市场C点可能试探或向下突破A点价位（空头市场相反）

**操作方法**：
- 在BC浪跌势中，**成交量必须缩小**
- 当价格向上突破BC浪的下降趋势线时 → 建立投机性头寸
- 止损点设定在A点与C点的较低价位
- 突破时若**大成交量** → 可能是主要多头市场第二只脚（非空头修正）

### 6.5 四天准则

**四天准则**：在中期走势中，当市场在高点或低点以**连续四天下跌或上涨**的走势呈现反转时，趋势很可能发生变化。

**统计验证**（1926~1985年道琼斯工业指数）：
- 25%的中期趋势立即呈现与四天排列相同方向的反转
- 41%在四天排列后6天内发生反转
- 75%在24天内发生反转
- 仅10%的中期趋势未出现有效的四天排列

**四天辅助准则**：中期走势发展到相当程度后，若出现顺势的四天（或以上）排列，**随后出现的第一天逆趋势行情**经常代表趋势变动的顶部或底部。

### 6.6 缺口准则与三天高低价准则

| 准则 | 内容 |
|------|------|
| **缺口准则** | 趋势线上方/下方出现缺口时，反映重大变化，显示趋势可能变动。缺口必须穿越趋势线才有效 |
| **三天高低价准则** | 运用最近三天的盘中高价与低价。价格反转穿越三天高价/低价时做多/卖空，以第三天低价/高价为止损点。止损被触发则反转头寸 |

**重要提示**：这两个准则必须配合另一个确认原则使用。

### 6.7 技术分析使用原则

1. **不可仅根据技术分析交易**——必须综合基本面分析与市场心理
2. 选择经时间考验的主要工具（1-2-3准则、2B准则）
3. **不可执著于任何技术指标**——任何工具被普遍认同后会失去效力
4. 当指标不再有效时，必须调整或抛弃
5. 辅助性工具告诉我们"不可以做什么"，而非"可以做什么"

## 7. 辅助性技术分析工具

### 7.1 移动平均线

#### 200天移动平均线（最理想）

**买卖准则**：
| 信号 | 条件 |
|------|------|
| **买进** | 200天MA由下降转为平坦或上升，且价格由下往上穿越MA |
| **卖出** | 200天MA由上升转为平坦或下降，且价格由上往下穿越MA |

**历史表现**：1917~1967年道琼斯工业指数，年平均回报率18.5%。

**个股筛选铁律**：
- 价格低于MA → **绝不买进**
- 价格高于MA → **绝不卖出**

#### 10周/30周移动平均线穿越准则

| 信号 | 条件 |
|------|------|
| **买进** | 10周MA穿越30周MA，两者斜率都向上，价格同时位于两条MA上方 |
| **卖出** | 10周MA穿越30周MA，两者斜率都向下，价格同时位于两条MA下方 |

### 7.2 相对强度（Relative Strength）

**核心概念**：衡量个股/类股与大盘指数之间的比率关系。

**关键原则**：
- **绝不因价格偏低而买进**——股价偏低都有理由
- 买进相对强度最高的股票（上涨速度快于一般股票）
- 上升趋势中，寻找同一天或稍早也创新高的股票（市场领导股）
- 等待回档在跌势中买进，迅速回升机会较高

**卖空选择**：
- ❌ 不卖空最强势股（判断错误损失惨重）
- ❌ 不卖空最弱势股（下档空间有限）
- ✅ 卖空**相对强度适中**的股票（风险收益平衡）

### 7.3 动能震荡指标

**核心理念**：市场也具有动能（如同钟摆），衡量市场参数在某期间内的变动率。

**股票市场两种震荡指标**：
- 以**行情宽度**为参数
- 以**价格**为参数

**商品市场震荡指标**：以两种移动平均值之差值为参数。

**使用原则**：
- 震荡指标越高 → 市场越超买
- 接近顶部时，震荡指标"迟缓下来"（趋于平坦做头）
- 价格反转时，震荡指标持续向下直至超卖

### 7.4 市场宽度（腾落线A/D线）

**定义**：计算每天上涨家数与下跌家数的差值并绘制为图。

**核心用途**：弥补加权指数缺失。道琼斯仅30种股票，若权重股异动，指数不能有效反映整体表现。

**背离信号**：腾落线与大盘指数走势背离 → 通常代表趋势即将变动。

**斯波朗迪自创指标**：10天约数期净变动移动平均行情宽度震荡指标——衡量中期"超买"或"超卖"情况。

### 7.5 辅助工具的定位

| 工具层级 | 工具 | 作用 |
|----------|------|------|
| **核心** | 1-2-3准则、2B准则、市场平均寿命 | 主要决策依据 |
| **辅助** | 成交量、200天MA、市场宽度、震荡指标 | 确认核心信号 |

## 8. 风险管理方法论

> "我用了很多年的时间思考如何衡量风险。在金融市场中，这是一个非常复杂的问题。"

### 8.1 风险的真正意义

**投机与赌博的根本差异**：
- **赌博**：盲目承担风险
- **投机**：仅在掌握有利胜算时承担风险

**对传统风险指标的批判**：
- 阿尔法系数、贝塔系数、市盈率、账面价值、收益率——都是**次要考虑因素**
- 首要任务：**判断市场趋势的最可能方向**
- 唯一根本问题：**"市场维持目前趋势的胜算有多少？"**

### 8.2 市场平均寿命架构（革命性方法）

**核心思想**：将人寿保险的死亡率统计方法应用于市场走势——市场走势的幅度与期限形成具有显著统计意义的正态分布。

**统计示例**（多头市场主要中期走势）：
- 涨幅平均数：**20%**
- 50.89%的走势涨幅介于15%~30%
- 最小幅度4.3%，最大幅度116.6%
- 25%的走势涨幅超过30%

**1987年崩盘前的风险评估**：
- 1987年5月20日~8月25日：96天内上涨22.9% → 已达历史平均水平
- 指标背离：工业指数创新高，腾落比率未创新高
- 市盈率21倍（1962年以来最高）
- 账面价值对股价比率高于1929年水平
- "市场不仅不是身心健康的人，还酒精中毒+肺炎第三期+每天三包无滤嘴烟"

**风险判断逻辑**：
1. 根据历史正态分布判断当前走势的"年龄"
2. 检查"病历表"（各项指标）
3. 年龄+体质 → 判断保费水平（风险水平）

### 8.3 风险/回报比率

**基本公式**：

```
风险/回报比率 = |买进价位 - 出场价位| / |目标价位 - 买进价位|
```

**铁律**：风险/回报比率**最多1:3**（即潜在回报至少是潜在损失的3倍）。

**操作实例**（1989年10月黄金）：
- 买进点：376.70美元
- 心理止损：375.70美元
- 目标价位：399.50美元
- 风险/回报比率 = 1/22.8 ≈ **1:22.8**

### 8.4 以胜算管理配置资本

**资金管理系统的三个部分**：
1. 评估风险/回报的方法
2. 判定交易成功几率的方法
3. 资产配置的系统

**三原则的执行流程**（以黄金交易为例）：

| 步骤 | 对应原则 | 操作 |
|------|----------|------|
| 1 | 保障资本 | 利用杠杆+止损，资金未暴露在偏高风险中 |
| 2 | 一致性获利 | 价格到目标价位时，了结相当比例头寸锁定获利 |
| 3 | 追求卓越 | 以一部分获利继续持有头寸追求更高业绩 |

**关键**：即使长期趋势判断错误，资金管理方法也能让你全身而退并保有部分获利。

### 8.5 资本配置的8个评估问题

在评估股票市场风险时，必须回答：

1. 长期趋势如何？（上升/下降/窄幅盘整/变化中）
2. 目前长期趋势的期限与幅度如何？（初期/末期/中间阶段）
3. 目前的中期趋势如何？
4. 道氏理论对目前行情有何看法？（背离？成交量？行情宽度？）
5. 移动平均线情况如何？（买进/卖出/持有）
6. 震荡指标显示什么？（超买/超卖/中间）
7. 经济面情况如何？（通胀、美联储政策、负债、利率、美元、消费者心态、产业强弱、潜在问题）
8. 是否存在可供运用的重大谬论？

## 9. 交易准则体系（19条）

> "交易者发生亏损最常见的理由是未遵循交易准则，认为'这一次'例外。"

### 9.1 19条交易准则总览

| 编号 | 准则 | 核心要点 |
|------|------|----------|
| 1 | 根据计划进行交易，严格遵守计划 | 知道目标+如何达到；界定所有可能状况及对策 |
| 2 | 顺势交易，"趋势是你的朋友" | 知道根据什么趋势交易；1-2-3准则判断反转 |
| 3 | 尽可能采用止损单 | 预先确定错误价位；大单用心理止损 |
| 4 | 一旦心存怀疑，立即出场 | 持仓今天都应是买进/卖出机会；习惯性恐惧不适合交易 |
| 5 | 务必要有耐心，不可过度扩张交易 | 股票≤10种，期货≤5种；注意力问题 |
| 6 | 迅速认赔，让获利头寸持续发展 | **最重要最常被违反**；市场是法庭，被证明错误前你正确 |
| 7 | 不可让获利头寸演变为亏损 | 风险/回报变1:2时，止损移至成本之上 |
| 8 | 在弱势中买进，在强势中卖出 | 2B准则适用；双向运作是维持友谊最佳方式 |
| 9 | 多头初期做投资者，后期与空头做投机者 | 投资关心盈余股息；投机关心价格走势 |
| 10 | 不可摊平亏损——亏损头寸不可加码 | 摊平亏损=避免承认错误的借口；与计划性加码的区别在于有无止损 |
| 11 | 不可仅因价格偏低而买进，偏高而卖出 | 交易无"特价品"；除非见趋势变动征兆 |
| 12 | 只在流动性高的市场中交易 | 选取最近月份合约；成交量大而活跃的个股 |
| 13 | 价格变动迅速时不可建立头寸 | 快速市场报价不可靠；不要交易 |
| 14 | 不可根据特殊情报交易 | 99.9%是个人看法；不可提供免费消息 |
| 15 | 永远要分析自己的错误 | 错误与失败是最好的导师；错误源于恐惧 |
| 16 | 务必提防"错误的并购消息" | 唯一实现的特殊消息可能是takeunder |
| 17 | 交易成功与否取决于执行则不该从事 | 重复核对；养成确认报价的习惯 |
| 18 | 必须保有自己的交易记录 | 记录日期、时间、工具、买卖、报价、成交价 |
| 19 | 了解并遵守法则 | 每条准则至少5种违背方法；90种亏损新方法 |

### 9.2 准则6详解——最重要的准则

**"迅速认赔，让获利头寸持续发展"**

**市场法庭比喻**：
- 你是被告——在被证明有罪前，你是清白的
- 被证明为错误前，你是正确的
- 价格触及止损点 = 最高法院判决，不能上诉

**1:3风险/回报比率下**：每一次获利可抵消三次损失。

### 9.3 准则7详解——保护获利

**操作方法**：
1. 风险/回报变为1:2时 → 止损点调整至成本之上（持有必然获利的头寸）
2. 到达目标价格 → 了结1/3或一半获利，调高止损
3. 持续朝有利方向发展 → 继续调高止损
4. 长期趋势交易 → 适当价位扩大头寸规模

### 9.4 85%法则

> 没有任何人可以永远保持100%完美。如果实际上可以保持**85%**的平均水平，表现便相当不错了。

**含义**：
- 以100%准备，但只有85%可接受
- 永远有想不到的事情会让你亏损
- 心理上有所准备，可能因意想不到的原因而亏损

## 10. 情绪纪律方法论

> "差别并不在于智慧与知识，而是在于执行知识的意志力。"

### 10.1 史波克症候群——理性与情绪的战争

**核心问题**：大多数人接受情绪与理性互不相容的前提，认为它们必然冲突。这被称为"自我折磨的史波克症候群"。

**解决之道**：挑战上述前提——情绪与理性不是两种互不关联的人性层面。

### 10.2 情绪的本质——价值观的产物

**布兰登的定义**：情绪是人类通过评估现实环境某些层面对自身利弊关系，并根据经验所做的身心反应。

**核心认知**：
- 情绪是一种源于心理程序（价值评估）的生理反应
- **价值观是取舍的判断标准**
- 价值观存在于潜意识中，排列为井然有序的结构
- 我们在意识或潜意识中选择价值 → 成为情绪电脑的程式软件
- **价值观的有效性决定情绪的有效性**

**艾恩·兰德的观点**：
- 情绪机制中的价值标准并非先天存在
- 情绪的冲突起源于价值观的冲突
- 人类有能力通过理性思考修正价值观

### 10.3 情绪纪律的练习

**定义**：情绪纪律是接受情绪的持续性程序，试图辨识其在价值观上的根源，评估与当前事件的有效性，并在目标与行动之间建立经过综合的一致性关系。

**练习步骤**：
1. **接受情绪**——不与情绪搏斗，将其视为自我不可或缺的一部分
2. **辨识根源**——了解情绪在价值体系中的根源
3. **评估有效性**——评估情绪对当前事件反应的有效性
4. **设想后果**——设想情绪冲动反应可能造成的后果
5. **选择行动**——相信你有选择理性或非理性行为的余地

**意志力的本质**：
- "决定"在拉丁文中意思是"排除"（cut off）
- 真正做出决定时，心智已认定没有其他更理想的选择
- 执行计划不再是选择性问题，而是必要性问题

### 10.4 交易成功的四步法

| 步骤 | 内容 | 难度 |
|------|------|------|
| 1 | 建立目标 | 相对容易 |
| 2 | 获得市场的相关知识 | 相对容易 |
| 3 | 界定有效的交易准则 | 相对容易 |
| 4 | **严格遵循交易准则执行上述三者** | **非常困难** |

**前三点容易，第四点困难**——原因在于执行需要情绪纪律。

### 10.5 情绪纪律的长期效益

- 控制心智，培养决断力
- 降低情绪冲突的程度与频率
- 促成一致性的思想、行动与感觉
- 不仅对交易重要，也是一种积极的生活方式
- **唯一方法**：促使身心一致性并持续滋生自我尊严

## 11. 期权交易方法论

> "期权始终是我最偏爱的交易工具。我个人交易生涯的获利至少有40%来自于期权。"

### 11.1 期权的四大优势

| 优势 | 说明 |
|------|------|
| 绝对控制风险 | 下档风险有限，上档潜能无限 |
| 最大财务杠杆 | 小量资本控制大量市场资产 |
| 弹性策略 | 根据最低风险设计各种交易策略 |
| 卓越业绩潜能 | 行情波动剧烈时期权费波动幅度超过相应工具 |

### 11.2 维克托的期权交易准则

| 准则 | 内容 |
|------|------|
| 1 | 保持偏低交易规模：投入资金为风险资本的**2%~3%** |
| 2 | 只有掌握胜算时才交易 |
| 3 | 风险回报比率至少为**1:5** |
| 4 | 不可仅因价格偏低而买进，偏高而卖出 |
| 5 | 任何一个期权头寸规模**不超过风险资本的10%**（强烈认定时） |

### 11.3 期权定价的心理成分

**反对数学模型（如Black-Scholes）的理由**：
- 数学模型无法处理期权定价中最重要的因素——**期权费的心理成分**
- 价格波动剧烈时，期权费主要驱动力量是市场心理，而非时间因素
- **"你不能根据过去的资料来赚钱"**

**关键认知**：将期权价格视为"在某期间内从事纯粹杠杆操作的权利"，而非内涵价值+时间价值。

### 11.4 期权时机选择

**核心问题**：市场会在哪一段时间出现多大的走势？

**统计辅助**（道琼斯工业指数1939~1992年）：
- 单月至少2%波动几率：54.9%
- 单月至少4%波动几率：33.8%
- 两周内2%以上走势：43%
- 两周内5%以上走势：仅6%

**策略**：
- 在每个人都不想要期权时买进，每个人都想买进时卖出
- 顺着错误看法的趋势操作，在看法改变之前及时脱身（"索罗斯"的机会）

### 11.5 期权策略性计划——跨式价差交易

**适用场景**：涨跌互见而相对稳定的市场。

**操作方法**（以OEX指数为例）：
1. 销售未经抵补的跨式价差交易（同时卖出同执行价同到期日的看涨+看跌期权）
2. 在执行价格两翼建立保护性头寸（买入更高执行价看涨+更低执行价看跌期权）
3. 锁住风险，获取期权费净收益

**优势**：即使市场几乎没有变动也可获利，风险很低，杠杆倍数理想。

### 11.6 期权交易六要点

1. 偏高的期权费代表消息面非常敏感
2. 任何一个期权头寸规模绝对不超过风险资本的30%
3. 永远必须了解交易的胜算与风险/回报比率
4. 不可采用一般的数学模型
5. 虽然销售期权获利频率高，但长期而言通常发生亏损
6. 建立的交易策略应限制风险，风险回报比率至少1:3，最好1:5

## 12. 当日冲销专业交易方法

> "当日冲销的交易十分简单，因为我的个性非常适合，'认赔'从来都没有困扰过我。"

### 12.1 必要的心理特质

| 特质 | 说明 |
|------|------|
| **自尊** | 感觉自己能胜任、有价值 |
| **诚实** | 永远诚实地对待自己；不可欺骗自己 |
| **认赔能力** | 接受数以千或万计的损失，再次充满信心交易 |
| **内省能力** | 深刻了解自己是谁 |

**致命杀手——自我（谬误的自尊）**：
- 自我在潜意识中说"我不可能犯错，因为我很了不起"
- 绝对不可让自我取代自尊的地位
- **"你是你自己唯一的敌人；你是你自己的主人"**

### 12.2 必要的知识——预测行情的方法

**核心原则**："应该发生的事情没有发生，立即反转头寸。"

**预测的10个考虑因素**：
1. 长期、中期与短期趋势的知识
2. 对当天经济新闻的了解与感觉
3. 季节性知识（《股票交易者年鉴》）
4. 技术性因素（期权到期、程式交易、图形排列、机构动作）
5. 近期政治动向
6. 美联储政策及干预态度
7. 世界局势（《巴伦周刊》《华尔街日报》等）
8. 健全心智态度（不承受外来压力）
9. 良好生理状况（健康、清醒、机警）
10. 专注（留意观察所有市场及其相互关系）

### 12.3 S&P期货当日冲销的时间规律

| 时段 | 走势特征 |
|------|----------|
| 开盘~10:00 | 开盘与反开盘走势；跳空缺口95%在10~15分钟内反转 |
| 10:00~12:00 | 单方向震荡走势；关注美联储行动（通常11:40左右） |
| 12:00~14:30 | 反方向震荡走势；相对平静 |
| 15:10后 | 收盘走势；80%情况下3:10的方向也是收盘方向 |

**开盘价关键判断**：
- 后续发展高于开盘交易区间 → 买进，止损设区间高价下方一档
- 开盘价是当天最低价 → 向上走势非常强劲
- 开盘价上方1点了结部分获利（场内交易员默契）

### 12.4 16条交易员准则（当日冲销专用）

1. 不可过度交易
2. 不可把亏损带回家
3. 不理想的交易头寸不可加码
4. 不可让获利演变为亏损
5. 交易前务必先决定止损位置
6. 不可成为单方向交易员，保持弹性
7. 适当情况下可追加获利头寸（突破整理区间时）
8. 在场内仅专注于一个营业厅
9. 了解所有场内交易员与经纪人癖性
10. 除非行情非常活跃，否则不要在交易时段中间插进交易
11. 离开场内或报价机时，每个头寸必须设定止损
12. 涨势中买进或跌势中卖空时务必谨慎
13. 控制情绪：恐惧、贪婪、期待、焦虑、草率、陶醉、谬误自尊
14. 保持耐心
15. 迅速认赔，让获利头寸持续发展
16. 没有把握不要交易

### 12.5 短期趋势的7项判断指标

对股票或商品评估（每项多头+1分，空头-1分）：

1. 比昨天创新高/新低？
2. 开高/开低？
3. 收盘上涨/下跌？
4. 收盘价位于交易区间上半/下半部分？
5. 收盘价高于/低于今天开盘价？
6. 今天低价高于/低于昨天低价？高价低于/高于昨天高价？
7. 收盘价高于昨天高价/低于昨天低价？

**统计结果**：
- 7项全多头 → 90%情况下第二天高价高于今天高价
- 7项全多头 → 80%情况下第二天收盘价高于今天收盘价

## 13. 交易者的特质与个性

> "金融交易为何如此困难？因为你不能说谎、逃避、佯装或为失败找寻合理的借口。"

### 13.1 特质与个性的区分

| 概念 | 定义 |
|------|------|
| **特质（Character）** | 反映个人的道德伦理哲学；指引行为与抉择的价值观 |
| **个性（Personality）** | 个人如何行为；行为与情绪上的整体倾向 |

### 13.2 神经质交易者 vs 成功交易者

| 维度 | 神经质交易者 | 成功交易者 |
|------|-------------|-----------|
| **目的** | 获取赌博"快感" | 执行交易计划 |
| **面对困难** | 逃避（酗酒、赌博等麻醉） | 面对问题、执行止损 |
| **责任** | 归咎他人、运气 | 自己承担所有盈亏 |
| **计划** | 无计划、凭冲动 | 有明确计划 |
| **学习** | 拒绝承认犯错可能 | 从错误中学习改进 |

### 13.3 成功交易者的特质清单

**特质（10项）**：

| 英文 | 中文 |
|------|------|
| Judgment | 判断力 |
| Courage | 勇气 |
| Self-criticism | 自省 |
| Commitment | 投入 |
| Honor | 荣誉感 |
| Integrity | 正直 |
| Loyalty | 忠诚 |
| Self-truthfulness | 自信 |
| Devotion to winning | 获胜的决心 |
| Honesty | 诚实 |

**个性（10项）**：

| 英文 | 中文 |
|------|------|
| Discipline | 纪律 |
| Concentration | 专心 |
| Patience | 耐心 |
| Inspiration | 敏锐 |
| Passion | 热情 |
| Competition | 竞争 |
| Intensity | 专注 |
| Detachment | 客观 |
| Supreme desire to win | 获胜的强烈欲望 |
| Perfectionism-type A | 完美主义A型个性 |

**态度（6项）**：

| 英文 | 中文 |
|------|------|
| Self-esteem | 自尊（感觉自己很棒、有能力、有价值） |
| Optimism | 乐观（永不放弃，看见积极面） |
| Trust | 信心（胜任的感觉） |
| Flexibility | 弹性（应变能力） |
| Perseverance | 毅力（不断研究学习） |

**智识（5项）**：清晰（clarity）、智慧（wisdom）、想象力（imagination）、创造力（creativity）、明确（certainty）。

### 13.4 冲动性行为的特征（成功交易者的反面）

- **缺乏控制**：行为反复，易于受诱惑，明知故犯
- **缺乏计划**：想法与行动间隔极短，无预先规划
- **无法从失败学习**：无一致性标准判断方法有效与否
- **缺乏思考方法**：仅猜测、心存侥幸，不愿深入思考
- **放弃原则**：为立即满足放弃道德原则
- **生活失衡**：除追求立即满足外无任何目标

### 13.5 交易者的根本原则

> 艾恩·兰德："交易者是'理性'人类之间所有关系的象征，也是人类尊严的道德象征。交易者将以自己的力量赚取收入，不会要求或给予非分之物。交易者不会对自己的失败要求偿付，也不会要求人们赞许他的错误。"

**了解并接受它，或者离开这个行业。**

## 14. 核心检查清单

### 14.1 交易前检查清单

- [ ] 风险/回报比率≤1:3？
- [ ] 是否根据计划进行交易（准则1）？
- [ ] 顺势交易——了解根据什么趋势交易（准则2）？
- [ ] 预先确定止损价位并设定止损单（准则3）？
- [ ] 对交易有信心，无心存怀疑（准则4）？
- [ ] 耐心等待适当机会，未过度扩张交易（准则5）？
- [ ] 在流动性高的市场交易（准则12）？
- [ ] 价格变动迅速时未建立头寸（准则13）？
- [ ] 未根据特殊情报交易（准则14）？
- [ ] 1-2-3准则或2B准则是否确认趋势变动？
- [ ] 道氏理论——所有相关指数是否相互确认？
- [ ] 市场平均寿命架构——走势"年龄"是否偏高？
- [ ] 200天移动平均线是否支持交易方向？
- [ ] 成交量关系是否配合趋势发展？
- [ ] 完成资本配置8个评估问题？

### 14.2 持仓中检查清单

- [ ] 风险/回报变为1:2时，止损是否移至成本之上（准则7）？
- [ ] 到达目标价位是否了结部分获利？
- [ ] 是否持续调高止损点保护获利？
- [ ] 是否仅顺着交易方向移动止损？
- [ ] 是否未摊平亏损（准则10）？
- [ ] 一旦心存怀疑是否立即出场（准则4）？
- [ ] 动力系统颜色是否仍支持持仓方向？

### 14.3 交易后检查清单

- [ ] 是否分析了交易错误（准则15）？
- [ ] 是否保有自己的交易记录（准则18）？
- [ ] 是否遵循了所有适用的交易准则？
- [ ] 如有违背准则，是否识别了违背原因？
- [ ] 情绪纪律——是否诚实面对自己的情绪反应？

### 14.4 风险管理速查

| 指标 | 标准 |
|------|------|
| 单笔交易风险/回报 | ≤1:3（最好1:5） |
| 期权单笔头寸规模 | ≤风险资本10%（通常2%~3%） |
| 期货最初头寸 | ≤总资本10% |
| 单笔潜在损失 | ≤风险资本1%~2% |
| 季度操作 | 亏损后按比例缩小，获利后部分运用新头寸 |
| 走势"年龄"评估 | 对照历史正态分布，超过平均数需提高警惕 |

## 结语：方法论整合的精髓

斯波朗迪方法论的精髓在于**整合**与**纪律**：

**整合层面**：
- 七大知识领域（胜算、市场工具、技术分析、统计概率、经济学、政治学、心理学）的整合
- 技术分析+基本面分析+市场心理的整合
- 1-2-3准则+2B准则+市场平均寿命+辅助指标的整合
- 保障资本+一致性获利+追求卓越的整合

**纪律层面**：
- 知识本身不是成功的保证——38位培养的交易员仅5人稳定获利
- 差别在于**执行知识的意志力**
- 情绪纪律是连接知识与行动的唯一桥梁
- "如果可以根据基本要点思考，将交易方法与经过时间考验的原理整合起来，那么你可以适应不断变化的任何市场情况"

**最终目标**：如斯波朗迪所追求的——"经年累月地稳定赚钱"，在心理上与财务上界定一种从市场中获得成功的方法，实现真正的自由。

> **"成功与快乐来自于自我潜能的实现，并认识到错误是不可避免的过程，甚至是生命的一部分。"**
