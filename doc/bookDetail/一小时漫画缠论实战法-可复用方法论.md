## 1. 总体框架：缠论的底层逻辑

### 1.1 缠论是什么

缠论源于网络隐世高手"缠中说禅"在博客上发表的"教你炒股票"系列文章，是一套**完全适用于 A 股市场**、也可用于任何金融品种技术分析的完备交易系统。

| 维度 | 缠论的立场 |
|------|-----------|
| **本质** | 科学（含交易心理学、形态学原理），而非实证归纳 |
| **核心思想** | 把行情走势分类，结合不同级别时间周期，做到"不测而测" |
| **技术面基础** | 在无序走势中建立有序结构，用数学递归找出级别 |
| **确定性来源** | "一个级别的走势一定会完成"——股市不确定性中的确定性 |
| **交易性质** | 绝对的**右侧交易**，哪怕预测到最高点也要等信号出现再操作 |
| **适用范围** | 超短线、短线、中线、长线都适用（通过级别切换） |

### 1.2 缠论的两大组成部分

缠论由**技术面 + 基本面**两部分组成，本书聚焦技术面。

**基本面要素**（缠论认为确定后在一段时间内不会有太大变化）：
- 行业现状（夕阳 or 热门）
- 财务数据
- 国家政策
- 公司市值在行业中的地位
- 板块总体形势
- 股价处于历史高位还是低位

### 1.3 缠论 vs 其他理论

| 对比项 | 缠论 | 道氏理论 / 江恩理论 |
|--------|------|---------------------|
| 结论客观性 | 正确答案唯一，严谨 | 因人而异，"千人千浪" |
| 适用市场 | 专为 A 股设计，适用任何金融品种 | 普适 |
| 分析精度 | 可用 1 分钟周期图形分析 | 通常大周期为主 |

### 1.4 作者的市场观（重要前提）

> 缠论创始者把市场看作**投机场所**，所有走势都是资金驱动的结果。

但作者（管鹏）补充了一个价值投资视角的交叉点：
- **中枢的形成 = 多空资本博弈 = 市场对公司基本面或未来业绩的看法产生分歧**
- 分歧不断升级 → 中枢扩张；分歧消失 → 趋势出现
- **周线以上级别的走势反映的就是市场对基本面的看法**
- 决定股价走势的根本动力是公司基本面与业绩的变化，资金只是表层因素

## 2. 基础组件体系：从 K 线到走势的递进

缠论的所有形态都有专有名称，甚至 K 线都已不是原来的定义。整个体系是一个**逐级递进**的结构，理解每一层是理解下一层的前提。

### 2.1 递进层级图

```
K线（兵）→ 分型（班）→ 笔（排）→ 线段（连）→ 中枢 → 走势类型（营）
     ↑                                              ↓
     └────────── 结合律贯穿所有层级 ←── 多义性允许灵活组合
```

| 层级 | 比喻 | 组成 | 最小数量 | 说明 |
|------|------|------|----------|------|
| K 线 | 兵 | 单根 | 1 | 只看价格区间，不看开收盘 |
| 分型 | 班 | K 线组合 | 3 根 K 线 | 4 种类型 |
| 笔 | 排 | 顶/底分型 + K 线 | ≥5 根 K 线 | 最小分析单元 |
| 线段 | 连 | 笔 | ≥3 笔（需重叠） | 体现趋势 |
| 中枢 | — | 线段/笔 | ≥3 根（本体）+ 进出段 | 筹码密集区，重中之重 |
| 走势类型 | 营 | 中枢 | 盘整1个/趋势≥2个 | 大佬级别，走势必完成 |

### 2.2 学习路径的严格顺序

> 缠论是**连环知识点，一环扣一环**，其中一环漏掉就很难接上。

**必须按顺序掌握**：K 线 → 分型 → 包含处理 → 笔 → 线段 → 中枢 → 走势类型 → 级别 → 背驰 → 区间套 → 买卖点 → 结合律 → 多义性

## 3. K 线与分型方法论

### 3.1 缠论的 K 线定义（与传统不同）

| 维度 | 传统 K 线 | 缠论 K 线 |
|------|-----------|-----------|
| 关注点 | 开盘价、收盘价、上影线、下影线 | **只看股价所到达的价格区间** |
| 确定方式 | 四个价格 | 找到最高点和最低点即可 |

### 3.2 四种分型

单看 K 线没有实质用处，要看 K 线组合形成的**分型**。分型涵盖所有 K 线形态：

| 分型类型 | 特征 | 形象记忆 |
|----------|------|----------|
| **上升 K 线** | 出新高、不出新低 | 上楼梯 |
| **下降 K 线** | 出新低、不出新高 | 下楼梯 |
| **顶分型** | 中间高，两边低 | 像"品"字 |
| **底分型** | 中间低，两边高 | 顶分型的反面 |

**关键认知**：
- 顶分型 / 底分型是上升 / 下降 K 线的**拐点**
- 出现顶分型 → 顶部形成，股价要下降；反之亦然
- 小级别 K 线图上股价波动太频繁，**不结合其他指标单独判断操作没有意义**

### 3.3 包含关系处理（合并规则）

当两根相邻 K 线中，一根被另一根**完全包含**时，不符合四种分型条件，必须合并。

**合并铁律：永远先判断方向！**

| 前走势方向 | 合并取值 |
|------------|----------|
| **上升**（分型上升） | 取两根 K 线中**最高的高点**和**最高的低点** |
| **下降**（分型下降） | 取两根 K 线中**最低的高点**和**最低的低点** |

- 多根 K 线包含 → 依次处理
- 理解后实际操作基本可以"看一眼瞬间合并"

### 3.4 分型的实战定位

分型的确立是为找到"笔"做准备。分型本身只是中间组件，**不要在小级别上用单一分型做操作决策**。

## 4. 笔的绘制方法论

### 4.1 笔的定义

笔是缠论中**最小的分析单元**。

**组成部件**（三个）：
- 顶分型 + K 线 + 底分型（或反过来）

**方向公式**：
```
上升笔 = 底分型 + 上升K线 + 顶分型
下降笔 = 顶分型 + 下降K线 + 底分型
```

### 4.2 笔的硬性规则（必须全部满足）

| 规则 | 说明 |
|------|------|
| 顶底必须对应 | 永远不会出现"底分型 + K线 + 底分型"，没有平行笔 |
| 中间 K 线最少 1 根 | 上不封顶，且**必须不属于顶分型和底分型** |
| 最少 5 根 K 线 | 最小的一笔也要由 5 根 K 线组成 |
| 笔是连续的 | 向上笔结束后一定跟向下笔，**不会出现隔空笔** |
| 延续性 | 下跌不足 5 根 K 线无法确认底分型 → 笔继续延续，直到新笔出现 |
| 确认滞后 | 新笔要等**更新的笔出现**后才能确立 |

### 4.3 为什么至少 5 根 K 线

- 牵扯到**结合律**问题（后续章节）
- 原理：任何走势都是多空博弈的结果
  - 多方力量足够强，股价才上升
  - 顶分型说明空方力量开始占优
  - 空方力量太弱（没走几步就被顶回）→ 不具分析意义 → 股价往往被多方再推一波
- 这样画笔**大大加强走势判断的确定性**

### 4.4 笔的实战建议

> 后期熟练后，基本一眼能看出整个级别的走势。只要这根笔不是太短，一般不太关注包含关系；只有五六根 K 线时才去数有没有包含。

**关键认知**：现在的精细，才会让将来的模糊更准确。缠论的应用是**熟能生巧**的过程——由精细变为模糊，但模糊基于前期扎实的精细功底。

## 5. 线段的构建方法论

### 5.1 线段的定义

线段由笔组成，意义在于：当你在 K 线图中到处找笔时，线段给你的笔一个直观体现。

| 要素 | 规则 |
|------|------|
| 最少组成 | **3 笔** |
| 重叠要求 | 三笔在价位上**必须有重叠部分** |
| 方向 | 只有向上、向下两种，**没有横向** |
| 起止 | 向上线段起始于向上笔，终结于向上笔（向下同理） |
| 禁忌 | 线段永远不会始于向下笔终于向上笔 |

### 5.2 线段的终结规则

- 向上线段的终结 = 下一个**向下线段**的产生
- 没有出现向下线段 → 线段一直延续（向上趋势没走完）
- 本质：线段体现趋势，趋势没走完就只能延续

### 5.3 笔与线段的关系（可相互转换）

| 派别 | 观点 |
|------|------|
| 笔派 | 笔最重要，线段可有可无 |
| 线段派 | 线段最有用，笔是废物 |

**作者立场**：不管哪种方式，能赢就行。笔和线段可相互转换：
- 5 分钟 K 线图的笔太乱 → 在 30 分钟 K 线图画笔
- 30 分钟 K 线图的笔看不清细节 → 在 5 分钟 K 线图的笔里找细节
- **大级别的一笔 ≈ 小级别的一个线段**

### 5.4 线段特征序列（了解即可）

原著中提到"线段特征序列"——把笔转变成 K 线，再通过 K 线分型确定线段结束，且转化的 K 线存在包含关系。

**作者建议**：这块作用较小，不必浪费时间，有兴趣可自行深造。

## 6. 中枢的画法方法论

> 中枢是缠论的**重中之重**，几乎所有分析都要用到它，买卖点也要根据它来定。

### 6.1 中枢的本质

- **筹码密集区**：多空力量势均力敌、你来我往形成的区域
- 表面是平台，实际充斥能量对抗，最终以一方获胜结束盘整
- **你来我往的时间越长，能量越大；能量越大，爆发的走势越强**
- 中枢的意义在于：**结束后会有一段行情的延续**

### 6.2 中枢的结构

理论上中枢以线段为基础画，**至少看 5 根线段**：

```
进入段(方向) + [中枢本体: ≥3根线段重叠] + 走出段(方向)
```

- 进入段和走出段**方向必须一致** → 这个方向就是中枢方向
- 中枢本体 = 中间线段的**重叠部分**
- 除去代表方向的两根线段，一个中枢至少由 **3 根线段**组成

### 6.3 中枢的方向规则（关键）

| 走势方向 | 中枢结构 | 永远不会出现 |
|----------|----------|--------------|
| **上升走势** | 下-上-下-上……最后以下-上-下结束 | — |
| **下降走势** | 上-下-上…… | 永远不会以上-下-上……上-下结束 |

### 6.4 画中枢的标准步骤

```
第一步：先判断方向（上升 or 下降）
第二步：找到中枢的长度
第三步：找重叠部分
         → 找高点里的最低 + 低点里的最高
         → 这两个点就是 ZG、ZD
第四步：画出中枢区间 [ZG, ZD]
```

> **缠论原著标准**：取中枢中**前三根线段**作为参考来画中枢。

### 6.5 笔中枢（作者推荐）

实际操作中，中枢也可由笔组成，画法与线段中枢一模一样，只是把线段替换成笔。

**作者偏好笔中枢的理由**：
- 方便又好认
- 看不出走势 → 缩小级别
- 看着太乱 → 放大级别

### 6.6 奔走中枢（特殊形态）

| 特征 | 意义 |
|------|------|
| 重合部分特别特别少 | 压制力量小 |
| 容易被遗漏 | 被原有力量瞬间秒杀 |
| — | 原走势非常强势 |

**实战提醒**：别看奔走中枢小，以后要根据它做买卖点！

### 6.7 中枢的确认与完结

- 中枢走完走出段后，又出现一个线段**没有回到中枢内** → 中枢确认完成
- 这个点就是**第三类买卖点**（后续详解）

## 7. 走势类型分析方法论

> 走势类型是**大佬级别**，中枢都是走势类型的一部分。打开 K 线图，里面全是走势。

### 7.1 两种走势类型

| 类型 | 定义 | 子类 |
|------|------|------|
| **盘整型** | 只有一个中枢的完整走势 | 上升盘整、下降盘整 |
| **趋势型** | 至少两个**同级别、同方向**中枢组建的完整走势 | 上升趋势、下降趋势 |

**关键认知**：缠论里**没有横向走势**，只有上升和下降！盘整也只是上升盘整或下降盘整。

### 7.2 趋势走势的硬性规则

1. **至少两个同方向中枢**：且两个中枢之间**不能有重合部位**
2. **重合 = 中枢扩张**：出现重合就变成一个中枢了，不能称为上升趋势
3. **同级别**：两个中枢必须是同级别（体型差不多 / 相同组件构成且数量差不多）
   - 笔中枢和线段中枢**肯定不在一个级别**
   - 大小差异很大的两个中枢没法组成一个走势

### 7.3 走势必完成（核心确定性）

> 缠论核心：**一个级别的走势一定会完成**。股价千变万化，都离不开这两种走势类型，且都会以这两种走势类型完结。

### 7.4 走势的组成部分命名

```
a（进入段）→ A（第一个中枢）→ b（走出段）→ B（第二个中枢）→ c → C → d ...
```

- 一个盘整走势只有 a、A、b 三个部分
- **绝大多数情况下，一个趋势走到 c 就差不多了**，不建议追高
- 很少有 C、d 阶段（由交易习性决定，除非全智能投资时代改变人性习惯）

### 7.5 走势的生长（重要概念）

走势会**由小级别生长成大级别**：

| 原状态 | 新状态 | 说明 |
|--------|--------|------|
| 原走势 a、A、b 走得好好的 | B 中枢级别变大 | 原 a、A、b 变成大走势中的 a |
| 原来的走势 | 大级别 a 里的上升盘整走势 | 原 B 变成大级别的 A |
| — | 将来出现同级别 b、B、c | 这段趋势类型才算走完 |

**实战意义**：
- 走势看似千变万化，其实都离不开两种走势类型
- 大小级别同时出现中枢时，**以大级别为主**
- 认清走势 → 知道什么时候该追、不该追、该卖、不该卖

### 7.6 盘整走势的经验特征（非绝对）

- 噌噌噌拉上来 → 一个中枢走上很长时间 → 再拉一下就结束走势
- 盘整走势可以出现一买、二买，但**出现不了三买就掉头向下完结走势**

## 8. 级别分析方法论

### 8.1 级别的两种含义

**含义一：时间周期**

| 使用 | 不使用 |
|------|--------|
| 1 分钟、5 分钟、30 分钟、日线、周线、月线 | 15 分钟、60 分钟 |

> 月线级别最大，1 分钟级别最小。

**含义二：形态级别**（同一时间周期内比较）

```
走势 > 线段 > 笔
（营）  （连） （排）
```

> 必须在**同一个时间周期内**比较。不同时间周期不好比：1 分钟图里的一个走势，在 30 分钟图里可能连一笔都构不成。

### 8.2 为什么不看 15 分钟和 60 分钟

| 级别关系 | K 线数量 | 分析意义 |
|----------|----------|----------|
| 5 分钟图 1 根 K 线 = 1 分钟图 5 根 | ≥5 根可成一笔 | 有意义 |
| 15 分钟图 1 根 K 线 = 5 分钟图 3 根 | <5 根 | **分析意义不大** |
| 日线 1 根 = 60 分钟图 4 根 | <5 根 | 没法构成最小分析单位，看多会乱 |

### 8.3 级别升级规则（走势生长的本质）

> 一个级别的升级，是由**该级别走势的完成**决定的。

| 升级路径 | 触发条件 |
|----------|----------|
| 1 分钟走势 → 5 分钟走势 | 1 分钟走势完结 |
| 5 分钟走势 → 30 分钟走势 | 5 分钟走势完结 |
| 30 分钟走势 → 日线走势 | 30 分钟走势完结 |
| 日线走势 → 周线走势 | 日线走势完结 |

**关键规则**：
- 如果 1 分钟 K 线图的走势走了几个月都没走完，哪怕在月线中都快成一笔了，这个走势**也只能是 1 分钟级别走势**
- 当把笔走势画出来时，它就是一条线段；**线段组成的走势类型才是该级别最大的形态级别**
- **以线段走势的完成作为级别升级的条件**

### 8.4 笔中枢升级为线段中枢

> 当一个笔中枢由 **9 根或 9 根以上笔**构成时，它就升级为**线段中枢**了。

### 8.5 级别的交易决策原则

| 原则 | 说明 |
|------|------|
| **各级别都要看** | 追求利益最大化，避免错过最佳买卖点 |
| **交易决策只按一个级别** | 30 分钟买入就等 30 分钟卖出，**不能混级别操作** |
| **大级别看位置** | 知道当下位置，游刃有余 |
| **小级别看细节** | 通过小级别提前确认大级别买卖点 |

### 8.6 不同交易风格的级别选择

| 交易风格 | 推荐级别 |
|----------|----------|
| 做 T（超短线） | 1 分钟、5 分钟 |
| 中短线 | 30 分钟、日线 |
| 长线 | 周线、月线 |

> **作者强烈建议**：先用大周期，熟练后再尝试小周期。

### 8.7 小级别看大级别的逻辑

- 小级别走势终究是大级别走势的一个组成部分
- 大级别的每一笔，在小级别中都有走势体现
- 如果没有，就一定会体现在更小的级别中

## 9. 背驰判断方法论

### 9.1 背驰的本质

- 一个上升走势代表多方力量获胜，但力量不会一直延续
- 跌下来之前，多方力量会有**撑不住、脱力**的表现 → 这就是背驰
- **背驰的意义：用来判断一个走势是否快要走完了**
- 出现背驰 → 走势呈衰弱趋势，可以开始找买卖点

### 9.2 背驰的分析位置

> **必须在 a、b、c 之间比较**（中枢的进入段和走出段）。

⚠️ **严禁**与中枢内的线段或笔比较——本来已经背驰了，看成没有背驰，重仓杀进去就会被套。

### 9.3 判断背驰的三种方法

| 方法 | 操作 | 备注 |
|------|------|------|
| **MACD 红绿柱面积** | 取红绿柱最高拐点处的面积 ×2，大体估算能量强弱 | 最常用 |
| **成交量** | 看量能缩减 | 缠论不太看重（涨停组成的上涨段成交量无参考意义） |
| **绝对长度法** | 比较中枢进入段和走出段（a、b、c）的绝对长度 | 后人研究，简单好用 |

### 9.4 绝对长度法的判断逻辑

> 中枢像一块磁铁，谁想离开都会被吸回来，只有力量足够大才能脱离。

| 情况 | 判断 |
|------|------|
| 走出长度越来越短 | 脱离磁力的力量越来越弱，走势可能即将被重新吸回中枢 → 新走势开启 |
| **b 的绝对长度 < a** | B 走完，走势可能随时完成，**不建议确定任何买卖点（尤其买点），风险较大** |
| c 没有背驰 | 接下来可能出现 C、d，直到背驰产生走势才慢慢结束 |

### 9.5 背驰的局限

- 背驰建立在走势分析之上，**走势没研究明白就先搞基础**
- 一般走势走到 c 基本完结，很少有能走出 C、d 的走势
- 背驰只能判断"快走完"，**精确判断需要区间套**（下一章）

## 10. 区间套应用方法论

### 10.1 区间套的本质

> 大级别单独拿出一根线段，在小级别中可能就是一个走势。从大级别往小级别找走势的过程，就叫**区间套**。

**别名**：区间放大镜——哪里不清楚，放大哪里。

### 10.2 区间套的核心应用：判断背驰

**问题**：如何判断走势是否正在发生背驰？

**方法**：用区间套放大。

| 大级别状态 | 小级别状态 | 结论 |
|------------|------------|------|
| 30 分钟 b 走得和 a 差不多高 | 5 分钟走势丝毫没有完结样子 | **目前 b 不会背驰** |
| 30 分钟 b 还没走多远 | 5 分钟走势已走到 c，马上完结 | **30 分钟的 b 已经背驰** |
| 30 分钟走到 b | 5 分钟也走到 b | 再放大到 1 分钟判断 |

### 10.3 区间套的手机地图类比

> 像用手机地图找景区：大级别看四川省 → 放大看到成都市 → 再放大看到都江堰景区 → 需要时继续放大。区间套只是把"找景区"变成了"找背驰、找买卖点"。

### 10.4 区间套的实战流程（以日线 b 段为例）

```
1. 日线走到 b 段，不确定是否背驰
2. 看 MACD 红绿柱面积，a 与 b 差不多 → 不确定
3. 找出 b 在 30 分钟图上的走势
4. 30 分钟走势已快走完 → 日线 b 该结束了
5. 接下来来一个同级别中枢 + c → 走势结束
6. 判断 b 已背驰 → 准备卖出
```

### 10.5 下跌走势的区间套应用

```
1. 日线看出小级别正在走一个走势，有明显背驰，正在走 c 段
2. 想参与 → 先在 30 分钟图画走势
3. 找到 c 段，用区间套找 5 分钟级别走势
4. 5 分钟正在走下降趋势的 B → 股价还会跌，不要着急出手
5. 等 c 走出来并完结走势 → 此刻才是入手时机
6. 走势完结迎来新走势开始 → 获利
```

### 10.6 区间套与买卖点的关系

> 通过放大判断大级别走势是否要完结，就可以寻找**绝佳的买卖点**。

**核心规律**：
- 大级别的买点 = 小级别的一买
- 大级别的卖点 = 小级别的一卖

## 11. 买卖点方法论（三买三卖）

> 缠论的买卖点根据**走势位置**确立，走势走到一定位置就会出现交易点，按指示操作即可。

### 11.1 三类买点详解

#### 第一类买点（真正的底）

| 要素 | 说明 |
|------|------|
| **位置** | 下降走势完结后的转折点 |
| **触发** | 由前面下降走势的**背驰**引发 |
| **前提** | 没有下降走势的完结，就没有一类买点 |
| **难度** | 极难准确抓到，"神仙才能做到" |
| **建议** | 新手**先不要参与**一类买点 |

#### 第二类买点

| 要素 | 说明 |
|------|------|
| **位置** | 一类买点后上涨，产生的回落低点 |
| **特征** | 回落低点**不会低于一类买点**的位置 |
| **意义** | **二买出现，一买才能确立** |
| **风险** | 容易赚不到钱 |

#### 第三类买点（赚钱神器）

| 要素 | 说明 |
|------|------|
| **位置** | 上升走势中枢走完，回落**不进入中枢** |
| **意义** | 排除处于盘整走势的个股（盘整时间长且没利润） |
| **重要性** | **买点里最重要的就是第三类** |
| **建议** | **基本一定要参与** |

> 我们赚钱就是要赚**趋势走势**的钱，三买的意义就在于排除盘整走势的个股。

### 11.2 三类买点的关系

```
没有一类买点 → 就没有二类买点 → 就没有三类买点
（但不参与不代表没有）
```

### 11.3 特殊情况：二三买合一

> 下降走势完结后，上升走势异常凶猛，形成的二类买点已经在前一个走势的中枢之上。

此时可以**二类、三类买点一起买**。这种情况下，哪怕出现盘整，也会有一定利润。

### 11.4 三类卖点

> 反过来就是三类卖点，道理完全一样。

| 卖点 | 位置 |
|------|------|
| 第一类卖点 | 上升走势完结后的转折点（真正的顶） |
| 第二类卖点 | 一卖后下跌反弹，高点不高于一卖 |
| 第三类卖点 | 下降走势中枢走完，反弹不进入中枢 |

### 11.5 买卖点与级别的关系（难点）

> 只看一个级别的 K 线图，三买三卖可以轻松找到。但缠论有级别，大级别的一个买卖点在小级别会出现好几个三买三卖。

**操作原则**：

| 原则 | 说明 |
|------|------|
| **分清各级别走势** | 知道什么级别目前是什么走势，买卖点只属于这个级别 |
| **确定操作级别** | 30 分钟买入就等 30 分钟卖出 |
| **新手挑大级别** | 30 分钟或以上，太小级别涨幅小、时间短、稍延误就错过 |
| **用区间套确认** | 大级别买卖点用小级别一买/一卖确认 |

### 11.6 大级别买卖点的确认流程

```
1. 大级别 K 线图即将走到买点
2. 用区间套看小级别走势
3. 小级别出现一买 → 确认大级别买点出现
4. ⚠️ 必须在小级别一买确立后，才能在大级别交易
5. 一买怎么确立？→ 等二买出现
6. 哪怕等到小级别二买，大级别也不过刚走高一点点，完全不用可惜
```

### 11.7 仓位分配建议

| 买点 | 仓位 | 说明 |
|------|------|------|
| 一买 | 1 成 | 风险大，轻仓试探 |
| 二买 | 3 成 | 中等仓位 |
| 三买 | 6 成 | 重仓主攻 |

### 11.8 买卖点的方向规则

| 买点 | 形成于 |
|------|--------|
| 第一类买点 | **下降走势结束后**形成 |
| 第二、第三类买点 | **上升走势中**形成 |

> 反之则是卖点。第一类买卖点最难掌握，不建议参与；大家可以在第二类买卖点下手，第三类买点基本一定要参与。

## 12. 结合律方法论

> 结合律像空气一样——看不见、摸不着，但无比重要。它是保证缠论分析结论**客观与唯一性**的绝招。

### 12.1 结合律的本质

结合律讲的是"结合的铁律"：多根 K 线结合成笔，多个笔结合成线段，多个线段结合成走势，多个走势结合成更大级别的走势。**结合是缠论的基础。**

### 12.2 结合律的数学公式

```
a + b + c = (a + b) + c = a + (b + c)

但 a + b + c ≠ (a + c) + b
```

- b 之所以可以和 a 或 c 组合，是因为 b 是**单独的个体**，不属于 a 和 c
- 缠论的分析就像 1+1 一样有客观答案，等于 2，而不是有人想等于 3、有人愿意等于 4
- 没有结合律 → 出现"千人千浪"

### 12.3 结合律的五大规则

| 规则 | 说明 |
|------|------|
| 1. 组件独立 | 结合的组件要独立，**任何组件不能重复使用** |
| 2. 顺序结合 | 必须按顺序结合，**不能跳过中间结构** |
| 3. 走势独立 | 可以结合的走势一定要相对独立 |
| 4. 走势完整 | 走势必须要完整 |
| 5. 整体合并 | 可以合并的走势必须是一整个走势的合并，**不可能两个走势的局部合并** |

### 12.4 结合律在走势划分中的应用

**典型问题**：当前走的是新走势，还是中枢的扩张？

**判断方法**：取决于接下来怎么走。

| 后续走势 | 结论 | 操作 |
|----------|------|------|
| 中枢走完继续上涨 | 要合并了（原走势没结束，走势生长了） | — |
| 中枢走完向下走 | 目前走势是新走势的一部分，不能合并 | — |

**如何判断中枢走完向上还是向下**：看回落或反弹有没有回到中枢内 → **第三类买卖点**。

> 出现第三类买卖点，也就确认了走势，这正是第三类买卖点的重要作用之一。

### 12.5 结合律的实战场景

遇到疑似下跌结束的股票，价格又回到中枢内震荡时：
1. 极有可能是**中枢生长** → 下跌走势还未完成
2. 只能**等**，等走势清晰才能判断买还是观望
3. 走势的清晰取决于**第三类买卖点的出现**
   - 出现三买 → 新的上涨走势，可以介入
   - 出现三卖 → 走势生长，下跌没走完，继续观望

## 13. 多义性方法论

### 13.1 多义性的定义

> 只要满足结合律，满足之前讲的所有知识点，就可以**随意组合**走势，这叫做走势的**多义性**。

面对同一段走势，可能有多种合法的划分方式——它们**都对**。

### 13.2 多义性的核心特征：不变性

> 在满足结合律的前提下，不管怎么划分，**买卖点都不变**。

| 特征 | 说明 |
|------|------|
| **客观性** | 买卖点一定，不因划分方式改变 |
| **不变性** | 随意结合但结果不变 |
| **可操作性** | 不影响操作，对应着操作就可以 |

### 13.3 多义性的实战价值

缠论是**右侧交易理论**，只有形态全部走完才能确定走势。但多义性允许我们：

```
1. 总结出几个接下来可能的走势（基于多义性）
2. 根据当前整体市场环境、利好利空，适当预测走势
3. 根据第三类买卖点的出现，确定走势
4. 做到"不测而测"
```

> 缠论本身不是用来预测的，但根据多义性可以总结可能走势，再根据第三类买卖点确认。

### 13.4 多义性常出现的位置

- 大级别 K 线图（如日线）正在进行的走势
- 接下来的路还很长，个股受外因影响
- 走势可能生长，也可能快速背驰用盘整形态结束

### 13.5 实战案例（视源股份）

| 阶段 | 分析 | 操作 |
|------|------|------|
| 关注时 A 在构建中，三买卖点未出现 | 做出几种预测 | 不操作 |
| 预测一：结束上涨开始下跌 | 以 2018 年末市场环境判断该走势已终结 | 暂时不关注，等三卖 |
| 三卖没出现，紧接 2019 年初大涨 | 基本判定要走趋势走势 | — |
| 三买出现，确认走势 | — | **买入** |
| 中枢 B 如约而至 | 利用这段时间在小级别交易，做低成本 | 做 T |
| c 出现，通过背驰分析 c 位置不会很高 | — | — |
| 出现卖点 | 利用区间套卖出 | **离场** |

## 14. 实盘交易执行方法论

### 14.1 交易决策的完整流程

```
① 基本面筛选（行业、财务、政策、市值、板块、股价位置）
        ↓
② 技术面分析：确定操作级别（如 30 分钟）
        ↓
③ 画笔 → 画线段 → 画中枢 → 确定走势类型
        ↓
④ 判断当前处于走势的哪个阶段（a/A/b/B/c）
        ↓
⑤ 用背驰 + 区间套判断走势是否快完结
        ↓
⑥ 等待对应级别的买卖点出现
        ↓
⑦ 用区间套在小级别确认买卖点（小级别一买/一卖）
        ↓
⑧ 按仓位规则执行交易
        ↓
⑨ 等同级别卖点/买点出现再平仓（不混级别）
```

### 14.2 级别操作纪律

| 规则 | 说明 |
|------|------|
| **买入卖出同级别** | 30 分钟买入 → 30 分钟卖出，**绝不混级别** |
| **新手用大级别** | 30 分钟或以上，避免小级别涨幅小、时间短 |
| **大级别看位置，小级别看细节** | 各级别都要看，但决策只按一个级别 |
| **用区间套确认** | 大级别买卖点 = 小级别一买/一卖 |

### 14.3 走势分析的关键问题

面对一只股票，依次回答：

1. **当前是什么级别的什么走势？**（盘整 or 趋势，上升 or 下降）
2. **走势走到哪个阶段了？**（a、A、b、B、c）
3. **走势是否快完结？**（用背驰 + 区间套判断）
4. **有没有出现买卖点？**（三买三卖）
5. **买卖点是否已确认？**（小级别一买/一卖出现）

### 14.4 常见错误与规避

| 错误 | 后果 | 规避方法 |
|------|------|----------|
| 混级别操作（30 分钟买、5 分钟卖） | 乱，亏损 | 买卖同级别 |
| 用小级别单一分型做操作 | 频繁误判 | 结合中枢、走势综合判断 |
| 背驰与中枢内线段/笔比较 | 看错背驰，重仓被套 | 只在 a、b、c 之间比较 |
| 没熟练就在各级别买卖 | 越来越乱 | 先掌握一个级别再扩展 |
| 似懂非懂就操作 | 觉得缠论是骗人的 | 先模拟/研究一段时间 |
| 追一类买点 | 很难抄到底 | 新手从二买、三买下手 |
| 日线级别以下频繁操作 | 散户劣势放大 | 多从基本面着手，用大级别 |

### 14.5 做低成本的技巧

在中枢 B 形成期间（走势已确认但未完结），可以利用这段时间在**小级别里交易**，做低成本。

## 15. 缠论的边界与作者忠告

### 15.1 缠论的适用边界

| 维度 | 说明 |
|------|------|
| **理论性质** | 右侧交易理论，形态走完才能确定走势，**不预测** |
| **精度边界** | 入门级别，只靠阅读书面文字很难理解透彻 |
| **能力要求** | 操作熟练后才可实盘，需配合其他书籍进修或自己研究 |
| **理念冲突** | 缠论创始者视市场为投机场所，与价值投资理念不完全一致 |

### 15.2 作者的核心忠告

1. **即使学会了缠论，也尽量不要做日线级别以下的操作**
2. **要多从基本面着手**，缠论只是帮你厘清思路、找出买卖点的工具
3. **投资还是要靠心**
4. 散户与专业投资者厮杀，大多数人很难改变注定失败的结果——散户劣势：知识有限、资金有限、消息面资源有限
5. 决定股价走势的根本动力是公司基本面与业绩变化，资金只是表层因素
6. 不要似懂非懂就胸有成竹地去操作

### 15.3 缠论的价值定位

> 这本书讲的是入门级别的内容，却是市面上难得的通俗易懂的缠论读本。

- 缠论是一个很系统的理论，中心思想是分类 + 级别 + 不测而测
- 包含交易心理学、形态学原理，本质是科学
- 周线以上级别走势反映市场对基本面的看法，理论核心很有价值
- 缠论可作为**没有完整交易系统的人**的起步框架

## 16. 核心检查清单

### 16.1 分析前自检

- [ ] 基本面是否筛选过？（行业、财务、政策、市值、板块、股价位置）
- [ ] 操作级别是否确定？（推荐 30 分钟或以上）
- [ ] 是否熟悉笔、线段、中枢、走势的画法？

### 16.2 画图自检

- [ ] K 线包含关系是否处理？（先判断方向再合并）
- [ ] 笔是否满足 ≥5 根 K 线？顶底是否对应？
- [ ] 线段是否 ≥3 笔且有重叠？
- [ ] 中枢是否先判断方向再画？ZG、ZD 是否正确？
- [ ] 走势类型是否确认？（盘整 1 个中枢 / 趋势 ≥2 个同级别中枢不重合）

### 16.3 买卖点自检

- [ ] 当前走势处于哪个阶段？（a/A/b/B/c）
- [ ] 是否用背驰判断走势快完结？（只在 a、b、c 间比较）
- [ ] 是否用区间套在小级别确认？（小级别一买/一卖）
- [ ] 买卖是否在同一级别？（不混级别）
- [ ] 仓位是否按规则分配？（1:3:6）
- [ ] 是否等到了第三类买卖点确认走势？

### 16.4 风险自检

- [ ] 是否在做日线级别以下操作？（不建议）
- [ ] 是否脱离基本面只看技术面？（不建议）
- [ ] 是否似懂非懂就操作？（不建议）
- [ ] 是否在小级别单一分型上做决策？（不建议）

## 17. 术语速查表

| 术语 | 含义 |
|------|------|
| **缠论** | "缠中说禅"博客"教你炒股票"系列总结的交易体系 |
| **不测而测** | 不预测，但通过分类和级别做到客观判断 |
| **走势必完成** | 一个级别的走势一定会完成（核心确定性） |
| **分型** | K 线组合形态，4 种：上升K线、下降K线、顶分型、底分型 |
| **包含** | 两根相邻 K 线一根被另一根完全包含，需合并 |
| **笔** | 最小分析单元，顶分型+K线+底分型，≥5 根 K 线 |
| **线段** | ≥3 笔且有重叠，有方向（向上/向下），无横向 |
| **中枢** | 筹码密集区，多空博弈势均力敌，≥3 根线段重叠 |
| **ZG / ZD** | 中枢的上沿 / 下沿（高点里的最低 / 低点里的最高） |
| **笔中枢** | 用笔代替线段画的中枢（作者推荐） |
| **奔走中枢** | 重合部分特别少的中枢，原走势非常强势 |
| **中枢扩张** | 两个中枢重合变成一个中枢 |
| **走势类型** | 盘整型（1个中枢）/ 趋势型（≥2个同级别同方向中枢） |
| **a/A/b/B/c** | 走势组成部分命名：进入段/中枢/走出段/第二中枢/第三段 |
| **级别** | 时间周期（1分/5分/30分/日/周/月）+ 形态级别（笔<线段<走势） |
| **背驰** | 走势力量脱力，判断走势快完结 |
| **区间套** | 从大级别往小级别找走势，像放大镜判断背驰/买卖点 |
| **三买三卖** | 第一类、第二类、第三类买卖点 |
| **结合律** | a+b+c=(a+b)+c=a+(b+c)≠(a+c)+b，保证客观唯一 |
| **多义性** | 满足结合律下可随意组合，但买卖点不变 |
| **右侧交易** | 等信号出现再操作，不预测 |
| **千人千浪** | 其他理论因人而异的弊端，缠论通过严谨规则避免 |
