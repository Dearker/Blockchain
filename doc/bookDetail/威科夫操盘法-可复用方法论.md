## 1. 总体框架：威科夫交易体系

威科夫方法的本质是**透过价量表象，解读市场自身行为（供求关系），洞察CM意图，在趋势反转前的准备过程中低风险介入**。

| 维度 | 公众做法（失败） | 聪明钱做法（威科夫） |
|------|------------------|----------------------|
| 分析工具 | 技术指标、消息、传言、几何图形 | 仅用价格、成交量、走势速度 |
| 解读逻辑 | 指标信号直接交易 | 价量→供求关系→CM意图→方向 |
| 交易时机 | 突破时追涨、反弹时抄底 | 趋势起飞前的准备过程末端 |
| 风险控制 | 无预案，套牢死扛 | 危机管理，进场1~3根即甄别 |
| 信息层面 | 第一层面（表象） | 第二层面（本质/市场自身行为） |

**三大核心原则**（贯穿全书）：

| 原则 | 内容 | 作用 |
|------|------|------|
| **供求关系** | 趋势动力来自供求不平衡；供过于求跌，供不应求涨 | 判断方向、控盘方、趋势延续/反转 |
| **因果关系** | 趋势反转前必有准备过程（吸筹/派发震荡区）；"因"是准备过程，"果"是趋势 | 避免盲目抄底，在最低风险位介入 |
| **努力与结果** | 努力=成交量，结果=价格进展幅度；努力无结果是停止行为 | 提前预判走势停止/反转 |

**关键认知**：
- 交易是**负和游戏**，公众的亏损流入CM口袋
- 任何消息最终都通过交易员的手反映到价量里，看懂价量即看懂外部消息
- 威科夫方法**独立完整**，不可与技术指标组合使用（指标是表象的数学公式，无法撼动市场规律）
- 不是市场每一刻都值得分析，CM不在场时价格移动随意，应休息而非盲目交易

## 2. 三大基本要素与看盘顺序

### 2.1 三大基本要素

聪明钱解读市场只用**价格、成交量、走势速度**，不使用任何技术指标。

| 要素 | 可读出的信息 |
|------|--------------|
| 价格 | 支撑/阻力、趋势角度（涨跌速度）、谁在控盘、超买/超卖 |
| 成交量 | CM制造狂热/恐慌、支撑阻力质量、吸筹/派发起止、阻力上卖盘吸收情况 |
| 走势速度 | 趋势线角度、急涨/急跌背后的急切心态、超买/超卖 |

### 2.2 聪明钱的看盘顺序（五步法）

面对走势图，按以下顺序分析，而非等待指标信号：

| 步骤 | 内容 | 关键问题 |
|------|------|----------|
| ① 背景 | 当前大趋势、所处阶段 | 牛市/熊市/震荡？供过于求/供不应求？ |
| ② 价量形态 | 蜡烛长度、成交量高度、变化速度 | 形态本身是什么？ |
| ③ 性质 | 走势强弱、原因与后果 | 这是什么性质的行为？意味着什么？ |
| ④ 结论/预测 | 结合供求关系洞察CM意图 | 方向判断 |
| ⑤ 措施/行动 | 进场/出场/收紧止损 | 是否行动？如何行动？ |

**示例**：牛市中回调，蜡烛与成交量都递减→性质：回落力度弱→结论：供应未进场，价格将再涨→行动：看到停止行为或需求接盘即买入。

### 2.3 CM观察走势遵循的三大原则

- 趋势动力来自**供求关系不平衡**
- 趋势形成前需要**准备过程**（因果关系）
- 成交量增长但价格未大幅增长=**走势停止行为**（努力与结果）

## 3. CM行为逻辑方法论

### 3.1 CM的本质

CM（Composite Man）= 掌握庞大资金、能短期控制价格走势的投资人或机构。市场上每一波走势都是CM精心策划执行的。公众无需知道CM是谁，只需**解读其行为**。

### 3.2 CM的行为习惯（核心规律）

| 行为 | CM操作 | 公众反应 | 结果 |
|------|--------|----------|------|
| 吸筹 | 低价位让走势枯燥无味，制造恐慌 | 绝望抛售 | 筹码转入CM手中 |
| 拉升初期 | 严控市场，走势缓慢清淡 | 无法进场/被震出 | 公众踏空 |
| 出货前 | 制造急涨、利好消息、媒体唱多 | 贪婪抢购 | 风险转移给公众 |
| 出货时 | 顶部悄悄派发，维持支撑 | 以为回调是便宜货 | 满仓接盘 |
| 出货后 | 撤掉支撑，做空加速下跌 | 套牢无救 | 深度亏损 |

### 3.3 CM迫使公众抛售的三大策略

| 策略 | 手段 | 适用阶段 |
|------|------|----------|
| **恐慌抛售** | 制造悲观情绪+长阴+坏消息，公众心理崩溃抛掉筹码 | 熊市末期 |
| **疲劳战术** | 震荡区内每次回升立即打压，漫长枯燥折磨公众心理 | 吸筹震荡区 |
| **震仓** | 价格急速穿透所有支撑，摆出大跌姿态 | 吸筹末期/牛市回调 |

### 3.4 CM与公众的关键区别

| 对比项 | 公众 | 专业人士（聪明钱） |
|--------|------|---------------------|
| 分析依据 | 技术指标、消息、传言、基本面 | 仅价格、成交量、变化速度 |
| 解读方式 | 指标信号直接交易 | 价量→市场自身行为→供求关系 |
| 决策配合 | 无 | 结合大背景做结论/推断 |
| 风控 | 无危机管理，套牢 | 使用危机管理，风险最低 |

## 4. 供求关系方法论

### 4.1 供求关系基本原理

| 供求状态 | 价格行为 | 市场阶段 |
|----------|----------|----------|
| 需求>供应 | 上涨趋势 | 牛市 |
| 供应>需求 | 下降趋势 | 熊市 |
| 供需平衡 | 区间震荡 | 危险区（易亏钱） |

**核心能力**：在不使用技术指标的情况下，通过价格和成交量的细节变化识别当下供求关系，判断谁在控盘（买方/卖方）。

### 4.2 供求关系与上涨趋势

| 阶段 | 供求变化 | 价量表现 |
|------|----------|----------|
| 牛市延续 | 需求>供应，买方抬价抢筹 | 价格持续抬高 |
| 牛市预警 | 高成交量但价格上涨无进展 | 供应开始增加（CM抛盘） |
| 牛市终止 | 供需平衡→供过于求 | 上涨停止，派发开始 |

**回调的本质**：需求逐步减弱，市场降价刺激新需求。关键看回调是否引起**供应扩大**（CM抛盘）。

### 4.3 供求关系与下降趋势

| 阶段 | 供求变化 | 价量表现 |
|------|----------|----------|
| 熊市延续 | 供应>需求，卖方降价找买家 | 价格持续下跌 |
| 熊市预警 | 天量+超长阴线=恐慌抛售 | CM接盘，需求扩大 |
| 熊市终止 | 需求吸收供应，供应枯竭 | 二次测试缩量确认 |

### 4.4 抄底的理论基础与价量确认

| 理论基础 | 价量确认 |
|----------|----------|
| 供应枯竭 | 天量抛售后低量测底 |
| 需求增加并主导市场 | 震仓驱离浮动供应，或长阳带量离开吸筹区间 |

**铁律**：没有看到上述理论根据成立，绝不盲目抄底（看指标背离抄底=抄在半山腰）。

## 5. 因果关系方法论

### 5.1 核心定义

- **因**：趋势出现前的准备过程（吸筹/派发的震荡区交易）
- **果**：趋势形成

**目的**：判断趋势反转，在趋势最初期介入，降低风险。没有准备过程的趋势不会持续太久。

### 5.2 反转的理论基础

| 反转方向 | 理论基础（必须在价量上体现） |
|----------|------------------------------|
| 熊→牛 | ①供应枯竭（卖单流入稀少）②需求扩大并保持力度 |
| 牛→熊 | ①需求减弱②供应大量进入市场③需求无力继续主导 |

### 5.3 因果关系的应用原则

| 原则 | 说明 |
|------|------|
| 不要急于抄底 | 看到大跌不等于能抄底，必须等待吸筹过程 |
| 准备过程=震荡区 | 新趋势反转前必有震荡区交易（吸筹/派发） |
| 短暂准备过程 | 短线反转的准备过程较短，可从更低时间框架寻找 |
| 左手边vs右手边 | 左手边（熊市第一次反弹）不抄底；右手边（二次测试后）开始找进场点 |

### 5.4 公众抄底失败的原因

1. 怕错过趋势大底
2. 缺乏市场行为知识
3. 缺乏耐心

## 6. 努力与结果关系方法论

### 6.1 核心定义

- **努力** = 成交量（市场流入订单总量）
- **结果** = 价格的上涨/下跌幅度

**作用**：判断走势是否出现停止行为（SOT，Stopping of Thrust）。

### 6.2 努力与结果的四种组合

| 努力（成交量） | 结果（价格幅度） | 含义 | 行动 |
|----------------|------------------|------|------|
| 大 | 大 | 正常推进 | 顺势 |
| 大 | 小（滞涨/滞跌） | 停止行为！供应/需求介入 | 警惕反转 |
| 小 | 大 | 真空突破（无阻力） | 顺势 |
| 小 | 小 | 僵持/枯燥 | 等待 |

### 6.3 两种典型停止行为

| 类型 | 表现 | 含义 |
|------|------|------|
| **放量滞涨**（牛市） | 成交量扩大但价格涨幅缩小 | CM抛盘，供应增加，危险信号 |
| **放量滞跌**（熊市/支撑） | 成交量扩大但价格跌幅缩小 | 需求接盘，停止行为，可能反弹 |

### 6.4 波段交易量分析

除单根蜡烛成交量外，可用**一波的累积成交量**判断努力程度。若一波上涨总成交量远大于前几波，但价格未创新高→CM出货，警告。

## 7. 支撑与阻力的本质解读

### 7.1 关键认知：支撑/阻力与图上的线无关

均线、黄金分割线、指标线都不能单独代表支撑/阻力。市场对这些线一无所知。

| 概念 | 本质定义 |
|------|----------|
| **支撑** | 某价位购买力>抛售压力（需求吸收全部供应） |
| **阻力** | 某价位抛售压力>购买力（供应超过需求） |

### 7.2 支撑有效性的判断方法

| 测试信号 | 含义 |
|----------|------|
| 测试蜡烛小+低成交量 | 供应枯竭，支撑有效 |
| 触底后强烈反弹 | 需求强劲，支撑有效 |
| 触底后无力反弹（小蜡烛小量） | 无需求，支撑失效 |
| 放量但振幅缩小（停止行为） | 需求吸收卖盘，可轻仓进场 |

### 7.3 阻力位突破的判断

| 情况 | 含义 |
|------|------|
| 突破时低量小蜡烛回调 | 供应缺乏，需求控制，可进场 |
| 突破后放量阴线跟随 | 供应扩大，突破失败 |
| 阻力位收盘价相近（递减） | 需求减弱，阻力有效 |
| 阻力位放量阴线+跟随 | 供应扩大，无法突破 |

### 7.4 关键概念

| 概念 | 定义 |
|------|------|
| **垂直需求柱** | 放量长阳，需求控制市场，底部形成支撑，价格常回测 |
| **垂直供应柱（VSB）** | 放量长阴，供应控制市场，顶部形成阻力，价格常回测 |
| **冰线** | 上升趋势最后一道防线（新高前反弹低点）；破冰后下跌幅度大 |
| **供应线** | 下降通道上轨；突破=需求吸收供应，可能反转 |
| **死角** | 支撑/阻力位蜡烛高低点不断收窄，供求都弱；预示大动作，提前进场好时机 |

## 8. 关键市场行为信号体系

### 8.1 底部/吸筹相关信号

| 信号 | 英文/缩写 | 特征 | 含义 |
|------|-----------|------|------|
| 初次支撑 | PS | 扩大成交量使下跌停止 | CM需求进场，熊市终止第一步 |
| 恐慌抛售 | SC (Selling Climax) | 超长阴线+超高成交量 | 公众恐慌抛售，CM接盘，牛市前提 |
| 自动反弹 | AR | 恐慌抛售后自然反弹 | 空头回补，不持续，不抄底 |
| 二次测试 | ST | 缩量小幅测试 | 成功=熊市终止；失败=继续等 |
| 强势出现 | SOS (Sign of Strength) | 长阳+高成交量（震荡区内） | 需求控制市场，震荡区终止 |
| 跳离震荡区 | JOC (Jump over Creek) | 放量长阳突破震荡区顶部 | 需求越过供应区，牛市开始 |
| 终极震仓 | | 急速深跌突破所有支撑后快速反弹 | CM清除最后浮动供应，吸筹结束 |
| 弹簧效应 | Spring | 小幅突破支撑后迅速返回 | 测试支撑附近供应量 |

### 8.2 顶部/派发相关信号

| 信号 | 英文/缩写 | 特征 | 含义 |
|------|-----------|------|------|
| 初次供应 | | 放量下跌或急涨后放量跌 | CM首次出货，牛市预警 |
| 抢购高潮 | BC (Buying Climax) | 天量+超长阳线+陡峭角度 | 公众疯狂接盘，CM出货 |
| 自然回落 | | 抢购高潮后自然下跌 | 形成派发支撑（冰线） |
| 上冲回落 | UT (Upthrust) | 突破阻力后立刻跌回（1~3根） | CM测试需求，扫空头止损 |
| 弱势出现 | SOW (Sign of Weakness) | 放量长阴（震荡区右手边） | 供应控制市场，熊市信号 |
| 破冰 | | 放量长阴突破冰线（支撑） | CM撤掉支撑，派发完成 |
| 最后派发点 | LPSY | SOW后无量回升 | 最佳做空点/最后出逃点 |

### 8.3 通用停止行为

| 信号 | 英文/缩写 | 特征 | 含义 |
|------|-----------|------|------|
| 突破幅度缩小 | SOT | 波峰/波谷突破幅度递减 | 停止行为，趋势可能停止 |
| 50%原则 | | 回调超过前一波50% | 趋势减弱；牛市中是警告 |
| 超买 | | 价格突破上升通道上轨 | 至少有回落，看是否有供应 |
| 超卖 | | 价格突破下降通道下轨 | 至少有反弹，看是否有需求 |

### 8.4 信号确认原则

任何信号都需**后续行为确认**，不可凭单一信号行动：

| 信号 | 需要的确认 |
|------|------------|
| 恐慌抛售 | 二次测试（缩量） |
| SOS | 缩量回调测试 |
| JOC | 低量回测突破点 |
| SOW | 无量回升确认 |
| UT | 放量阴线跟随 |
| 震仓 | 快速带量反弹 |

## 9. 吸筹过程方法论（熊转牛）

### 9.1 吸筹四阶段总览

| 阶段 | 市场行为 | 供求状态 | 操作 |
|------|----------|----------|------|
| **第一阶段** | 初次支撑→恐慌抛售→自动反弹→二次测试 | 供应>需求→需求开始吸收 | 不抄底，等二次测试 |
| **第二阶段** | 震荡区交易，CM疲劳战术吸筹 | 供需平衡，需求渐强 | 右手边找进场点 |
| **第三阶段** | 终极震仓/Spring测试供应 | 供应枯竭确认 | 震仓后/二次测试进场 |
| **第四阶段** | SOS/JOC+测试，进入牛市 | 需求>供应 | SOS/JOC回测进场 |

### 9.2 第一阶段：熊市终止的四个行为

| 行为 | 价量特征 | 关键点 |
|------|----------|--------|
| 初次支撑 | 扩大成交量使下跌停止 | CM试探进场，暂时的 |
| 恐慌抛售 | 超长阴线+超高成交量 | 公众抛售，CM接盘，牛市前提 |
| 自动反弹 | 恐慌抛售后自然反弹 | 空头回补，**不抄底** |
| 二次测试 | 缩量小幅测试 | 成功（缩量）=熊市终止 |

**二次测试的判断**：
- 成交量和波动都远小于恐慌抛售→成功，熊市终止
- 成交量仍大→供应还在，需等待进一步测试
- 成交量介于两者之间→也属二次测试，需等供应耗尽

### 9.3 第二阶段：震荡区吸筹

| 要点 | 说明 |
|------|------|
| 为何有震荡区 | CM需吸收足够筹码才能允许价格上涨 |
| 早期特征 | 价格波动和成交量较大 |
| 后期特征 | 波动和成交量逐渐减小（CM收购后卖单减少） |
| 正常吸筹特征 | 上涨波的力度和成交量>下降波 |
| CM疲劳战术 | 顶部布置卖单控价；每次回升立即打压；漫长枯燥折磨公众 |

### 9.4 第三阶段：测试供应（确认吸筹）

**终极震仓**（吸筹末期CM的最后手段）：

| 维度 | 内容 |
|------|------|
| 位置 | 吸筹阶段末期/交易区间末期 |
| 形态 | 急速大幅深跌，冲破所有支撑，随后快速反弹 |
| 目的 | ①吸收剩余股票②迫使弱者割肉③制造下跌假象 |
| 卖方 | 恐慌的公众 |
| 买方 | CM（验证浮动供应是否耗尽） |
| 震仓前预警 | 震荡区内反弹高点台阶式下降（浮动供应还在） |
| 确认 | 震仓后连续阳线+递增成交量 |

### 9.5 第四阶段：进入牛市

| 信号 | 特征 | 进场点 |
|------|------|--------|
| SOS+测试 | 震荡区内长阳+高成交量 | 缩量回调测试时 |
| JOC+测试 | 放量长阳突破震荡区顶部 | 低量回测突破点时 |

**JOC（跳离震荡区）的"跳跃小溪"比喻**：
- 小溪=充满卖单的供应区
- 男孩跳跃=需求越过供应区
- 找水流缓窄处=找供应薄弱点
- 助跑=短暂回调
- 跳过后休息=突破后回测

### 9.6 吸筹阶段的三个进场位置

| 位置 | 条件 | 风险 |
|------|------|------|
| SOS/JOC后的回测 | 回测必须小蜡烛+低成交量 | 最低 |
| 震荡区底部Spring | 弹簧效应小幅突破支撑，可直接进场；大幅突破需二次测试 | 低 |
| 终极震仓中/后 | 震仓后二次测试进场风险更小 | 中 |

### 9.7 没有恐慌抛售的熊市终止

中国A股特点：底部形成前很少出现恐慌抛售。CM用两种替代办法：

| 方法 | 说明 |
|------|------|
| 拖长震荡区时间 | 持续数年枯燥行情折磨公众割肉 |
| 终极震仓 | 多次大幅震仓制造恐慌 |

### 9.8 普通震仓（与终极震仓区别）

| 维度 | 普通震仓 | 终极震仓 |
|------|----------|----------|
| 位置 | 上升趋势中（刚启动个股） | 吸筹末期 |
| 原因 | 突发消息致公众临时恐慌 | CM主动策划清除浮动供应 |
| 后续 | 二次测试缩量=牛市恢复 | 连续阳线递增=牛市开始 |

## 10. 派发过程方法论（牛转熊）

### 10.1 派发四阶段总览

| 阶段 | 市场行为 | 供求状态 | 操作 |
|------|----------|----------|------|
| **第一阶段** | 初次供应→抢购高潮→自然回落→二次测试 | 供应开始进入 | 启动出逃计划 |
| **第二阶段** | 震荡区派发，CM建支撑出货 | 供需平衡，供应渐强 | 等待右手边 |
| **第三阶段** | UT/SOW测试需求是否耗尽 | 供应>需求 | UT/SOW做空 |
| **第四阶段** | 破冰+SOW确认，进入熊市 | 供应完全控制 | LPSY做空/离场 |

### 10.2 牛市停止的四个行为

| 行为 | 价量特征 | 警告级别 |
|------|----------|----------|
| 初次供应 | 放量下跌或急涨后放量跌 | 预警（不忽视） |
| 抢购高潮 | 天量+超长阳线+陡峭角度 | 明显触顶信号 |
| 自然回落 | 抢购高潮后自然下跌 | 形成派发支撑（冰线） |
| 二次测试 | 成功（低量）=牛市终止 | 确认牛市结束 |

### 10.3 牛市到顶的预警信号

| 信号 | 含义 |
|------|------|
| 上升突破幅度递减（SOT） | 需求减弱或供应压力增大 |
| 50%原则：回调超过前一波50% | 供应扩大，趋势减弱 |
| 趋势线角度突然变陡（垂直上涨） | CM急切拉升引诱公众 |
| 牛市中天量上涨（抢购高潮） | CM出货，供应涌入 |
| 上涨中成交量猛涨但价格无进展 | CM隐蔽派发 |

### 10.4 派发第二阶段：CM出货手法

| 手法 | 说明 |
|------|------|
| 建立派发支撑（冰线） | 在高位设定支撑，出货完成前用买单保护 |
| 引诱公众追高 | 媒体唱多+上市公司利好+价格急涨 |
| 震仓驱离公众 | 突破后制造枯燥行情，让跟风者失望离场 |
| UT扫清空头 | 向上突破扫止损，然后跌回（CM本意做空） |

### 10.5 派发第三阶段：测试确认

| 测试行为 | 特征 | 确认 |
|----------|------|------|
| 上冲回落（UT） | 突破阻力后跌回，成交量增加 | 放量下挫跟随=供应强劲 |
| 弱势出现（SOW） | 震荡区右手边放量长阴 | 无量回升确认=SOW有效 |

### 10.6 派发第四阶段：破冰入熊

| 行为 | 特征 |
|------|------|
| 破冰 | 放量长阴突破冰线（派发支撑） |
| LPSY | SOW后无量回升，最佳做空点/最后出逃点 |
| 跟随 | 破冰后任何回测冰线遇供应=做空点 |

### 10.7 特殊派发模式

| 模式 | 特征 |
|------|------|
| 无抢购高潮的派发 | 蜡烛成交量逐步递减（需求耗尽），然后放量下挫 |
| 低位派发 | SOW后形成的派发区间，非吸筹（回升无需求） |
| 头肩顶 | 左肩=初次供应，头=抢购高潮，右肩=二次测试（需供求确认） |
| UT派发模式 | 震荡区右手边缩量上冲回落+SOW确认 |

## 11. Spring弹簧效应交易法

### 11.1 Spring定义

价格迅速突破支撑后迅速返回支撑上方，本质是**下跌走势的停止行为**。CM用它评估突破支撑时的供应量，以及供应是否被全部吸收。

### 11.2 Spring的三种类型

| 类型 | 突破时价量 | 含义 | 操作 |
|------|------------|------|------|
| **第1种（坏）** | 大蜡烛+大成交量 | 支撑附近大量卖单，需求无法吸收 | 不进场，等二次测试（震仓） |
| **第2种（最佳）** | 小蜡烛+小成交量 | 突破时缺乏供应，需求将积极进入 | **直接进场** |
| **第3种（待确认）** | 略扩大的蜡烛+成交量 | 有供应但不足以压垮需求 | 等二次测试，缩量=进场 |

**第3种衍生**：突破时成交量扩大但蜡烛小且收在50%之上（下影线）→高成交量代表强劲需求，可直接进场。

### 11.3 Spring的位置与意义

| 位置 | 意义 |
|------|------|
| 牛市回调震荡区底部 | 震荡区结束，牛市恢复（成功率最高） |
| JOC后回测突破点 | 进场信号 |
| 回测垂直需求柱底部 | 进场信号 |
| 吸筹末期 | CM制造恐惧收购，后跟SOS=吸筹结束 |
| **熊市中** | **陷阱！不能进场** |

### 11.4 Spring使用要点

- Spring不是每天都有，没出现不要刻意制造，**等待也是策略**
- 成功率高的Spring发生在**上升趋势**中
- 卖盘控制背景下Spring经常失败，是**做空信号**而非抄底信号
- 突破支撑时成交量相对扩大，**必须等二次测试**（缩量）
- 进场后必须看到持续新高阳线+递增成交量，否则立刻出场
- 牛市中连续出现失败的Spring=预警信号

### 11.5 Spring成功与失败的判断

| 结果 | 表现 | 含义 |
|------|------|------|
| 成功 | 持续新高阳线+递增成交量 | 需求占上风，持有 |
| 失败 | 反弹无力，无跟随 | 需求匮乏，立刻离场 |
| 失败（熊市背景） | 突破后继续下跌 | Spring是陷阱 |

### 11.6 震仓是特殊的Spring

震仓与Spring目的相同（迫使公众恐慌，CM吸收卖盘），区别在突破幅度：震仓大幅突破所有支撑，Spring小幅突破。

## 12. 交易计划与进场点方法论

### 12.1 市场本质交易法

每次分析走势，**先考虑理论基础，再在图中找确认理论基础的价量**：

| 步骤 | 内容 |
|------|------|
| ① 理论基础 | 如：突破阻力的理论基础=阻力上任何卖盘都被需求吸收且需求占上风 |
| ② 价量确认 | 在图中找吸收行为的价量证据 |
| ③ 判断 | 理论基础成立→判断阻力会被突破 |
| ④ 行动 | 制订交易计划并执行 |

### 12.2 进场点的核心原则

| 原则 | 说明 |
|------|------|
| 右手边进场 | 必须在震荡区右手边进场（测试过程确认吸筹/派发后） |
| 低量回测进场 | SOS/JOC后低量小蜡烛回测=最佳进场点 |
| 停止行为进场 | 支撑/阻力位出现停止行为（SOT/Spring）=进场点 |
| 死角交易法 | 死角中供求都弱，需求稍作努力即突破=低风险进场 |
| 不与公众为伍 | 突破是公众行为，不在突破时追，等回测 |

### 12.3 进场后的确认要求

进场后必须看到（做多）：

| 确认行为 | 含义 |
|----------|------|
| 连续阳线 | 更高的高点、低点、收盘价 |
| 递增成交量 | 需求持续 |
| 缩量回调 | 回调时供应未扩大 |

若进场后看到**放量阴线**=遇到大量供应，立刻出场。

### 12.4 典型交易计划模板

| 要素 | 内容 |
|------|------|
| 市场背景 | 牛市/熊市/震荡？供求谁控盘？ |
| 理论基础 | 进场的理论根据是什么？ |
| 价量确认 | 图上哪些价量行为确认了理论基础？ |
| 进场点 | 具体在哪个价位/什么信号进场？ |
| 止损位 | 设在需求来源下方（如JOC蜡烛底部下方1点） |
| 第一目标 | 供应线/阻力位/前高 |
| 危机管理 | 出场条件是什么？ |

## 13. 危机管理方法论

### 13.1 危机管理的本质

危机管理=对进场时机的甄别，发生在进场后**1~3根或3~5根蜡烛内**。一旦发现进场过早或判断失误，**没有任何借口地立刻出场**。

### 13.2 进场后的甄别清单

| 情况（做多） | 行动 |
|--------------|------|
| 连续阳线+递增成交量 | 持有/加仓 |
| 缩量小蜡烛回调 | 持有（供应未扩大） |
| 放量阴线 | **立刻出场** |
| 小蜡烛小量反弹（无需求反弹） | **立刻出场** |
| 突破支撑后无需求反弹 | **立刻出场** |
| 出现SOW（放量长阴） | **立刻出场** |

### 13.3 各阶段的危机管理

| 阶段 | 危机信号 | 行动 |
|------|----------|------|
| 牛市持仓 | 抢购高潮（天量上涨） | 预警，准备出逃 |
| 牛市持仓 | SOT+50%突破+放量跌 | 离场 |
| 支撑买入 | 支撑上无需求反弹 | 离场 |
| Spring买入 | 反弹无跟随 | 离场 |
| JOC回测买入 | 放量阴线回测 | 放弃进场 |
| 做空 | 长阳突破冰线 | 离场 |

### 13.4 危机管理的纪律

- 不用情绪和希望否定市场本质行为
- 看到与进场依据相反的行为，立刻出场
- 任何出场后可重新评估，但不可死扛
- 止损放在**需求来源下方**（如JOC蜡烛底部、垂直需求柱底部下方）

## 14. 选股方法论：相对强弱分析

### 14.1 相对强弱选股原理

利用**大盘与个股的相对强弱**选股，在消息公布前跟随CM行动。

| 步骤 | 操作 |
|------|------|
| ① 对比大盘与个股 | 大盘走弱时，个股是否坚守支撑？ |
| ② 回调幅度对比 | 大盘回调接近50%，个股回调仅30%=个股强 |
| ③ 大盘创新低时 | 个股是否坚守高位支撑？（有大资金收购） |
| ④ 大盘恢复上涨时 | 个股涨势是否超过大盘？ |

### 14.2 选股前的知识储备

- 停止行为的识别
- 价格强弱表现
- 市场供求关系表现
- 支撑的意义

### 14.3 经典案例

巴菲特2008年斥资50亿美元买入高盛（初次支撑），市场嘲笑他看走眼，一年后高盛翻倍。CM凭借资金优势提前获知消息，在图上留下行为痕迹，公众只需读懂这些痕迹即可跟随。

## 15. 时间框架分析

### 15.1 多时间框架原则

日线出现反转前，小时线会**提前**出现上涨行为。看不同时间框架是为了发现日线看不到的吸筹阶段。

| 时间框架 | 作用 |
|----------|------|
| 周线/月线 | 判断大背景、大趋势 |
| 日线 | 主要交易决策 |
| 小时线 | 发现日线不明显的吸筹过程（如V型反转的吸筹在小周期可见） |

### 15.2 短暂准备过程的处理

日线反转过程短暂时（不符合因果关系的明显震荡区），从**更低时间框架**寻找准备过程。如日线看不到吸筹，3小时图可能看到明显震荡区。

## 16. 交易纪律与成长路径

### 16.1 核心纪律

| 纪律 | 说明 |
|------|------|
| 不与技术指标组合 | 威科夫方法独立完整，组合=走回头路 |
| 不盲目抄底 | 必须等吸筹过程，看到理论根据成立 |
| 不在熊市找Spring | 熊市Spring是陷阱 |
| 不与公众为伍 | 突破追涨是公众行为 |
| CM不在场时休息 | 价格随意移动时不要交易 |
| 危机管理无借口 | 进场后判断失误立刻出场 |

### 16.2 学习路径

| 阶段 | 内容 |
|------|------|
| ① 傻瓜式学习 | 抛掉小道消息、高手解读、媒体观点、政治形势 |
| ② 掌握供求关系 | 从价量细节识别供求，成为习惯 |
| ③ 掌握1~2个停止行为 | 如SOT、Spring |
| ④ 掌握吸筹/派发过程 | 识别CM布局 |
| ⑤ 掌握危机管理 | 进场后1~3根甄别 |
| ⑥ 坚持交易笔记 | 每天记录进场、出场、结果、感悟 |

### 16.3 熟练运用的标志

当有人介绍新发明的指标用法时，你已没兴趣花时间研究→说明注意力已定位于市场本质，用供求关系解读市场已成为习惯。

### 16.4 交易笔记

必不可少的工具（电脑/博客），每天记录进场、出场、交易结果、感悟，坚持下去会发现自己让自己进步。**学的不是方法，学的是盈利。**

## 17. 核心检查清单

### 17.1 看盘检查清单

- [ ] 当前大背景是什么？（牛市/熊市/震荡）
- [ ] 供求谁控盘？
- [ ] 处于吸筹/派发/趋势的哪个阶段？
- [ ] 左手边还是右手边？
- [ ] 有无停止行为（SOT/Spring）？
- [ ] 进场理论根据是否成立？
- [ ] 价量是否确认了理论根据？

### 17.2 进场前检查清单

- [ ] 是否在右手边？
- [ ] 是否有低量回测/停止行为？
- [ ] 止损位设在哪里（需求来源下方）？
- [ ] 第一目标价位？
- [ ] 危机管理方案？（出场条件）

### 17.3 进场后检查清单（1~3根蜡烛内）

- [ ] 是否有连续阳线+递增成交量？（做多）
- [ ] 回调是否缩量？
- [ ] 有无放量阴线？（有=离场）
- [ ] 反弹是否有力？（无力=离场）

### 17.4 顶部预警检查清单

- [ ] 是否出现SOT（突破幅度递减）？
- [ ] 回调是否超过50%？
- [ ] 是否出现抢购高潮（天量+超长阳+陡峭）？
- [ ] 趋势线角度是否突然变陡？
- [ ] 上涨是否放量但无进展（CM派发）？

### 17.5 底部确认检查清单

> **总结**：威科夫方法的核心是**放弃一切技术指标**，仅用价格、成交量、走势速度，通过供求关系解读市场自身行为，洞察CM意图。在吸筹/派发的准备过程中，于右手边的低风险位置（SOS/JOC回测、Spring、死角、停止行为）介入，并以危机管理保障每笔交易。熟练掌握1~2个方法并反复实战，不与任何指标组合，方能立足于市场少数赚钱者之列。
