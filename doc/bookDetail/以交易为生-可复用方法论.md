## 1. 总体框架：四大支柱

埃尔德将交易成功归结为四个相互支撑的支柱，缺一不可：

| 支柱 | 作用 | 失败后果 |
|------|------|----------|
| **心理** | 控制情绪，保持客观 | 情绪化交易，自我毁灭 |
| **市场分析** | 识别趋势与反转时机 | 盲目交易，逆势操作 |
| **风险管理** | 限制损失，保护本金 | 单次或连续亏损毁掉账户 |
| **交易记录** | 强化纪律，从经验中学习 | 重复犯错，无法成长 |

**关键认知**：
- 交易是**负和游戏**（佣金+滑点持续抽走资金），仅"好于平均"不够，必须出类拔萃
- 交易对智力的要求适度，但对**情绪管理**的要求极高
- 好的交易者目标不是赚钱，而是**做好每一笔交易**；交易做好了，钱自然来

## 2. 心理管理方法论

### 2.1 交易者的"酗酒"类比

埃尔德（精神病医师出身）提出：**亏损者与酗酒者在心理模式上高度相似**。

| 酗酒者 | 亏损交易者 |
|--------|------------|
| 否认酒精控制了自己 | 否认亏损源于自己的失控 |
| 换啤酒代替烈酒 | 换市场/系统/大师寻找"秘诀" |
| 触及谷底才承认问题 | 账户爆仓才面对现实 |
| 第1步：承认无力驾驭酒精 | 第1步：承认无力驾驭亏损 |
| "一次一天"保持清醒 | "一次一天"保持纪律 |

**核心方法**：将匿名戒酒会（AA）的12步法理念迁移到交易——
- **承认**：承认自己无法控制亏损冲动
- **停损**：像戒酒者永远不再碰酒一样，永远不让小亏变大亏
- **一次一天**：每天只专注当天的纪律执行，不幻想一夜暴富

### 2.2 情绪管理准则

1. **不在持仓时计算浮盈/浮亏**：计算得失会干扰决策，如同律师在庭审中盘算酬金、外科医生在手术中想着购物
2. **只关注收益/亏损的百分点，不转化为金额**：把精力集中在管理交易上，钱随后自然会来
3. **不因单次交易成败而生气**：市场有大量随机成分，每步都对仍可能失败，应追求**正期望值的交易方法**
4. **成功的交易应该"有些无聊"**：耗时枯燥的功课才是成功的基础，刺激感以损失为代价
5. **连胜后警惕自大**：业余者小赚几次就自认天才，开始盲目承担风险，这正是毁灭的开始

### 2.3 自我毁灭的识别与控制

有自毁倾向的交易者特征：
- 账户反复大起大落而非稳步增长
- 赚钱时过度兴奋，亏钱时陷入绝望
- 违反自己制定的规则
- 追求刺激而非稳健
- 将失败归咎于坏运气、经纪人或其他外部因素

**自我毁灭的典型模式**：
- 赌徒模式：交易上瘾，无法停止一个月不交易；测试运气而非执行计划；胜利时获得权力感如同小孩饱餐一顿
- 撞车模式：在当日最高点买入如同高速路上打开车门；市场没有安全网，提供无限的自毁机会
- 与市场对抗：如同与父亲抗争般对抗市场，拒绝承认市场远比自己强大

**控制方法**：
1. **测试是否上瘾**：试着停止交易一个月。如果渴望强烈到无法做到，应拜访匿名赌徒会或运用AA原则
2. **坚持写交易日志**：记录每次建仓/清仓理由，寻找成功和失败的重复模式
3. **建立心理安全网**：严格的资金管理法则 + AA原则 + 交易日志，如同登山运动员的求生设备
4. **为自己负责**：不把损失归咎于坏运气或他人，为最终结果负责

### 2.4 交易者的三个目标排序

**埃尔德强调的目标优先级（大多数人本末倒置）**：

| 优先级 | 目标 | 说明 |
|--------|------|------|
| 第一目标 | **长期生存** | 保护本金，留在游戏中 |
| 第二目标 | **资本稳定增长** | 稳步积累而非大起大落 |
| 第三目标 | **赚取高额利润** | 大多数人将其错置于第一位 |

### 2.5 成功交易者的七条准则

埃尔德从业余者进化为专业交易者的经验总结：

1. **坚定长期意念**：从现在开始至少交易20年
2. **海绵式学习 + 有益怀疑**：关注专家观点但刨根究底，不简单接受
3. **不贪婪，不急于交易**：市场一直在这里，未来有无尽机会
4. **培养分析市场的方法**："如果A发生，那么B很可能发生"，用多种方法确认
5. **建立资金管理计划**：生存第一，增长第二，高利润第三
6. **认识到交易者是系统最薄弱的环节**：去AA学习或建立自己的情绪控制方法
7. **改变思考、感受与行动方式**：胜利者与失败者在三方面完全不同，必须深探内心

## 3. 群体心理方法论

> 技术分析是以盈利为目的的社会心理学。每个价格都是市场参与者价值共识的货币表现，图表是观测群体心理的窗口。

### 3.1 市场的本质——群体行为

**核心认知**：
- 市场是一大群人的集合，每个人都试图从他人身上赚取利润
- 每个价格都是所有市场参与者在某时点对价值达成的**暂时性共识**
- 你的利润来自其他交易者——市场获利只有一种原因：由其他交易者向你提供
- 内幕交易者也在图表中留下足迹，技术分析可以发现内幕交易行为

**四类市场参与者**（华尔街动物比喻）：
- **牛**（多头）：判断价格上涨而买入
- **熊**（空头）：判断价格下跌而卖出
- **猪**（贪婪者）：持有超承受能力的头寸或持有时间过久 → 被屠杀
- **羊**（从众者）：被动胆小，跟从趋势、小道消息和大师 → 被屠杀

**关键启示**："牛赚钱，熊赚钱，唯有猪羊被屠杀"——如何管理交易比选择什么股票和方向更重要。

### 3.2 群体对个体的吞噬机制

**人类加入群体的本能**：
- 远古时代群体生存率高于独行 → 基因植入加入群体的倾向
- 未来不确定性越大，越渴望加入群体、追随领袖
- 交易中价格涨跌超出控制 → 恐惧膨胀 → 迫切寻找领袖

**加入群体后的变化**（勒庞《乌合之众》）：
- 独立思考能力显著降低
- 变得更容易轻信、更冲动
- 急切寻找领导者
- 行为情绪化而非理智化

**群体行为的简单性**：
- 群体心智简单原始 → 交易策略也应该简单
- 价格走势与预测相反 → 止损退出，永远不要与群体叫板
- 利用判断力决定何时加入群体、何时离开观望

### 3.3 价格——市场群体的领导者

**核心洞察**：价格是市场群体的主要领导者（非专家、非大师）。

- 全世界交易者都在疯狂追随价格的涨跌
- 价格似乎在说："跟随我吧，我将带你走向财富之路"
- 大部分交易者自认独立，却未意识到自己多关注价格走势
- 有利趋势如慷慨父亲分享快乐；不利趋势如愤怒父亲随时惩罚

**趋势反转的心理学**：
- **上涨反转**：多头被奖励→加仓→新多头涌入→空头受罚→平仓加入多方→价格波动出现（大卖单无足够多头消化）→多头受虐→空头受鼓舞→顶背离出现→反转
- **下跌反转**：空头被奖励→加仓→新空头涌入→多头受罚→平仓加入空方→价格波动出现（大买单消化所有卖单）→空头受虐→多头受鼓舞→底背离出现→反转

**恐惧比贪婪强3倍**（卡尼曼研究）：
- 空头平仓造成的上涨速度更迅疾（恐惧驱动），但持续时间短
- 这解释了为什么下跌速度是上涨速度的两倍

### 3.4 保持独立的实操方法

**塞壬比喻——把自己绑在桅杆上**：
- 奥德赛为听塞壬歌声又不被诱惑，把自己绑在桅杆上
- 交易者应将自己绑在**交易计划和资金管理规则**的桅杆上
- 持仓时坚决不修改计划，让规则替你抵御群体情绪

**具体方法**：
1. **交易前**写下计划（纸面化使其更确切）
2. **持仓时**不修改计划，只执行预先设定的入场/出场条件
3. **监控自己的精神状态**：若发现失去独立性，暂时远离市场观望
4. **独立制定决策**：参考群体智慧（如交易竞赛）但必须独立选股，不可窥探他人操作

**积极的群体**（群体智慧）：
- 并非所有群体都有害——聪明群体的成员各自独立决策
- 领袖作用是维护团队结构和主持投票，而非指挥操作
- 关键：所有选股和方向必须独立做出，交流在结果收齐后进行

### 3.5 个人交易者 vs 机构交易者

**机构优势**：低佣金、顶级研究员、信息网络、心理优势（不是自己的钱）

**个人交易者的唯一优势**：
- **可以自由选择交易或不交易**——机构必须交易，个人可以静待良机
- 大多数个人交易者因过度频繁交易而浪费了这个优势
- 目标是交易获利，而不是频繁交易

**机构交易员单干失败的原因**：
- 离开机构 = 离开交易经理（纪律和风险控制的管理者）
- 自己的钱冒险 → 恐惧、贪婪、兴奋、恐慌涌现
- 证明：心态才是交易成败的关键，纪律需要外在约束

## 4. 市场分析方法论

### 4.1 分析的五大基础数据

**只需分析五种数据**：开盘价、最高价、最低价、收盘价、成交量。大道至简，在此基础上堆加大量指标只会徒增困扰。

### 4.2 趋势跟随指标 vs 震荡指标的冲突处理

这是技术分析的核心困境：

| 指标类型 | 趋势市表现 | 震荡市表现 |
|----------|------------|------------|
| 趋势跟随（MA、MACD） | 盈利 | 双重损失 |
| 震荡指标（RSI、随机） | 过早危险信号 | 盈利 |

**关键认知**：不同指标在同一市场给出矛盾信号是常态，不能简单平均化处理，否则会被主观操纵。解决方案见[4.1 三重滤网系统](#41-三重滤网交易系统triple-screen)。

### 4.3 指数移动平均（EMA）

**为什么用EMA而非简单MA**：简单MA在去掉旧数据时会再次变动（"叫两次的看门狗"），EMA为最近价格分配更大权重且旧数据逐渐消失。

**推荐参数**：两条EMA，比例约2:1
- 周线图：26周 + 13周
- 日线图：26日 + 13日（或22日 + 11日）
- 在所有时间周期使用相同参数组

**价值区间**：两条EMA之间的空间定义为"价值区间"——
- 价格在价值区间内 → 市场定价公允
- 价格在下EMA下方 → 被低估（买入机会）
- 价格在上EMA上方 → 被高估（卖出机会）

**交易规则**：
1. EMA上升时做多，价格回调到MA附近时买入，低点下设保护性止损
2. EMA下降时做空，价格反弹到MA附近时卖出，高点上设保护性止损
3. EMA走平时不使用趋势跟随方法

### 4.4 MACD与MACD柱状线

**MACD线**：12日EMA − 26日EMA（快线）
**信号线**：MACD线的9日EMA（慢线）
**MACD柱状线**：MACD线 − 信号线

**核心原则：柱状线的斜率比正负更重要**
- 斜率向上 → 多方力量增强（买入时机）
- 斜率向下 → 空方力量增强（卖出时机）
- 最佳买入信号：柱状线低于0但斜率转向上（空方强弩之末）
- 最佳卖出信号：柱状线高于0但斜率转向下（多方力量耗尽）

**背离信号（最强信号）**：
- **牛市背离**：价格创新低，但MACD柱状线底部比前一次浅 → 市场底部信号
- **熊市背离**：价格创新高，但MACD柱状线顶部比前一次矮 → 市场顶部信号

### 4.5 震荡指标使用要点

**随机指标**：低于30为超卖（买入区），高于70为超买（卖出区）
**RSI**：类似随机指标，进入买卖区域时发出信号

**使用原则**：震荡指标在周线上升趋势中只采用日线买入信号；在周线下降趋势中只采用日线卖出信号（配合三重滤网使用）。

### 4.6 经典图表分析

#### 支撑位与阻力位
- 支撑位：买单比卖单密集处（价格下跌停止）——如同地板
- 阻力位：卖单比买单密集处（价格上涨停止）——如同天花板
- 支撑/阻力被突破后会**角色互换**（阻力变支撑，支撑变阻力）

#### 支撑/阻力的心理学根源
- 支撑/阻力依赖于交易者的**记忆、痛苦和悔恨**
- 在某价位赔钱的人 → 痛苦 → 等待回本后卖出 → 形成阻力
- 错过买卖机会的人 → 悔恨 → 等待第二次机会 → 形成支撑
- 突破区间时痛苦和悔恨最强 → 支撑/阻力最强

#### 支撑/阻力的强度评估（三维法）
强度取决于三个因素（长、宽、深）：

| 因素 | 规律 |
|------|------|
| **持续时间**（长度） | 越久越强。两周=微弱；两月=中等；两年=重要 |
| **高度** | 密集成交区高度占市价1%=弱；3%=中等；7%+=可碾碎大趋势 |
| **成交量**（深度） | 成交量越大，交易者参与越活跃，感情投入越深，支撑/阻力越强 |

**注意**：过于久远的支撑/阻力会减弱——失败者被洗出，新进者无感情投入。

#### 趋势与交易范围
- 上升趋势：一系列更高的高点和更高的低点
- 下降趋势：一系列更低的高点和更低的低点
- 交易范围：价格在支撑和阻力之间横向波动

**关键区别**：
- 趋势中 → 跟着强势方走（上升做多，下降做空），用宽松止损
- 震荡区间 → 做反向操作（买入弱势，卖出强势），用收紧止损

#### 假突破（最重要的图表模式之一）
**定义**：价格穿过支撑/阻力线，在线外待一两天后又返回线内。

| 类型 | 特征 | 信号 |
|------|------|------|
| 向下假突破 | 价格微降至支撑位下，然后重回支撑区 | 买入信号 |
| 向上假突破 | 价格微升至阻力位上，然后重回阻力区 | 卖出信号 |

**真假突破的判断**：
- 真突破伴随**巨额成交量**；假突破成交量很少
- 真突破时技术指标在趋势方向达到新极限；假突破常出现**价格与指标背离**
- 真正的向上突破后不会跌回密集交易区（"火箭不应落回发射台"）

**专业交易者热爱假突破**：
- 等突破停止创新高/低后扑进，逆势买卖
- 在极限价附近设保护性止损，风险小
- 价格返回密集交易区时利润巨大
- 即使一半时间判断错误也值得出手

#### 袋鼠尾
- 单根长长的价格刺穿，随后迅速回归
- 反映群体情绪的极端反应，常预示反转
- 出现在阻力位上方的袋鼠尾 → 卖出信号
- 出现在支撑位下方的袋鼠尾 → 买入信号

### 4.7 成交量分析

**核心原则**：成交量反映群体行为的强度和情感投入程度

成交量=金钱和情感的卷入程度，如同疼痛程度。成交量越大，市场包含的痛苦越大。

#### 成交量与趋势的关系

| 场景 | 含义 |
|------|------|
| 趋势中成交量增加 | 趋势健康，新输家替代出局的交易者 |
| 趋势中成交量减少 | 输家供应减少，趋势即将反转 |
| 价格创新高+成交量创新高 | 价格将保持高位或再创新高 |
| 价格创新低+成交量创新高 | 底部将再次确认或创新低（随后小成交量底部=买入机会） |
| 上升趋势中回调成交量缩水 | 抛压耗尽，上升趋势将重新开始 → 买入机会 |
| 下降趋势中反弹成交量缩水 | 脆弱空头被消灭，下降趋势将继续 → 卖空信号 |

**关键经验法则**：
- "高成交量"= 高于过去两周平均值25%以上
- "低成交量"= 低于过去两周平均值25%以上
- 如果今天成交量高于昨天，今天趋势很可能持续
- "将价格拉上去需要大家来买入，但价格会自行下降"

#### 恐惧 vs 贪婪的成交量特征
- **下降趋势中成交量骤增**：恐惧爆发（强烈但短暂）→ 趋势很可能反转
- **上升趋势中成交量骤增**：贪婪驱动（缓慢且带来快感）→ 趋势可能短暂停止后继续
- **低成交量下的突破**：情感投入不足 → 价格倾向于回归原区间

#### 成交量指标

**能量潮指标（OBV）**：
- 收盘价高于前日 → 当日成交量加入OBV
- 收盘价低于前日 → 当日成交量从OBV减去
- OBV创新高 → 确认多头，买入信号
- OBV创新低 → 确认空头，卖出信号
- **OBV与价格背离**：强烈信号（价格创新高但OBV没创新高=熊市背离=卖出）

**集散指标（A/D）**：
- 追踪开盘价与收盘价的关系
- 收盘价高于开盘价 → 多头获胜，A/D为正
- 收盘价低于开盘价 → 空头获胜，A/D为负
- 按当日获胜程度分配成交量比例
- A/D追踪专业投资者（收盘主导）vs 业余投资者（开盘主导）的对抗

**强力指数（Force Index）**：
- 2日EMA强力指数用于三重滤网第二重
- 降到0以下（非几周新低）→ 买入信号
- 升到0以上（非几周新高）→ 卖出信号
- 13日EMA在中心线上方/下方，判断牛市/熊市

### 4.8 持仓量分析（期货/期权专用）

**定义**：多空双方各自持有的合约总数（期货期权市场多空持仓总是相等）。

#### 持仓量变化的含义

| 持仓量变化 | 市场含义 |
|------------|----------|
| 上升 | 新血液注入，多空分歧加大，现有趋势将持续 |
| 持平 | 输家供给不再增长，趋势进入末期（黄灯警告） |
| 下降 | 输家认赔+赢家获利了结，分歧下降，趋势即将反转 |

#### 具体交易准则

| 价格趋势 | 持仓量变化 | 操作建议 |
|----------|------------|----------|
| 上涨 | 增加 | 确认上升趋势，可加仓多头 |
| 下跌 | 增加 | 空头加码，可加仓空头 |
| 上涨 | 减少 | 多头获利了结+空头认赔，趋势将反转 → 了结多头，准备做空 |
| 下跌 | 减少 | 空头获利了结+多头认赔，趋势将反转 → 回补空头 |
| 上涨 | 走平 | 涨势缺乏后劲，收紧止损，不开新多仓 |
| 下跌 | 走平 | 跌势即将完结，收紧空头止损 |
| 横盘 | 增加 | 商业避险者抛空 → 空头信号，避免交易 |
| 横盘 | 减少 | 商业避险者回补 → 买入信号 |

**持仓量越高**：市场越活跃，滑点越小。短线交易者应专注于持仓量最高的市场。

### 4.9 指标的季节性

将季节概念应用于MACD柱状线，判断市场周期位置：

| 季节 | MACD柱状线状态 | 市场含义 | 操作 |
|------|----------------|----------|------|
| **春季** | 中线下方但斜率向上 | 下跌记忆犹新，多数人不敢买 | **最佳买入时机**，风险低（止损近） |
| **夏季** | 中线上方且上升 | 多数人意识到上升趋势，买入容易 | 获利空间低于春季，风险较高 |
| **秋季** | 中线上方但斜率向下 | 少有人看出变化，仍在买进 | **最佳做空时机**，需远离看多群体 |
| **冬季** | 中线下方且下降 | 多数人意识到下跌，做空容易 | 风险收益比对空方不利，获利空间收窄 |

**核心原则**：春季和秋季操作最难（情感上），但潜在获利最大；夏季和冬季操作容易，但风险收益比恶化。

### 4.10 市场指标（用于判断大盘方向）

**新高-新低指数（NH-NL）**：埃尔德认为这是股市最佳先行指标
- NH-NL为正且上升 → 牛市健康
- NH-NL为负且下降 → 熊市健康
- NH-NL与指数背离 → 转折预警
- "尖峰反弹信号"：NH-NL跌至-500以下后反弹回升 → 大盘买入信号

**50日均线上股票占比**：衡量市场宽度的指标

### 4.11 一致性指标与投入指标（反向指标）

**核心原理**：当群体达成高度共识时，趋势将反转。

| 指标类型 | 来源 | 用法 |
|----------|------|------|
| 一致性指标 | 投资顾问民意调查 | 顾问强烈看多→卖出；强烈看空→买进 |
| 媒体封面 | 主流财经杂志封面 | 封面画牛→了结多头；画熊→底部不远 |
| 广告信号 | 报纸/杂志广告 | 3+广告忽悠同一"机会"→顶部将临 |
| 套期保值者持仓 | CFTC报告 | 跟随套期保值者（长期成功） |
| 大投机者持仓 | CFTC报告 | 大投机者（商品基金）整体表现糟糕 |
| 公司内部人交易 | SEC报告 | 3+内部人一个月内持续买入→看涨；持续卖出→看跌 |
| 空头净额 | 交易所数据 | 高卖空比例→上涨时空头恐慌回补→推升价格 |

**重要提示**：
- 一致性指标不适合精确择时，只提示趋势已衰竭
- 得到信息后用技术指标把握精确时点
- 内部人买入比卖出更有预测价值（卖出理由多，买入只有一个：预期上涨）
- 恐惧比贪婪强烈 → 空头回补的上涨尤其迅猛

### 4.12 多时间周期分析原则

**5倍法则**：每个时间周期对大一号或小一号时间周期约为5倍关系
- 周线 → 日线 → 1小时（或更短）
- 25分钟 → 5分钟 → 1分钟
- 月线 → 周线 → 日线

**必须先看大周期再回到小周期**，避免先入为主被小周期误导。

**市场时间比个人时间慢**：
- 市场由大量人群构成，反应速度更慢
- 新手往往进场过迟（趋势已持续很久）
- 老手往往进场过早（反转信号出现就急于行动）
- 顶部反转比底部反转需要更长时间形成
- 应对：不贪婪、小头寸交易、运用多重时间周期

## 5. 交易系统方法论

> 交易系统是寻找、进入和退出交易的一整套规则。每一个专业的交易者都有一个或多个系统。系统让你在闭市冷静时设计规则，成为市场波动中的理性行为之锚。

### 5.0 机械交易 vs 自主决策交易

| 类型 | 特点 | 优势 | 劣势 |
|------|------|------|------|
| **机械交易** | 严格规则，历史数据测试，系统自动提示 | 情绪化少，可离开屏幕 | 市场变化后失灵，不适应新形态 |
| **自主决策** | 每日更新策略，多因素不同权重 | 对新机会保持开放 | 压力下决断力易出错 |

**高级交易者的做法——两者结合**：
- 机械性交易者：用几个系统但自主调整资产分配比例
- 自主决策交易者：遵循几个不可违反的机械准则（如不逆动力系统交易、不在通道外买卖）
- 性格匹配：冷静执着者倾向机械，虚张声势者倾向自主

**无论哪种方式，系统必须被写下来**——市场压力下重要步骤容易被遗忘（参考《清单革命》）。

### 5.1 三重滤网交易系统（Triple Screen）

埃尔德1986年开发的经典系统，核心思想：**用多重时间周期和技术指标进行三重过滤**，保留趋势跟随与震荡指标各自优势，过滤其缺陷。

#### 步骤一：选择时间周期（5倍关系）

| 你的中期时间周期 | 长期时间周期 | 短期时间周期 |
|------------------|--------------|--------------|
| 日线 | 周线 | 1小时/日内 |
| 5分钟 | 25/30分钟 | 1分钟 |
| 39分钟（美股日内） | — | 8分钟 |

#### 第一重滤网——市场潮流（长期图表）

- **先看长期图表**（不看中期图表），做出牛市/熊市的战略决策
- 使用**动力系统**或周线MACD斜率判断大趋势
- **只允许顺大趋势方向交易**：
  - 周线上升 → 只能买入或观望
  - 周线下降 → 只能卖空或观望
- 第一重滤网排除"卖"或"买"中的一个选项

#### 第二重滤网——市场波浪（中期图表）

- 在中期图表上使用**震荡指标**，识别与大趋势相反的回调
- 周线上升 + 日线震荡指标下降 → 买入机会
- 周线下降 + 日线震荡指标上升 → 卖空机会
- 推荐：2日强力指数EMA、RSI、随机指标

#### 第三重滤网——买入技术（短期图表或日内）

两种方法：
1. **突破法**：在前一日高点上加一个最小单位设买入限价单（卖空反之）
2. **平均EMA下跌穿透法**（推荐）：
   - 测量过去4-6周回调中价格穿透短期EMA的平均深度
   - 买入价 = 估算明日EMA值 − 平均穿透值
   - 利用回调以折扣价买入，避免突破时支付溢价
   - 每天调整买入价，直到成交或周线信号反转

#### 止损与止盈设置
- **止损**：在中期图表（日线）上设置
- **止盈**：在长期图表（周线）上设置
- 三重滤网要求相对保守的止损，跟随潮流或马上退出

### 5.2 动力系统（Impulse System）

**核心思想**：用"惯性"和"能量"两个概念描述市场运动——
- **惯性**：短期EMA的斜率
- **能量**：MACD柱状线的斜率

**颜色规则**：

| EMA斜率 | MACD柱斜率 | 颜色 | 含义 | 禁止 |
|---------|------------|------|------|------|
| 上升 | 上升 | 绿色 | 牛市 | 禁止卖空 |
| 下降 | 下降 | 红色 | 熊市 | 禁止买入 |
| 上升 | 下降 | 蓝色 | 中性 | 无禁止 |
| 下降 | 上升 | 蓝色 | 中性 | 无禁止 |

**关键认知：动力系统是监测系统，不是自动交易系统！**
- 它不告诉你"应该做什么"，只告诉你"**不该做什么**"
- 想买入时：等待周线和日线都不再是红色
- 想卖空时：等待周线和日线都不再是绿色

**入场时机**：
- 日内动量交易者：两个时间周期都绿色时买入，其中一个变蓝或红时止盈
- 抓拐点交易者：最佳信号不是绿色/红色本身，而是红色/绿色**开始消失**时

**退出时机**：
- 一旦动力系统颜色不再支持交易方向（即使只有一个周期变色），立即了结
- 动力系统鼓励"小心谨慎地入场，快速地退出"

### 5.3 通道交易系统（Channel Trading）

**理论基础**：曼德博的发现——"价格围绕价值上下波动"。价值区间在两条EMA之间，通道界定正常波动与异常波动的边界。

#### 构建通道
- **对称通道**（推荐）：以长期EMA为核心，上下各画一条平行线
  - 上通道线 = EMA + 通道系数 × EMA
  - 下通道线 = EMA − 通道系数 × EMA
  - 通道系数从3%或5%开始调整，直到包含最近100根线柱约95%的价格
- **多重ATR通道**：1倍、2倍、3倍ATR三组通道线

#### 交易规则
1. **通道平缓时**：底部买入，顶部卖出
2. **通道上升时**：只在多头方向交易，价值区间买入，上通道线卖出
3. **通道下降时**：只在空头方向交易，价值区间卖空，下通道线回补
4. **突破处理**：大多数突破以失败告终，与突破方向做反向交易

#### 通道+背离组合（最强信号）
1. 价格到达上通道线 + MACD柱状线熊市背离 → 卖出信号
2. 价格到达下通道线 + MACD柱状线牛市背离 → 买入信号

#### 铁律
- **永远不要在上通道线之上买入**
- **永远不要在下通道线之下卖出**
- 此准则会错过一些趋势机会，但安全性极大提升

### 5.4 系统开发与测试原则

1. **手工测试优于自动回测**：每次前进一天，记录信号和决策，最接近真实交易环境
2. **不要过度优化系统**：稳定的系统能经受市场变化，过度优化的系统在真实交易中反而脆弱
3. **系统一旦稳定运行，不要再改来改去**：想修修补补就再开发一个新系统
4. **最好的系统简单而稳定**：构成要素越少，出错风险越小
5. **测试三步走**：模型回测 → 小规模实盘 → 逐步加仓

## 6. 风险管理方法论

### 6.1 风险控制铁三角

这是全书最核心的风险管理工具，用于计算每笔交易的最大规模：

```
A. 单笔交易最大风险额度 = 账户总资金 × 2%（上限）
B. 每股风险 = |入场价 − 止损价|
C. 最大可交易股数 = A ÷ B
```

**交易规模必须通过公式计算，而非随意决定。**

#### 计算示例
- 账户：50,000美元
- 2%原则最大风险：1,000美元
- 买入价：40美元，止损价：38美元
- 每股风险：2美元
- 最大可买股数：1,000 ÷ 2 = **500股**

#### 账户规模调整原则
- 每月第一天统计账户资产，重新计算2%限额
- 账户盈利 → 下月限额提高（可交易更大规模）
- 账户亏损 → 下月限额降低（强制缩小规模）
- 这将交易规模与操作表现自动挂钩

### 6.2 2%法则——防"鲨鱼"

**定义**：单笔交易承担的最大风险不超过账户资产的2%。

| 账户规模 | 单笔最大风险 |
|----------|--------------|
| 5万美元 | 1,000美元 |
| 10万美元 | 2,000美元 |
| 100万美元 | 2,000美元（专业人士常降至0.5%-1%） |

**要点**：
- 2%是**上限**，不是目标；好的交易者常把风险控制在远低于2%
- 2%指的是"账户风险"（入场到止损的价差×股数），不是交易规模
- 新手嫌2%太小，专业人士嫌2%太大——永远选择专业人士那边
- 仅指交易账户资金，不包括储蓄、退休金等

### 6.3 6%法则——防"食人鱼"

**定义**：当月总损失+持仓头寸风险额度之和达到账户资产的6%时，本月剩余时间停止新交易。

**计算方法**：
1. 加总本月所有已实现亏损
2. 加总所有持仓头寸的风险额度（入场价−止损价 × 持仓量）
3. 两者之和若达到月初账户资产的6%，停止新交易

**"可用风险"概念**：
- 每月可用风险 = 账户资产 × 6%
- 每次交易前问：考虑本月所有持仓和已了结交易，我还有足够的可用风险吗？

**6%法则的运作**：
- **连续盈利时**：可将止损提到盈亏平衡点，释放可用风险，允许增加新交易
- **连续亏损时**：强制暂停交易，保护账户主要资产，下月全新开始
- **想交易但无额度时**：可平掉部分盈利持仓释放风险额度，或收紧持仓止损线

**示例流程**（5万美元账户，每笔2%风险）：
- 买入股票A（风险2%）→ 买入B（风险2%）→ 买入C（风险2%）= 已达6%，不能买D
- A上涨，止损提到盈亏平衡（A风险=0）→ 可用风险恢复到4% → 可买D
- B被止损（已亏2%）→ C+D风险4% + 已亏2% = 6% → 不能买E

### 6.4 从下降中恢复——阶梯式规模管理

**核心原则**：风险越高，交易能力越差；盈利后逐步加仓，亏损后快速减仓。

**恢复路径**（以100股为最小单位示例）：
1. 亏损后减到最小规模（如100股）
2. 连续两周盈利（盈利天数>亏损天数且整体盈利）→ 加到200股
3. 再连续两周盈利 → 加到300股
4. 以此类推，每两周盈利增加100股
5. **任何一周亏损 → 退回上一周规模**

**心理学原理**：如同走木板——在桌上走轻松，在10层楼间走会僵化。更高风险妨碍交易能力，需要通过计划好的步骤慢慢接受风险。

### 6.5 两种致命错误

1. **从不使用止损**：暴露在无限损失中
2. **过度交易**：相对于账户持有过高仓位，如同小船装大帆

## 7. 实践执行方法论

### 7.1 盈利目标设定——"足够"是关键词

**核心原则**：盈利潜力应至少是风险的**2倍**（收益风险比 ≥ 2:1）

**不同交易周期的目标设定**：

| 交易类型 | 盈利目标参考 |
|----------|--------------|
| 波段交易 | 两条移动均线的价值区间 |
| 日内交易 | 上/下通道线，配合震荡指标退出 |
| 长线投资 | 前期支撑位/阻力位 |

**"足够"的哲学**：
- 不追求最大化每笔利润，而是通过一个又一个"足够"的交易积累长期成果
- 在价值区间内兑现利润 → 减少不确定性，减少暴露于风险的时间
- 快速赚0.25美元好过慢慢赚1美元

### 7.2 止损设定方法

#### 原则一：在"市场噪声"之外设定
- 止损太近 → 被无意义波动触发
- 止损太远 → 保护作用减弱
- **安全区域止损**：度量平均穿透值（噪声水平），止损设在噪声区域外2倍以上

#### 原则二：不要设在明显位置
- 大众把止损设在近期最低点下方 → 市场会先打穿再反转
- **尼克止损法**：止损设在**次低点**（而非最低点）附近
  - 多头：止损设在近期次低点下方1美分
  - 空头：止损设在近期次高点上方几美分
- 避免整数价位：80美元买入，止损设77.94美元而非78美元

#### 原则三：ATR止损法
- 止损设在离当前线柱极值至少1倍ATR处（2倍更安全）
- 可作为移动止损，随线柱转变而移动

#### 原则四：只顺着交易方向移动止损
- **永远不要"给下跌更大空间"**——调低止损线等于放弃纪律
- 交易走坏时：接受损失，保持关注，等待底部重新买入

#### 原则五：灾难性止损
- 每笔交易都应设一个"灾难性止损"（硬止损），如同救生衣
- 软止损（心中止损）可灵活使用，但灾难性止损始终有效
- 新手必须使用硬止损，专业交易者可酌情使用软止损

#### 不要让盈利变为亏损
- 盈利目标1000美元的交易，浮盈300美元时开始保护利润
- 浮盈达到300美元 → 止损移到盈亏平衡点（"为交易翻边"）
- 之后保护一定比例的浮盈（如1/3）

### 7.3 A级交易评估系统

#### 交易评级公式

**买入评级** = (最高价 − 买入价) / (最高价 − 最低价)
- 买入价越接近当日最低点，评级越高
- >50%为不错，100%为买在最低点

**卖出评级** = (卖出价 − 最低价) / (最高价 − 最低价)
- 卖出价越接近当日最高点，评级越高
- >50%为不错

**交易评级** = (卖出点 − 买入点) / (通道线高点 − 通道线低点)
- 通道线包含过去100天约90%-95%的价格
- **获利达到通道高度的30%或以上 = A级交易**

#### A级交易的意义
- 必须为自己定义什么是"A级交易"——如果不知道，就没有业务模式
- 每次下单前问自己：**"这是一笔A级交易吗？"**
- 答案为"是"才计算风险、规模和买入价；为"否"则翻过这一页
- 个人交易者的优势：可以在喜欢时交易，不喜欢时不交易——充分利用这个优越条件

### 7.4 交易策略的三种模式

埃尔德自己的三种策略（可参考但不必复制）：

1. **价格背离 + 假突破**（首选）：牛市/熊市背离伴随假突破
2. **回调到价值区域**：强势趋势中回调到价值区间时买入
3. **极值交易**：在过度扩张的趋势反转时下注（偶尔使用）

**原则**：只参与符合自己策略标准的交易机会，"在我看来还不错"不是一个系统。

### 7.5 交易计划的三要素

每笔交易计划必须包含三个关键数字：

| 要素 | 说明 |
|------|------|
| **买入价位** | 你愿意以什么价格成交 |
| **盈利目标价位** | 期望从市场获得的收益 |
| **止损价位** | 愿意遭受的风险 |

**没有这三个数字的交易不是交易，而是赌博。**

收益风险比应 ≥ 2:1，唯一可偏离的时机是技术指标信号特别强烈时——但不要捏造目标来把不确定的交易转为可接受的交易。

### 7.6 搜寻交易机会的方法

1. 先开发交易系统，再搜寻机会（没有系统，你寻找什么？）
2. 对每个机会第一个问题："这是A级交易吗？"
3. 扫描程序找候选目标，但扫描不能替代分析
4. 半自动MACD背离扫描：寻找A+B形态，C开始浮现时警示
5. 比较看涨清单和看跌清单的规模，判断市场整体方向
6. 添加"负面规则"：剔除低成交量股、极端高价/低价股

### 8.2 自我测试（5问，开市前30秒）

| 问题 | 0分 | 1分 | 2分 |
|------|-----|-----|-----|
| 1.身体感觉 | 生病 | 正常 | 好极了 |
| 2.昨日交易 | 亏损 | 有盈有亏/没交易 | 盈利 |
| 3.交易计划 | 没做 | 还凑合 | 准备很好 |
| 4.情绪状态 | 低落 | 正常 | 棒极了 |
| 5.行程安排 | 很忙 | 正常 | 很闲 |

**评分解读**：
- ≤4分（红色）：今天最好不要交易
- 5-6分（黄色）：非常小心地交易
- 7-8分（绿色）：可以交易
- 9-10分（黄色）：条件太完美，警惕过度自信

**铁律**：任何一题得0分，当天不进行交易。

### 8.3 交易阿氏评分（Apgar Score）

借鉴新生儿阿氏评分，对交易计划进行客观打分。以下为"假突破+背离"策略的示例（需根据自身策略定制）：

| 问题 | 0分 | 1分 | 2分 |
|------|-----|-----|-----|
| 1.周线动力系统 | 红色 | 绿色 | 蓝色 |
| 2.日线动力系统 | 红色 | 绿色 | 蓝色 |
| 3.日线价格位置 | 价值区间之上 | 价值区间内 | 价值区间之下 |
| 4.假突破 | 没有 | 已发生 | 很可能将发生 |
| 5.完备性 | 无周期符合 | 一个符合 | 两个都完备 |

**决策规则**：
- 总分 ≥ 7分 且 无任何0分项 → 继续执行交易计划
- 总分 < 7分 → 放弃，寻找其他机会
- 计算不超过1分钟

### 8.4 交易单（Trade Ticket）

完整的交易计划表格，包含5个部分：

**第一部分：交易鉴定**
- 股票代码、财报披露日期、除息日、计划日期

**第二部分：交易阿氏评分**
- 五个问题打分，回答"这是A级交易吗？"

**第三部分：市场、买入点、目标价、止损价、风险控制**
- 市场基本状况（尖峰反弹信号、均线指标、空头净额等）
- **三个核心数字**：买入价、目标价、止损价
- 资金风险（≤账户2%）、持仓规模（铁三角计算）

**第四部分：买入之后**
- A级盈利目标 = 买入价 + 日通道线高度 × 30%
- 软止损/硬止损/灾难性止损
- 止损移到盈亏平衡点的价格水平
- 检查清单：设置止损、创建日志、下达止盈单

**第五部分：版权信息**

### 8.5 交易日志

交易日志的三大好处：

1. **即时好处**：秩序感与结构感——记录计划、入场、出场强化纪律
2. **中期好处**（1-2个月后）：回顾已结束交易，当时模糊的信号在图表中间变得清晰
3. **长期好处**：积累几十个记录后，多种方法分析，从资金曲线中成长

**日志核心要素**：
- A部分：为何决定交易这只股票（附合成线图）
- B部分：入场/退出日期、价格、滑点、交易量、交易等级
- C部分：退出原因（附标注入场/出场点的线图）
- D部分：退出策略清单（到目标价/止损/价值区间/不再延续趋势等）
- E部分：退出两个月后的回顾分析

**"两个月后回顾"是最佳学习方式**：
- 在3月底回顾1月退出的交易
- 用箭头标注入场/出场点
- 写下评价：哪些做对了，哪些需要改变
- 从中发现自己系统性的问题（如止损太紧、错过大趋势等）

## 9. 交易工具选择

### 9.1 工具对比

| 工具 | 优势 | 劣势 |
|------|------|------|
| 股票 | 流动性好、信息透明、可长期持有 | 需关注财报、分红 |
| ETF | 分散风险、流动性好 | 个股表现可能远超ETF |
| 期权 | 杠杆高、风险可控 | 复杂、有时间价值损耗 |
| 期货 | 杠杆高、可双向 | 杠杆致命、需极敏感风控 |
| 外汇 | 24小时交易 | 透明度低、经纪商问题多 |

### 9.2 期货交易特别提示
- 期货通过小保证金比例引诱过度交易，这是致命的
- 必须用2%原则筛选：如果一份合约的风险超过2%限额，不要交易
- 用铁三角判断：账户2%÷单合约风险，若结果<1，不交易

### 9.3 交易时间周期选择

**三种交易类型对比**：

| 类型 | 持仓期 | 优点 | 缺点 | 适合人群 |
|------|--------|------|------|----------|
| **长期投资** | 月-年 | 长期收益可观，避免每日盯盘 | 跌势中持有过久亏损严重，需极强信心和耐心 | 有基本面判断力者 |
| **波段交易** | 日-周 | 机会多，风险可控，不留隔夜风险 | 可能错过主升浪 | **推荐初级和中级投资者** |
| **日内交易** | 分钟-小时 | 短线机会多，无隔夜风险 | 需快速反应，交易成本高 | 专业交易者 |

**埃尔德的推荐**：
- 波段交易最适合学习——比长期投资快，比日内交易有思考时间
- 多时间周期交易者应为不同操作建立**不同账户**，分开评价
- 长期投资应结合技术分析选择买卖时机，基本面选股+技术面择时

### 9.4 交易软件的三种类型

| 类型 | 特点 | 风险 |
|------|------|------|
| **工具箱** | 显示数据、绘图、标指标、调参、测试系统 | 最安全，自己决策 |
| **黑匣子** | 输入数据即给出交易信号，运作保密 | 历史数据好看但会失灵，市场变化后失效 |
| **灰匣子** | 工具箱+黑匣子结合，架构公开，可调参数 | 较好，兼顾透明与便利 |

**警示**：不要购买黑匣子系统，几乎所有黑匣子的历史记录都好看（因按历史数据设计），但市场变化后会失灵。

### 9.5 限价指令原则
- **建仓用限价指令**（保证价格，不保证成交）
- **止损用市价指令**（保证成交，不保证价格）
- **"慢慢建仓，快速清仓"**
- 滑点成本平均是佣金的3倍以上，务必重视

## 10. 成本控制方法论

### 10.1 佣金控制
- 货比三家，选择佣金最低的经纪商
- 设计不用交易太频繁的交易系统
- 活跃交易者每年佣金可达账户20%，这是致命的

### 10.2 滑点控制
- 尽可能用限价指令，除非止损
- 选择流动性好、成交量高的市场
- 在市场平静时交易
- 记录每次滑点成本

### 10.3 买卖价差
- 市价指令总是以不利一方成交
- 市场波动剧烈时价差扩大
- 尽量用限价指令

## 11. 交易者成长路径

### 11.1 三个阶段

| 阶段 | 特征 | 时间分配 |
|------|------|----------|
| 新手 | 不做过多思考，盲目交易 | 几乎不分析 |
| 中级 | 精力放在市场分析上 | 分析市场为主 |
| 专业 | 大部分时间花在风险控制上 | 风控占1/3时间 |

### 11.2 成长原则
1. **成为自己的经理**：机构交易员表现好是因为有经理强制纪律；个人交易者必须自我管理
2. **从模拟到实盘**：模拟交易测试纪律和系统，但微小金额实盘学到的比几个月模拟更多
3. **先小后大**：小规模测试系统，盈利持续良好后逐步扩大规模
4. **质疑和检验**：用自己的数据检验每个观点，亲自交易实践

### 11.3 纪律建立
- 纪律的第一步：完成每日功课
- 纪律的进阶：写下交易计划
- 纪律的高潮：执行计划并完成交易日志
- "交易纪律和控制体重一样——不知道今天的体重，怎么控制？"

## 附录：关键公式速查

### 指标公式

| 公式 | 计算 |
|------|------|
| EMA | EMA = P今日 × K + EMA昨日 × (1−K)，其中 K = 2/(N+1) |
| MACD线 | 12日EMA − 26日EMA |
| 信号线 | MACD线的9日EMA |
| MACD柱状线 | MACD线 − 信号线 |
| 通道上轨 | EMA + 通道系数 × EMA |
| 通道下轨 | EMA − 通道系数 × EMA |
| OBV | 收盘涨→+成交量；收盘跌→−成交量；持平→不变 |
| 集散指标A/D | [(收盘−开盘)/(最高−最低)] × 成交量 |
| 强力指数 | 今日成交量 × (今日收盘−昨日收盘) 的EMA |

### 风险管理公式

| 公式 | 计算 |
|------|------|
| 2%限额 | 账户资产 × 2%（单笔最大风险） |
| 6%限额 | 月初账户资产 × 6%（当月最大风险） |
| 最大持仓 | 单笔最大风险 ÷ 每股风险（铁三角C=A÷B） |
| 每股风险 | \|入场价 − 止损价\| |
| 可用风险 | 6%限额 − 本月已实现亏损 − 持仓头寸风险之和 |

### 交易评级公式

| 公式 | 计算 |
|------|------|
| 买入评级 | (最高价 − 买入价) / (最高价 − 最低价) |
| 卖出评级 | (卖出价 − 最低价) / (最高价 − 最低价) |
| 交易评级 | (卖出点 − 买入点) / (通道高 − 通道低) |
| A级交易标准 | 交易评级 ≥ 30% |
| A级盈利目标 | 买入价 + 日通道线高度 × 30% |

### 入场与止损公式

| 公式 | 计算 |
|------|------|
| 平均EMA穿透值 | Σ\|每日EMA穿透深度\| ÷ 穿透线柱数 |
| 买入触发价 | 估算明日EMA值 − 平均EMA穿透值 |
| 安全区域止损 | 平均穿透值 × 系数（≥2），设在噪声区域外 |
| ATR止损 | 离当前线柱极值 ≥ 1倍ATR（2倍更安全） |

### 持仓量判断规则

| 价格趋势 | 持仓量 | 含义 |
|----------|--------|------|
| 上涨/下跌 | 增加 | 趋势将持续 |
| 上涨/下跌 | 走平 | 趋势老化，收紧止损 |
| 上涨/下跌 | 减少 | 趋势将反转 |
| 横盘 | 增加 | 空头信号（避险者抛空） |
| 横盘 | 减少 | 买入信号（避险者回补） |

## 结语：四大支柱的整合

埃尔德方法论的精髓在于**整合**：

- **没有心理**，再好的系统也会被情绪毁掉
- **没有分析**，无法识别交易机会
- **没有风控**，一次或一连串亏损就能终结交易生涯
- **没有记录**，无法从经验中成长，会重复犯错

四大支柱通过**交易记录**这一粘合剂结合在一起：记录功课强化心理，记录计划落实分析，记录风险执行风控，记录回顾促进成长。

**最终目标**：像那位得克萨斯专业交易者一样——"如果你坐到桌子对面看我操盘，你肯定无法分辨我今天是赚了还是赔了2000美元。"达到"胜不骄，败不馁"的境界，只关注正确地交易与提升技巧，钱不再影响情绪。
