## 1. 总体框架：缠论的底层逻辑与学科结构

### 1.1 缠论是什么

缠论是缠中说禅在新浪博客"教你炒股票"108 篇中构建的**原创技术分析体系**。它不是指标的堆砌，而是一套**纯逻辑推导**的理论，其有效性如同几何一般，前提是"价格充分有效市场 + 非完全绝对趋同交易"。

| 维度 | 缠论的立场 |
|------|-----------|
| **本质** | 几何理论 + 动力学（能量结构的几何化） |
| **核心思想** | 走势必完美：任何级别的任何走势类型终要完成 |
| **方法论根基** | 完全分类 + 递归定义 + 当下直观（非预测） |
| **确定性来源** | "走势终完美"——股市不确定性中的确定性 |
| **交易性质** | 右侧交易，等买卖点信号出现再操作 |
| **适用范围** | 任何可连续交易、交易算数的金融品种 |

### 1.2 两大组成部分：形态学 + 动力学

缠论本质分为两部分，缺一不可：

| 分支 | 研究对象 | 是否需前提 | 关键产物 |
|------|----------|-----------|----------|
| **形态学** | 分型、笔、线段、中枢、走势类型 | 无须前提（纯几何） | 第二、三类买卖点 |
| **动力学** | 背驰、盘整背驰、中枢能量结构 | 需"价格充分有效市场+非完全绝对趋同交易"前提 | 第一类买卖点 |
| **两者结合** | 形态学 + 动力学联立 | — | 完整三类买卖点体系 |

> **重要**：光用形态学即可形成有效操作体系（第二、三类买卖点 + 中枢震荡）。动力学缺失时抓不住第一类买卖点。形态学是基础，"形态分析不好，也动力不起来"。

### 1.3 缠论 vs 其他技术理论

| 对比项 | 缠论 | 波浪/江恩/指标派 |
|--------|------|------------------|
| 推导方式 | 纯逻辑推导（如几何证明） | 先验前提 + 经验归纳 |
| 结论客观性 | 唯一客观，不因人而异 | "千人千浪"，因人而异 |
| 预测性 | 当下理论，反对预测 | 预测性理论，易被庄家制造骗线 |
| 可解释性 | 可严格推出波浪等所有图形理论并指出其缺陷 | 各自为政，互不兼容 |

### 1.4 懒人学习路径（缠中说禅官方推荐）

```
分型 → 笔 → 线段 → 最小级别中枢 → 各级别中枢 → 走势类型
```

> 这是形态学最基本、无法再简略的链条。**一环扣一环，一环漏掉就接不上**。

### 1.5 三大独立系统

缠论强调，技术分析只是三个独立系统之一，三者依据概率原则组合才保证数学有效性：

1. **基本面**（变化慢，稳定）
2. **比价关系**（确立后基本稳定）
3. **技术面**（变化最快，操作上起决定作用）

> 介入品种靠三个系统组合，介入后技术面起决定作用。

## 2. 两大原理与走势分解定理

这是缠论的"宪法"，所有操作都建立其上。

### 2.1 走势的基本分类

> **所有走势，都能完全分解成上涨、下跌、盘整三种基本情况。** 上涨、下跌合称趋势。

| 类型 | 定义（基于高低点） |
|------|-------------------|
| 上涨 | 最近一个高点比前一高点高，且最近一个低点比前一低点高 |
| 下跌 | 最近一个高点比前一高点低，且最近一个低点比前一低点低 |
| 盘整 | 高低点方向不一致（高高低低 或 低低高高） |

### 2.2 两大基本原理

> **缠中说禅技术分析基本原理一**：任何级别的任何走势类型终要完成。——即"走势终完美"。

> **缠中说禅技术分析基本原理二**：任何级别任何完成的走势类型，必然至少包含一个缠中说禅走势中枢。

**原理一的深刻含义**（四层）：
1. 走势可分为上涨、下跌、盘整三种类型
2. 任何走势类型都要完成
3. 任何完成的走势类型必然包含一个中枢（三段次级别走势）
4. 任何走势类型完成后，必然转化为其他两种走势类型之一

### 2.3 两大走势分解定理

> **缠中说禅走势分解定理一**：任何级别的任何走势，都可以分解成同级别盘整、下跌与上涨三种走势类型的连接。

> **缠中说禅走势分解定理二**：任何级别的任何走势类型，都至少由三段以上次级别走势类型构成。

### 2.4 原理的可操作化

| 静态结论 | 动态转化（原理一使其可操作） |
|----------|------------------------------|
| 走势可分解为趋势与盘整 | "走势类型终要完成"→ 完成后必然转化为其他类型 |
| 难以实用 | 可实用：下跌完成 → 只能转上涨或盘整 → 多头必然获利 |

> **操作意义**：一旦判断下跌结束，随后面对的必然是盘整与上涨，对多头都必然产生利润，区别只在大小与快慢。

## 3. 形态学基础：K线、包含处理与分型

### 3.1 缠论的 K 线观

| 维度 | 传统 K 线 | 缠论 K 线 |
|------|-----------|-----------|
| 关注点 | 开盘、收盘、上下影线 | **只看价格到达的区间**（最高点、最低点） |
| 阴阳 | 区分阳线阴线 | **不分阴阳** |

### 3.2 K 线包含关系处理（关键预处理）

相邻两根 K 线若一根的高低点全在另一根范围内，需合并处理：

| 方向 | 合并规则 |
|------|----------|
| **向上** | 取两根 K 线的**最高点**为高点，**低点中较高者**为低点 |
| **向下** | 取两根 K 线的**最低点**为低点，**高点中较低者**为高点 |

**方向判定**：若第 n 根与 n+1 根包含，而 n 与 n-1 不包含，则 gn≥gn-1 为向上，dn≤dn-1 为向下。

**处理三原则**：
1. **结合律**必须遵守
2. **包含关系不满足传递律**：必须按时间顺序逐对处理（先 1-2 合并新 K 线，再用新 K 线与 3 比较）
3. 处理后所有 K 线无包含关系

### 3.3 四种基本形态（完全分类）

经非包含处理后，相邻三根 K 线只有四种组合：

| 形态 | 特征 |
|------|------|
| **顶分型** | 第 2 根 K 线高点是三根中最高，低点也是三根中最高 |
| **底分型** | 第 2 根 K 线低点是三根中最低，高点也是三根中最低 |
| 上升 K 线 | 三根依次上升 |
| 下降 K 线 | 三根依次下降 |

- 顶分型的最高点叫**顶**，底分型的最低点叫**底**
- **核心推论**：所有顶必然是顶分型，所有底必然是底分型
- **操作推论**：没有顶分型，没有顶；没有底分型，没有底 → 操作级别 K 线图上没有顶分型就可持有睡觉

### 3.4 分型的心理本质

一个顶分型 = 卖方分力三次阻击买方分力三次努力后的胜利（"一鼓作气，再而衰，三而竭"）。第三根 K 线买方攻击被完全击败，小级别上大致出现第二类卖点走势。

## 4. 笔与线段方法论

### 4.1 笔的定义

> **两个相邻的顶和底之间构成一笔。** 一笔内部的波动都可忽略不计。

**严格构成条件**：
1. 必须是**相邻**的顶和底（隔了几个不行）
2. 顶和底之间**至少有 1 根独立 K 线**（不共用 K 线，违反结合律）
3. 先顶后底 = 向下一笔；先底后顶 = 向上一笔

**两种异常处理**：
- 两顶/底中间有其他顶底 → 把多笔当一笔，继续用一顶一底原则拆分
- 两顶/底中间无其他顶底 → 第一个顶/底转折级别太小，可忽略

> **缠中说禅笔定理**：任何当下，在任何时间周期 K 线图中，走势必然落在一确定的具有明确方向的笔当中（向上笔或向下笔）。在笔中的位置只有两种：（1）分型构造中；（2）分型确认后延伸为笔的过程中。

### 4.2 线段的定义

> **线段至少由 3 笔组成。** 线段的前 3 笔必须有重叠部分。

| 性质 | 说明 |
|------|------|
| 笔数 | 必为**奇数且 ≥ 3**（3、5、7…） |
| 方向 | 第一笔与最后一笔方向一致 |
| 重叠 | 前 3 笔必须有重叠部分，否则不构成线段 |

### 4.3 线段破坏定理

> **缠中说禅线段分解定理**：线段被破坏，当且仅当至少被有重叠部分的连续 3 笔的其中一笔破坏。换言之，**线段只能被线段破坏，单纯一笔不能破坏线段**。

**两种破坏情况**（用特征序列分型精确化，第 67 课）：
- **第一种**：笔直接突破线段（无缺口或笔内有第一类破坏）
- **第二种**：笔突破后需延伸出特征序列分型才算破坏

### 4.4 线段划分操作要点

1. 必须从较大趋势的**起涨点/起跌点**开始划分
2. 30 分钟线段完成 → 在日线相应趋势上标记 → 该线段即日线上中枢的构件
3. 缺口处理：顺着线段方向的缺口归入笔内；**突然的逆向缺口**可当成一个线段（0=0+0+0，三个缺口叠加成 3 笔）
4. 精度预设（如取整）一旦设定，**必须一路保持**

## 5. 走势中枢：定义、生灭与三大定理

> **中枢是缠论技术分析的核心问题。中枢问题一旦解决，很多判断难题迎刃而解。**

### 5.1 走势中枢的定义

> **缠中说禅走势中枢**：某级别走势类型中，被至少三个连续次级别走势类型所重叠的部分。

**区间公式**：三个连续次级别走势 A、B、C，高低点分别为 a1/a2、b1/b2、c1/c2，则中枢区间为：
```
[max(a2, b2, c2), min(a1, b1, c1)]
```
（三个低点中最高者 ~ 三个高点中最低者，可目测）

**递归底线**：对最低不可分解级别，中枢定义为**至少三根该级别 K 线重叠的部分**（一般设 1 分钟或 5 分钟）。

### 5.2 中枢的关键变量

| 变量 | 含义 |
|------|------|
| ZG | min(g1, g2) —— 中枢区间上沿 |
| ZD | max(d1, d2) —— 中枢区间下沿 |
| GG | max(gn) —— 中枢所有次级别走势高点最高者 |
| DD | min(dn) —— 中枢所有次级别走势低点最低者 |
| G | min(gn) |
| D | max(dn) |
| Zn | 中枢中按时间顺序的第 n 个与中枢方向一致的次级别走势段 |
| gn/dn | Zn 的高点/低点 |

### 5.3 中枢的"生住坏灭"

| 阶段 | 含义 |
|------|------|
| 生 | 前三个连续次级别走势类型重叠确定（已解决） |
| 住（维持） | 任何离开中枢的走势都必须以次级别以下走势类型返回 |
| 坏（破坏） | 次级别走势离开后，次级别回抽不重新回到中枢内 |
| 灭 | 破坏后形成新的同级别中枢或更大级别中枢 |

### 5.4 中枢三大定理

> **缠中说禅走势中枢定理一**：在趋势中，连接两个同级别缠中说禅走势中枢的，必然是次级别以下级别的走势类型。
> （不必然是趋势，也不必然是次级别；可以是缺口。相连走势级别越低，力度越大——这是缺口技术含义的理论依据）

> **缠中说禅走势中枢定理二**：在盘整中，无论是离开还是返回缠中说禅走势中枢的走势类型，必然是次级别以下的。

> **缠中说禅走势中枢定理三**：某级别缠中说禅走势中枢的破坏，当且仅当一个次级别走势离开该中枢后，之后的次级别回抽走势不重新回到该中枢内。
> （两个次级别走势组合只有三种：趋势+盘整、趋势+反趋势、盘整+反趋势。**最有力的破坏是"趋势+盘整"**）

### 5.5 中枢的延伸、扩张与新生

| 演化 | 条件 | 结果 |
|------|------|------|
| **延伸** | 任意 [dn,gn] 与 [ZD,ZG] 有重叠 | 中枢延续，不升级 |
| **扩张** | Zn 使 dn>ZG 或 gn<ZD，且与前一中枢波动区间重叠 | 形成更大级别中枢 |
| **新生** | Zn 使 dn>ZG 或 gn<ZD，且与前一中枢波动区间不重叠 | 形成同级别新中枢 → 趋势 |

> **缠中说禅走势中枢中心定理一**：中枢延伸等价于任意 [dn,gn] 与 [ZD,ZG] 有重叠。若有 Zn 使 dn>ZG 或 gn<ZD，则必然产生高级别中枢或趋势及延续。

> **缠中说禅走势中枢中心定理二**：
> - 后 GG < 前 DD ⇔ 下跌及其延续
> - 后 DD > 前 GG ⇔ 上涨及其延续
> - 后 ZG < 前 ZD 且后 GG ≥ 前 DD，或 后 ZD > 前 ZG 且后 DD ≤ 前 GG ⇔ 形成高级别中枢

### 5.6 中枢延续与级别扩张的判别

> **缠中说禅走势级别延续定理一**：在更大级别缠中说禅走势中枢产生前，该级别走势类型将延续（只能是该级别盘整或趋势的延续）。

> **缠中说禅走势级别延续定理二**：更大级别缠中说禅走势中枢产生，当且仅当围绕连续两个同级别缠中说禅走势中枢产生的波动区间产生重叠。

**实操要点**：
- 中枢震荡**超过 9 段** → 等于形成新盘整 → 原趋势结束
- 中枢突破后继续形成第三类买点 → 观察是否进入背驰段；不背驰则新中枢（趋势延伸）；背驰则按背驰后三种转化处理

## 6. 走势类型与级别方法论

### 6.1 走势类型的精确定义

> **缠中说禅盘整**：在任何级别的任何走势中，某完成的走势类型**只包含一个**缠中说禅走势中枢。

> **缠中说禅趋势**：在任何级别的任何走势中，某完成的走势类型**包含两个以上依次同向**的缠中说禅走势中枢。向上为上涨，向下为下跌。

**关键约束**：趋势中的中枢之间**绝对不存在重叠**。

### 6.2 走势类型延伸的本质

| 类型 | 延伸实质 | 结束判据 |
|------|----------|----------|
| 盘整 | 不能产生新的中枢（围绕中枢上下延伸） | 次级别突破后回抽不回中枢 |
| 趋势 | 同级别同向中枢不断产生 | 趋势方向不再产生新中枢 |

> **判断走势类型是否结束 = 判断是否产生新中枢**。这是技术分析核心问题。

### 6.3 级别的本质与"显微镜"比喻

> **级别是自相似性自组出来的，是生长出来的**，如同基因按图谱自动生长。

| 关键认知 | 说明 |
|----------|------|
| 级别客观存在 | 由中枢递归定义保证存在性 |
| 图 ≠ 级别 | 什么级别的图和什么级别的中枢无必然联系；图是显微镜，中枢是观察物 |
| 倍数选择 | 1 分钟图倍数最大看得最清；也可从日线/周线入手粗选再精细看 |
| 联立观察 | 多级别联立 = 联立方程，解大幅减少，边界条件异常简单 |

### 6.4 级别的递归构建

选定最低级别（如 1 分钟）后：
```
1分钟分型/笔/线段 → 1分钟中枢 → 1分钟走势类型
  → 5分钟中枢 → 5分钟走势类型
    → 30分钟 → 日 → 周 → 月 → 季 → 年
```

**大级别图上标记次级别走势类型的好处**：
1. 帮助记忆（1 分钟线段成千上万会抓狂）
2. 看大级别 MACD 一目了然（避免 1 分钟图上累加柱子面积）

### 6.5 各级别对应的最少延伸时间（经验）

| 级别 | 一笔最少 K 线 | 对应均线 |
|------|--------------|----------|
| 笔 | ≥ 6 根基本 K 线 | 连 5 日均线都碰不到就不是笔反弹 |
| 趋势中同向中枢 | — | 对应的压制均线大致相同（如第一中枢被 89 日压制，后续中枢大概率同均线） |

## 7. 背驰与盘整背驰方法论

### 7.1 背驰的前提：没有趋势，没有背驰

> **背驰只出现在趋势之中。盘整中无所谓背驰**（盘整中类似方法称"盘整背驰"）。

### 7.2 背驰的精确定义

> **缠中说禅趋势力度**：前一缠绕结束与后一缠绕开始，由短期均线与长期均线相交所形成的面积。

> **背驰**：在前后两个同向趋势中，当缠中说禅趋势力度比上一次弱，就形成了背驰。

> **缠中说禅趋势平均力度**：当下价格与前一收敛/缠绕结束时短长期均线形成面积 ÷ 时间。即时可判断，转折点基本即时抓住（风险稍大，需高技巧）。

### 7.3 MACD 辅助判断背驰（标准方法）

**前提结构**：A、B、C 三段在一个大趋势里，A 之前已有一个中枢，B 是大趋势的另一个中枢（一般把 MACD 黄白线回拉 0 轴附近），C 段完成时对应 MACD 柱子面积 < A 段对应面积 → 标准背驰。

**操作技巧**：
- 看 MACD 柱子面积**不需等 C 段红柱全出来**：柱子伸长力度变慢时，把已出现面积 ×2 近似该段面积
- 一般可抛到最高价/买在最低价附近

**两中枢上涨的标准 MACD 形态**：
1. 第一段：黄白线从 0 轴下穿上穿，0 轴上方停留形成第一个中枢（同时第二类买点）
2. 突破中枢：黄白线快速拉起（最有力度的一段，指标钝化多发于此），次级别背驰结束 → 第二中枢形成，黄白线回 0 轴
3. 突破第二中枢：黄白线/柱子不能创新高或面积/高度不能突破 → 背驰 → 结束

### 7.4 盘整背驰

> 盘整背驰 = 企图脱离中枢的运动力度有限，被阻止重新回到中枢里。

**判断方法**（以上涨为例）：
- C 段**不破中枢**：MACD 柱子面积 C < A → 必有回跌
- C 段**突破中枢**但面积 C < A：**先出来**，其后两种情况：
  - 回跌不重新跌回 B 段中枢顶部以下 → 次级别第一类买点回补（构成第三类买点）
  - 回跌跌回 → 继续盘整

### 7.5 背驰的级别与力度

> **缠中说禅背驰-买卖点定理**：任一背驰都必然制造某级别的买卖点，任一级别的买卖点都必然源自某级别走势的背驰。

| 关键认知 | 说明 |
|----------|------|
| 背驰必有逆转 | 逆转直到新的次级别买卖点出现为止 |
| 逆转≠永远持续 | 日线顶背驰 → 回跌后 5 分钟/30 分钟底背驰 → 买点 → 可重新上涨甚至创新高 |
| 小级别背驰价值 | 与所处大级别位置相关：日线上涨中段 5 分钟顶背驰意义不大；背驰段中 5 分钟顶背驰可能引发趋势逆转 |
| 两中枢后背驰占比 | 日线以上级别第二个中枢后背驰占 90% 以上 |

### 7.6 防狼术（MACD 0 轴法则）

> **最基本的防狼术**：回避所有 MACD 黄白线在 0 轴下面的市场或股票。

- 选定最低时间周期（如 60 分钟/30 分钟）
- 该周期 MACD 黄白线进入 0 轴下 → 彻底离开，直到重新站上 0 轴
- 第一类买点往往对应 0 轴下金叉；第三类买点往往对应 0 轴上金叉

## 8. 三类买卖点体系与完备性定理

### 8.1 三类买卖点定义

| 买卖点 | 定义（买点为例） | 位置 | 形成机理 |
|--------|------------------|------|----------|
| **第一类买点** | 空头模式最后一次缠绕后出现的背驰式下跌的终点 | 中枢**之下** | 趋势背驰构成 |
| **第二类买点** | 多头模式第一次缠绕后出现的下跌的终点（第一类买点后第一次次级别回调低点） | 中枢上/中/下皆可 | 多由盘整背驰构成 |
| **第三类买点** | 次级别走势向上离开中枢后，次级别回试低点不跌破 ZG | 中枢**之上** | 中枢扩张或新生产生 |

> **卖点反之**。第一类卖点 = 多头最后一次缠绕后背驰式上涨终点。

### 8.2 买卖点定律与完备性

> **缠中说禅买卖点定律一**：任何级别的第二类买卖点都由次级别相应走势的第一类买卖点构成。→ 所有买卖点都可归结为不同级别的第一类买卖点。

> **缠中说禅趋势转折定律**：任何级别的上涨转折都是由某级别的第一类卖点构成的；任何级别的下跌转折都是由某级别的第一类买点构成的。

> **缠中说禅买卖点完备性定理**：市场必然产生盈利的买卖点，只有第一类、第二类、第三类买卖点。

> **缠中说禅升跌完备性定理**：市场中的任何上涨与下跌，都必然从缠中说禅三类买卖点中的某一类开始以及结束。

### 8.3 三类买卖点与中枢的关系

| 中枢状态 | 中枢之上 | 中枢之下 |
|----------|----------|----------|
| 延续 | 只可能有卖点（延续必然转折向下） | 下跌确立后才可能有第一类买点 |
| 扩张/新生 | 第三类买点（扩张→更大中枢；新生→上涨趋势） | — |

**第二类买点的位置分类**：
- 在中枢上方 → 第二、三类买点重合 → **力度最强**（大级别上涨往往出现）
- 在中枢中 → 扩张与新生机会对半
- 在中枢下方（甚至低于第一类买点）→ 走势弱，扩张性中枢可能极大

### 8.4 第二、三类买点重合（最强信号）

**条件**：第一类买点后，次级别走势凌厉直接上破前面下跌的最后一个中枢，然后其上次级别回抽不触及该中枢。

> 一旦出现，**大级别上涨往往就会出现**。典型如 1994 年 7 月 325 点后的跳空缺口行情。

### 8.5 第三类买卖点定理

> **第三类买卖点定理**：一个次级别走势类型向上离开中枢，然后以一个次级别走势类型回试，其低点不跌破 ZG，则构成第三类买点；向下离开后回抽高点不升破 ZD，则构成第三类卖点。

**操作要点**：
1. **必须第一次回调/回抽**才构成（第二次不算）
2. 第三类买卖点后**不必然是趋势**，也可能进入更大级别盘整
3. 一旦不能出现趋势，一定要在盘整高点出掉
4. 比第一、二类后知后觉，但抓得好不浪费盘整时间，适合短线技术好者

### 8.6 三类买卖点的操作策略

| 买点 | 风险 | 收益 | 适合资金 |
|------|------|------|----------|
| 第一类 | 逆势，需判断背驰 | 最高 | 大资金逆势重仓 |
| 第二类 | 中等 | 高 | 大资金温柔阻击 |
| 第三类 | 后知后觉 | 中高 | 短线技术好者、小资金 |

## 9. 区间套：精确转折点定位方法论

### 9.1 区间套原理

> **缠中说禅精确大转折点寻找程序定理**：某大级别的转折点，可以通过不同级别背驰段的逐级收缩范围而确定。即先找大级别背驰段，再在次级别图里找该背驰段的背驰段，反复进行直到最低级别，转折点就在该级别背驰段确定的范围内。

**数学类比**：高等数学的区间套定理。因级别非无限可分，达不到数学上唯一一点的精度，但可确认极精确的转折区间。

### 9.2 区间套操作步骤

```
大级别（如季线）发现背驰段
  ↓ 在次级别（月线）找该背驰段内的背驰段
    ↓ 在更次级别（周线）找背驰段
      ↓ … 一直递推到 1 分钟甚至每笔成交
        ↓ 转折点在最小级别背驰段范围内
```

### 9.3 四重背驰实例（第 61 课）

以 2007.6.20 大盘 72 点为例：
- 第一重：65-72 相对 55-60 的背驰
- 第二重：69-72 相对 65-66 的背驰
- 第三重：71-72 相对 69-70 的背驰
- 第四重：71-72 线段内部的背驰

→ 72 点的精确定位 = 65 开始背驰段的背驰段的背驰段的背驰段

### 9.4 区间套使用要点

1. **级别不宜过多**，复杂度越高越易出错。一般采用三级嵌套（如周线→日线→30 分钟）
2. 背驰段只是**假设**（假定离开第二中枢后的次级别走势为背驰段），可能被破坏使背驰不成立
3. 大资金用大级别（季线/月线）足够；小资金可精细到 1 分钟
4. 实际操作中，没证据否定背驰前，逐级观察内部背驰情况，**当下精确定位**

## 10. 转折力度与级别：背驰-转折定理

### 10.1 核心定理

> **缠中说禅背驰-转折定理**：某级别趋势的背驰将导致该趋势最后一个中枢的级别扩展、该级别更大级别的盘整或该级别以上级别的反趋势。

**背驰后的三种完全分类**（以下跌背驰为例）：

| 情况 | 反弹力度 | 含义 | 概率 |
|------|----------|------|------|
| ① 最后一个中枢级别扩展 | 最弱，反弹只触及最后中枢 DD=min(dn) | 未完成走势的延续 | 很少 |
| ② 更大级别盘整 | 反弹重新触及最后中枢 | "下跌+盘整"两走势类型连接，盘整级别 > 下跌级别 | 常见 |
| ③ 以上级别反趋势 | 反弹突破最后中枢 | "下跌+上涨"，上涨级别 ≥ 下跌级别 | 常见 |

### 10.2 三种情况的判别

**关键**：看某级别背驰后第一次的次级别回抽是否能进入前面趋势最后一个中枢的中枢区间。

- **不能进入** → 情况①（级别扩展）可能性大，反弹力度可疑
- **能进入甚至突破，且回调形成第三类买点** → 情况③（反趋势）可能性大
- **能进入但未突破** → 情况②（盘整）

### 10.3 小级别背驰引发大级别转折

> **缠中说禅小背驰-大转折定理**：小级别顶背驰引发大级别向下的**必要条件**，是该级别走势背驰段的最后一个次级别中枢出现第三类卖点；小级别底背驰引发大级别向上的**必要条件**，是最后一个次级别中枢出现第三类买点。

> 注意：只有必要条件，无充分条件。常发生在**趋势冲顶或赶底**时。

**全仓操作处理**（30 分钟级别 a+A+b+B+c，c 中 1 分钟背驰）：
- 5 分钟向下走势未跌破构成中枢 B 第三类买点的 5 分钟向上走势高点 → 不理睬
- 跌破该高点 → 任何向上回抽都先离开

**分仓操作**：c 段最后中枢出现第三类卖点先出一部分；之后跌破上述高点则在反弹中出清，否则回补（权当短差）

### 10.4 走势类型分解原则

> **缠中说禅走势类型分解原则**：一个某级别的走势类型中，不可能出现比该级别更大的中枢。一旦出现，证明这不是一个某级别的走势类型，而是更大级别走势类型的一部分或几个该级别走势类型的连接。

## 11. 走势分解：结合律、多义性与同级别分解

### 11.1 结合律（缠论最基础定律）

> **结合律必须贯穿所有层级**：K 线包含处理、笔、线段、中枢、走势类型都遵守。

**含义**：走势的分解可按不同方式组合，但组合必须满足结合律。同一走势有多种合法分解方式，**不改变走势本身**。

### 11.2 多义性

**多义性 ≠ 模糊性**：多义性是同一走势的多种合法分解视角，每种分解都正确，且可相互验证。

**多义性的应用**：
- 不同分解方式下，买卖点位置可能不同 → 选择最有利于操作的分解
- 利用多义性可重新组合走势，使当下走势类型的边界更清晰

### 11.3 同级别分解

**核心思想**：把走势严格按同一级别分解为走势类型的连接，拒绝跨级别混合。

**"下跌+上涨"买卖法（中小资金高效法）**：

六种基本走势组合（陷阱式/反转式/中继式）中，只参与唯一一种：**下跌+上涨**。

**标准买入程序**：
1. 只选出现"下跌+盘整+下跌"走势的股票
2. 在第二段下跌出现第一类买点时介入
3. 介入后一旦出现盘整走势，**坚决退出**（肯定盈利，用低一级别第一类卖点退出）

> 盘整浪费时间且一半概率下跌，中小资金没必要参与。

### 11.4 同级别分解的多重赋格

不同级别可同时进行同级别分解，形成多级别联立分析。这是区间套与两重表里关系的理论基础。

## 12. 两重表里关系：多级别联立诊断

### 12.1 笔的状态数组

任何当下，在任何时间周期 K 线图中，走势状态用两变量数组（方向, 位置）表示：

| 数组 | 含义 |
|------|------|
| (1, 1) | 向上笔延伸中 |
| (-1, 1) | 向下笔延伸中 |
| (1, 0) | 向上笔出现顶分型构造 |
| (-1, 0) | 向下笔出现底分型构造 |

**状态连接规则**（不可随意连接）：
- (1,1) → 只能接 (1,0)
- (-1,1) → 只能接 (-1,0)
- (1,0) → (1,1) 或 (-1,1)
- (-1,0) → (-1,1) 或 (1,1)

### 12.2 未病-欲病-已病（中医类比）

| 阶段 | 含义 | 对应买卖点 |
|------|------|-----------|
| 未病 | 小级别出现警告，但未破坏大级别结构 | — |
| 欲病 | 小级别警告确认，大级别状态开始改变 | 第一类买卖点 |
| 已病 | 大级别状态确认改变 | 第二、三类买卖点 |

**联立诊断示例**（5 分钟与 1 分钟）：
- 5 分钟 (1,1) + 1 分钟 (1,0) → 小警告（未病）
- 1 分钟 (1,0) 确认 → 欲病向已病发展
- 5 分钟 (1,0) 确认并向 (-1,1) 发展 → 已病

### 12.3 多级别矩阵监控

可构建 8 行矩阵（1 分钟/5 分钟/30 分钟/日/周/月/季/年），每行状态数组，共 4⁸ 种状态。

**实操简化**：只关心三个连续级别（如 1 分钟/5 分钟/30 分钟）→ 64 种状态。

> **多级别联立 = 联立方程**，解大幅减少，走势边界条件异常简单。看走势必须立体地看，否则浪费自相似性结构的有利条件。

## 13. 中枢震荡监视器与布林辅助

### 13.1 中枢震荡监视器

| 变量 | 含义 |
|------|------|
| Z | 中枢区间 [A,B] 的一半位置（震荡中轴） |
| Zn | 每个次级震荡区间的一半位置 |

**核心规律**：
- Zn 在 Z 之上 → 震荡偏强；Z 之下 → 偏弱
- **Zn 必然要超越 A 或 B**（否则永远不出现第三类买卖点，不可能）
- Zn 超越 A/B **不一定**构成第三类买卖点（可能多次超越，只有最后一次构成）
- Zn 超越 A/B 但未构成第三类买卖点 → 一般构成中枢震荡级别扩展（概率极高）
- Zn 数量不超过 9 个（超过则中枢升级）

### 13.2 操作要点

| 情景 | 处理 |
|------|------|
| Zn 在 Z 下甚至在 A 下 | 介入风险很大 |
| Zn 缓慢提高但无力突破 B | 小心突然变盘（上升楔型诱多） |
| 次级别是趋势 + Zn 配合 | 注意变盘可能 |
| 最后一个次级别中枢在中枢之外 + 下一次级别走势在该中枢区间内完成 | 震荡很可能变盘 |

### 13.3 布林通道辅助

布林通道对中枢震荡结束时间的辅助判断，与中枢震荡监视器配合可极大提高变盘预见性。

## 14. 分型的心理本质与辅助操作

### 14.1 分型的心理构成（顶分型为例）

| K 线 | 买方分力 | 卖方分力 |
|------|----------|----------|
| 第一根 | 努力上攻，高点被阻击回落 | 阻击 |
| 第二根 | 更高高点，但与第一根高点在小级别背驰 | 阻击制造上影线 |
| 第三根 | 再次攻击被完全击败，不能创新高 | 完全胜利（小级别第二类卖点） |

### 14.2 分型力度的判别

| 分型形态 | 力度 | 后续 |
|----------|------|------|
| 第一根长阳 + 二三根小阴小阳 | 弱 | 多为中继分型 |
| 第二根长上影/长阴 + 第三根不能收复第二根一半之上 | 强 | 延续成笔可能性极大 |
| 包含关系直接把阳线以长阴吃掉 | 最坏 | 强烈调整 |

### 14.3 分型辅助操作（日线以上）

**核心原则**：分型都是某小级别第一类/第二类买卖点成立后出现的。

**顶分型操作**：
1. 形成顶分型那天冲高时（不破前高）→ 看小级别卖点坚决出掉
2. 跌破 5 日均线后看小级别是否盘整背驰
3. 盘整背驰 → 回补；非盘整背驰 → 等待背驰才回补

**关键判别**：
- 顶分型后**有效跌破 5 日均线** → 多形成一笔（较大调整）
- 顶分型后**未有效跌破 5 日均线** → 多为中继分型（小级别盘整背驰回补点）

> 大级别分型与某小级别第一类/第二类买卖点**非绝对对应**：有分型必有后者，有后者不一定有前者。分型只是辅助。

## 15. 均线系统与板块强弱指标

### 15.1 均线缠绕分类（早期课程基础）

| 类型 | 特征 |
|------|------|
| 多头模式 | 短期均线长期运行在长期均线之上 |
| 空头模式 | 短期均线长期运行在长期均线之下 |
| 弱收敛 | 短期均线略走平后继续原趋势 |
| 强收敛 | 短期均线靠近长期均线但不跌破/升破，继续原趋势 |
| 交叉缠绕 | 短期均线跌破/升破长期均线，甚至反复缠绕 |

> 一切大的转折，基本都从**交叉缠绕**开始。均线系统本质是辅助工具，与中枢等概念完全不同，不要混在一起。

### 15.2 缠中说禅短差程序

> **大级别买点介入的，在次级别第一类卖点出现时先减仓，其后在次级别第一类买点出现时回补。**

对趋势操作者必要：任何下跌都由次级别开始，一个次级别下跌很可能引发该级别走势结束。

### 15.3 缠中说禅板块强弱指标

**计算方法**：
1. 设 n 条均线（如 5/13/21/34/55/89/144/233，斐波那契数列），按"本次反弹未攻克的最小周期均线"分类，n 条均线分 n+1 类
2. 计算某板块个股类别值的算术平均数 = 该板块强弱指标
3. 对所有板块执行
4. 比较所有板块指标值 → 强弱对比与轮动次序

**应用**：
- 最强板块 = 领涨板块，动态关键
- 各板块指标列图 → 轮动次序与节奏一目了然
- 一旦某趋势级别走势在 n 类上出现顶背驰 → 先出来，找调整到位或补涨股票

### 15.4 均线与级别的经验对应

- 一个趋势中，n 个中枢对应的压制均线大致相同
- 反弹只能达到 34 日均线 → 与第一中枢（被 89 日压制）同级别概率很小
- 均线系统设置必须与实际走势吻合（如 6124 点下跌用 5/13/21/34/55/89/144/233 极端吻合）

## 16. 操作心法与钢铁战士标准

### 16.1 持股与持币：两种最基本的操作

> 买入卖出 1 秒完成，**填充其间的持股与持币才是更重要的操作**。所有股票操作归根结底只有两个字：**等待**。

| 操作级别 | 买点后 | 卖点后 |
|----------|--------|--------|
| 30 分钟 | 持股，等 30 分钟卖点 | 持币，等下一个 30 分钟买点 |

**核心认知**：买卖点是**生长出来的**（走势类型生灭过程），不可预测，只可观察当下选择并反应。

### 16.2 钢铁战士七条标准

1. **买点在恐慌下跌中形成**，出现就义无反顾买进
2. **上涨中抵抗情绪干扰**，钢铁意志持住，决不中途下车
3. **卖点在疯狂上涨中形成**，出现就手起刀落
4. **失误只跌倒一次**，绝不在同一错误上犯两次
5. **买错比卖错严重**，确认买错就手起刀落；错过第二类买卖点买豆腐回家，错过第三类直接磨墙
6. **市场只有自己能救自己**
7. **踏准节奏**，刀山火海中逍遥游

### 16.3 预测毫无意义（当下理论）

- 走势是市场合力结果，不是上帝事先确定
- 大资金/高技巧可引导市场，但不能控制所有细节
- 唯一可做：**观察市场当下走势，让市场选择结构与类型，根据选择操作**
- 技术分析的最终意义不是预测市场要干什么，而是**市场正在干什么**

### 16.4 远离聪明，机械操作

> 股市死掉的大半是聪明人。机械化操作本质 = **目无全牛而合其关节**。

- 分解市场的标准不重要，关键能实现**完全分类**
- 挣钱 = 良好习惯 + 操作策略的结果，不用费力
- 用最简单的分型 + 能否延伸为笔，就足以处理震荡行情

### 16.5 防狼术与风险回避

| 风险类型 | 应对 |
|----------|------|
| 政策性风险 | 不可预测，只能防范；选长期价值个股、控制仓位、降低成本 |
| 资金期限风险 | 资金必须无期限；有期限则降操作级别 |
| 交易停止风险 | 唯一不能控制的死穴；不参与将停止交易的品种 |
| 市场波动风险 | 各级别买卖点操作，不断降成本至负 |

## 17. 资金管理与市场获利绝对分类

### 17.1 市场获利的绝对分类

> 任何市场获利机会只有两种：**中枢上移** 与 **中枢震荡**。

| 获利模式 | 含义 | 短差机会 |
|----------|------|----------|
| 中枢上移 | 该级别上涨走势 | 不存在理论短差机会（除非上移结束） |
| 中枢震荡 | 该级别盘整或上涨中新中枢形成延续 | 短差的理论天堂 |

### 17.2 0 成本：唯一安全状态

> 任何成本为正的股票都不信任，只有一个想法：尽快搞成负数。

**操作方式**：
- 持仓量始终不变（一开始买够，不加仓）
- 卖点出现变少，买点出现恢复原数量
- 任何卖点都是卖点，任何买点都是买点，唯一控制的是**量**
- 1 分钟卖点也参与（可能只卖 5 万股），差价再小也降低成本

### 17.3 大资金操作模式

| 模式 | 说明 |
|------|------|
| 第二类买点阻击 | 温柔介入，利用之后中枢不断降成本 |
| 第三类买点阻击 | 硬来，需顶住庄家打压，要求资金实力与功力 |
| 留机动部队 | 一定比例资金/稳健股票随时变现增援 |

### 17.4 小资金快速成长路径

> **一笔足够长的钱 + 缠论熟练运用 = 战无不胜。**

- 全仓进出，游走于不同凭证之间
- 只选波动大的股票
- 利用小级别快速积累（资金量大后小级别无意义）

### 17.5 资金量与操作方式分类（第 79 课）

| 资金类型 | 操作方式 |
|----------|----------|
| 无时间散户 | 60% 指数基金定投 + 40% 成长股基金定投 |
| 有时间散户 | 第三类买点买卖法（见下） |
| 大散户 | 对长期价值股抽骨吸髓式中枢震荡降成本 |
| 猎手 | 对选定目标猎杀式攻击（吃大鱼） |
| 战役级 | 对板块围歼式攻击 |
| 全局战争 | 立体金融战争，超出市场本身 |

**第三类买点买卖法（散户标准程序）**：
1. 选定反应级别（30 分钟/5 分钟/日线）
2. 只介入该级别出现第三类买点的股票
3. 买入后新次级别向上不能创新高或盘整背驰 → 坚决卖掉
4. 未出现则持有到上移背驰，至少卖一半
5. 尽量只介入**第一个中枢**的第三类买点（第二个中枢后大级别中枢概率剧增）
6. 不对任何股票有感情

## 18. 核心检查清单

### 18.1 走势分析清单

- [ ] 当前走势在哪个级别？方向（上涨/下跌/盘整）？
- [ ] 是否已形成中枢？中枢区间 [ZD, ZG] 在哪？
- [ ] 当前是中枢延伸/扩张/新生？
- [ ] 是否进入背驰段？背驰段是否被破坏？
- [ ] 当前位置对应哪类买卖点？是否已确认？
- [ ] 多级别联立：大级别状态如何？小级别是否出现警告？
- [ ] 区间套定位：转折点在哪个范围内？

### 18.2 操作执行清单

- [ ] 操作级别已选定？资金期限是否匹配？
- [ ] MACD 黄白线是否在 0 轴上（防狼术）？
- [ ] 买卖点是否 100% 确认（不被预测左右）？
- [ ] 仓位是否合理（机动部队是否留存）？
- [ ] 买入后持股/卖出后持币的等待是否到位？
- [ ] 失误是否总结（不在同一错误上犯两次）？
- [ ] 成本是否在降低（向 0 成本乃至负成本推进）？

### 18.3 误区警示清单

- [ ] 是否在预测市场？（应改为当下观察）
- [ ] 是否根据盈亏进出？（应改为根据走势是否在可操作范围）
- [ ] 是否一根筋思维？（应大胆假设、小心求证）
- [ ] 是否混淆级别？（图 ≠ 级别，显微镜 ≠ 观察物）
- [ ] 是否在盘整中找背驰？（盘整只有盘整背驰）
- [ ] 是否参与将停止交易的品种？（理论前提失效）
- [ ] 是否资金有期限？（最常见失败原因）
