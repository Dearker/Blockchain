## 1. 总体框架：超短线交易体系

超短线交易的本质是：**在一幅70笔交易走势图上，仅用价格与一条20期EMA，识别7种结构形态，在概率优势下以固定10点目标/10点止损反复"刮头皮"，以复利做大账户。**

| 维度 | 业余做法（失败） | 专业做法（沃尔曼） |
|------|------------------|---------------------|
| 分析工具 | 指标、趋势线、回归带、震荡指标 | 仅价格 + 1条20期EMA |
| 时间框架 | 1分钟图/5分钟图 | 70笔交易走势图（按交易笔数绘制） |
| 交易目标 | 抓大趋势、可调整目标 | 固定10点利润 |
| 风险控制 | 随意止损、情绪化 | 固定10点止损 + 临界点移动 |
| 入场方式 | 限价单、犹豫、追涨杀跌 | 市价单、一键式、突破即入 |
| 心态 | 证明每笔交易正确 | 概率思维，单笔结果无关紧要 |
| 账户管理 | 固定手数、取出利润 | 复利计算、2%风险模型 |

**四大核心原则**（贯穿全书）：

| 原则 | 内容 | 作用 |
|------|------|------|
| **概率游戏** | 交易不是赢或输的游戏，是概率游戏；单笔结果无关紧要 | 消除恐惧贪婪，坚持执行 |
| **双重压力** | 逆势者平仓 + 顺势者入场 = 同方向双重订单压力 | 交易优势的来源 |
| **结构至上** | 7种结构是入场工具，但首先要评估整体价格行为 | 不盲目交易形态 |
| **复利增长** | 每日根据账户余额调整交易量，利润立即投入 | 小账户爆炸式增长 |

**关键认知**：
- 超短线交易是**职业**，不是娱乐，需要商人式的结构化操作
- 交易者**不能依靠指标**，算法代码比不上良好的观察
- 市场不是可被打败的对象，只能打败技术不如自己的其他交易者
- 交易者要**先经历自我毁灭再重塑**，这是行业不可分割的一部分
- 超短线交易**不一定适合每个人**，但任何决心训练的人都可以做到

## 2. 交易基础设施

### 2.1 交易标的：欧元/美元

| 特性 | 说明 |
|------|------|
| 高度重复的日内特征 | 适合反复"刮头皮" |
| 较低交易点差 | 行业标准约1个点 |
| 资金门槛低 | 小型交易者可参与 |
| 普适性 | 价格行为原理可迁移至相似波动率、低成本市场 |

### 2.2 走势图：70笔交易走势图

| 维度 | 说明 |
|------|------|
| 绘制方式 | 按交易笔数（非时间）绘制，每70笔形成一根蜡烛线 |
| 优势 | 趋势期与震荡期均可服务交易者；消息发布时仍能呈现市场兴衰 |
| 对比 | 1分钟图在快速波动时只呈现一根巨大蜡烛线，70笔图更精细 |
| 蜡烛单位 | 所有蜡烛线波动单位为完整1个点（不接受小数点） |
| 独立性 | 强烈建议使用独立图表软件，与交易平台分离监控 |

### 2.3 订单类型与执行

| 订单类型 | 用途 | 说明 |
|----------|------|------|
| **市价单** | 入场、手动出场 | 唯一采用的入场订单，瞬间执行 |
| 限价单 | 止盈 | 设于入场价±10点 |
| 止损单 | 止损 | 设于入场价∓10点，像市价单执行 |
| **包围单（OCO）** | 自动管理 | 止盈/止损二择一，触发一个自动取消另一个 |

**执行原则**：
- 入场只用手动市价单（一键式），无限价单
- 包围单可预设，让市场决定胜负
- 更好方式：入场后紧密追踪价格，用临界点技术主动出场
- 平台需支持"always-on-top"窗口，一个显示屏即可
- 隐藏账户盈亏信息，避免贪婪恐惧干扰决策

### 2.4 固定参数（刚性，不可随意更改）

| 参数 | 设定 | 理由 |
|------|------|------|
| 盈利目标 | **10个点** | 抵消1点差，符合70笔图典型波幅，固定以消除恐惧贪婪 |
| 止损幅度 | **10个点** | 与目标对称，可在目标方向调整（移动止损） |
| 点差假设 | **1个点** | 行业标准，竞争将使其进一步下降 |

## 3. 概率原理与交易心理

### 3.1 概率思维的核心

| 维度 | 误解 | 正解 |
|------|------|------|
| 交易本质 | 赢或输的游戏 | 概率游戏，单笔结果无关紧要 |
| 目标 | 每笔盈利 | 做需要做的事，交给"优势"完成 |
| 亏损 | 垂头丧气 | 经营成本，随机分布的正常变化 |
| 盈利 | 兴高采烈 | 同样只是数据 |
| 预测 | 力图预测市场方向 | 观察者而非参与者，保持中立 |

### 3.2 "优势"的本质

| 维度 | 说明 |
|------|------|
| 定义 | 一个小小的、可提高成功概率的技术优势 |
| 获得方式 | 热血、汗水、眼泪 + 不计其数的研究时间 |
| 局限 | 不能一直打败市场，只能打败技术不如自己的交易者 |
| 验证 | 较长一段时间内分析成百上千次交易来评估 |
| 寻找位置 | 其他采用同样方式训练的交易者的局限处 |

### 3.3 胜率与盈亏比的数学关系

| 关系 | 数值 |
|------|------|
| 单笔盈利 | +10点 |
| 单笔亏损 | 约-6点（含点差，未达目标即出） |
| 盈亏平衡胜率 | **约40%**（10×40% = 6×60%） |
| 实际优势 | 若胜率>40%，长期即盈利；若>50%，盈利可观 |

**关键洞察**：即使胜率仅40%，只要盈亏比有利（10:6），长期仍盈利。这允许承受较高点差仍能盈利。

### 3.4 交易心理的三大陷阱

| 陷阱 | 表现 | 后果 |
|------|------|------|
| **方向偏见** | 被止损后埋怨市场"犯错"，坚持原方向 | 错过反方向交易机会 |
| **证明自己** | 必须证明每笔交易正确 | 情绪化、不按概率思维 |
| **报复交易** | 亏损后立即反手或加仓 | 打爆账户 |

### 3.5 心态重塑路径

| 阶段 | 内容 |
|------|------|
| ① 自我毁灭 | 探访绝望与沮丧的心灵深处 |
| ② 怀疑一切 | 深深怀疑所知的一切 |
| ③ 重塑自我 | 以完全不同的角度看待交易 |
| ④ 概率思维 | 完全接受风险、亏损及任何市场结果 |
| ⑤ 持续执行 | 信任结构，不放过任何有效结构 |

## 4. 20期指数移动平均线（EMA）的角色

### 4.1 EMA的定义与作用

| 维度 | 说明 |
|------|------|
| 计算 | 指数式赋予最近收盘价更高权重，标绘最近20根蜡烛线平均收盘价 |
| 替代选择 | 15~25期任何移动均线（指数或简单）均可，影响不大 |
| **核心定位** | **指导，不是准则**；视觉辅助，非支撑/阻力 |
| 作用 | 指示单边市vs横盘整理；预测价格方向转变；将价格"推出"整理形态 |

### 4.2 EMA与市场状态

| 市场状态 | EMA表现 | 交易选择 |
|----------|---------|----------|
| 上升趋势 | EMA上升，大部分蜡烛线在其上方 | 做多，结构最好在EMA附近 |
| 下跌趋势 | EMA下跌，大部分蜡烛线在其下方 | 做空，结构最好在EMA附近 |
| 价格筑顶 | 价格无法进一步上涨 | 考虑做空，留心做多 |
| 价格筑底 | 价格无法创出新低 | 考虑做多，留心做空 |
| 横盘整理 | EMA水平，蜡烛线上下交替 | 需更多甄别，等待区间突破类结构 |

### 4.3 EMA周期选择

| 周期 | 特点 | 适用 |
|------|------|------|
| 较快（≤15期） | 与价格更近，但易被个别蜡烛线击穿 | 趋势未变时频繁误判 |
| 20期（推荐） | 平衡灵敏度与稳定性 | 超短线交易标准 |
| 较慢（≥30期） | 更好指示趋势，但滞后/结构形态少 | 太慢，不适合超短线 |

### 4.4 EMA的局限

- EMA只是经过调整的20个收盘价移动平均线，**不提供支撑/阻力**
- 它指出价格逆势波动时倾向停留的动态水平
- 体现"走两步退一步"的市场特性
- 有强烈的自我实现预言倾向
- EMA可能滞后（急剧波动由少数长蜡烛线构成时跟不上）
- 有时必须忽略EMA，永远基于价格本身做决策

## 5. 双重压力原理（核心优势）

### 5.1 原理定义

> 当逆势交易者平仓（必须递出顺势订单）+ 顺势交易者入场 = **同方向双重订单压力**，价格在该方向上狂奔不止。

| 参与方 | 行为 | 方向 |
|--------|------|------|
| 逆势交易者 | 被迫止损/平仓 | 递出顺势订单 → 变成顺势者 |
| 顺势交易者 | 在回撤末端入场 | 顺势订单 |
| 场外观望者 | 看到突破加入 | 顺势订单 |
| **结果** | **三方同方向** | **价格狂奔** |

### 5.2 双重压力的两种情形

| 情形 | 描述 | 结果 |
|------|------|------|
| **顺势双重压力**（有利） | 回撤后顺势突破，逆势者止损+顺势者入场 | 趋势继续，交易获利 |
| **逆势双重压力**（不利） | 在大部分对手会平仓的价位逆势交易 | 窄幅止损被快速触及 |

### 5.3 双重压力的应用

- **核心任务**：找到失败一方最可能止损的价位 + 与市场主导者持类似观点的场外观望者最可能入场的价位
- 超过某价位1个点即可激发群体行动
- 超短线交易者需提前确定关键点位，在突破瞬间交易
- 若无法预期双重压力，**不要拿资本冒险**

### 5.4 逆势交易者的本质

| 问题 | 解答 |
|------|------|
| 逆势者为何对抗趋势？ | 趋势在较大时间框架可能只是回撤，视角不同 |
| 市场为何有流动性？ | 价格和价值上永恒的分歧提供流动性 |
| 逆势交易可行吗？ | 可以，但要像夜里的小偷一样敏捷谨慎 |
| 逆势交易目标 | 快速"刮走"10点，"预期寿命"很短 |

## 6. 七大交易结构总览

### 6.1 结构的本质

| 维度 | 说明 |
|------|------|
| 定义 | 指示入场的工具，结构本身几乎无意义 |
| 首要目标 | 评估市场未来方向的整体价格行为 |
| 入场前提 | 某方向市场力量超过另一方向时才考虑入场 |
| 命名目的 | 减少鲁莽行事概率，给形态命名以识别 |
| 适用性 | 有些适合单边市，有些适合横盘整理 |
| 频率 | 大部分结构日内出现多次，但交易条件不一定每次最佳 |

### 6.2 七大结构速查表

| 结构 | 简称 | 适用市场 | 核心特征 | 章节 |
|------|------|----------|----------|------|
| 双十字星突破 | DD | 顺势（回撤末端） | 2+十字星在EMA附近，信号蜡烛线被突破 | 第7章 |
| 首次突破 | FB | 顺势（新趋势首次回撤） | 强劲趋势+斜向回撤+首次回撤，回撤中首根蜡烛线被突破 | 第8章 |
| 二次突破 | SB | 顺势（"W"形态） | 放弃FB，等待回撤最低点测试后二次突破 | 第9章 |
| 箱体突破 | BB | 单边+震荡+顶部+底部 | 箱体内力量蓄积，信号线被突破 | 第10章 |
| 区间突破 | RB | 横盘整理/筑顶筑底 | 区间边线被突破（含"挤压"） | 第11章 |
| 区间内突破 | IRB | 区间内部 | 区间内小箱体突破，常含杯柄形态 | 第12章 |
| 高级区间突破 | ARB | 区间突破+回撤 | RB后回撤至边线，信号蜡烛线被突破 | 第13章 |

### 6.3 通用术语

| 术语 | 定义 |
|------|------|
| **信号蜡烛线** | 结构中在趋势方向上具有极值（最高点/最低点）的蜡烛线 |
| **入场蜡烛线** | 突破信号蜡烛线极值的那根蜡烛线，触发入场 |
| **突破确认** | 信号蜡烛线极值被另一根蜡烛线超过1个点 |
| **回撤** | 价格与趋势相反方向的波动，趋势"稍作休息" |
| **鱼钩形态** | 强劲趋势+有力回撤形成的走势图形态 |

## 7. DD结构：双十字星突破

### 7.1 结构定义

> 在强劲趋势回撤末尾，20期EMA附近出现2个或多个十字星，当信号蜡烛线极值被突破时入场。

### 7.2 结构要素

| 要素 | 要求 |
|------|------|
| 趋势 | 强劲趋势（可趋势中途或新趋势刚走出） |
| 回撤 | 价格回撤至20期EMA区域 |
| 十字星 | 2个或多个相邻十字星"靠"在EMA上（下影线最好穿过均线） |
| 信号蜡烛线 | 十字星群中在趋势方向上有极值的蜡烛线 |
| 突破 | 入场蜡烛线超过信号蜡烛线极值1个点 |
| 道路 | 入场后前方无阻力/支撑阻挡10点目标 |

### 7.3 十字星的灵活性

| 类型 | 特征 | 有效性 |
|------|------|--------|
| 严格十字星 | 开盘价=收盘价 | 有效 |
| 短小蜡烛线 | 长度≤3个点 | 同样有效，表达市场犹豫 |
| 评价标准 | 与整体趋势蜡烛线对比 | 越短小（相对趋势），犹豫情绪体现越好 |

### 7.4 回撤接近EMA的三种情况

| 情况 | 表现 | 可交易性 |
|------|------|----------|
| ① 十字星最高点不触及均线 | 价格未触及均线即反转 | 可交易，但可能过早入场，回撤未完成 |
| ② 十字星穿越均线后立即回落 | 穿越均线但快速收盘于下方 | **最佳交易方式** |
| ③ 十字星穿越均线但未快速回落 | 多根蜡烛线"停靠"在均线上 | 风险较高，可能趋势转变，等SB/BB结构 |

### 7.5 DD结构的质量评估

| 质量维度 | 优秀 | 较差 |
|----------|------|------|
| 十字星极值 | 两个十字星极值相等或差1点 | 极值相差2点以上 |
| 趋势强度 | 强劲趋势（蜡烛线长且激进） | 微弱趋势 |
| 回撤形态 | 斜向有序回撤 | 横向聚集阻碍道路 |
| 位置 | EMA附近 | 远离EMA |

### 7.6 放弃DD交易的情形

- 趋势微弱 + 极值相差超过1点 → 放弃
- 极值相差超过2点 → 确定放弃
- 回撤表现为横向聚集（阻碍未来道路） → 谨慎
- 前方阻力/支撑离入场点<10点 → 放弃
- 回撤中有较高的价格底部（多头力量增强信号）→ 谨慎

### 7.7 入场与止损

| 步骤 | 操作 |
|------|------|
| 识别 | 上升趋势确认 → 等待回撤至EMA |
| 信号 | 2+十字星靠在EMA上 |
| 入场 | 入场蜡烛线超过信号蜡烛线最高点1个点时，市价做多 |
| 止损 | 入场价下方10点 |
| 止盈 | 入场价上方10点 |

## 8. FB结构：首次突破

### 8.1 结构定义

> 强劲趋势后的首次回撤中，回撤里在原趋势方向上被突破的第一根蜡烛线即信号蜡烛线，突破时入场。

### 8.2 三个必要条件

| 条件 | 要求 | 理由 |
|------|------|------|
| ① 趋势 | 单一方向急升/暴跌，最好发生在横盘整理之后；蜡烛线长于整理期，连续快速绘制 | 大时间框架交易者参与，大量交易者陷在错误方向 |
| ② 回撤 | 斜向有序，回撤幅度达趋势的40%~60%；回撤中蜡烛线单一方向收盘 | 成熟回撤，非虚弱尝试 |
| ③ 首次 | 该趋势的第一次回撤 | 首次回撤最易被顺势者扳回 |

### 8.3 鱼钩形态

| 组成 | 说明 |
|------|------|
| 鱼钩柄 | 强劲的单方向趋势 |
| 鱼钩弯 | 有力的斜向回撤（40%~60%） |
| 入场点 | 回撤中首根被反向突破的蜡烛线 |

### 8.4 信号蜡烛线的要求

| 维度 | 要求 |
|------|------|
| 位置 | 回撤中，最好在20期EMA区域 |
| 长度 | ≤7个点（以容纳10点止损：7点蜡烛+2点突破+1点点差） |
| 颜色 | 无要求（十字星或反向蜡烛线更佳） |
| 突破 | 极值被另一根蜡烛线超过1个点 |

### 8.5 FB结构的特殊性

- **大多数情况下应放弃**（首次回撤要求严格）
- 急剧波动常由消息发布引发，蜡烛线绘制速度可达平常13倍
- 快速波动市场中，10点止损可能被快速触及
- FB本质也是DD结构的一种（若有2个相等极值十字星）

### 8.6 入场后管理

| 情况 | 行动 |
|------|------|
| 成交滑移价差1~2点 | 保持冷静，按原计划管理 |
| 入场后小幅回撤 | 保持冷静，技术条件许可时用临界点摆脱困境 |
| 错过入场 | 不追涨杀跌，等待回撤再造访突破价位 |
| 信号蜡烛线极度短小 | 止损可适当放宽（考虑技术点位） |

## 9. SB结构：二次突破

### 9.1 结构定义

> 放弃首次突破，等待回撤测试前一个最低点（"W"形态）后，二次突破信号蜡烛线时入场。

### 9.2 与FB的关系

| 维度 | FB | SB |
|------|----|----|
| 突破次数 | 首次回撤突破 | 二次回撤突破 |
| 信号蜡烛线 | 回撤中首根被突破的蜡烛线 | 二次回撤中首根被突破的蜡烛线 |
| 胜率 | 较低（大多放弃） | 较高（"W"形态确认） |
| 原理 | 趋势恢复 | 回撤最低点被测试并守住 |

### 9.3 "W"形态原理

| 要素 | 说明 |
|------|------|
| 第1个最低点 | 首次回撤的最低点 |
| 第2个最低点 | 二次回撤测试第1个最低点 |
| 守住 | 第2个最低点≥第1个最低点 |
| 意义 | 趋势很强，价格将不负众望奔向目标 |

### 9.4 SB结构的优势

| 优势 | 说明 |
|------|------|
| 胜率更高 | 基于"价格测试回撤最低点"的前提 |
| 风险更可控 | 信号蜡烛线明确，止损位清晰 |
| 放弃FB的合理性 | 首次突破可能蜘蛛犹豫，SB更可靠 |

### 9.5 放弃SB交易的情形

- 二次突破入场价高于首次入场价7点以上，且未看到价格测试前一最低点及EMA → 不能算高胜率交易
- 前方有重大阻力（如被跌破的支撑位变阻力位） → 谨慎

### 9.6 假突破与SB结构

| 概念 | 定义 | 意义 |
|------|------|------|
| 假突破 | 突破前期极值后立即失败回落 | 诱骗交易者像真突破一样交易 |
| SB中的假突破 | 信号蜡烛线原本看涨，最后变成看跌十字星 | 强烈震慑逆势者，加速假突破形成 |
| 危险性 | 假突破外形最具欺骗性 | 看法一致的交易者越多，逆势成功概率越高 |

### 9.7 持仓中出新结构的处理

| 选择 | 说明 |
|------|------|
| ① 放弃新交易 | **作者偏好**，保持简单，遵守每笔最大风险 |
| ② 分开管理两笔 | 统计合理但操作复杂，可能意外建仓 |
| ③ 调整当前头寸止损/目标 | 数秒内难正确调整，风险高 |
| 兑现当前再入场 | 不考虑点差/滑移价差时可行 |

**作者建议**：坚持原交易直到结束。理由：①保持简单；②遵守每笔最大风险原则；③同向两个头寸=分两次建一个仓位，风险翻倍。

## 10. BB结构：箱体突破

### 10.1 结构定义

> 价格在箱体内聚集蓄积力量，箱体信号线（上/下边线）被突破时入场。适用于所有市场（单边/震荡/顶部/底部）。

### 10.2 箱体的绘制

| 维度 | 说明 |
|------|------|
| 信号线 | 穿过连续相等蜡烛线最高点（上边线）或最低点（下边线） |
| 箱体 | 上边线与下边线之间的价格区域 |
| 突破 | 入场蜡烛线超过信号线1个点 |
| 灵活性 | 信号线可低于绝对极值1个点（根据技术逻辑） |

### 10.3 突破前压力（核心概念）

| 维度 | 说明 |
|------|------|
| 定义 | 突破前在信号线下方/上方聚集的小蜡烛线（常为十字星） |
| 本质 | 与当前市场力量对抗的交易者阻止突破的最后尝试 |
| 意义 | 窗户纸被捅破后，多空双方立即行动 |
| 识别 | 箱体内更高的价格底部+信号线下力量蓄积 |
| 重要性 | 训练眼睛发现这些微妙之处对交易非常有利 |

### 10.4 BB结构的质量评估（6理由法）

以图10.4做空为例，6个理由支持交易：
1. 市场放弃了坚持很久的支撑区域
2. 逆势者缺乏测试前期支撑位的热情
3. 水平回撤中绘制了较低的价格顶部
4. 出现突破前压力的小十字星
5. 坚固的信号线屈服于下跌压力
6. 下方整数位已失去对价格的魔力作用

### 10.5 箱体突破后的回测

| 现象 | 说明 |
|------|------|
| 回测突破价位 | 突破后几根蜡烛线内常再度触及信号线 |
| 信号线变支撑/阻力 | "支撑位跌穿变阻力，阻力位突破变支撑" |
| 聚集蜡烛线力量 | 突破后仍有支撑，因聚集蜡烛线形成支撑位 |

### 10.6 逆势BB交易

| 维度 | 说明 |
|------|------|
| 前提 | 评估市场各方力量利弊，天平偏向激进一方 |
| 理想条件 | 突破立即在关键支撑位被跌破之后 |
| 目标 | 快速"刮走"10点，逆势波动"预期寿命"短 |
| 风险 | 承担更高风险，需小心评估整体市场状况 |
| 入场依据 | 最近价格行为情绪，非整体市场状况 |

### 10.7 BB交易的两种典型错误

| 错误 | 表现 | 后果 |
|------|------|------|
| 无理由入场 | 内心狂躁，不行动就不适 | 越来越多失败交易吞噬利润 |
| 无聊离场 | 长时间盘整后突然离开电脑 | 错过大发展的交易机会 |

## 11. RB结构：区间突破

### 11.1 结构定义

> 横盘整理市场的区间边线被突破时入场。区间需有明确的上下边线（经多个相等极值确认）。

### 11.2 区间的绘制

| 步骤 | 操作 |
|------|------|
| ① 等待 | 至少形成几个价格顶部和价格底部 |
| ② 绘制 | 连接相等的价格顶部为上边线，相等的价格底部为下边线 |
| ③ 调整 | 随价格发展可能需调整几次 |
| ④ 确认 | 两个独立价格顶点确认上边线，两个独立价格底点确认下边线 |

### 11.3 "挤压"状态（核心概念）

| 维度 | 说明 |
|------|------|
| 定义 | 区间内蜡烛线高低点不断收窄，多空双方僵持 |
| 本质 | 暂时的力量均衡，迟早一方放开绳子 |
| 意义 | 经过"挤压"后的突破最可靠 |
| 识别 | 区间内最后一个较为扁平的拱形，拱顶与区间边线重叠 |

### 11.4 三种突破陷阱

| 陷阱 | 定义 | 特征 |
|------|------|------|
| **假突破** | 突破后立即回落，无力量蓄积 | 最危险，外形最具欺骗性 |
| **捉弄式突破** | 突破前有少许力量蓄积但仍失败 | 介于假突破与真突破之间 |
| **真空效应** | 突破后价格被快速吸入整数位"真空地带" | 缺乏对手盘，价格快速移动 |

### 11.5 假突破的分类

| 类型 | 定义 | 可交易性 |
|------|------|----------|
| 逆势假突破 | 与趋势对抗的突破失败 | 大玩家设置的陷阱，机警者可利用 |
| 顺势假突破 | 顺势突破失败 | 较难证明是假突破，真顺势浪潮随时可能涌入 |

### 11.6 突破前压力的识别

| 信号 | 含义 |
|------|------|
| 区间内逐步升高的价格底部 | 多头力量增强 |
| 区间内逐步降低的价格顶部 | 空头力量增强 |
| 信号线下小十字星聚集 | 突破前压力，力量蓄积 |
| 缺乏"跟进动作" | 突破失败可能性增高 |

### 11.7 偶然事件与显著事件

| 概念 | 定义 | 应用 |
|------|------|------|
| 显著事件 | 顺势突破（与趋势同向） | 顺势交易依据 |
| 偶然事件 | 逆势突破失败（假突破） | 逆势交易依据 |
| 原理 | 显著事件失败后，随后的不那么显著的事件更可信 | 逆势交易需更多技术支持 |

### 11.8 区间突破的交易原则

| 原则 | 说明 |
|------|------|
| 等待"挤压" | 经过充分力量蓄积的突破最可靠 |
| 不抢跑 | 不在绝对极值被突破前入场（即使只抢1点） |
| 接受所有结果 | 突破是真还是假无法预先知道 |
| 避免典型陷阱 | 识别假突破、捉弄式突破 |
| 整数位"真空" | 突破后价格可能被吸入整数位 |

## 12. IRB结构：区间内突破

### 12.1 结构定义

> 在区间内部形成的小型箱体被突破时入场。常包含杯柄形态，位于区间内部。

### 12.2 与BB结构的区别

| 维度 | BB | IRB |
|------|----|----|
| 位置 | 区间边线 | 区间内部 |
| 规模 | 较大箱体 | 较小箱体 |
| 意义 | 突破区间 | 区间内力量再蓄积 |

### 12.3 杯柄形态（经典IRB）

| 组成 | 说明 |
|------|------|
| 茶杯 | 大的"U"形价格形态 |
| 手柄 | 茶杯旁较小的"U"形 |
| 连接点 | 茶杯与手柄之间的高点 |
| 突破 | 连接点（高点）被突破时入场 |
| 多手柄 | 茶杯旁出现多个手柄，加强力量蓄积 |

### 12.4 IRB结构的优势

| 优势 | 说明 |
|------|------|
| 位置灵活 | 区间内任何位置都可形成 |
| 信号明确 | 小箱体信号线清晰 |
| 风险可控 | 手柄越短，止损越经济 |
| 真空效应 | 突破后价格被吸入最近区间边线"真空地带" |

### 12.5 IRB交易的评估

以图12.5为例，5个理由支持交易：
1. 确定无疑的"W"形态意味着多头力量持续
2. IRB结构底部在前期两个价格顶部找到技术支撑
3. 整数位被多头掌控并为结构提供支撑
4. IRB结构位置处于之前上升波段50%回撤位
5. 区间上边线可能存在真空效应，将价格快速吸上去

### 12.6 负面因素评估

| 因素 | 影响 |
|------|------|
| 双顶形态 | 可能吓退交易者，减少"跟进动作" |
| 入场蜡烛线过长 | 增加交易风险（手柄长→止损宽） |
| 前方阻力 | 阻力最少的盈利路线更佳 |

### 12.7 力量对比原则

> 支持交易的力量是否强大到足以对抗反对它的力量？回答越肯定，胜率越高。

| 评估 | 决策 |
|------|------|
| 支持力量>>反对力量 | 交易 |
| 支持力量≈反对力量 | 谨慎/放弃 |
| 支持力量<<反对力量 | 放弃 |

## 13. ARB结构：高级区间突破

### 13.1 结构定义

> 区间边线突破后，价格回撤至边线，在边线区域形成信号蜡烛线，信号蜡烛线极值被突破时入场。是RB结构的"高级"版本。

### 13.2 ARB的两种形式

| 形式 | 定义 | 特征 |
|------|------|------|
| **ARB直接突破** | 区间边线被突破时直接入场（类似RB） | 适合保守者 |
| **ARB回撤交易** | 突破后等待回撤至边线，信号蜡烛线被突破时入场 | **作者推荐**，胜率更高 |

### 13.3 拱顶测试（核心概念）

| 维度 | 说明 |
|------|------|
| 定义 | 区间内最后一个拱形的拱顶（区间内最后一个较为扁平的拱形顶部） |
| 位置 | 拱顶与区间边线重叠 |
| 意义 | 突破后回撤至边线时，拱顶价位提供支撑/阻力 |
| 测试 | 信号蜡烛线最高点成功测试最近一个拱顶 |
| 理想情况 | 信号蜡烛线最高点测试拱顶，最低点落在区间之外 |

### 13.4 回撤幅度的判断

| 回撤幅度 | 判断 |
|----------|------|
| ≥40% | 理想，可交易ARB回撤 |
| 20%~40% | 谨慎，回撤可能未结束 |
| <20% | 放弃，类似SB结构中放弃首次突破 |

**法则**：形态的右肩越扁平，发生典型回撤的概率越高；右肩越高耸，越需测量回撤与回撤前波段的相对关系。

### 13.5 ARB回撤交易的止损

| 维度 | 说明 |
|------|------|
| 标准止损 | 信号蜡烛线极值下方/上方1个点 |
| 拱顶测试 | 临界点可设在拱顶价位（更经济） |
| 额外保障 | 技术显著性存疑时，额外加1个点 |

### 13.6 ARB与SB的类比

| 维度 | SB | ARB回撤 |
|------|----|----|
| 突破次数 | 放弃首次，等二次 | 放弃直接突破，等回撤后突破 |
| 原理 | 回撤最低点测试 | 拱顶测试 |
| 胜率 | 高 | 高 |
| 适用 | 趋势市场 | 区间市场 |

### 13.7 三种ARB交易方式

以图13.9为例：
1. **普通RB**：将首次边线突破当做RB结构交易
2. **BB类型**：等待箱体信号线被突破
3. **ARB回撤**：等待信号蜡烛线在边线区域形成，交易其突破

## 14. 临界点技术（交易管理）

### 14.1 临界点的定义

> 临界点（Critical Point）是交易管理中的技术性止损位，允许以和入场相同的精度为交易出场择时。本质是**移动止损**。

| 维度 | 说明 |
|------|------|
| 本质 | 技术性止损位/移动止损 |
| 目的 | 以技术精度管理头寸，减少亏损、保护利润 |
| 依据 | 走势图上的关键技术点位 |
| 调整 | 随价格发展逐步移向更经济价位 |

### 14.2 临界点确定原则

| 原则 | 说明 |
|------|------|
| 技术显著性 | 临界点必须位于走势图关键技术点位 |
| 最经济价位 | 在技术有效前提下，尽可能靠近入场价 |
| 黄金法则 | 支撑位被测试前，止损设在最低点下方；测试后形成较高低点，新支撑位让位于旧支撑位 |
| 额外1点 | 技术显著性存疑时，额外加1个点保障 |

### 14.3 典型临界点位置

| 结构 | 最初临界点 |
|------|------------|
| DD | 结构最高点/最低点 |
| FB/SB | 信号蜡烛线另一端1个点外 |
| BB | 箱体信号线（或箱体绝对极值） |
| RB | "挤压"的十字星最低点 |
| IRB | 箱体信号线 |
| ARB | 最后一个拱形的拱顶 |

### 14.4 临界点移动规则

| 规则 | 说明 |
|------|------|
| ① 技术事件优先 | 新临界点在技术性事件发生后才生效（如支撑被测试并形成较高低点） |
| ② 突破确认 | 等蜡烛线跌破/突破前根蜡烛线极值后再移动 |
| ③ 不在EMA上方了结多头 | 黄金法则，但有例外（如双顶阻力） |
| ④ 盈亏平衡止损 | 大忌，会被立即止损出局 |
| ⑤ 区间边线保护 | 简单区间突破后，临界点留在区间内部 |

### 14.5 临界点移动示例

以图14.5为例（做多）：
1. **最初临界点**：SB结构最低点（约8点止损）
2. **首次回撤**：形成较高低点(5)，但不移动临界点（支撑未被测试）
3. **支撑测试后**：蜡烛线(7)突破蜡烛线(6)最高点，临界点移至(6)最低点
4. **二次测试**：蜡烛线(9)突破蜡烛线(8)最高点，临界点再提高
5. **止损出场**：临界点被跌破，止损1~2点出场(x)

### 14.6 激进vs保守的临界点

| 风格 | 特点 | 适用 |
|------|------|------|
| 激进 | 临界点紧跟价格，止损窄 | 波动快速市场（更可能假突破） |
| 保守 | 临界点留有余地，止损宽 | 波动缓慢市场（更尊重关键点位） |
| 选择依据 | 市场波动速度 + 交易者风险偏好 | — |

### 14.7 放弃交易的原则

> 如果适当的技术止损位太远，超过心理承受力，**宁愿放弃交易，也不要将止损设在较近位置**。

## 15. 不利市场条件与放弃原则

### 15.1 放弃交易的总原则

| 原则 | 说明 |
|------|------|
| 评估阻力/支撑 | 通往目标价格路上有无价位为对手提供极好机会 |
| 整数位 | "00"价位可能将价格拉回，"跟进动作"减少 |
| 双重压力 | 若双重压力不利于头寸，放弃 |
| 力量对比 | 支持力量不足以对抗反对力量，放弃 |
| 逆势交易 | 需更多技术支持，谨慎 |

### 15.2 典型不利市场条件

| 条件 | 表现 | 决策 |
|------|------|------|
| 前方有强大阻力 | 整数位/前期密集区/反转形态 | 放弃 |
| 逆势双重压力 | 在大部分对手会平仓的价位逆势交易 | 放弃 |
| 回撤无序 | 回撤中价格底部升高（多头力量增强） | 谨慎做空 |
| 缺乏"跟进动作" | 突破后无人跟随 | 突破失败概率高 |
| 100%回撤原理 | 单方向快速波动缺乏跟进，价格回撤至起始位置 | 谨慎 |

### 15.3 逆势交易的评估

| 评估维度 | 问题 |
|----------|------|
| 突破位置 | 突破是否立即在关键支撑/阻力被跌破之后？ |
| 道路通畅 | 通往10点目标的路上有无阻力？ |
| 力量蓄积 | 突破前有无适当力量蓄积？ |
| 偶然事件 | 是否有顺势假突破（偶然事件）支持逆势？ |
| 整体趋势 | 逆势交易"预期寿命"短，能否快速获利？ |

### 15.4 保守vs激进的抉择

| 维度 | 保守 | 激进 |
|------|------|------|
| 入场 | 等待完美结构 | 抓住任何有效结构 |
| 临界点 | 留有余地 | 紧跟价格 |
| 逆势交易 | 多放弃 | 抓住机会 |
| 长期收益 | 较低但心态平静 | 较高但风险大 |
| 适用 | 新手、心理脆弱者 | 老手、心理强大者 |

**作者建议**：不确定时选择保守。随着技术水平提高，可逐步激进。

## 16. 账户管理：复利与风险模型

### 16.1 2%风险模型

| 维度 | 说明 |
|------|------|
| 定义 | 每笔交易最大风险=总资金的2% |
| 计算 | 10点止损允许每点价值=总资金×2%÷10 |
| 标准 | 5000美元账户 → 每点10美元 → 1手标准合约（100000单位） |
| 杠杆要求 | 汇率<2.000时，40:1杠杆即可支持2%模型 |
| 灵活性 | 可选1%模型（保守）或逐步提高至2% |

### 16.2 复利计算公式

| 公式 | 说明 |
|------|------|
| 每日交易量 = 账户余额 × 风险系数 × 10 | 系数10 = 100 ÷ 止损点数 |
| 2%模型 | 账户余额 × 2 × 10 |
| 1%模型 | 账户余额 × 1 × 10 |
| 调整时机 | 每个交易日结束时计算，非交易期间 |

### 16.3 复利的威力（示例）

| 交易者 | 方法 | 200个交易日后 |
|--------|------|---------------|
| A | 10000美元，每日5点，**不复利** | 30000美元（+20000） |
| B | 10000美元，每日5点，**复利** | **73160美元**（+63160） |

### 16.4 复利的双重功能

| 功能 | 说明 |
|------|------|
| 增长 | 持续盈利时账户指数式增长 |
| 保护 | 亏损时自动降低交易量，防止超风险 |

### 16.5 账户管理的原则

| 原则 | 说明 |
|------|------|
| 2%上限 | 持续盈利者每笔风险≤2%，新手≤1% |
| 复利 | 利润立即投入下一轮交易，不取出 |
| 逐步提高 | 1%→1.1%→1.2%→...→2%，每周提升 |
| 最大成交量 | 在风险额度内选择最大交易量 |
| 不持双倍头寸 | 同向两个头寸=分两次建一个仓位，风险翻倍 |
| 日常费用分离 | 账户做大前不取利润，生活费另备 |

### 16.6 复利计算器的使用

| 变量 | 输入 |
|------|------|
| 当前本金 | 账户余额 |
| 天数 | 交易日数（视为年） |
| 利率 | 2%模型+每日5点 → 1%；每日10点 → 2% |
| 年增长数 | 0 |

**示例**：5000美元，200天，1%利率 → 36580.09美元

### 16.7 现实考量

| 考量 | 说明 |
|------|------|
| 均值假设 | 每日固定收益是均值，实际有波动 |
| 亏损交易量更大 | 盈利后提高交易量，次日亏损则亏损更大 |
| 心理压力 | 交易量随账户增长，需逐步适应 |
| 经纪商限制 | 账户做大后可能需更换经纪商 |
| 不切实际幻想 | 每日10点复利314天达百万美元，保持头脑清醒 |

## 17. 交易纪律与核心检查清单

### 17.1 核心纪律

| 纪律 | 说明 |
|------|------|
| 不用指标 | 仅价格+20期EMA，指标增加混乱 |
| 固定目标 | 10点利润，不可调整 |
| 固定止损 | 10点止损，可用临界点移动 |
| 市价单入场 | 无限价单，一键式执行 |
| 概率思维 | 单笔结果无关紧要，坚持执行 |
| 无方向偏见 | 观察者而非参与者，保持中立 |
| 不追涨杀跌 | 错过机会不追，等待回撤再造访 |
| 不报复交易 | 亏损后不立即反手 |
| 保守优先 | 不确定时选择保守 |
| 每笔最大风险 | 不超2%（或1%），不持双倍头寸 |
| 复利管理 | 每日根据账户调整交易量 |

### 17.2 看盘检查清单

- [ ] 当前多空力量谁占主导？
- [ ] 有无明确的区间/箱体？
- [ ] 有无"挤压"状态？

- [ ] 有无即将发布的重大消息？

- [ ] 信号蜡烛线是否明确？
- [ ] 入场蜡烛线是否突破信号蜡烛线极值1个点？

### 17.5 放弃交易检查清单

- [ ] 前方几个点内有强大阻力/支撑？
- [ ] 逆势双重压力是否对头寸不利？
- [ ] 回撤是否无序（如价格底部升高）？
- [ ] 突破前是否缺乏力量蓄积？
- [ ] 整数位是否可能将价格拉回？
- [ ] 100%回撤原理是否适用（缺乏跟进动作）？
- [ ] 技术止损位是否太远，超过心理承受力？
- [ ] 市场整体状况是否不支持交易？

### 17.6 心态检查清单

- [ ] 是否完全接受交易的风险？
- [ ] 是否以概率思维而非预测思维交易？
- [ ] 单笔亏损是否影响对方法/能力的信心？
- [ ] 是否信任结构，不放过任何有效结构？
- [ ] 是否避免证明每笔交易正确的冲动？
- [ ] 是否保持中立，无方向偏见？
- [ ] 情绪是否稳定，不受盈亏波动影响？
- [ ] 不确定时是否选择保守？

## 附录：核心术语速查表

| 术语 | 定义 |
|------|------|
| 超短线交易 | 在70笔交易走势图上反复"刮取"小额利润的交易方式 |
| 70笔交易走势图 | 按交易笔数（非时间）绘制的走势图，每70笔形成一根蜡烛线 |
| 20期EMA | 20期指数移动平均线，唯一的指标，指导非准则 |
| 包围单（OCO） | 止盈/止损二择一订单，触发一个自动取消另一个 |
| 双重压力 | 逆势者平仓+顺势者入场=同方向双重订单压力 |
| 信号蜡烛线 | 结构中在趋势方向上具有极值的蜡烛线 |
| 入场蜡烛线 | 突破信号蜡烛线极值的那根蜡烛线 |
| 突破前压力 | 突破前在信号线下聚集的小蜡烛线（常为十字星） |
| "挤压" | 区间内蜡烛线高低点收窄，多空僵持 |
| 拱顶 | 区间内最后一个较为扁平的拱形顶部 |
| 假突破 | 突破后立即失败回落，诱骗交易者 |
| 捉弄式突破 | 有少许力量蓄积但仍失败的突破 |
| 真空效应 | 突破后价格被快速吸入缺乏对手盘的区域 |
| 鱼钩形态 | 强劲趋势+有力回撤形成的走势图形态 |
| "W"形态 | 两次回撤测试同一最低点，趋势强劲的信号 |
| 杯柄形态 | 大"U"形（茶杯）+小"U"形（手柄），连接点被突破时入场 |
| 临界点 | 交易管理中的技术性移动止损位 |
| 2%模型 | 每笔交易最大风险=总资金2%的账户管理模型 |
| 复利系数 | 所有新收益投入运用，与本金共同作用于账户增长 |
| 偶然事件 | 逆势假突破，可给出更多逆势交易指示 |
| 显著事件 | 顺势突破，顺势交易依据 |

> **总结**：沃尔曼超短线交易方法的核心是**放弃一切指标**，仅用价格与20期EMA，在70笔交易走势图上识别7种高胜率结构（DD/FB/SB/BB/RB/IRB/ARB），以"双重压力原理"为优势来源，在10点固定目标/10点固定止损的刚性框架下，用临界点技术管理交易，以2%风险模型+复利管理账户，以概率思维对待每一笔交易。结构是工具，价格行为是根本，概率是思维，复利是增长引擎。熟练掌握1~2个结构并反复实战，不与任何指标组合，方能立足于超短线交易少数赚钱者之列。
