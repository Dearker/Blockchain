## 1. 总体框架：江恩方法论的底层逻辑

### 1.1 三大根本要素

江恩认为，研判市场只需研究三种因素，无需被消息和传闻干扰：

| 要素 | 作用 | 关键认知 |
|------|------|----------|
| **时间** | 最重要因素，可使价格失去平衡；时间到了，成交量放大迫使价格变动 | 时间的变化比价格的反转更重要 |
| **价格** | 通过百分比阻力位、顶底关系判断买卖点 | 顶变底、底变顶；50%是最重要阻力位 |
| **成交量** | 验证趋势真伪，研判反转时机 | 大成交量往往意味着头部；底部成交量萎缩 |

**核心原则**：让市场自我表白。不要有成见，不凭希望或恐惧买卖，研究三大因素并按规则得出的确定迹象交易。

### 1.2 市场运作的底层信念

| 信念 | 内涵 |
|------|------|
| **历史重演** | 人的本性不变，股市在某些条件下年复一年运行极为相似 |
| **自然法则** | 市场运动受自然法则支配，波动法则可测定价格涨跌的准确位置 |
| **趋势为王** | 只要时间周期指明上升趋势，什么也阻挡不了；指明下降趋势，什么也阻挡不了下跌 |
| **消息服从趋势** | 股票可以因利空上涨，因利多下跌；消息不能改变趋势 |

### 1.3 江恩规则体系的层级

```
心理管理（基础）→ 资金管理（保障）→ 交易规则（核心）→ 技术工具（辅助）
    ↓                ↓                ↓                ↓
 克服人性弱点     十分之一法则      24条+12条规则    百分比/时间周期/3日图
```

**关键认知**：江恩能给你最好的规则，但你仍会因"人的因素"——最大弱点——而输钱。没有遵守规则、凭希望和恐惧行事、犹豫迟疑、失去耐心、仓促行动、延误时机，这些才是失败的根源。

## 2. 亏损三大原因与盈利哲学

### 2.1 多数人输钱的三大原因

| 原因 | 说明 |
|------|------|
| **① 过度交易** | 相对于资本，买卖过于频繁 |
| **② 不设止损** | 没有下止损单限制损失 |
| **③ 知识不足** | 对市场知之甚少——最重要的原因 |

**深层分析**：多数人买股票只因希望上涨，按小道消息或他人想法购买，自己无任何具体知识。错误入市后不能认识错误或及时改正，最后因害怕再跌而卖在底部——犯了两个错误：错误时间入市 + 错误时间出市。其中一个本可避免：错误入市后及时脱身。

### 2.2 盈利哲学

| 原则 | 内容 |
|------|------|
| **目标优先级** | 不是获得多大利润或造成多大损失，而是与市场保持一致、顺应趋势；利润自会滚滚而来 |
| **稳健复利** | 每年获25%收益，1000美元10年变9313美元；不贪暴利，市场大多数时候正常运动 |
| **利润>损失** | 利润必须永远大于损失，规则是减小损失、累积利润 |
| **不值得的交易不做** | 认为不可能获得3-5点利润时不要买卖（除非止损仅1-2点）；不为3-5点利润冒3-5点风险 |
| **知识>金钱** | 最宝贵财富是时间，对时间最好的利用是获取知识，知识远比金钱可贵 |

### 2.3 交易者的三个目标排序

| 优先级 | 目标 | 说明 |
|--------|------|------|
| 第一 | **保护资本** | 资金安全是首要考虑 |
| 第二 | **限制损失** | 通过止损单自我保护 |
| 第三 | **获取利润** | 顺势而为，利润自然来 |

## 3. 二十四条常胜规则

江恩基于45年经验总结的24条规则，"任何人只要遵守就会获得成功"。

| 序号 | 规则 | 核心要点 |
|------|------|----------|
| 1 | **资金十分之一** | 将资金分成十等份，永不在一次交易中使用超过十分之一 |
| 2 | **必用止损单** | 永远在离成交价3-5点处设置止损单 |
| 3 | **不过度交易** | 这会破坏资金使用规则 |
| 4 | **不让盈利变损失** | 获得3点以上利润立即用止损单保护，不损失资本 |
| 5 | **不逆势而为** | 无法确定趋势时就不买卖 |
| 6 | **看不准就退出** | 看不准行情时不入市也不持仓 |
| 7 | **只交易活跃股** | 避免运动缓慢、成交稀少的股票 |
| 8 | **分散风险** | 交易4-5只股票，避免全部资金投一只 |
| 9 | **不限制委托** | 不固定买卖价格，用市价委托 |
| 10 | **无好理由不平仓** | 用止损单保护利润 |
| 11 | **累积盈余** | 成功交易后将部分资金划入盈余账户，备恐慌时使用 |
| 12 | **不为分红买股** | 永不为获得一次分红而买进 |
| 13 | **不平均分摊损失** | 这是交易者最糟糕的错误之一 |
| 14 | **不因耐心出/入市** | 不因失去耐心而出市，不因急不可耐而入市 |
| 15 | **避免赢小亏大** | — |
| 16 | **不撤销止损单** | 交易时不撤销已设置的止损单 |
| 17 | **避免频繁出入市** | — |
| 18 | **愿卖也愿买** | 目的与趋势一致并从中获利 |
| 19 | **不因价低买/价高卖** | 永不因股价低而买入，或因股价高而卖出 |
| 20 | **小心错误时加码** | 等股票活跃并冲破阻力位后加码买入；跌破派发区域后加码放空 |
| 21 | **小盘股做多/大盘股做空** | 挑选小盘股做多，大盘股放空 |
| 22 | **永不对冲** | 多头下跌时不卖另一只补仓，应离场认赔等机会 |
| 23 | **无好理由不变换多空** | 必须有好的理由或明确计划，未现明确转势迹象前不离场 |
| 24 | **盈利后不增加交易** | 避免在长期成功或赢利后增加交易 |

**使用方法**：每次交易前确认没有触犯任何一条；割肉时回顾违反了哪条，不再重蹈覆辙。

## 4. 十二条交易规则

### 规则1：研判趋势

- 研判道琼斯30种工业股平均指数等，在股票群中挑选个股，观察趋向指标是否与平均指数一致
- 使用**3日图**和**9点转向图**，应用所有决定正确买卖时机的规则

### 规则2：在单底、双底和三重底买入

| 形态 | 买卖 | 关键认知 |
|------|------|----------|
| 单底/双底/三重底 | 买入（接近前底、顶或阻力位） | 顶被突破后反抽，原卖点变买点 |
| 单顶/双顶/三重顶 | 卖出 | 顶被突破若干点后反弹达该位置=卖点 |
| **第四次到达同价位** | 不安全 | 几乎总要向上突破（顶）/破位下跌（底） |
| **三重顶/底** | 最安全买卖位置 | 市场离开时非常快 |

**双头双底的范围**：平均指数双头3-5点范围（多数1-2点）；个股2-3点范围。止损单设在双头/三重头之上1-3点，或双底/三重底之下1-3点。

### 规则3：按百分比买卖

| 百分比 | 重要性 |
|--------|--------|
| **50%（一半）** | 最重要阻力位 |
| **100%** | 第二重要，及成比例部分（200%、300%…） |
| **25%** | 第三重要 |
| **12.5%** | 第四重要（极限最高价和最低价间） |
| **6.25%** | 仅在极高价位交易时使用 |
| **33.3% / 66.7%** | 在25%和50%之后考虑 |

**应用**：从任何高位下跌50%买入，从任何低位反弹50%卖出（须在主要趋势中）。还可使用3%-5%、10%-12%、20%-25%、33%-37%、45%-50%、62%-67%、72%-78%、85%-87%。

### 规则4：按三周上涨或下跌买卖

| 时间周期 | 操作 |
|----------|------|
| 牛市3周调整 | 买入（大牛市平均调整时间） |
| 熊市3周反弹 | 卖出 |
| 上涨/下跌30天后 | 留心6-7周的买卖点 |
| 反弹/下跌45-49天以上 | 下一个周期约60-65天（熊市反弹/牛市调整最常见平均时间） |

### 规则5：市场分段运动

| 趋势 | 段数 | 认知 |
|------|------|------|
| 牛市 | 三至四段 | 刚走完第一段不要认为到头，真正牛市至少运行三段，很可能四段 |
| 熊市 | 三至四段 | 刚走完第一跌不要认为到底，熊市前会运行三段甚至四段 |

### 规则6：按5至7点运动买卖

| 运动幅度 | 含义 |
|----------|------|
| 个股调整5-7点 | 买卖位置（强势时调整5-7点，最多不超9-10点） |
| 10-12点反弹/下跌 | 重要关注位置 |
| 18-21点上涨/下跌 | 从重要头/底开始，常表示一轮行情结束 |

### 规则7：成交量结合时间周期

- 研究纽约证券交易所成交量及个股成交量
- 成交量有助于研判趋势何时反转（详见第11节）

### 规则8：时间周期

时间因素最重要——时间可使价格失去平衡，时间到了成交量放大迫使价格变动。

**转势日期**（季节性变化，详见第10节）+ **失去平衡的市场**：

| 失衡信号 | 含义 |
|----------|------|
| 下跌时间比前一次长 | 意味转势 |
| 下跌点数比前一次多许多 | 市场失去平衡，转势即将发生 |
| 反弹时间首次超过前一次 | 熊市中趋势正改变（至少暂时） |
| 反弹幅度首次超过前一次 | 空间运动失去平衡，转势已开始 |
| 末期涨幅减小+时间缩短 | 转势即将出现的信号（牛市末期） |
| 末期跌幅减小+时间缩短 | 时间循环即将结束（熊市末期） |

**核心**：时间的变化比价格的反转更加重要。

### 规则9：在高低点上移时买入

| 趋势信号 | 操作 |
|----------|------|
| 高低点依次上移 | 买入（主要趋势向上） |
| 高低点依次下移 | 卖出（主要趋势向下） |

注意以前头部至头部、底部至底部的时间长度，以及最低至最高、最高至最低的运动时间。

**图表选择**：运动缓慢波幅狭窄→月线图；股价活跃→周线图；高位运行→日线图；**3日转向图远比日线重要**。

### 规则10：牛市中趋势的变化

趋势反转常出现在节日附近：1月3日、5月30日、7月4日、9月初劳动日后、10月10-14日、选举年11月3-8日及11月25-30日、感恩节、12月24-28日。

**趋势变化的第一信号**：
- 价格突破9点转向图的前一个最低点，或3日转向图的前一个最低点 → 趋势改变
- 熊市中股价超过9点图/3日图的前一个反弹最高点 → 趋势变化第一信号

**重要时间周期检查**：市场是否离任何极限最高/最低价相距1、2、3、4或5年；离任何极限最低价是否是15、22、34、42、48或49个月。

### 规则11：最安全的买卖点

| 方向 | 最安全买卖点的形成过程 |
|------|------------------------|
| **最安全买点** | 股票筑底→反弹→次级调整→更高底部获支撑→开始上涨→冲破第一次反弹头部=最安全买点。止损单挂在第二个底下方 |
| **最安全卖点** | 长期上涨→最后高价→第一次快速垂直下跌→反弹形成较低的第二顶→从该顶下跌并跌穿第一次下跌的最低点=相对安全卖点 |

**2日回调与2日反弹**（活跃市场最重要时间周期）：
- 回调仅持续两天，第三天不继续跌 → 多头气势很强
- 下跌迅速的市场，反弹常陡直且仅持续两天

**铁律**：趋势向上时股价永远不会高得不能买；趋势向下时永远不会低得不能抛。但必须永远用止损单保护，永远顺势而为，强势中买股，弱势中卖股。

### 规则12：快速运动中的利润

| 运动速度 | 判断 |
|----------|------|
| 每日历日运动1点 | 市场非常活跃、上涨下跌迅速时的平均速度 |
| 每日历日运动2点以上 | 已远离正常轨道，不可能持续长久 |

牛市中这种快速运动出现在短期迅速回调中；熊市中出现在短期迅速反弹中。

## 5. 资金管理与风险控制

### 5.1 资本安全——十分之一法则

**唯一安全稳妥的规则**：将资金分成十等份，不在任何一次交易中拿超过资本的十分之一（10%）去冒险。

| 起步资金 | 单笔最大风险 | 心理优势 |
|----------|--------------|----------|
| 1000美元 | 100美元 | 即使有损失也不会心神不安 |
| 10股损失3点=30美元 | — | 比100股损失300美元好很多 |

**核心逻辑**：一开始就冒巨大风险会使资本处在危险中且有损判断力。只要有资本可供操作，总能发现新的获利机会。

### 5.2 保护盈利

保护盈利与保护资本同等重要——一旦获得盈利，不能让它化为乌有变成损失。

| 盈利幅度 | 止损单设置 |
|----------|------------|
| 朝有利方向运动3点 | 按当前价格设置止损单（即使已有利润） |
| 活跃高价股4-5点利润 | 止损单改在打平手的位置 |
| 利润持续增长 | 止损单步步跟随，保护并增加利润 |

### 5.3 期待多大的盈利

| 认知 | 内容 |
|------|------|
| 稳健复利 | 每年25%收益，10年1000→9313美元 |
| 不可翻倍念头 | 市场存在天赐良机可短期大赚，但不可臆想永远如此 |
| 正常市场 | 大多数时候正常运动，不可能获得超常利润 |
| 交易前提 | 认为不可能获3-5点利润时不交易（除非止损仅1-2点） |
| 不跑差价 | 跑差价的人赚不了钱，只能获蝇头小利 |

### 5.4 追加保证金的应对

**核心原则**：接到追加保证金通知时，大多数情况要做的不是存入更多保证金，而是按市价平仓。

| 错误做法 | 后果 |
|----------|------|
| 不断追加保证金 | 死捂股票至第二、第三个通知，在一笔交易中损失所有资本 |
| 存钱后判断力更差 | — |

**正确做法**：清仓离场，在判断力好时把资金用在有更好理由的新交易上。

### 5.5 联合账户的警示

- 尽量不与他人合伙交易——平仓时很难在时间和价格上取得一致
- 两人共同成功的唯一途径：一人负责买卖，另一人专门负责下止损单
- 不与配偶开联合账户，出入市应由一人负责

## 6. 止损单方法论

江恩将止损单视为"保护投资者和交易商的唯一安全阀"。

### 6.1 止损单的核心认知

| 认知 | 说明 |
|------|------|
| **十有八九正确** | 止损单十次中有八九次是正确的，能让你在与期望背道而驰时脱身 |
| **一次错误离场可接受** | 如果哪次止损单让你错误离场，下面九次它会发挥作用让你正确脱身 |
| **永不撤销** | 一旦设置止损单就不撤销——撤消止损单非基于正确判断而是因为希望，希望只会导致损失 |
| **第一件事** | 进行一笔交易后首先要做的就是设置止损单 |

### 6.2 止损单的位置设置

| 场景 | 止损单位置 |
|------|------------|
| 一般情况 | 离成交价3-5点 |
| 双头/三重头 | 比头部高1-3点 |
| 双底/三重底 | 比底部低1-3点 |
| 不知在哪设 | 不要交易 |

### 6.3 转变观念

| 智者 | 愚者 |
|------|------|
| 先调查再做决定 | 仅做决定 |
| 市场与判断背道而驰时更改止损单（不撤销） | 撤销止损单（基于希望而非判断） |
| 设止损单后不改变 → 十有八九是最佳结果 | 撤销止损单 → 损失 |

**铁律**：如果你不能遵循某种规则，就不要开始投机，因为你将血本无归。必须遵守而且永不偏离的规则之一：交易的同时设置止损单而且不撤销。

## 7. 百分比阻力位理论

江恩"最伟大的发现之一"——计算平均指数和个股最高价和最低价的百分比，以指出未来阻力位。

### 7.1 阻力位的重要性排序

| 排序 | 百分比 | 说明 |
|------|--------|------|
| 1 | **50%** | 最重要阻力位 |
| 2 | **100%** | 第二重要（及200%、300%、400%、500%、600%…） |
| 3 | **25%** | 第三重要 |
| 4 | **12.5%** | 第四重要（极限最高价和最低价间） |
| 5 | **6.25%** | 仅在极高价位交易时使用 |
| 6 | **33.3% / 66.7%** | 在25%和50%之后考虑 |

### 7.2 顶底关系原理

| 关系 | 应用 |
|------|------|
| 每个最低价→未来最高价 | 最低价的百分比指出下一最高价位（小风险卖出/做空位） |
| 每个最高价→未来最低价 | 最高价的百分比指出未来最低价位（小风险买入位） |
| 顶变底/底变顶 | 突破后阻力位变支撑位，支撑位变阻力位 |

### 7.3 50%关键位的应用

| 场景 | 判断 |
|------|------|
| 股票跌破极限最高价和最低价间的50% | 非常重要——若未获支撑站稳，走势很弱，会跌到75%或更低 |
| 股票跌破最高价的50% | 走势很弱——若要获支撑上涨，应在从最高点下跌50%时企稳 |
| 不要买跌破50%的股票 | 直到根据所有规则看到止跌迹象 |

### 7.4 中点（50%）的计算与应用

计算极限最高价和极限最低价间的中点，验证其对估计顶底的作用：

| 历史区间 | 中点 | 验证 |
|----------|------|------|
| 1896年最低28.50→1929年最高386.10 | 207.28 | 重要阻力位 |
| 1932年最低40.56→1946年最高213.36 | 126.96 | 重要阻力位 |
| 1942年最低92.69→1946年最高213.36 | 153.02 | 重要阻力位 |

### 7.5 突破确认规则

**市场必须在阻力位之上5点，或在老的底部/阻力位之下5点，才能算是有确切的转势迹象。**

| 突破情况 | 判断 |
|----------|------|
| 突破阻力位5点以上 | 确切转势迹象 |
| 未达5点 | 不能确认转势 |
| 整数关口（100、200、300点） | 总存在大量抛压和某种阻力 |

### 7.6 百分比验证实例

江恩用大量历史数据验证百分比理论，例如：
- 1929年最高386.10的50%=193.05 → 1929年11月最低195.35（在阻力位上方2点获支撑）
- 1932年最低40.56加上100%=81.12 → 1932年9月最高81.50（正好在50%位置做头）
- 1937年最高195.50减去50%=97.75 → 1938年3月最低97.50

**核心方法**：从每个重要的最高点和最低点计算百分比点和阻力位，预测将来下一个顶部或底部可能出现的位置。

## 8. 时间周期理论

时间因素和时间周期在研判趋势变化时最为重要。

### 8.1 重要转势时间周期

从重要顶部或底部开始，留心以下天数附近是否出现转势：

| 时间周期（天） | 说明 |
|----------------|------|
| 7-12天 | 短期 |
| 18-21天 | 短期 |
| 28-31天 | 约一个月 |
| 42-49天 | 约6-7周 |
| 57-65天 | 约2个月，熊市反弹/牛市调整最常见平均时间 |
| 85-92天 | 约3个月 |
| 112-120天 | 约4个月 |
| 150-157天 | 约5个月 |
| 175-185天 | 约6个月 |

**规则**：这些时间周期开始时的顶部和底部越重要，转势就越重要。

### 8.2 涨跌行情持续时间统计

江恩统计1912-1949年共292次涨跌行情：

| 持续时间 | 次数 | 比例 | 重要性 |
|----------|------|------|--------|
| 3-11天 | 41次 | 约1/7 | — |
| **11-21天** | 65次 | 约1/4.7 | 重要 |
| **22-35天** | 65次 | 约1/4.7 | 重要 |
| 11-35天合计 | 130次 | 1/3以上 | 留心极限最高/最低点的重要时段 |
| 36-45天 | 31次 | 约1/9.5 | — |
| 43-60天 | 33次 | 约1/9 | — |
| 61-95天 | 20次 | 约1/14.5 | — |
| 96-112天 | 13次 | 约1/22.5 | — |
| 超过112天 | 12次 | 约1/24 | — |

### 8.3 失去平衡的市场

| 失衡判断 | 应用 |
|----------|------|
| 上涨/下跌相当长时间后失去平衡 | 这段时间越长，调整/反弹力度越大 |
| 某次下跌时间比前一次长 | 意味转势 |
| 股价下跌点数比前一次多许多 | 市场失去平衡，转势即将发生 |
| 反弹时间首次超过前一次（熊市） | 趋势正改变（至少暂时） |
| 反弹幅度首次超过前一次（熊市） | 空间运动失去平衡，转势已开始 |
| 末期涨幅减小+时间缩短（牛市末期） | 转势即将出现的信号 |
| 末期跌幅减小+时间缩短（熊市末期） | 时间循环即将结束 |

**核心**：时间的变化比价格的反转更加重要。当反转出现时，应用所有规则验证转势是否必然发生。

### 8.4 时间周期的关键认知

- 市场突破头部或跌破底部的时间越长，今后行情就涨得越高或跌得越低
- 行情末期指数可创出新高的略低位置，然后止步不前——因为时间周期已到
- 永远记住市场上升中最长的调整时间；永远记住熊市中最长的反弹时间
- 遵循这些时间周期可帮助研判市场趋势

## 9. 3日图与9点转向图

### 9.1 3日图（3-Day Chart）

**定义**：记录3日或3日以上的市场运动，作为趋势指标。

**使用规则**：
| 信号 | 含义 |
|------|------|
| 指数跌破3日底 | 意味着会跌得更低 |
| 指数突破3日头部 | 意味着会继续上扬 |
| 上涨市场3日图突破上一最低点 | 小趋势已反转向下（至少） |
| 下跌市场3日图突破前一最高点 | 市场已暂时反转向上 |

### 9.2 绘制3日图的方法

1. 市场从一个最低点开始连续3天高低点上移 → 3日图到达第三天头部
2. 随后调整2天不必记录
3. 市场运动到第一个头部以上时 → 趋势线延长至每天头部，直至出现3天更低的底部
4. 画趋势线至第三天最低点，只要指数一直跌就画下去
5. 市场振荡剧烈时可记录2日运动

**关键**：必须考虑市场开始上扬时的最后一个最低点，以及开始下跌时的最后一个最高点——这些主要转折点非常重要。

### 9.3 9点转向图（9-Point Swing Chart）

**定义**：市场上涨时图也上涨，直至出现9点或以上调整；下跌时图也下跌，直至出现9点或以上反弹（在9点图上是反转）。

**市场运动幅度统计**（1912-1949年37年）：

| 运动幅度 | 次数 | 占比 | 重要性 |
|----------|------|------|--------|
| 9-21点 | 271次 | 约50% | **最重要**，大多数重要趋势运动 |
| 21-31点 | 61次 | 约1/4 | 重要 |
| 31-51点 | 36次 | 约1/8 | — |
| 大于51点 | 6次 | 极少 | 均出现在1929年最疯狂市场 |
| 小于9点 | 54次 | 次要地位 | — |

**应用规则**：
| 场景 | 判断 |
|------|------|
| 从低位反弹未超9点 | 市场处于弱势，指数还将走低 |
| 回调小于9点 | 市场处于强势，指数还将走高 |
| 大势反转后上涨10点以上 | 应能从低位上涨20点以上 |
| 熊市后跌幅越过10点 | 会下跌20点或以上 |
| 运动超过21点 | 下一个注意幅度是30-31点 |

**需要留心的运动幅度序列**：9-10点 → 18-20点 → 30点左右 → 45点左右 → 50-52点。

### 9.4 道琼斯平均指数作为趋势指标

| 认知 | 说明 |
|------|------|
| 工业股平均指数 | 反映大多数个股趋势，确定买卖阻力位有效 |
| 铁路股平均指数 | 慢慢绝迹，不再与工业股协调一致 |
| 公用事业股平均指数 | 处于强势，与工业股协调性更强 |
| 建议 | 跟上道琼斯30种工业股平均指数，追踪成分股趋势，盯有同样趋势的个股 |

**注意**：平均指数因考虑分红和拆股而失真，但仍可用于判断趋势。真正的平均指数是购买100股这30种股票的实际成本。

## 10. 季节性与周年纪念日

### 10.1 全年重要转势日期

股票市场指数和个股的趋势呈季节性变化，以下日期附近常出现转势：

| 月份 | 重要日期 | 重要性 |
|------|----------|--------|
| **1月** | 7-10日、19-24日 | 一年开始时最重要，持续数周至数月的趋势常在此变化 |
| **2月** | 3-10日、20-25日 | 重要性仅次于1月 |
| 3月 | 20-27日 | 小转势，有时也有重要顶底 |
| 4月 | 7-12日、20-25日 | 后半部分常相当重要 |
| **5月** | 3-10日、21-28日 | 与1、2月同等重要 |
| 6月 | 10-15日、21-27日 | 小转势，某些年份出现极限最高/最低点 |
| **7月** | 7-10日、21-27日 | 重要性仅次于1月（年中，派发红利、季节变化） |
| 8月 | 5-8日、14-20日 | 与2月同等重要 |
| **9月** | 3-10日、21-28日 | **一年中最重要的时期**，最高点最多出现在9月 |
| **10月** | 7-14日、21-30日 | 相当重要，重要转势常出现 |
| 11月 | 5-10日、20-30日 | 选举年转势常在月初，其他年份最低价常在20-30日 |
| **12月** | 3-10日、15-24日 | 后半部分至1月转势百分比非常高 |

### 10.2 极限最高点出现的月份统计

1881-1949年35轮行情中，各月出现极限最高点次数：

| 月份 | 次数 | 月份 | 次数 |
|------|------|------|------|
| 1月 | 4 | 7月 | 4 |
| 2月 | 4 | 8月 | **0** |
| 3月 | 4 | **9月** | **8（最多）** |
| 4月 | 4 | 10月 | 4 |
| 5月 | 2 | 11月 | 4 |
| 6月 | 4 | 12月 | 2 |

**结论**：牛市已运行很长时间后，9月是最需注意顶部的月份；8月从未有牛市见顶。

### 10.3 极限最低点出现的月份统计

36轮下跌行情中：

| 月份 | 次数 | 月份 | 次数 |
|------|------|------|------|
| 1月 | 1 | 7月 | **5** |
| 2月 | 3 | 8月 | 2 |
| 3月 | 2 | 9月 | 1 |
| 4月 | 4 | 10月 | 3 |
| 5月 | 3 | **11月** | **6（最多）** |
| 6月 | 4 | 12月 | 4 |

**结论**：股市下跌相当长时间后，可预计在7月或11月见底；其次是4月、6月、12月。

### 10.4 周年纪念日（Anniversary Date）

**核心发现**：股票会在它们已经到达过最高价和最低价的月份发生重要的转势。

| 重要日期 | 事件 |
|----------|------|
| 1929年9月3日 | 历史最高点 |
| 1932年7月8日 | 自1897年来最低位 |

**每年留心这些周年纪念日**，以及其他创下极限最高/最低点的日子（如1937年3月8日、1938年3月31日、1942年4月28日、1946年5月29日等）。

### 10.5 重大消息与市场走势

**核心原则**：重大消息出现时，要考虑消息显现时平均指数和个股的走势，而非消息本身。

| 消息 | 市场实际走势验证 |
|------|------------------|
| 1914年7月战争爆发 | 12月24日才到最低53.17点 |
| 1939年9月1日二战爆发 | 最低127.51→9月13日最高157.75（上涨30点） |
| 1945年8月14日日本投降 | 指数继续上扬 |

**关键认知**：为什么平均指数在几乎相同的位置多次出现最高点和最低点？因为在这些价值附近存在某种百分比点。要从每个重要的最高点和最低点计算百分比点和阻力位。

## 11. 成交量分析

### 11.1 成交量的核心原则

| 原则 | 内容 |
|------|------|
| 大成交量=头部 | 大家在市场上涨时买入过度，内幕人士正在抛售 |
| 底部成交量萎缩 | 抛压已结束，市场处于底部区域 |
| 成交量验证趋势 | 研究月成交量和周成交量，结合其他所有规则 |

### 11.2 成交量与牛熊市的对应

| 市场阶段 | 成交量特征 | 含义 |
|----------|------------|------|
| 牛市末期 | 成交量巨放 | 行情的终点信号 |
| 熊市末期 | 成交量持续萎缩至极低 | 套现压力消失，底部区域 |
| 牛市启动 | 成交量开始放大 | 买盘少量介入 |
| 上升途中 | 成交量配合放大 | 趋势健康 |
| 顶部区域 | 月成交量创阶段新高 | 头部信号，卖点 |

### 11.3 成交量分析的实例规律

| 时期 | 成交量 | 指数 | 判断 |
|------|--------|------|------|
| 1936年 | 4.96亿股（全年） | 牛市 | 1-2月成交量最大 |
| 1937年3月后 | 逐步萎缩至8月1721万股 | 下跌 | — |
| 1937年10月 | 放大至5100万股以上 | 大突破 | — |
| 1942年2-4月 | 约800万股或更低 | 底部 | 抛压结束 |
| 1946年1月 | 5151万股 | 顶部将至 | 自1937年3月来最大月成交量 |
| 1949年6月 | 1776万股 | 底部 | 套现盘消耗殆尽，买点 |

### 11.4 成交量应用要点

- 牛市最后一年成交量巨放 → 行情终点
- 市场到达以往卖压区时巨大成交量 → 正在做头
- 底部区域成交量极度萎缩 → 买点
- 研究月成交量和周成交量，与其他所有规则结合

## 12. 个股选择与独立行情

### 12.1 独立行情的识别

许多股票在其他股票创新低时创新高，走出独立于平均指数的行情。通过研究几年来走势图，可在独立行情开始时就辨认出来。

**识别要点**：
| 特征 | 含义 |
|------|------|
| 股票在某价格区间运行数年 | 抛压已很小，只有内部人士愿买进 |
| 价格区间极窄（如仅1.5美元） | 即将突破的蓄势 |
| 突破数年前的高点（如8年后突破） | 会出现更高价格，应立即买入 |
| 高低点继续抬高 | 主要趋势向上无疑 |

### 12.2 买入一只做空另一只

利用股票的"相反倾向"——一些股票上涨创新高，同时另一些下跌创新低：

| 操作 | 条件 | 止损 |
|------|------|------|
| 高位卖空一只 | 高点不断下移，趋势向下 | 设在前期高点上方 |
| 低位买入另一只 | 低价区获良好支撑 | 设在支撑位下方 |
| 两只股票价格走到一起 | 两头赚钱 | — |

**优势**：如果两个止损单都成交，总损失有限（如400美元）；但如果方向正确，可两头获利。

### 12.3 小盘股 vs 大盘股

| 类型 | 特征 | 操作 |
|------|------|------|
| **小盘股** | 支配供需的资金量少，供给减少时不需要太大购买力即可推升 | 牛市中涨幅惊人，远超大盘股；**做多首选** |
| **大盘股** | 运动缓慢，需大量资金推动 | **做空首选** |

### 12.4 阻力位与个股位置

当平均指数到达重要阻力位、涨至以前顶部、或跌至以前底部时，应研究你买卖的个股的位置，将所有规则应用在这些个股上。

## 13. 加码与平仓时机

### 13.1 加码的两种方法

| 方法 | 内容 |
|------|------|
| **突破加码法** | 市场突破进入新价格区域、创新高/新低时立即加码 |
| **调整加码法** | 判断调整位置，从短期顶部调整了多少点/从短期底部反弹了多少点 |

**调整加码法的具体规则**：
- 找出调整是否已运动3、5、7、10或12个点
- 在从顶部开始的调整的第一、第二、第三或第四次加码
- 根据过去调整幅度等待3、5或10个点
- 熊市中规则反过来适用

### 13.2 时间规则辅助加码

**江恩的时间规则**：测定第一次重要调整的时间。

| 应用 | 说明 |
|------|------|
| 通用汽车例 | 从1924年上涨时仅调整3周，所以每次从任何顶部调整2-3周时买入较可靠，直至最终顶部 |
| 测定调整时间并计算 | 可大幅提高利润，紧跟主要趋势数年，获100-200点利润 |
| 适用范围 | 对成交活跃、价格高起的股票尤其有效，应用于活跃市场 |

### 13.3 加码与止损单的关系

| 加码阶段 | 止损单位置 |
|----------|------------|
| 加码早期 | 止损单紧贴市价，保护初始资本 |
| 已有100点利润 | 止损单可远离市价20点（因以前曾回调20点） |
| 利润越多 | 可承受的波动空间越大，止损单可越远 |

**核心**：无论何种加码方法，应永远设置止损单保护利润。

### 13.4 何时了结获利

- 遵循规则，不要在明确的转势出现前平仓
- 看出趋势发生变化时毫不迟疑地出货
- 不要买卖太早或太迟：
  - **太早离场**：股价第一次上升至新空间时匆忙卖出（错误）
  - **太迟离场**：捂股希望创新高，结果跌后再"希望"反弹至前高，最终大跌后才清仓（错误）

### 13.5 延误的危险

| 认知 | 内容 |
|------|------|
| 行动产生利润 | 延误不能产生利润，希望毫无用处 |
| 停止希望，开始思考 | 凭希望赌博的人总要破产 |
| 延误越久判断越糟 | 犯错误可能性越大 |
| 反应迟钝=死亡 | 行动才是生命 |
| 保持身心健康 | 身心不适时判断总会出错，健康就是财富 |

## 14. 心理管理：人性弱点

### 14.1 人的因素——最大的弱点

**江恩的核心警示**：我能给你世上最好的规则，但你还是会因为"人的因素"——最大弱点——而输钱。

| 弱点表现 | 后果 |
|----------|------|
| 没能遵守规则 | — |
| 凭希望和恐惧行事 | 而非凭事实 |
| 犹豫迟疑 | 错失时机 |
| 失去耐心 | 错误时间出/入市 |
| 仓促行动 | — |
| 延误时机 | — |

### 14.2 赚钱与赔钱时的态度差异

| 情境 | 态度 | 问题 |
|------|------|------|
| 赚钱时 | 褒奖自己，认为判断正确全凭本事 | 自大，开始盲目承担风险 |
| 赔钱时 | 很少责怪自己，找借口 | "如果没听信别人建议本该赚钱" |
| 找借口 | "如果"、"而且"、"但是" | 认为不是自己的过错 |

**自救原则**：投资者必须设法自救，从自己而非他人身上寻找损失原因。是你自己的行动造成了你的损失，必须寻找内在原因并加以改正。

### 14.3 交易者不愿知道什么

| 普通投资者的心态 | 应有心态 |
|------------------|----------|
| 不愿听到痛苦的事实 | 只有事实才能帮助 |
| 需要迎合希望的东西 | 必须相信事实 |
| 买入后相信所有利好消息 | 也要看不利消息 |
| 犯错后说"下次不再" | 但仍重蹈覆辙 |

**华尔街真相**：人们很少谈论损失这一真正的内幕。交易者无论大小，总把利润挂在嘴上，夸大成功交易，绝口不谈损失。

### 14.4 独立思考

| 错误 | 正确 |
|------|------|
| 从不自己思考，让别人替自己思考 | 自己进行调查研究 |
| 听从并不比自己高明的人的建议 | 从"羊羔"变成思想者，渴求知识 |
| 盲目跟随他人 | 即使他人正确，不知根据也无信心 |

**只有你帮助自己的时候，别人才能帮助你。**

### 14.5 转变观念

| 智者 | 愚者 |
|------|------|
| 先调查然后做决定 | 仅做决定 |
| 会转变观念 | 不会转变观念 |
| 在华尔街不转变观念的人很快头脑僵化 | — |

**注意**：一旦你决定进行一笔交易且有确定的理由，就不要无缘无故地改变。所指的"转变观念"是：在市场与判断背道而驰时**更改或取消止损单**——这是愚蠢的。

## 15. 大炒家的教训

江恩回顾历史上聚敛成百上千万美元后又输光的大炒家，提炼教训。

### 15.1 大炒家失败的共同原因

| 炒家 | 失败原因 |
|------|----------|
| 莱特（Lighthouse，囤积小麦） | 不相信大量小麦会涌入压低价格；未知数导致败落 |
| 萨利（Sully，棉花） | 认为自己有很大能耐，开始拉抬价格 |
| 普莱斯（Theodore Price，棉花） | 过度买入，对突发事件估计不足 |
| 斯凯勒斯（Eugene Scales，棉花） | 权力欲望+操纵市场野心；从未想过市场会见顶 |
| 利文摩尔（Jesse Livermore） | 贪心、权力欲；赚大钱后不再稳妥交易；试图让市场跟意愿走 |
| 克劳福博士（Dr. Crawford） | 过度交易，将谨慎抛到九霄云外；没考虑意外出现 |
| 乔丹（Jordan，棉花） | 相信价格会不断上涨；看不到顶部；满仓后大家都要抛售时没买家 |

### 15.2 大炒家失败的共同模式

```
少量资金起家（小心谨慎，遵循规则）→ 聚敛大笔财富 → 改变判断方法
→ 失去均衡感 → 渴望金钱带来的力量/妄图垄断市场 → 过度交易
→ 未料到意外出现 → 爆仓破产
```

### 15.3 成功保住盈利的大炒家

| 炒家 | 成功原因 |
|------|----------|
| 伯纳德·巴鲁克（Bernard Baruch） | 遵循确定规则 |
| 本·史密斯（Ben Smith） | 赚了钱也保住了钱 |
| 伯特·卡斯特里斯（Bert Castles） | 建仓时总在离买入/卖空价5点处设止损单 |

### 15.4 成功者与失败者的差异

| 成功者 | 失败者 |
|--------|--------|
| 遵循确定理财规则 | 赚大钱后输个精光 |
| 研判股票/商品趋势，正确时候买进 | 鲁莽，过度交易 |
| 知道何时了结获利 | 贪心，权力欲，操纵市场 |
| 知道天有不测风云 | 没料到意外出现 |
| 从不过度交易 | 过度交易（最愚蠢之处） |
| 别人买进时卖出，别人卖出时买进 | 跟风，满仓后无买家 |

### 15.5 核心教训

| 教训 | 内容 |
|------|------|
| 不要过度交易 | 所有投机炒家最愚蠢的地方 |
| 使用止损单 | 自动执行，保住本钱和盈利 |
| 按事实交易 | 消除希望和恐惧心理（交易者大敌） |
| 凭希望持仓 | 最终在最恐惧时平仓，那时已太晚 |
| 面对事实 | 任何想要成功的人必须消除希望 |
| 记住趋势变化 | 必须在趋势变化时做出相应变化 |
| 世上无捷径 | 投机可成为获利职业，但必须遵守规则、永远意识到可能发生意外 |

## 16. 期权与权证

### 16.1 看涨期权（Call）

| 要素 | 内容 |
|------|------|
| 定义 | 30/60/90/180天内以固定价格买入某种股票的权利 |
| 权酬 | 140-250美元（根据股价和市场情况） |
| 最大损失 | 支付的权酬（固定且有限） |
| 潜在利润 | 无限（股票上涨越多利润越大） |

**抵补交易**：持有看涨期权获利后可做空50股锁定部分利润；若股价继续上涨，用50股多头仓位赚钱；若下跌，回补50股空头仓位获利。

**保护投资**：持有多头时买入看跌期权对冲下跌风险。

### 16.2 看跌期权（Put）

| 要素 | 内容 |
|------|------|
| 定义 | 期权生存期内以固定价格卖出100股或更多股票的权利 |
| 最大损失 | 权酬（如187.50-200美元） |
| 应用 | 预期下跌时买入，跌后买进股票按执行价交割获利 |

### 16.3 权证（Warrant）

| 特征 | 说明 |
|------|------|
| 生存期 | 更长（有些可交易数年） |
| 本质 | 类似长期股票看涨期权 |
| 杠杆优势 | 价格波幅比普通股大，百分比尤其明显 |
| 最大损失 | 权证价格 |
| 最佳策略 | 萧条时期低价买进，牛市末期高价抛出 |

**实例**：三角洲公司权证1941/42年低至1/32美元，1946年卖5美元，1000美元投资→160000美元（4年赚159倍）。

### 16.4 期权/权证的核心优势

**低风险、高利润**：最大损失是权酬/权证价格，而一旦股票与交易同向运动，利润无限。江恩认为这是"有利可图的、安全的交易途径"。

## 17. 核心检查清单

### 17.1 交易前检查清单

- [ ] 是否违反24条常胜规则中的任何一条？
- [ ] 资金使用是否不超过十分之一？
- [ ] 已确定止损单位置（离成交价3-5点）？
- [ ] 趋势方向是否明确（高低点上移/下移）？
- [ ] 是否顺势而为（不逆势）？
- [ ] 是否在双底/三重底买入或双顶/三重顶卖出？
- [ ] 是否考虑了百分比阻力位（尤其50%）？
- [ ] 是否检查了时间周期（3周、60-65天等）？
- [ ] 是否处于重要转势日期附近？
- [ ] 收益风险比是否合理（利润大于损失）？
- [ ] 交易的是活跃股票？
- [ ] 是否独立思考，未盲从他人建议？
- [ ] 身心状态是否良好？

### 17.2 持仓中检查清单

- [ ] 止损单是否仍有效且未被撤销？
- [ ] 是否有3点以上利润（应设止损单保护）？
- [ ] 是否只顺交易方向移动止损？
- [ ] 是否在持仓时计算浮盈/浮亏（不应计算）？
- [ ] 3日图趋势是否仍支持持仓方向？
- [ ] 是否出现2日回调/反弹（强势/弱势信号）？
- [ ] 成交量是否验证趋势？

### 17.3 平仓信号检查清单

- [ ] 3日图/9点转向图是否出现转势信号？
- [ ] 是否到达重要百分比阻力位？
- [ ] 是否处于重要时间周期尾声？
- [ ] 是否到达重要转势日期？
- [ ] 成交量是否出现头部特征（巨量）？
- [ ] 市场是否失去平衡（时间/价格失衡）？
- [ ] 是否出现明确的转势迹象（而非凭希望）？
- [ ] 接到追加保证金通知（应平仓而非追加）？

## 结语：江恩方法论的整合

江恩方法论的精髓在于**规则至上、时间为王**：

- **没有知识**，对市场一无所知就入市，必输大部分本钱
- **没有止损单**，暴露在无限损失中，一次意外即可爆仓
- **没有规则**，凭希望和恐惧行事，人性的弱点会毁灭账户
- **没有时间周期**，无法研判趋势变化，错过最佳买卖时机
- **没有百分比**，无法预测未来阻力位和买卖点

江恩方法论的终极价值：
1. **研判趋势**——通过3日图、9点转向图、高低点移动判断主要趋势方向
2. **预测转势**——通过时间周期、季节性日期、周年纪念日预判趋势变化
3. **精确买卖**——通过百分比阻力位、双底/三重底、最安全买卖点确定进场位置
4. **控制风险**——通过十分之一法则、止损单、不过度交易保护资本
5. **战胜人性**——通过独立思考、遵守规则、消除希望和恐惧克服最大弱点

**最终目标**：像江恩一样，将投机和投资从赌博变为一种有利可图的职业。通过学习规则、耐心等待机会、在正确的时候买卖，战胜华尔街。

> "我能给你世上最好的规则，以及研判股票位置的最佳方法。但你还是会因为人的因素，也就是你最大的弱点，而输掉账户上的钱。"——W·D·江恩

> "在华尔街，历史不断重演，以往发生的事会在未来再次出现。"
