## 1. 第一原则与风险度量：本金保护优先

### 1.1 第一原则

> **重要的不是你能赚多少，而是你准备亏损多少。**

为弥补亏损，必须获取比所受损失更大的收益百分比，且亏损与获利的先后顺序无关：

| 亏损幅度 | 扳平所需盈利 | 亏损幅度 | 扳平所需盈利 |
|----------|------------|----------|------------|
| -20% | +25% | -50% | +100% |
| -25% | +33% | -77.9% | +352.5% |
| -33% | +50% | | |

**核心结论**：为确保长期投资成功，保护本金远比获取偶然暴利重要得多。

### 1.2 风险与波动率的线性关系

阿佩尔将共同基金按波动率分为9组（第9组波动率最大），基于1983—2003年数据得出三条铁律：

| 关系 | 结论 |
|------|------|
| **收益/损失比率** | 波动率越大，收益/损失比越小（第9组为1.5，第1组为2.7） |
| **最大回撤** | 波动率越大，回撤越深（第9组68%，第1组15%） |
| **20年平均收益** | 低波动率组与高波动率组最终收益基本一致，高风险并未换来高收益 |

**反常识结论**：对买进并持有策略而言，**越高的波动率在历史中产生的收益越低**。低波动率的共同基金通常产生更高回报和更小回撤。

### 1.3 回撤幅度——最有效的风险度量

**回撤**：投资组合从最高值到下一个新高之前所发生的最大下跌幅度。

- 评估任何投资组合时，应确保历史最大回撤不超过自己可承受的范围
- 可通过平衡高低风险资产调整整体回撤（如50%中期国债基金+50%高波动基金≈30%综合回撤）
- 精准择时能降低高波动品种的风险，纪律严明的活跃投资者可适度持有25%高波动标的

## 2. 相对强度投资法：时时持有最优组合

### 2.1 核心理念（赛马类比）

> 像赛马一样：开始可下注任何马，但每过一个弯道都能把赌注转移给当前最领先的马。

**基本原则**：
- 找出领涨股
- 买进领涨股
- 只要它们保持领先就一直持有
- 当领涨股减缓上涨速度时，卖出并买进新的领涨股

### 2.2 三步操作流程

| 步骤 | 操作 |
|------|------|
| **第一步** | 获取至少500只（越多越好）共同基金最近一个季度的价格和波动率数据 |
| **第二步** | 开设多元化投资账户，买入**上季度业绩排名前10%**、且**波动率低于或约等于标普500指数**的基金，选4—5只分散持有，优先选持有90天以上免申赎费的基金 |
| **第三步** | 每个新季度开始，剔除跌出前10%的基金，用新进前10%的基金替换 |

### 2.3 历史验证（1990—2003年）

| 策略 | 年化收益 | 最大回撤 |
|------|---------|---------|
| 前10%基金（1—5组低波动率） | +14.1% | 20.3% |
| 标普500买进并持有 | +10.8% | 44.7% |
| 前10%基金（1—9组全市场） | +14.4% | 40.2% |

**关键发现**：
- 增加波动率范围（1—5组→1—9组）收益仅微升（14.1%→14.4%），但回撤从20%飙升至40%
- **结论：坚持重点投资低于平均波动率的共同基金**
- 持有周期通常以两个季度（6个月）为宜，过于频繁调仓会增加费用并可能被基金公司禁止交易
- 共同基金比ETF更适合此策略（ETF平均波动率更高）

## 3. 两个简便易用的股市行情指标

> 这两个指标每周只需15分钟维护，在过去20多年历史中表现出色地预测了适宜投资的时期。

### 3.1 纳斯达克/纽约证券交易所相对强度指标

**逻辑**：当投资者乐观时倾向成长型/科技股（纳斯达克领先），悲观时转向防御型（纽交所领先）。纳斯达克领先时几乎所有市场板块都产生较高收益。

**四步计算法**（每周一次）：

1. 获取纳斯达克综合指数与纽约证券交易所指数的**周收盘点数**
2. 计算比率：`Nasdaq/NYSE = 纳斯达克收盘点数 ÷ 纽交所收盘点数`
3. 计算最近**10周**的比率移动平均值
4. 比较最新比率与10周均线：
   - **比率 > 10周均线 → 纳斯达克领先 → 牛市信号（买进）**
   - **比率 < 10周均线 → 纽交所领先 → 中性信号（减仓/持有低波动品种）**

**历史表现**（仅纳斯达克领先时投资）：
- 纳斯达克年化12.0%（买进并持有仅9.2%），投资时间仅占55%
- 最大回撤：纳斯达克39.7%、标普500 24.2%（买进并持有分别为77.4%、48.0%）

### 3.2 货币过滤器指标（利率趋势）

**逻辑**："不要和美联储作对"——利率下降股市表现好，利率上涨股市表现差。

**计算与规则**：

1. 每周读取**三年期国债**与**五年期国债**收益率，取平均值
2. 累积至少34周数据后，将最新周平均值与前**33周**平均值比较
3. **最新值 < 前33周均值 → 利率下降 → 有利于股市（持有）**
4. **最新值 > 前33周均值 → 利率上升 → 不利于股市（谨慎）**

### 3.3 两个指标的综合运用

两个指标可独立使用，也可组合：当**两者同时发出牛市信号**时，投资环境最为有利，可信度最高（协同效应）。

## 4. 移动平均线与变动率：跟踪趋势与动量

### 4.1 三层移动平均线体系

| 周期 | 参数 | 用途 |
|------|------|------|
| **短期** | 10日 | 反映短期趋势，捕捉每日价格变化 |
| **中期** | 50日（10周） | 反映中期趋势 |
| **长期** | 200日（40周或30月） | 反映市场主要趋势 |

**核心读图法则**：
- **斜率即力量**：脉冲持续时间越长、斜率越垂直，持续趋势概率越大；斜率温和则逆转迫近
- **上涨动能递减 → 调整将至；上涨动能增强 → 进一步上涨**
- 移动平均线**加速上扬形成抛物线**（如1980年黄金、2000年股市）→ 高度投机期，常以凌厉下跌收场
- 周线30周均线、月线30月均线常提供关键支撑；价格跌破后趋势反转

### 4.2 移动平均线交叉策略的局限

1970—2004年道琼斯指数回测：200日/100日均线交叉策略年化收益仅6.0%—6.6%，略低于买进并持有的7.8%。单纯交叉策略价值有限，需配合指数移动均线、波段交易法（见第13章）使用。

### 4.3 变动率指标（Rate of Change）

**计算**：`变动率 = 今日收盘价 − N天前收盘价`（可用点数或百分比）

**参数选择**：
- **10日变动率**：短期交易
- **21—25日变动率**：中期分析
- 结合使用：短期率先预测方向，长期确认方向

**背离的核心判读**：

| 类型 | 定义 | 信号 |
|------|------|------|
| **顶背离** | 价格创新高，变动率未创新高（动量衰竭） | 熊市预警 |
| **底背离** | 价格创新低，变动率未创新低（下跌动能衰竭） | 牛市预警 |

**超买/超卖的环境依赖**：
- 牛市中变动率很少跌至极低负值，一旦跌至低位常迅速反弹
- 熊市中变动率很难触及高位正值，一旦触及常引发凌厉下跌
- **必须结合市场价格趋势、移动平均线方向调整超买超卖参数**
- 强趋势中"超买"可继续超买，不宜简单反向操作
- 双顶+变动率双顶 → 熊市；双底+变动率双底 → 绝佳介入点

## 5. 市场周期的四个阶段

移动平均线形态可清晰划分市场周期四阶段，对应不同的投资组合策略：

| 阶段 | 名称 | 移动平均线特征 | 操作策略 |
|------|------|--------------|---------|
| **一** | 筑底/吸筹 | 短期均线先反转，长期均线仍在跌但斜率趋缓；价格上穿关键均线 | 从容累积头寸，调整组合，需数周至数月 |
| **二** | 上涨（最佳持股期） | 短/中/长期均线依次反转并加速上涨；价格在关键均线处找到支撑 | 满仓持股，买进脉冲比卖出脉冲更久更陡 |
| **三** | 筑顶/派发 | 短期→中期→长期均线依次失去动能并走平；价格跌破关键均线 | 部署卖出策略，用停损单保护利润，卖出失去相对强度的标的，反弹即清仓机会 |
| **四** | 下跌（最弱阶段） | 短期→长期均线依次转头向下并加速；反弹受阻于下跌均线 | 持有现金为主；严重下跌常伴随高利率；激进者可短线交易 |

**关键认知**：股市往往比经济新闻早9个月至1年预测经济形势改变。第三阶段媒体仍播放利好消息时，市场本身已发出警告。

## 6. 三重动量纳斯达克择时模型

> 一个"简洁到优雅完美"的短期择时模型，1972—2004年32年中有20年远超买入并持有，每笔盈利是亏损的5倍。

### 6.1 指标构建

维护三个纳斯达克综合指数收盘价的**百分比变动率**：
- **5日变动率**
- **15日变动率**
- **25日变动率**

**三重动量值 = 5日变动率 + 15日变动率 + 25日变动率**（均为百分比）

### 6.2 买卖规则（唯一规则）

| 信号 | 条件 |
|------|------|
| **买入** | 三重动量自下而上突破 **+4%** |
| **卖出** | 三重动量自上而下击穿 **+4%** |

**特点**：投资时间仅占45.9%，无其他规则。短期5日变动率在方向改变上领先于长期的15日和25日。

## 7. 图表形态工具：协同、楔形、头肩、支撑阻力

### 7.1 协同效应（贯穿全书的核心原则）

> **协同效应**：将两种以上分析工具结合使用，其作用大于单独使用之和（1+1>2）。

**应用准则**：当多个指标（图表形态、周期分析、动量、利率结构、相对强度）发出**一致的买卖信号**时，可信度远高于单一指标。当信号相互矛盾时，往往预示市场处于震荡期。

### 7.2 楔形形态

**上涨楔形**（熊市预兆）：
- 两条收敛的上升趋势线（阻力线斜率小于支撑线斜率）
- 成交量逐步减少（买压稳定但卖压增强）
- **最可能向下突破**
- 操作：上边界卖出、跌破下边界卖出、交叉点设为止损

**下降楔形**（牛市预兆）：
- 两条收敛的下降趋势线
- 成交量减少（卖压消除、买方壮大）
- **通常以突破性上涨终结**
- 操作：下边界买进、突破上边界买进

楔形是阿佩尔**最喜爱的逐日交易形态**，与其他指标佐证时尤其可信。

### 7.3 头肩形态

**头肩顶**（牛转熊）形成条件：
1. 上涨后达高点（左肩）→ 回调 → 再创新高（头）
2. 头部回调至颈线位 → 再上涨但不超过头部（右肩）
3. 价格向下击穿颈线 → 形态完成
4. **关键验证**：成交量必须递减（左肩最大→头部减少→右肩明显萎缩）

**最小跌幅目标**：头部顶端到颈线的垂直距离，从颈线向下度量。

**头肩底**（熊转牛）为头肩顶的倒影，颈线被向上突破时完成，最小涨幅目标同理。止损位设在右肩最低点。

**有效性判断**：若跌破颈线后反弹超过右肩最高点，头肩顶失效，随后常伴随强劲上涨。

### 7.4 支撑位与阻力位

**核心法则**：
- 支撑区域被击穿 → 转变为阻力区域
- 阻力区域被突破 → 转变为支撑区域
- 牛市特征：不断提升的支撑位和上涨高点
- 熊市特征：持续降低的高点和低点
- **击穿前期阻力区域 = 早期趋势反转的重大信号**

**趋势线技巧**：
- 连接的点数越多越重要
- 上涨趋势线被击穿后转为阻力线；下跌趋势线被突破后转为支撑线
- 趋势线不应单独使用，需配合其他指标

### 7.5 通道线

- 通道上轨平行于主支撑趋势线
- 价格快速突破上轨 → 市场全面好转、有创新高能力
- 上涨斜率减小且无法触及上轨 → 市场疲软预警
- 击穿支撑趋势线 → 画对称的延伸线预测下跌目标

### 7.6 假突破和假回撤

> 假突破通常先于强烈的市场上涨或下跌行情发生。

**熊市预警序列**：
1. 价格在可交易区间波动
2. 突破交易区间上轨，看似上涨进行中
3. 上涨迅速终止，价格回落至突破前区间
4. 进一步下跌击穿区间下轨 → **随之而来的是凌厉且持续的下跌**

**关键点**：虚假突破只能吸引很少投资者跟随，其欺骗性反而确认了真实方向。

### 7.7 成交量法则（重要纠正）

> **错误观念**：缩量下跌意味着多头即将到来。
> **正确认知**：市场下跌在出现**显著的成交量增加**（抛售高峰期）之前不会结束。

- 缩量下跌仅是中性指标，可能持续很长时间
- 成交量峰值常出现在市场底部（2001年9月、2002年7月、2002年10月）
- 激进买家在恐惧卖盘中跑步进场形成抛售高峰期 → 反转信号

## 8. 政治、季节与时间周期

### 8.1 季节性规律

| 规律 | 内容 |
|------|------|
| **月内** | 每月开始的2—3天和结束的2—3天是买股最佳时机；重大节假日前表现好 |
| **年内** | 11月至次年4月是持股最佳期；5月至10月收益率低于银行短期收益（"Sell in May"） |
| **选举周期** | 总统竞选前一年收益最好；竞选后两年收益最低；竞选年纳斯达克常先于主流市场见顶 |

### 8.2 关键时间周期

按2倍递增的周期族（每个周期约为上一个的一半、下一个的两倍）：

| 周期 | 长度 |
|------|------|
| 4年市场周期 | 48个月（最具操作意义，1962年起规律性惊人） |
| 1年市场周期 | 12个月 |
| 中期周期 | 22—24周 |
| 短期周期 | 11—12周（约53个交易日） |
| 更短 | 5—6周、15—17天、7—10天、4—5天、17—20小时 |

### 8.3 周期的核心概念

- **周期长度**：从一个最低点到下一个最低点
- **A波段与B波段**：每个周期分两段；B波段高于A波段=牛市，反之熊市
- **周期嵌套（Nesting）**：多个周期同时到底部时影响最大，可能引发严重衰退或最强烈牛市
- **周期自我纠正**：一个周期长于名义长度，下一个会短一些，平均值保持稳定
- **18个月周期**：1995—2004年间表现规律，A-B波段斜率反映主趋势方向

## 9. T型形态：终极周期预测工具

> 阿佩尔"最喜欢的工具之一"，基于时间周期对称性，可预测市场反转的**时间点**（不预测价格水平）。

### 9.1 核心原理

> **T型形态左侧时间长度 = 右侧时间长度**（A-X = X-B）

在中性市场周期中，周期波段的时间长度基本相等。已知周期起点到中心的时间，即可预测中心到终点的时间。

### 9.2 构造步骤

1. **识别形态起点A**：周期开始的低点（或高点）
2. **确定中心X**：从A上升到最高点（或下降到最低点）时，由最高点向下做垂线
   - 单顶/单底：直接取该点
   - **双顶/双底：取两个顶点的正中间M作为X**（关键规则）
3. **测量A-X的水平距离**（时间长度）
4. **从X向右延伸相同长度到B**：B点对应的时间即预测的周期结束时间

### 9.3 应用要点

- T型形态**只预测时间，不预测价格**；价格可从运动斜率大致估算
- 适用所有时间框架：日线、周线、月线、日内小时图
- **镜像模式**：中性市场中右侧走势常是左侧的完美镜像，T型形态此时特别有效
- **周期逆转**：当市场不遵循T型预测的方向（如预测下跌却强劲上涨），称为周期逆转，此时顺势而为可快速获利
- **必须与其他指标协同使用**，不可单独作为买卖依据
- 出现频率极高，"想发现就一定能发现"

## 10. 市场广度指标体系

> 市场指数（加权平均）常被少数权重股主导，市场广度指标反映"内部市场"真实强度。

### 10.1 创新高/创新低指标

**核心规则**：
- 市场上涨伴随创新高股票数量增加 → 上涨有群众基础，可持续（确认）
- 指数新高但创新高股票数量减少 → **顶背离**，危险信号
- 指数下跌但创新低股票数量减少 → **底背离**，牛市征兆
- 创新高股票数量峰值通常比牛市最终顶点早约1年
- 三重底形态（如2002年三次创新低激增峰值）后反转迅猛

### 10.2 腾落线（Advance-Decline Line）

**计算**：累积每日（上涨股票数 − 下跌股票数）

**21日变动率判读**（2004年参数，约3500只股票时）：
- **[+9000, +10000]**：高度超买，常在牛市完成50%—65%时出现（买进已迟，卖出尚早）
- **[−10000, −9000]**：超卖，市场低点附近，上涨行情很快来临
- 牛市底部更迅猛尖锐；顶部更缓慢温和

**10日变动率**（更灵敏）：
- 波动范围通常在+5000（超买）到−8000（超卖）
- 与腾落线新高背离 → 短期/中期调整预警
- 存在6—7周的市场周期，连续上涨3周以上需警惕

### 10.3 指数移动平均线（EMA）

**平滑常数** = 2 ÷ (数据个数 + 1)

| 数据个数 | 平滑常数 |
|---------|---------|
| 10日 | 0.1818（0.18） |
| 19日 | 0.10 |
| 39日 | 0.05 |
| 6日 | 0.286 |

**公式**：`新EMA = 平滑常数 × (当日数据 − 前日EMA) + 前日EMA`

**特殊属性**：
- 价格穿越EMA时，EMA会快速转头（简单均线无此特性）
- 涵盖所有历史数据（近期权重更大）
- 仅需前一日EMA和当日收盘即可计算

**初始化方法**：取待计算天数（如10日）的简单移动平均值作为初始值。

### 10.4 周脉冲信号指标

> 完全客观的周线择时模型，1970—2003年间仅用22%时间即获得标普500 56.4%的收益。

**计算步骤**：
1. 每周收集纽交所上市交易股票总数、上涨数、下跌数
2. 净上涨股票数 = 上涨数 − 下跌数
3. 周净广度百分比 = 净上涨股票数 ÷ 交易股票总数
4. 计算6周指数平均值（平滑常数0.286）

**信号规则**：
- **6周指数平均值 ≥ 25% → 买进**
- **6周指数平均值 < −5% → 卖出（退出持现金）**

**特点**：信号不频繁但极少亏损，约1/3买进信号出现在牛市开端或重大上涨起点。

### 10.5 日线广度脉冲信号

> 阿佩尔见过的成功率最高的择时模型之一：34年内年化41.6%，84%交易获利。

**构建规则**：
1. 每日获取纽交所上涨股票数和下跌股票数
2. 计算日比率 = 上涨股票数 ÷ (上涨股票数 + 下跌股票数)
3. 计算该比率的**10日指数平均值**（平滑常数0.1818）
4. **10日指数平均值 ≥ 0.615（61.5%）→ 买进**
5. **10日指数平均值 ≤ 0.490（49%）→ 信号消失**（不发出卖出信号，仅表示强势广度结束）

**中长期升级版**（MACD过滤）：卖出条件需同时满足——广度脉冲10日指数平均值≤49% **且** 19-39日MACD读数≤1%（MACD读数=（19日EMA−39日EMA）÷39日EMA×100）。两个条件必须同时满足才卖出。

## 11. 成交量极值、TRIN与波动率指数

### 11.1 交易者指数（TRIN / 阿姆斯指数）

**公式**：`TRIN = (上涨股票数 ÷ 下跌股票数) ÷ (上涨成交量 ÷ 下跌成交量)`

| TRIN值 | 含义 |
|--------|------|
| **= 1.00** | 中性（上涨股票比例与上涨成交量比例匹配） |
| **> 1.00** | 上涨股票缺乏成交量支持（派发/熊市特征） |
| **< 1.00** | 上涨股票获得巨幅成交量支持（吸筹/牛市特征） |
| **极高（恐慌）** | 投资者过度悲观，**绝佳买进时机** |

**读数基准**（随市场演变需调整）：
- 强势市场10日均值：0.80—0.90
- 10日均值 > 1.50（近年）或 > 1.30（早期）→ 恐慌性抛售，买进信号
- 35日均值 > 1.30—1.35 → 极端区域，买进持股待反转
- 日内 < 0.70 难维持，< 0.50 为"爆买"，难以持续

**预警信号**：
- 价格上涨但TRIN走弱 → 早期反转警告
- 价格与TRIN顶背离（TRIN越低净买越强）→ 警惕
- TRIN判定底部优于判定顶部

### 11.2 波动率指数（VIX）

基于标普500期权价格的反向情绪指标。

**操作规则**：
- **VIX > 35 → 准备买进**（投资者极度悲观，绝佳买点）
- **VIX从35跌至18—20之前 → 安全持股阶段**
- **VIX跌破18 → 安全持股阶段结束**（本身非卖出信号）
- VIX较低（< 18）常出现在稳步攀升的持续牛市中

**注意**：VIX只能提供买进信号后的安全持股区间，不提供可靠的离场信号。

### 11.3 主要反转波动率模型

**直接测量**纳斯达克综合指数的波动率（不依赖TRIN/VIX间接指标）。

**计算**：
1. 每周末计算纳斯达克指数的周变化率（取绝对值，涨跌无关）
2. 计算10周移动平均值

**买进信号条件**（需同时满足）：
- 10周平均变化率在 **3%—6%** 之间（>6%表示极度不稳定，需先跌穿3%再反弹突破3%后从峰值下跌0.5%）
- 10周移动平均值从最高点**下跌0.5%以上**

**特点**：
- 信号不频繁（每十年约2—6次）
- 1970—1999年共12次信号，全部获利
- **不提供卖出信号**，仅确认买进时机
- 2000—2002年熊市中首次出现失败信号，提示波动率上升时需调整参数

## 12. MACD：终极市场择时指标

> 阿佩尔20世纪70年代末发明，全球最流行的技术分析工具之一，适合所有时间周期（月/周/日/小时/分钟），特别擅长发现市场长期低迷后的买进点位。

### 12.1 基本概念

**MACD = 短期指数移动平均值 − 长期指数移动平均值**

- 市场转强 → 短期均线涨更快 → MACD向上
- 市场转弱 → 短期均线走平并下穿长期均线 → MACD降至0轴下
- **信号线** = MACD的9日（或6日）指数平均值
- **0轴**：短期与长期均线交叉点（力量平衡点）

### 12.2 参数组合体系

| 参数组合 | 用途 |
|---------|------|
| **6-19日** | 市场非常有利时快速发出买进信号 |
| **12-26日** | 中性市场买进信号；非常弱势市场买卖信号 |
| **19-39日** | 卖出信号（最常用慢线） |

**黄金法则**：
- **至少使用两种MACD组合：快线买进，慢线卖出**
- 市场强有利：6-19买进，19-39卖出
- 中性偏强：12-26买进，19-39卖出
- 明显弱势：12-26同时买卖（须在19-39下穿0轴前卖出）
- 50日（或50周/50月）移动均线判定主趋势

### 12.3 买卖信号规则

**买进信号**：
1. 检查50单位移动均线趋势
2. MACD转头向上并穿越信号线
3. 检查是否底背离
4. 检查周期、T型形态、楔形、V型反转、趋势线突破等确认因素

**零轴法则**（重要先决条件）：
- 买进信号前，MACD必须先跌至0轴以下（除非长期趋势非常有利+明显底背离+重大下跌趋势被突破）
- 卖出信号前，MACD必须先上涨至0轴以上

**卖出信号**：
- MACD自上而下穿越信号线（须经零轴法则确认）
- 伴随顶背离 → 即使长期趋势走强也应减仓
- **必须遵循19-39日MACD发出的第二个卖出信号**

**首信号忽略规则**（强势市场延迟卖出）：
- 市场处于上涨趋势（50日均线上行）
- 第一次卖出信号**无顶背离**确认
- 可忽略第一个卖出信号，但**必须遵循第二个**
- 辅助止损：价格跌破50日移动均线时无条件清仓

**止损信号**：买进信号MACD跌至前一个低点之下时止损（基于MACD读数，非价格）。

### 12.4 背离的运用

| 类型 | 定义 | 可靠度 |
|------|------|--------|
| **顶背离** | 价格创新高，MACD未创新高 | 伴随卖出的顶背离信号可信度更高 |
| **底背离** | 价格创新低，MACD未创新低 | 伴随买进的底背离信号可信度更高 |

**经背离确认的信号可靠度远高于无背离确认的信号。**

### 12.5 MACD的局限与失灵

- 在窄幅稳定上涨/下跌中可能失灵（如1994—1995年稳步上扬时MACD发出卖出后无法再发买进信号）
- 强势上涨中可能过早发出卖出信号
- 解决方案：配合10日新高股/（新高+新低股）比率约90%、上涨股票占比周比率等广度指标中和

### 12.6 MACD的周期四阶段映射

MACD平滑形态清晰对应市场周期四阶段：
- 阶段一（筑底）：MACD底背离后转头向上
- 阶段二（上涨）：MACD稳步上扬穿越0轴
- 阶段三（筑顶）：MACD顶背离
- 阶段四（下跌）：MACD跌至0轴下方运行

### 12.7 多时间框架共振

- **周线/月线MACD确认日线信号** → 准确性大增
- 短期和长期指标在周期性低点和高点完全吻合 → 绝佳投资机会
- 月线MACD比实际高低点晚约3个月产生信号
- 周线MACD可明确规避1987年大崩盘类事件

### 12.8 资金管理建议

- 分仓买进/分批卖出，而非一次满仓/清仓
- 温和上涨+低波动基金组合时不宜完全清仓
- 股市约75%时间在涨，牛市短期卖出信号可忽略（短暂下跌后会重拾升势，再进场位置相近）

## 13. 移动平均线交易通道

> 阿佩尔与MACD并列推崇的实战工具，可应用于任何时间框架。

### 13.1 通道构建

1. 选择移动平均线参数（21日—50日最适宜，反映7—8周或更长周期）
2. 设定偏离率（百分比），使**85%—90%的价格数据包含在通道内**
3. 上轨 = 移动平均线 × (1 + 偏离率)；下轨 = 移动平均线 × (1 − 偏离率)
4. 市场波动率越大，偏离率越大；均线参数越大，通道越宽

**示例**：21日均线1000点，偏离率4% → 上轨1040、下轨960。

### 13.2 八大交易规则

| 规则 | 内容 |
|------|------|
| **规则一** | 价格击穿通道下轨 → 通常会反弹至买进位置或更高（可考虑买进） |
| **规则二** | 击穿下轨后的**第一次反弹**往往在通道中轨（均线）遇阻；若击穿幅度过大则反弹止于下轨 |
| **规则三** | 从下轨反弹若突破中轨，下一阻力位极可能是上轨 |
| **规则四** | 价格抵达上轨后，回调的第一次目标位极可能是中轨（除非市场异常弱） |
| **规则五** | 上涨未能像前一次一样触及/突破上轨 → 价格和动量走弱（顶背离） |
| **规则六** | 上涨不能触及上轨 → 接下来的下跌很可能击穿中轨 |
| **规则七** | 通道出现顶部/底部形态且被其他指标确认 → 趋势反转概率大增 |
| **规则八** | 高点未超前高 → 随后回调低点可能跌破两高点间最低点；低点比前低更低 → 反弹无法超越两低点间高点 |

### 13.3 动量对比分析法

每次市场波动中，对比当前趋势脉冲与前一次同趋势脉冲的强度，回答以下问题：

**下跌时**：MACD下行动量增减？价格与下轨距离增减？下跌斜率增减？通道下行动量增减？

**上涨时**：上涨动量消失还是增强？斜率？MACD形态？价格与上轨关系？

### 13.4 离差振荡指标

可将通道数据转换为振荡图：0轴对应移动均线，±偏离率线对应上下轨。便于识别背离和绘制趋势线。

### 13.5 投资规则总结

- 计算和使用不同时间长度（主趋势/中级趋势/短期趋势）的通道
- 通道须包含85%—90%价格数据
- 价格跌破下轨可考虑买进；第一次反弹多在中轨受阻
- 突破中轨后看上轨；到上轨后回调看中轨
- 突破上轨幅度递减 → 动量减弱 → 回撤预警
- 未能触及上轨 → 可能击穿中轨
- 顶部形态被确认 → 趋势反转
- **务必协同MACD、RSI、周期、T型形态等其他指标**

## 14. 四步整合：构建自己的赢家策略

> 将所有"武器"按步骤和顺序组合运用，形成完整的决策流程。

### 第一步：确定主要趋势和周期阶段

| 工具 | 方法 |
|------|------|
| **40周/200日移动均线** | 判定主趋势方向和斜率变化 |
| **长期移动均线通道** | 观察通道方向和价格运行模式 |
| **月线/周线MACD** | 作为长期强度指标 |
| **四阶段模型** | 判断当前处于筑底/上涨/筑顶/下跌哪一阶段 |
| **长期底部指标** | 主要反转波动率模型、新高新低、TRIN、市场底部形态 |
| **时间周期/政治周期** | 4年周期自1931年来19个周期中16个在大选前年上涨 |

### 第二步：检查市场行情指标和季节性周期

- 持续跟踪 Nasdaq/NYSE 相对强度指标（领先时几乎所有板块都涨）
- 持续跟踪货币过滤器指标（利率趋势）
- 关注季节性：11月至次年1月最易发生上涨；股市下跌常在10月结束

### 第三步：确定中期趋势方向和强度，预估下次反转

**判定方向与强度的工具**：
- MACD（周线+日线）：趋势、背离、斜率、转折点（慢速MACD最有效）
- 移动平均线交易通道：可与MACD结合或独立使用
- 变动率指标：10日、21日，发现动量背离
- 市场广度指标：腾落线、新高新低数据

**预估拐点位置的工具**：
- 中期时间周期理论和T型形态
- 图表形态（角度变化、反转形态、楔形、趋势线支撑阻力）
- 季节性考量
- 情绪指标（VIX）

**中线交易者的特殊模型**：
- 三重动量纳斯达克模型
- MACD修正的广度脉冲模型
- 新高新低+周脉冲持续信号

### 第四步：用短期日线/小时读数重新调整中期策略

- MACD、变动率、图表形态、短期小时/日线周期用于研判短期趋势和动量
- 短期趋势日线广度脉冲信号触发模型：频率低但极可靠
- 共同基金选择：相对强度投资法（阿佩尔最喜爱，强力推荐）

## 15. 40年交易生涯的核心心得

1. **新闻媒体无价值**：事后分析而非预测。当畅销杂志头条长篇累牍讲述股市低迷时，往往是买进好时机
2. **自行摸索独有策略**：沙龙论坛有帮助，但更重要的是依据自身条件领悟决策方略
3. **私密保存投资纪录**：四处吹嘘成功却害怕提及失败毫无帮助
4. **警惕天性陷阱**：过早卖出强势头寸（锁利）、过久持有弱势头寸（不卖不亏）——应"截断亏损让利润奔跑"
5. **择时工具只在部分时间获利**：但平均而言盈利更大、亏损更小
6. **投资本质是获取收益而非自我感觉良好**：1999—2000年泡沫中"买什么都赚钱"的心态让人不愿离场
7. **股票上涨≠投资天才**：不要搞混
8. **频繁交易不一定优于中长期投资**：费用和滑点往往吞噬收益
9. **错过盈利机会远比遭受损失好**
10. **避免开盘时段操作**：上午11:30左右和下午1:15—1:30市场相对平静，决策更明智
11. **进场即准备离场**
12. **投资资金须在可承受损失范围内**
13. **连续3次成功后警惕亏损**（自我膨胀时风险最大）
14. **记录失败经历**：分析哪些非理性因素使你偏离基本法则
15. **证券市场本质是概率游戏**：并非每次亏损都源自错误决策
16. **市场不明朗时就离开休息**：没有必要一直待在市场里

## 16. 核心检查清单与参数速查

### 16.1 买卖决策检查清单

**买进前确认**：
- [ ] 50日/40周移动均线方向（上涨或走平）
- [ ] Nasdaq/NYSE相对强度领先？
- [ ] 货币过滤器有利（利率下降）？
- [ ] MACD零轴法则（已跌至0轴下）？
- [ ] 底背离出现？
- [ ] 市场周期处于阶段一末/阶段二初？
- [ ] T型形态预测的时间窗口？
- [ ] 广度指标（新高新低、腾落线）确认？
- [ ] VIX > 35 或 TRIN 极高（恐慌买点）？
- [ ] 多指标协同确认？

**卖出前确认**：
- [ ] MACD零轴法则（已涨至0轴上）？
- [ ] 顶背离出现？
- [ ] 通道上轨未能触及/突破幅度递减？
- [ ] 市场周期处于阶段三/四？
- [ ] 广度顶背离（新高股票减少）？
- [ ] 支撑位被击穿转为阻力位？
- [ ] 是否为19-39日MACD的第二个卖出信号？

### 16.2 关键参数速查表

| 指标 | 参数 | 信号阈值 |
|------|------|---------|

### 16.3 全书方法论一句话总结

> **以"保护本金"为第一原则，用"相对强度"选品种，用"Nasdaq/NYSE+货币过滤器"判环境，用"移动均线+变动率"跟趋势，用"T型形态+时间周期"测拐点，用"广度指标+TRIN+VIX"探顶底，用"MACD+移动均线通道"定买卖，所有工具通过"协同效应"相互确认，按"四步整合"流程执行，最终实现低风险下的超额收益。**
