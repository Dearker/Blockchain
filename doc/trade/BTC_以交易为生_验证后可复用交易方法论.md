# BTC 以交易为生：验证后可复用交易方法论

> 数据基础：Binance `BTCUSDT` 最近两年 K 线，验证区间 2024-06-28 至 2026-06-27。
> 适用对象：BTCUSDT 日线波段交易，周线用于战略过滤，4小时用于执行优化；不构成投资建议。

## 1. 方法论定位

这套方法保留《以交易为生》中经过 BTC 两年 K 线验证后仍有执行价值的部分：四大支柱、三重滤网、动力系统、EMA价值区、MACD柱状线、强力指数、成交量确认、假突破/袋鼠尾、通道、2%/6%风控和交易日志。

它不追求预测每一根 K 线，只解决五个问题：

1. 我现在的心理和账户状态是否允许交易？
2. 周线背景允许做哪个方向？
3. 日线是否给出回调、突破或反转候选？
4. 4小时是否给出可执行触发？
5. 如果判断错了，账户最多亏多少？

本次样本中，日线买入并持有总收益 -0.66%、最大回撤 -52.03%；动力系统过滤总收益 70.06%、最大回撤 -15.85%；三重滤网代理总收益 27.11%、最大回撤 -5.47%。结论是：埃尔德体系的价值不在某个神奇指标，而在“方向过滤 + 回调等待 + 风险约束 + 记录复盘”的闭环。

## 2. 证据边界

本方法只使用 OHLCV K 线。心理、纪律、日志、账户风险、滑点、手续费、资金费率、订单簿、链上数据和宏观新闻无法由本次 K 线直接验证。

因此，所有指标只能给出概率加权，不能替代计划。凡是不能写出入场价、止损价、目标价、仓位和失效条件的交易，一律视为赌博。

## 3. 证据等级与取舍

把验证结果分成三层，避免把所有信号等权使用：

| 等级 | 保留内容 | 执行地位 |
| --- | --- | --- |
| A级：流程级通过 | 四大支柱、三重滤网顺序、动力系统禁行灯、2%/6%风控、交易日志 | 必须执行，缺一项则不交易 |
| B级：信号级可用 | 2日强力指数普通回调、放量突破、MACD柱斜率、OBV/A-D确认、4小时结构确认 | 作为入场或加仓的加权条件 |
| C级：预警级保留 | MACD背离、通道偏离、假突破/袋鼠尾、价值区单独触发、缩量突破 | 只做观察或减仓提示，不能单独开仓 |
| 剔除/外置 | 新高-新低指数、VIX、CFTC、内部人交易、宏观新闻 | 不进入本K线系统，可另建外部风险面板 |

本方法最终只允许三类交易：三重滤网回调、放量突破、假突破/袋鼠尾反转。其他信号只能为这三类交易加分、减分或触发退出。

## 4. 四大支柱

| 支柱 | BTC执行含义 | 不满足时的动作 |
| --- | --- | --- |
| 心理 | 不因连胜加杠杆，不因亏损复仇，不在持仓中反复看盈亏金额 | 降仓或停止交易 |
| 市场分析 | 周线定方向，日线找位置，4小时做触发 | 没有多周期一致性就等待 |
| 风险管理 | 单笔风险、月度风险、止损和杠杆先于收益 | 风险算不清就不下单 |
| 交易记录 | 每笔交易写计划、评分、截图/价位、复盘结论 | 未记录的交易下次禁止加仓 |

## 5. 三层时间框架

| 层级 | 周期 | 任务 | 工具 |
| --- | --- | --- | --- |
| 战略层 | 周线 | 只做多、只做空或观望 | MACD柱斜率、13周EMA斜率、周线动力系统 |
| 计划层 | 日线 | 找价值区、突破、假突破和止损 | 13/26 EMA、MACD柱、2日强力指数、20日高低点、OBV/A-D |
| 执行层 | 4小时 | 优化入场和失败退出 | 动力系统、短通道、前高前低、放量收回/跌回 |

大周期优先。日线信号若逆周线方向，只能降级为短线候选；周线明显红色时，不开新的日线趋势多单。

## 6. 市场状态分类

每日先分类，再找交易：

| 状态 | 判定 | 行动 |
| --- | --- | --- |
| 多头潮流 | 周线MACD柱斜率向上，13周EMA不下行 | 只找做多或观望 |
| 空头潮流 | 周线MACD柱斜率向下，13周EMA不向上 | 现货空仓；合约只允许小风险顺势空 |
| 中性潮流 | 周线EMA和MACD柱斜率冲突 | 降仓，等待日线结构更清晰 |
| 高波动失控 | 日线ATR扩张、长影线和失败突破频繁 | 缩小风险，暂停追单 |

动力系统只做禁行灯：

- 周线红色：禁止开多。
- 周线绿色：禁止开空。
- 日线红色：已有多仓减仓或等待重新转蓝。
- 日线绿色：已有空仓减仓或等待重新转蓝。
- 蓝色：不是信号，只是允许继续观察。

## 7. 标准入场模型

### A. 三重滤网回调买入

条件：

1. 周线 MACD柱斜率向上，或周线动力系统不是红色。
2. 日线回到 13/26 EMA 价值区附近。本次“周线向上/回到价值区”样本命中率为 50.00%。
3. 日线 2日强力指数跌破0但没有创20日新低。本次该条件命中率为 65.26%。
4. 日线 MACD柱由降转升，或4小时动力系统由红转蓝/绿。
5. 止损能放在结构低点或1.5-2.5ATR外，单笔风险不超过账户权益上限。

执行：先试探仓，4小时重新转强或日线收回价值区后再恢复标准仓位。

### B. 放量突破买入

条件：

1. 周线不逆向。
2. 日线收盘突破20日高点或关键阻力。
3. 成交量高于20日均量25%以上。本次“20日高点放量真突破”命中率为 53.33%。
4. 收盘在当日振幅上半区，不能是长上影失败。
5. 突破后1-3根不能快速跌回区间。

执行：突破当日不满仓追高，优先等待4小时回踩不破或次日延续。缩量突破只观察，不开标准仓位。

### C. 假突破/袋鼠尾反转

条件：

1. 价格刺穿20日低点或关键支撑后收回区间。
2. 下影线明显长于实体，或收盘重新站回支撑上方。
3. MACD柱或强力指数没有同步创极端新低。
4. 4小时形成更高低点后再入场。
5. 止损放在袋鼠尾极值外侧；若止损过远，放弃。

执行：这是候选反转，不是抄底许可。初始仓位不超过标准仓位50%。

## 8. 最小交易闭环

任何交易必须同时回答 6 个问题，缺一个就不下单：

| 环节 | 必须写清 |
| --- | --- |
| 背景 | 周线允许的方向，是否红色禁行 |
| 模型 | 属于三重滤网回调、放量突破、假突破/袋鼠尾中的哪一类 |
| 触发 | 日线收盘条件和4小时确认条件 |
| 止损 | 结构止损或ATR止损，不能盘中临时放宽 |
| 仓位 | 单笔亏损占账户权益比例，加仓后总风险 |
| 退出 | 失效退出、盈利保护、复盘时间 |

最小闭环不是为了提高胜率，而是为了让每笔交易可复盘、可归因、可停用。

## 9. 离场模型

硬退出：

1. 跌破入场计划中的结构止损或ATR止损。
2. 周线动力系统转红，且日线无法重新转蓝。
3. 放量突破后3根内跌回原区间。
4. 三重滤网买入后，2日强力指数继续创20日新低且价格跌破价值区。
5. 入场理由中趋势、位置、动能、成交量有三项同时失效。

盈利保护：

1. 到达1.5R-2R先减仓或移动止损。
2. 触及动态上通道不直接清仓，但若再次上攻无法创新高且MACD柱走弱，减仓。
3. 出现价格新高而MACD柱/OBV未确认，收紧止损。
4. 只顺交易方向移动止损，不能因害怕止损而放宽。

## 10. 仓位与风险

BTC 高波动下，2%是上限，不是默认值：

| 项目 | 规则 |
| --- | --- |
| 单笔计划亏损 | 账户权益的0.25%-1%，极佳机会最多1.5%，不把2%当常态 |
| 月度风险上限 | 已实现亏损 + 未实现持仓风险达到月初权益6%时，停止新交易 |
| 同向总风险 | 不超过账户权益2%-3% |
| 杠杆 | 强平价必须远离结构止损；做不到就不用杠杆 |
| 加仓 | 只加盈利仓，且加仓后总风险仍低于上限 |

头寸公式：

| 步骤 | 公式 |
| --- | --- |
| 账户最大亏损 | 账户权益 × 单笔风险比例 |
| 每BTC风险 | 入场价 - 止损价，做空反向计算 |
| 可交易数量 | 账户最大亏损 / 每BTC风险 |
| 月度可用风险 | 月初权益 × 6% - 本月已实现亏损 - 持仓风险 |

## 11. 信号评分

硬门槛：

1. 周线红色不开新多。
2. 缩量突破不开标准仓位。
3. 止损、目标、仓位和失效条件必须写清。
4. 单笔风险超过账户权益1.5%时，分数作废。
5. 身体/情绪/时间状态任一项为0分，当天不交易。

评分表：

| 项目 | 分值 | 条件 |
| --- | --- | --- |
| 周线背景 | 2 | MACD柱斜率同向，动力系统不逆向 |
| 日线位置 | 2 | 回到价值区、突破关键位、或假突破收回 |
| 动能 | 2 | MACD柱斜率转向交易方向，动力系统不再逆向 |
| 强力指数 | 1 | 回调但未创20日极端新低/新高 |
| 成交量 | 1 | 突破放量、回调缩量、OBV/A-D未明显背离 |
| 结构 | 1 | 4小时形成更高低点/更低高点，或回测不破 |
| 风险收益比 | 1 | 第一目标至少1.5R，最好2R以上 |

执行阈值：

- 8-10分：标准仓位。
- 6-7分：半仓或试探仓。
- 低于6分：不交易。
- 反转类信号即使高分，也先按半仓执行，确认后再加。

## 12. 每日流程

1. 自我测试：身体、昨日交易、计划、情绪、时间安排，任一0分不交易。
2. 周线：标记13周EMA、MACD柱斜率、动力系统颜色。
3. 日线：标记13/26 EMA价值区、20日高低点、MACD柱、2日强力指数、OBV/A-D、动态通道。
4. 分类：多头潮流、空头潮流、中性潮流、高波动失控。
5. 机会：只登记三类信号，三重滤网回调、放量突破、假突破/袋鼠尾。
6. 过滤：剔除周线逆向、缩量突破、单指标背离、止损过远的交易。
7. 计划：写入入场价、目标价、止损价、仓位、失效条件。
8. 执行：用4小时触发，不用1小时噪声推翻日线计划。
9. 管理：盈利后移动止损，失败后立即退出。
10. 记录：交易后写入评级、执行偏差和下次改进。

## 13. 禁用规则

- 不在周线红色时开新的趋势多单。
- 不把MACD背离当成直接反手信号。
- 不在动态上通道上方追多，不在动态下通道下方追空。
- 不把RSI/随机指标超买超卖作为单独开仓理由。
- 不因浮亏金额而移动止损。
- 不把2%风险当作默认仓位。
- 不在达到月度6%风险上限后继续开新仓。
- 不做没有交易日志的加仓。
- 不用外部指标缺失来补脑确认，缺数据就降级处理。

| --- | --- | --- |
| 周线潮流 | 13周EMA斜率、12-26-9 MACD柱斜率 | 决定方向和禁行 |
| 日线价值区 | 13日EMA、26日EMA | 顺势回调位置 |

| 风险 | 单笔0.25%-1%，极佳机会最多1.5%；月度6%上限 | 保护生存权 |

参数是默认工作值，不是信仰。若市场波动结构明显变化，先降仓，再重验参数。

## 15. 方法停用与重验规则

出现以下情况，暂停新开仓，只允许复盘和重新验证：

1. 连续3笔计划内亏损，且亏损都来自同一入场模型。
2. 单月已实现亏损 + 持仓风险达到月初权益6%。
3. 最近20笔交易中，低于6分的交易超过3笔。
4. 最大回撤超过账户权益10%，或超过该方法历史回撤的1.5倍。
5. 市场状态从趋势转入高波动失控，突破后频繁跌回区间。
6. 交易日志缺失超过2笔，说明纪律已经失效。

重启条件：完成复盘，确认亏损来自执行错误还是模型失效；若是模型失效，必须降低仓位并用至少20个新信号重新评估。

| 字段 | 内容 |

| 周线背景 | 绿色/红色/蓝色，MACD柱斜率 |
| 日线位置 | 价值区/突破/假突破/通道偏离 |
| 信号类型 | 三重滤网/放量突破/袋鼠尾反转 |
| 信号评分 | 0-10分 |
| 入场价 | 计划价或触发条件 |
| 止损价 | 结构止损或ATR止损 |
| 目标价 | 1.5R、2R、上通道、前高/前低 |
| 计划亏损 | 账户权益百分比 |

| 评级 | 买入评级、卖出评级、交易评级 |
| 两个月后回顾 | 当时判断是否有效，是否应改进规则 |

## 17. 一句话版本

先检查自己能不能交易，再用周线决定方向，用日线等待价值区回调、放量突破或假突破收回，用4小时确认触发；所有交易都服从2%单笔风险上限、6%月度风险上限和交易日志。
