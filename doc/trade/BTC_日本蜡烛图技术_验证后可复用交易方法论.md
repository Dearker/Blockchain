# BTC 日本蜡烛图技术：验证后可复用交易方法论

> 数据基础：Binance `BTCUSDT` 最近两年 K 线，验证区间 2024-06-28 至 2026-06-27。
> 适用对象：BTCUSDT 波段交易；周线定背景，日线定计划，4小时定执行；不构成投资建议。

## 1. 方法定位

这套方法保留《日本蜡烛图技术》中经 BTC 两年K线验证后仍可复用的部分：蜡烛图作为趋势变化预警、关键区域择时、次根确认、多工具汇聚和形态止损。

它不把任何形态当作“必然反转”。日线全部蜡烛图信号在本次样本中的命中率为 54.37%；加入次根验证后为 62.90%；进一步要求主要趋势同向、关键区域和次根验证后为 71.79%。因此，交易价值来自过滤流程，而不是单一形态名称。

## 2. 核心改写

把原书语言改写成 BTC 可执行规则：

| 原书思想 | BTC执行版 |
| --- | --- |
| 反转形态 | 趋势变化预警；先减仓或等待确认，不直接假定反转 |
| 趋势之后才有意义 | 信号前必须有20根左右的可见上升/下降结构 |
| 等待验证 | 下一根日线收盘确认，或4小时收回/跌破形态关键位 |
| 多技术共同参照 | 周线背景、日线位置、形态、成交量、动量、风险收益同时打分 |
| 窗口/跳空 | BTC日线严格跳空样本为0，全部降级为概念，不做硬触发 |
| 形态止损 | 看涨看形态低点，看跌看形态高点，再叠加ATR缓冲 |

低样本高命中率不能升格为硬规则。日线、周线中很多单项形态样本很少，必须以“过滤后组合规则”而不是单项形态排名为准。

## 3. 三层时间框架

| 层级 | 周期 | 任务 | 蜡烛图用法 |
| --- | --- | --- | --- |
| 战略层 | 周线 | 判断只做多、只做空或观望 | 周线长影、吞没、十字只作为趋势衰竭预警 |
| 计划层 | 日线 | 定义交易区、止损区、目标区 | 只在关键区域寻找形态和次根验证 |
| 执行层 | 4小时 | 优化入场、减仓、失败退出 | 用4小时收回/跌回关键位确认，不用1小时噪声推翻日线 |

1小时形态只用于精细化执行和观察，不作为波段方向依据。

## 4. 市场状态先行

每日先判断状态，再解释形态：

| 状态 | 定义 | 蜡烛图行动 |
| --- | --- | --- |
| 多头趋势 | 价格在200日线上方，50日线不弱，日线高低点抬高 | 看涨形态可找买点；看跌形态只减仓或收紧止损 |
| 空头趋势 | 价格在200日线下方，50日线不强，日线低点降低 | 看跌形态可找空头/避险；看涨形态只平空或观察 |
| 震荡区间 | 价格反复穿越均线，20日高低点无延续 | 只在区间边界使用形态，中间位置不交易 |
| 高波动失控 | 长影线、放量失败、ATR抬升频繁 | 降仓或暂停，等待日线收盘确认 |

主趋势与形态冲突时，服从主趋势。

## 5. 证据等级

| 等级 | 条件 | 动作 |
| --- | --- | --- |
| S级 | 周线不逆向 + 日线关键区域 + A类形态 + 次根确认 + 风险收益达标 | 可执行标准仓位 |
| A级 | 周线不逆向 + 日线关键区域 + B类预警后价格结构确认 | 半仓或试探仓 |
| B级 | 有形态和位置，但缺少次根确认或成交量 | 记录观察，不入场 |
| C级 | 只有形态名称，没有趋势、位置、确认 | 忽略 |
| 风险级 | 反向A类形态被确认，或跌回突破区间 | 减仓、退出或取消计划 |

只有S级和A级能进入交易计划。B级以下只能写进观察列表。

## 6. 可交易形态分层

| 层级 | 形态/信号 | 动作 |
| --- | --- | --- |
| A类硬候选 | 看涨/看跌吞没、锤子/上吊线、刺透/乌云放宽口径、启明星/黄昏星放宽口径 | 必须在关键区域，且等次根确认 |
| B类预警 | 十字线、长腿十字、墓碑、流星、倒锤子、平头、孕线/十字孕线 | 先减仓或观察，不直接满仓入场 |
| C类持仓管理 | 白色三兵、三只乌鸦、上升/下降三法、前方受阻/停顿 | 用于加仓、持有、减仓或移动止损 |
| 降级形态 | 窗口、弃婴、三空、严格跳空星线 | BTC日线严格样本少，只保留概念，不做硬规则 |

日线严格窗口样本数为 0，所以凡依赖跳空的传统形态都要改写或放弃。

## 7. 形态客观口径

执行时用客观口径减少主观解释：

| 组件 | 建议口径 |
| --- | --- |
| 长实体 | 实体大于过去20根平均实体的1.2倍 |
| 小实体/星线 | 实体小于过去20根平均实体的0.65倍 |
| 十字线 | 实体小于当根全幅的8%，或小于过去20根平均振幅的3.5% |
| 锤子/上吊线 | 下影至少为实体2倍，上影不超过实体0.55倍，实体位于全幅上半区 |
| 流星/倒锤子 | 上影至少为实体2倍，下影不超过实体0.55倍，实体位于全幅下半区 |
| 吞没 | 第二根实体覆盖第一根实体，不要求吞没影线 |
| 关键区域 | 20/55日高低点、前高前低、50/200日线、趋势线、ATR容忍区 |
| 次根确认 | 看涨收高或收回形态高点；看跌收低或跌破形态低点 |

这些口径是执行标准，不是美学标准。形态不像教科书时，可以观察，但不能硬算为交易触发。

## 8. 标准入场模型

只在满足硬门槛后考虑入场：

1. 周线不明显逆向。
2. 日线价格到达关键区域：20日/55日高低点、前高前低、50/200日线、趋势线或ATR支撑阻力。
3. 出现A类形态，或B类预警后出现价格结构确认。
4. 下一根日线顺形态方向确认：看涨时收高或收回形态高点，看跌时收低或跌破形态低点。
5. 止损距离可接受，单笔计划亏损不超过账户权益0.5%-1.5%。

### 看涨模型

条件：

1. 主趋势多头或周线不逆向。
2. 日线回撤到支撑区，而不是在半空中追形态。
3. 出现锤子线、看涨吞没、刺透放宽口径、启明星放宽口径，或“十字/孕线预警后重新收强”。
4. 次日确认后才入场；若次日不能确认，信号降级为观察。
5. 止损放在形态最低点下方、关键支撑下方或1.5-2.5ATR外。

### 看跌/避险模型

条件：

1. 主趋势空头，或多头持仓已经到达强阻力区。
2. 日线反弹到阻力区，而不是在低位追空。
3. 出现上吊线、看跌吞没、乌云盖顶放宽口径、黄昏星放宽口径、墓碑/长腿十字。
4. 次日确认后做空或减仓；多头趋势中优先减仓，不直接反手。
5. 止损放在形态最高点上方、关键阻力上方或ATR外。

## 9. 形态确认规则

| 信号 | 必须确认方式 |
| --- | --- |
| 上吊线 | 下一根收低、跌破实体或跌破形态低点 |
| 倒锤子线 | 下一根收高、站上实体或站上形态高点 |
| 十字线/长腿十字 | 下一根给出方向，不提前押注 |
| 孕线 | 等待突破母线高低点，区间内不动 |
| 锤子线 | 若前面跌势猛烈，必须等收高确认 |
| 吞没形态 | 可视作较强信号，但仍要求位置和风险满足 |
| 启明星/黄昏星放宽口径 | 第三根深入第一根实体后，最好再看一根是否延续 |

确认失败时不允许把“预警”改口成“提前布局”。

## 10. 多工具汇聚评分

先过硬门槛，再打分。硬门槛包括：

1. 有前置趋势或清晰区间边界。
2. 价格处在关键区域。
3. 有可写清的止损位。
4. 单笔计划亏损不超过账户权益1.5%。
5. 跳空类形态不作为硬触发。

硬门槛通过后打分：

| 项目 | 分值 | 条件 |
| --- | --- | --- |
| 周线背景 | 2 | 与交易方向同向，或至少不逆向 |
| 日线位置 | 2 | 位于明确支撑/阻力、20/55高低点、50/200均线或趋势线 |
| 蜡烛图形态 | 2 | A类形态给2分，B类预警给1分 |
| 次根确认 | 1 | 下一根K线顺形态方向确认 |
| 成交量 | 1 | 形态或确认K线达到20日均量1.2倍以上 |
| 动量背离 | 1 | RSI/MACD背离与形态方向一致 |
| 风险收益比 | 1 | 第一目标至少1.5R，最好2R以上 |

执行阈值：

- 8-10分：标准仓位。
- 6-7分：半仓或试探仓。
- 低于6分：不交易。
- B类预警即使高分，也必须等价格结构确认。
- 缺少止损或风险收益比不足时，评分自动作废。

## 11. 仓位与风险

每笔交易先算亏损，再算数量：

| 项目 | 规则 |
| --- | --- |
| 单笔计划亏损 | 账户权益的0.5%-1.5%，高波动期降至0.25%-0.75% |
| 止损距离 | 形态高低点外侧、关键位外侧，或1.5-2.5ATR |
| 最低风险收益比 | 第一目标至少1.5R，最好2R以上 |
| 加仓 | 只加盈利仓，且必须出现新确认或回测守住 |
| 暂停条件 | 连续2次确认失败，或连续3笔计划亏损 |

头寸公式：

| 步骤 | 公式 |
| --- | --- |
| 账户最大亏损 | 账户权益 × 单笔风险比例 |
| 每BTC风险 | `abs(入场价 - 止损价)` |
| 可交易数量 | 账户最大亏损 / 每BTC风险 |
| 杠杆检查 | 强平价必须远离结构止损，否则放弃 |

## 12. 止损与退出

先定义失败，再定义盈利：

1. 看涨形态跌破形态低点或关键支撑，退出。
2. 看跌形态突破形态高点或关键阻力，退出。
3. 入场后3-5根日线没有延续，且回到原区间，退出或降仓。
4. 出现相反方向A类形态并被次根确认，至少减仓。
5. 盈利超过2R后，止损至少移动到入场价或最近结构低点/高点。
6. 不因单根反向小K线退出趋势仓，只因结构破坏退出。

## 13. BTC适配规则

- BTC连续交易，传统“窗口/跳空”不可作为日线硬规则。
- 启明星、黄昏星、刺透、乌云盖顶可使用“无跳空放宽口径”，但权重低于原版。
- 4小时适合确认日线形态，1小时只适合优化价格。
- 高杠杆会放大长影线噪声，止损必须按结构和ATR设置。
- 周末、低流动性时段的短周期形态降权。

## 14. 禁用规则

- 不在没有前置趋势的位置解释反转形态。
- 不把锤子线、十字线、孕线、流星线单独当作入场按钮。
- 不把BTC日线不存在的跳空硬套成窗口、弃婴或三空。
- 不在周线逆向时用日线形态重仓反手。
- 不因RSI超买/超卖直接交易，必须等蜡烛图和价格结构确认。
- 不为了提高仓位缩窄止损。
- 不在连续两次确认失败后继续交易同一方向的弱形态。
- 不把低样本形态的高命中率当成稳定优势。
- 不用4小时信号逆向推翻周线和日线计划。

## 15. 日常执行清单

1. 标记周线方向：20周均线、高低点、周线长影/吞没/十字。
2. 标记日线关键位：20/55日高低点、50/200日线、前高前低、ATR。
3. 判断状态：趋势、震荡或高波动失控。
4. 只在关键区域寻找形态。
5. 区分A类硬候选、B类预警、C类持仓管理和降级形态。
6. 等次根确认；没有确认就不执行。
7. 写入入场、止损、目标、仓位和失效条件。
8. 入场后3-5根日线复核是否延续。
9. 复盘记录：形态、位置、确认、成交量、评分、执行是否违规。

## 16. 复盘模板

| 字段 | 内容 |
| --- | --- |
| 市场状态 | 多头/空头/震荡/高波动失控 |
| 周线背景 | 同向/逆向/中性 |
| 日线关键区域 | 支撑、阻力、均线、前高前低、ATR区域 |
| 形态层级 | A类硬候选/B类预警/C类持仓管理/降级形态 |
| 次根确认 | 已确认/失败/等待 |
| 成交量 | 放量/正常/缩量 |
| 动量 | RSI/MACD同向、背离或无信号 |
| 评分与等级 | S/A/B/C/风险级 |
| 入场与止损 | 入场价、止损价、每BTC风险 |
| 仓位 | 账户风险比例、数量、杠杆检查 |
| 退出计划 | 失败退出、第一目标、移动止损 |
| 执行复盘 | 是否违反趋势、位置、确认、仓位或止损纪律 |

## 17. 一句话版本

BTC蜡烛图只做“关键区域里的趋势变化预警”：周线定方向，日线等形态，次日做确认，4小时优化执行；没有关键位、没有确认、没有止损的蜡烛图信号，一律不交易。
