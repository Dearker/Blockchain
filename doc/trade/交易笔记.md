# 斐波拉契

### 画法

1. **定方向**：涨→0在下1在上，跌→0在上1在下。"1"=价格要去的方向，"0"=价格开始的方向
2. **定0.618**：让0.618尽可能触碰K柱最密集区域，0.618~1之间作为缓冲垫
3. **取段规则**：优先取最近的完整趋势段（一段上涨+一段回调）。大趋势另画一套fib，两套**方向一致时优先小级别**
4. **验证**：画好后观察K线是否高频触碰各级别点位，若不贴合则调整0.618位置

### 关键回撤位

| 回撤位 | 强度 | 操作含义 |
|--------|------|----------|
| **0.618** | 强 | 核心位——趋势延续确认/反转关键 |
| **0.382** | 强 | 与0.618对称，同等重要 |
| 0.5 | 次 | 过渡位，不作为独立依据 |
| 0.236 / 0.786 | 弱 | 辅助参考 |

### 交易规则

| 场景 | 条件 | 操作 |
|------|------|------|
| **趋势延续** | 0.618/0.382处企稳（≥2根K线未破）+ 量能配合 | 顺势开仓 |
| **趋势反转** | 收盘价破0.618 + 后续K线不回落 + 量能放大 | 反向开仓 |
| **无效信号** | 0.5附近犹豫 / 0.618~0.786之间震荡 | 不操作 |
| **多层fib共振** | 大小级别fib关键位重合 | 信号升级 |

### 大小级别fib冲突处理

- 两套fib**方向一致**时优先小级别（更精确）
- 两套fib**方向矛盾**时（大级别看多、小级别看空）→ **等小级别转一致再入场**
- 永远不逆大级别fib方向开仓

### fib趋势时间

#### 核心节奏

三拍子华尔兹——**长短长**：0→0.382=长，0.382→0.618=短，0.618→1=长

#### 画法

1. 设**三个锚点**（第一、二锚点最关键），微调即"牵一发而动全身"
2. 让前面部分尽可能完美落在**有量能且波动较大的K柱**上
3. 适用周期：**1~2周**，不适合太长

#### 三条规律

1. **0.618位置是波动高发区**
2. 竖线前后**两个单位时间**均有效
3. 波动方向需结合量能和fib回撤综合判断

#### 时间量化规律

- **25-27小时规律**：1H快慢线长时间缠绕时，约25-27小时后出现最低点。缠绕时间较短且无量能 → 可能二次下跌；需观察4H是否金叉（至少3根4H K线确认），无量能则二次下跌概率大
- **4H缠绕判断**：4H MACD长时间缠绕无法转变为金叉 → 注意可能二次下跌，需对比多币种情况确认
- 适用周期1~2周，超出则参考价值下降

#### 关键限制

- **仅预测波动时间，不预测方向和点位**
- 不单独开仓，必须与fib回撤+量能共振
- 未到时间节点 + 价格已到回撤位 → **等待**，不提前入场

# 量能

### 基础判读

> **量在价先。量能放大的方向 = 真实方向。**

| 现象 | 操作指向 |
|------|----------|
| 放量上涨 | 看多 |
| 放量下跌 | 看空 |
| 放量后缩量 | 警惕转折 |
| 缩量盘整 | 方向未定，等突破 |
| 天量（极端放量） | 能量过载，准备反向 |
| 量价背离 | 假突破，不追 |

### 主次量能

| 级别 | 缩写 | 量化标准 | 作用 |
|------|------|----------|------|
| **主量能** | M | ≥ 近20根量柱均值的**2倍** | **决定趋势方向** |
| 次量能（同向） | H | 均值的1~2倍 | 为主量能铺路巩固 |
| 次量能（反向） | L | ≤ 均值 | 抵消对方，夯实趋势 |

运作逻辑：M定方向，H巩固大趋势、L消耗对方。马太效应——强者越强，至力竭方被逆转。

**要点**：
- **单看一方量能不能判断拐点**——需同时观察多空双方
- 判断的是**波浪起伏**，不是直角转弯
- **0.618/0.382不是拐点位置**——到达不等于可反向开单
- 放慢交易速度（按周），更容易判断趋势

### 多周期分析

4H定方向（1~4周）→ 2H确认（3~7天）→ 1H找入场（1~3天）。**大周期优先。**

周期冲突规则：4H看多但1H看空 → 等1H转多再入场；4H看空则不做多。

### 量能精进

焦点：**单个K柱**中多空的绝对/相对强势。价值 = **提供更优的开平仓时机和点位**，而非判断走势方向。

| 等级 | 识别 | 处理 |
|------|------|------|
| **绝对强势** | 长实心柱体或长影线收线 | 重点参考——可判断短期走势与开平仓点 |
| **相对强势** | 己方影线远长于实心K柱（即使对方收线）；或实心K柱与影线旗鼓相当 | 必须纳入参考 |
| **均势** | 十字线 | 做好拐点准备 |
| **平庸** | 无明显特征 | 可忽略 |

- 多头绝对强势 = 长下影线或长实心阳柱收线
- 空头绝对强势 = 长上影线或长实心阴柱收线
- 须结合fib/MACD，忌频繁激进

# MACD

### 短线规则

| 场景 | 条件 | 操作 |
|------|------|------|
| 4H金叉做多 | 4H放量金叉 + 1H DIF>0 | 回踩开多，跌破金叉时对应的4H K线最低点则止损 |
| 4H金叉做多(共振版) | 4H放量金叉 + 1H DIF>0 + 日线DIF>0 | 回踩开多，高胜率模式，跌破金叉K线最低点止损 |
| 1H金叉做多 | 1H放量金叉 | 回踩开多 |
| 死叉不做多 | 4H死叉不开多单；1H死叉+阻力位/放量 | 可看情况开空 |
| 4H十字星+底部盘整 | 4H十字星+无量+支撑位附近 | 最低点下方150点挂多单（止损取最低点，比特币） |
| 15min放量下跌 | 15min放量下跌 | 观察3根K线后决定，参考清算地图 |
| 4H死叉7根不跌 | 4H死叉后7根K线跌不下去 | 尝试开多 |
| 1H金叉后动能衰减 | 柱状图连续3根缩短但未翻绿 | 减仓或推止损 |
| 多周期金叉共振 | 4H零轴上金叉 + 1H零轴上金叉 | 做多信号强化 |
| 多周期冲突 | 4H零轴下 + 1H金叉 | 仅看反弹，不做多 |
| 日/周K已金叉+4H假金叉 | 日K或周K已金叉，4H快要金叉时放量下跌 | 一般最多出现3次，等第4次放量金叉做多（前3次是假金叉诱多） |

### 中长线规则

| 场景 | 条件 | 操作 |
|------|------|------|
| 3日线金叉 | 3日线金叉 | 做多，配合清算地图确定入场价 |
| 日线死叉 | 日线死叉 | 下跌趋势形成，不做多 |
| 日线连阳 | 日线4-5连阳 | 开多，不做空 |
| 4H高位死叉 | 4H零轴上高位死叉 | 反弹开空，观察2-3根K线；消化需3-5天，多头入场配合支撑位+清算地图 |
| 日线高位十字星 | 日线高位十字星 | 开空；低位则开多 |
| 日线顶背离 | 价格创新高但DIF未创新高 | 中长期顶部信号，做空或减仓 |
| 日线底背离 | 价格创新低但DIF未创新低 | 中长期底部信号，做多 |
| 多周期共振 | 3日线金叉 + 日线金叉同时出现 | 强多头信号，仓位可至10%；3日线死叉时任何反弹不做多 |
| 周线方向 | 周线零轴上方 | 中长期多头，回调即买点；零轴下方=空头，反弹即卖点 |

> **零轴上方死叉本质**：12EMA仍大于26EMA，整体仍处上升趋势，死叉仅代表短期动能减弱。距离零轴越远趋势越强，死叉可能只是回调；距离零轴越近趋势越弱，反转可能性增大。

### 柱状图实战规则

| 阶段 | 信号 | 操作 |
|------|------|------|
| 柱状图开始缩短 | 趋势减速预警 | 提高警觉，准备行动 |
| 柱状图缩短50% | 动能明显衰减 | 减仓50% |
| 柱状图翻色 | 趋势确认转向 | 全部平仓 |
| 柱状图反向扩大 | 新趋势确认 | 可考虑反向建仓 |
| 柱状图背离 | 价格创新高/低但柱状图未创新高/低 | **最早期的趋势衰竭警告**，比金叉/死叉更早 |

### 背离确认条件

**必须全部满足才有效**：

1. 价格创新高/低，但MACD（DIF或柱状图）未创新高/低
2. 至少识别**2个明显对应的高点/低点**
3. 等待MACD**金叉/死叉确认**——背离只是预警，不可直接入场
4. 配合价格反转K线形态（如Pin Bar/十字星）二次确认

> ❌ 常见错误：一看到背离就立即入场，没有等待确认信号，经常被套在趋势延续中。

### MACD与fib共振规则

| 场景 | MACD信号 | fib位置 | 操作 |
|------|----------|---------|------|
| 多头入场 | 4H金叉 + 1H金叉 | 0.618/0.382企稳 | 强做多，仓位5%-10% |
| 多头确认 | 4H金叉 | 0.382附近放量 | 标准做多，仓位5% |
| 空头入场 | 4H死叉 + 1H死叉 | 跌破0.618放量 | 强做空，仓位5%-10% |
| 假信号过滤 | 1H死叉但DIF<200 | 在0.382上方 | 不追空，观望 |

### 多周期MACD配合

| 层级 | 时间框架 | 功能 |
|------|---------|------|
| 战略层 | 日线 | 确定主趋势方向，只做顺势交易 |
| 战术层 | 4H | 寻找入场时机，等待同方向信号 |
| 执行层 | 1H | 精确入场点，金叉/死叉确认入场 |

三周期同时金叉/死叉 = **最高质量信号**。不一致时减仓或不做。

# 清算地图

### 工具

- [Coinglass清算热力图](https://www.coinglass.com/zh/pro/futures/LiquidationMap)
- [Chaincatcher流动性搜索](https://www.chaincatcher.com/search?search=若btc)

### 读取方法

| 元素 | 含义 |
|------|------|
| 当前价格线 | 上下分界线 |
| 亮色区域（黄/红） | 该价位清算资金量大 = **流动性密集区** |
| 价格上方亮色 | 空头清算密集区——价格涨至此 → 空头被强平 → 轧空上涨 |
| 价格下方亮色 | 多头清算密集区——价格跌至此 → 多头被强平 → 瀑布下跌 |

### 价格磁铁效应

高额清算区 = 大量流动性聚集 → 对价格有天然吸引力。巨鲸主动将价格推向清算区 → 触发连环强平 → 强平订单提供充足流动性 → 巨鲸以更优价格完成交易。**极高清算点1-2天内大概率被清算**。

### 实战操作规则

| 场景 | 条件 | 操作 |
|------|------|------|
| 辅助确认入场价 | fib关键位 + 清算密集区重合 | 入场价精确到清算密集区 |
| 止损避开清算区 | 计划止损位附近有密集清算 | 止损设在清算区**远端**，避免被插针打掉 |
| 顺势突破 | 放量突破大型清算区 | 该方向障碍已清除，可顺势跟进 |
| 3日线金叉入场 | 3日线金叉 + 清算地图确认 | 在清算密集区附近挂单入场 |
| 15min放量下跌 | 15min放量下跌 | 观察3根K线后参考清算地图找支撑 |
| 逆势预警 | 持仓方向前方有巨大清算墙 | 高度警惕，收紧止损 |

### 时间周期

- 短期以**1天和7天**为主
- 如果出现**非常高的清算点位** → 极大概率在1-2天内清算，重点关注
- 极高清算点可参考作为**具体的止盈位或开仓位**

### chaincatcher跌破/突破位

- chaincatcher中搜索到的**跌破位和突破位**，一般大概率会到达
- 可参考作为**开单点位**：跌破位附近挂空单，突破位附近挂多单
- 需配合量能确认：无量到达可能是假突破

### 与其他方法论的配合

- **fib共振**：fib 0.618/0.382 + 清算密集区重合 → 极强支撑/阻力，信号升级
- **MACD配合**：3日线/4H金叉 → 清算地图确定具体入场价
- **量能验证**：突破清算密集区需放量确认，无量=假突破
- **止盈/止损**：极高清算点作止盈目标；止损避开清算密集区

### 注意事项

- 清算地图是**概率工具**，非精确预测
- 数据有**滞后性**，反映已存在仓位，无法捕捉实时新建仓
- 清算密集区随市场**动态变化**
- 最适用于BTC、ETH等高流动性品种，山寨币参考价值有限
- **不单独产生交易信号**，用于验证判断 + 优化入场/止损

# 趋势线

### 基本规则

1. 时间周期1h、4h、1日
2. 画法：连接相邻低点（上升趋势线）或高点（下降趋势线），需至少3个触点确认有效
3. 趋势线被放量突破后，原支撑变阻力、原阻力变支撑
4. 趋势线 + fib关键位重合 → 信号强化（详见"与fib的配合"）

### 画法细节

**上升趋势线**：
- 连接**两个或以上逐步抬高的低点**，向右延伸
- 第3个触点确认有效，触点越多越可靠
- 不被K柱实体穿透，影线穿透不超过2次可接受

**下降趋势线**：
- 连接**两个或以上逐步降低的高点**，向右延伸
- 同样需要第3个触点确认

**趋势线角度**：
- 45度左右最稳健——过陡（>60度）容易被突破，过平（<30度）趋势力度弱
- 角度变陡 = 趋势加速，可能接近末端；角度变平 = 趋势减弱，可能转向

### 验证与失效

| 信号 | 判断 | 操作 |
|------|------|------|
| 放量突破趋势线 | 有效突破 | 原支撑变阻力/原阻力变支撑 |
| 缩量突破趋势线 | 假突破可能 | 观望，等回踩确认 |
| 趋势线上方回踩不破 | 趋势延续 | 在趋势线附近做多 |
| 趋势线下方反弹不破 | 趋势延续 | 在趋势线附近做空 |
| 连续3次触及未突破 | 趋势线有效 | 下一次触及大概率反弹 |
| 影线多次穿透趋势线 | 趋势线失效 | 需重新画线 |

### 与fib的配合

- 上升趋势线 + fib 0.618回撤位重合 = 多头最佳入场区
- 下降趋势线 + fib 0.618回撤位重合 = 空头最佳入场区
- 趋势线突破位置在fib关键位附近 = 突破更可靠

# 维加斯通道

### 参数设置

- 时间周期1h、1日
- 由EMA 144和EMA 169构成，两线之间区域为通道

### 核心用法

**趋势判断**：
- 价格持续在EMA 144上方 = 强多头，回调到EMA 144附近是买点
- 价格持续在EMA 169下方 = 强空头，反弹到EMA 169附近是卖点
- 价格在两线之间震荡 = 趋势不明，观望

**入场规则**：
- 多头入场：价格从下方穿越通道后，第一次回踩EMA 144不破 = 买点；EMA 144与fib 0.618重合时信号更强
- 空头入场：价格从上方穿越通道后，第一次反弹EMA 169不过 = 卖点；EMA 169与fib 0.618重合时信号更强
- 价格在通道中运行时不开仓，等明确穿越方向

**离场规则**：
- 多头持仓中，价格跌破EMA 169 = 趋势反转，止盈离场
- 空头持仓中，价格突破EMA 144 = 趋势反转，止盈离场
- 价格在通道内震荡超过3根K线 = 趋势减弱，减仓

### 多时间框架确认

- 日线维加斯通道方向定中期趋势
- 1H维加斯通道方向定短期趋势
- 日线多头 + 1H回踩通道不破 = 精确入场点

# 月度理论

### 月度时间节奏

1. 月底处高/低位 → 下月初第一周周三左右出现当月低点
2. 每月中下旬出现当月高点
3. 月度周期与fib趋势时间配合：月度时间节点与fib趋势时间节点重合 → 波动概率更高

### 月度规律细化

**月初规律（1日~10日）**：
- 上月收盘在高位 → 月初1-3日常出现回调，周三左右容易出现当月低点，具体开仓点位可找就近的支撑位
- 上月收盘在低位 → 月初直接反弹概率大，周三左右可能是加仓点，具体开仓点位可找就近的支撑位
- 月初的低点通常是当月最佳多头入场时机
- 月初如果在高位，则这可能是当月的最高位

**月中规律（10日~20日）**：
- 月中通常是趋势运行主阶段，月初确立的方向在此加速
- 月初反弹 → 月中往往出现当月高点
- 月中高点放量冲高后缩量 → 警惕转折

**月末规律（20日~月底）**：
- 月中下旬出现当月高点概率较大
- 月末高位放量 → 可能继续上涨；缩量 → 下月初回调概率大
- 月末低位 → 下月初反弹概率大

### 月度与周度共振

- 月末/月初时间节点 + 周五美股结算日 = 容易出现较大波动
- 月度时间节点 + CME期货交割日（当月最后一个周五）= 波动概率极高
- 月度低点 + fib 0.618企稳 + 4H量能M指向多头 = 月度级别最佳做多入场点
- **月K开盘价**作为月中支撑/阻力参考：价格在月K开盘价上方运行 → 月内偏多，回踩开盘价不破可加仓；价格在月K开盘价下方运行 → 月内偏空

# 支撑位和阻力位

### 基本规则

1. 时间周期1h、4h和1日
2. 下方出现盘整区且未跌破 → 后续作为支撑位；就近上方高点作为阻力，一般取上方4h盘整区最低点为具体阻力位
3. 盘整区跌破 → 下一盘整区作为支撑位；盘整区突破 → 回撤时该盘整区转为支撑位
4. 4h持续突破 → 以日线高点作为阻力位
5. 日线收盘在关键支撑位之上 → 可考虑开多
6. **周K和月K的最低点和最高点**，可作为支撑位和阻力位（详见下方使用规则）
7. 周期越大支撑阻力越强：1h < 4h < 1日 < 周K < 月K
8. 如果15分钟/1h级别放量跌破支撑位，则还后续还要下跌，如果是挂单则需要反弹之后直接离场，放量突破同理
9. 突破第一个阻力位后，一般会在第二个阻力位附近开始受阻回落，将之前的阻力位转换成支撑位
10. 同一个阻力位测试两次即以上无法突破，则大概率会继续下跌；同一个支撑位最多被测试2次，第3次必然跌穿
11. 4小时级别超过2根及以上k线在阻力位盘整无法突破，后续大概率会下跌

### 周K/月K高低点的使用规则

| 要点 | 说明 |
|------|------|
| 强度差异 | 月K > 周K > 日K，周期越大突破越难 |
| 需量能确认 | 放量企稳才有效，缩量触碰不可靠 |
| 需多方法共振 | + fib关键位重合 = 信号升级，单独出现仅参考 |
| 突破判定更严 | 需日线收盘+放量确认，1H/4H影线穿透不算 |
| 时效性 | 超过3个月未测试的效力明显衰减 |
| 假突破高发 | 流动性密集区易被插针，需等收盘确认 |

**实战操作**：
- 周K低点 + fib 0.618企稳 + 4H量能M指向多头 → 中线做多，止损放在周K低点下方
- 月K高点 + fib关键位受阻 + MACD死叉 → 中线做空，止损放在月K高点上方
- 周/月K高低点作为**止损参考**极有价值——突破意味着中期趋势转向

### fib关键位作为最强支撑阻力

- fib 0.618/0.382 = **最强支撑阻力位**——与盘整区/前高前低重合时效果倍增
- 量能确认：放量突破阻力位=有效突破可跟单；缩量突破=假突破不追；放量跌破支撑位=有效跌破趋势反转；缩量跌破=可能诱空等待确认

### 支撑阻力层级体系

| 层级 | 类型 | 强度 | 说明 |
|------|------|------|------|
| 1 | fib 0.618/0.382 + 盘整区重合 | 极强 | 多重确认，最可靠的操作依据 |
| 2 | 月K/周K高低点 + fib关键位重合 | 极强 | 大周期+关键位双重确认 |
| 3 | 日线盘整区 + 前高/前低 | 强 | 大周期确认，中长线参考 |
| 4 | 4H盘整区 | 中 | 短线操作的主要依据 |
| 5 | fib 0.5 / 0.236 / 0.786 | 弱 | 辅助参考，不独立使用 |
| 6 | 整数关口（如BTC万位关口） | 辅助 | 心理层面，量能配合才有效 |

**使用原则**：开仓优先用层级1-3；层级4作短线辅助；层级5-6不独立开仓，仅辅助确认

### 突破与假突破识别

**有效突破判定**：

| 条件 | 突破 | 假突破 |
|------|------|--------|
| K线收盘 | 收盘价站稳关键位上方 | 仅影线穿透，收盘回来 |
| 量能 | 放量突破 | 缩量突破 |
| 回踩确认 | 突破后回踩不破 = 有效 | 回踩跌破 = 假突破 |
| 时间 | 突破后2-3根K线维持 | 1根K线穿透后回落 |

**假突破的应对**：
- 缩量突破阻力位后回落 → 阻力位下方开空，止损放在突破影线上方
- 缩量跌破支撑位后回升 → 支撑位上方开多，止损放在跌破影线下方
- 假突破往往是主力测试流动性区域，方向通常与假突破方向相反

### 与维加斯通道的关系

- EMA 144 = 动态阻力位（空头反弹极限），EMA 169 = 动态支撑位（多头回调极限）
- EMA 144/169 + 静态阻力位重合 = 双重阻力，突破难度大但一旦放量突破空间也大
- EMA 144/169 + 静态支撑位重合 = 双重支撑，跌破难度大但一旦放量跌破下行空间也大

### 时效性

支撑阻力位随时间衰减——越久未被测试的支撑阻力位，效力越弱。最近1-2周内被测试过的支撑阻力位最有效。

# 开仓决策流程

### 开仓逻辑

开仓依赖以下方法论的综合判断，单一方法论不构成开仓依据：

| 方法论 | 作用 | 具体工具 |
|--------|------|----------|
| **斐波拉契** | 确定价格位置和方向 | fib回撤（0.618/0.382关键位）+ fib趋势时间（波动时间节点） |
| **量能** | 确认多空方向 | 主次量能M/H/L + 量能精进（K柱强弱）+ 量价判读 |
| **MACD** | 辅助确认趋势方向 | 金叉/死叉 + DIF数值 + 柱状图 + 顶底背离 |
| **支撑阻力位** | 确定具体入场价格 | 盘整区 + fib关键位重合 + 突破/假突破判定 |
| **趋势线** | 确认趋势方向和强度 | 趋势线画法 + 与fib重合判断 + 突破确认 |
| **维加斯通道** | 判断中期趋势 | EMA 144/169方向 + 穿越/回踩确认 |
| **月度理论** | 确认时间节点 | 月初低点/月中高点 + 与fib趋势时间共振 |
| **清算地图** | 确认流动性密集区 | coinglass/chaincatcher + 价格磁铁效应 |

### 开仓点位

#### 做多开仓点位

| 开仓场景 | 开仓点位 | 依赖方法论 | 确认条件 |
|----------|----------|------------|----------|
| fib回撤企稳做多 | fib 0.618 或 0.382 处企稳（≥2根K线未破） | 斐波拉契 + 量能 | 0.618/0.382处量能M指向多头 + MACD金叉确认 |
| 支撑位反弹做多 | 支撑位（盘整区与fib重合）放量反弹 | 支撑阻力位 + 斐波拉契 | 放量阳柱收线在支撑位上方 + 缩量回踩不破 |
| 趋势线回踩做多 | 上升趋势线 + fib 0.618重合处 | 趋势线 + 斐波拉契 | 回踩趋势线不破 + 量能精进K柱绝对/相对强势 |
| 维加斯通道回踩做多 | 价格穿越EMA 144后第一次回踩不破 | 维加斯通道 + MACD | 日线维加斯通道多头 + 1H回踩EMA 144不破 + 1H金叉 |
| MACD金叉做多 | 4H放量金叉回踩 / 1H放量金叉回踩 | MACD + 量能 | 回踩fib关键位不破 + DIF值确认（死叉DIF<200=假摔） |
| 月度低点做多 | 月初周三左右低点 + fib关键位企稳 | 月度理论 + 斐波拉契 | 月末处于低位 + 月初低点在fib 0.618企稳 + 4H量能M指向多头 |
| 底背离做多 | 日线MACD底背离 + fib 0.618企稳 | MACD + 斐波拉契 | 价格创新低但DIF未创新低 + 0.618处出现绝对强势K柱 |

#### 做空开仓点位

| 开仓场景 | 开仓点位 | 依赖方法论 | 确认条件 |
|----------|----------|------------|----------|
| fib回撤破位做空 | 收盘价跌破fib 0.618 + 后续K线不回落 | 斐波拉契 + 量能 | 跌破0.618放量 + MACD死叉确认 |
| 阻力位回落做空 | 阻力位（盘整区与fib重合）放量受阻回落 | 支撑阻力位 + 斐波拉契 | 放量上影线收线在阻力位下方 + 缩量反弹不过 |
| 趋势线反弹做空 | 下降趋势线 + fib 0.618重合处 | 趋势线 + 斐波拉契 | 反弹趋势线不过 + 量能精进K柱空头强势 |
| 维加斯通道反弹做空 | 价格跌破EMA 169后第一次反弹不过 | 维加斯通道 + MACD | 日线维加斯通道空头 + 1H反弹EMA 169不过 + 1H死叉 |
| MACD死叉做空 | 4H零轴上高位死叉反弹开空 / 1H死叉 | MACD + 量能 | 反弹至fib关键位受阻 + DIF>600真下跌 + 观察2-3根K线 |
| 月度高点做空 | 月中下旬高点 + fib关键位受阻 | 月度理论 + 斐波拉契 | 月末高位缩量 + 高点在fib关键位放量受阻 |
| 顶背离做空 | 日线MACD顶背离 + fib关键位受阻 | MACD + 斐波拉契 | 价格创新高但DIF未创新高 + 关键位出现空头绝对强势K柱 |

### 止盈点位

| 入场位置 | 止盈目标位 | 条件 | 操作 |
|----------|------------|------|------|
| 0.382入场做多 | 目标0.618 | 到0.618缩量 | 止盈半仓，剩余推止损到成本价 |
| 0.618入场做多 | 目标1.0（前高）或维加斯通道上轨 | 到前高缩量 | 止盈半仓，剩余推止损到成本价 |
| 支撑位入场做多 | 目标下一fib关键位或清算密集区 | 到目标位缩量 | 止盈半仓，剩余推止损到成本价 |
| 周K/月K低点入场做多 | 目标fib 0.382或前高 | 到目标位缩量 | 止盈半仓，剩余推止损到成本价 |
| 0.618入场做空 | 目标0.382 | 到0.382缩量 | 止盈半仓，剩余推止损到成本价 |
| 0.382入场做空 | 目标0.236或前低 | 到前低缩量 | 止盈半仓，剩余推止损到成本价 |
| 阻力位入场做空 | 目标下一fib关键位或清算密集区 | 到目标位缩量 | 止盈半仓，剩余推止损到成本价 |
| 周K/月K高点入场做空 | 目标fib 0.618或前低 | 到目标位缩量 | 止盈半仓，剩余推止损到成本价 |

**全仓止盈条件**：
- 维加斯通道边缘 + 量能异常（天量或极端缩量）
- 量价背离（价格继续走但量能不配合）
- 开仓依据被破坏（逻辑止损触发）

### 止损规则

- **硬止损**：开仓时设定，1%~2%必止——**开单前先设止损**
- **逻辑止损**：开仓依据被破坏（反向fib关键位突破+放量、支撑位放量跌破、维加斯通道反向穿越）
- **清算地图止损**：止损位避开清算密集区，设在清算区远端

### 决策流程

```
1. 定方向
   4H量能M指向何方？ MACD金叉/死叉方向？ 维加斯通道方向？
       →  方向不明 → 不交易
       ↓ 方向明确

2. 找入场点位
   fib回撤在0.618/0.382附近？ 支撑阻力位是否确认？
   趋势线是否提供方向？ 清算地图流动性密集区在哪？
       →  不在关键位 → 等待
       ↓ 价格到位

3. 确认时机
   fib趋势时间竖线附近？ 月度时间节点（月初低点/月中高点）？
       →  未到 → 等待
       ↓ 时机到达

4. 执行开仓
   按开仓点位表确定具体入场价格
   清算地图确认入场价（fib关键位+清算密集区重合=精确入场）
   同时设定硬止损1%~2%（开单前先设，避开清算密集区）
   按止盈点位表确定止盈目标
       ↓ 持仓中

5. 持仓管理
   价格按预期运行 + 量能持续 → 推止损到成本
   价格回落破开仓位 + 量能放大 → 逻辑止损离场
   维加斯通道中运行 → 持仓不动
   量能精进判K柱强弱 → 短周期调仓
   到止盈目标位 + 缩量 → 按止盈规则操作
   量价背离 / 开仓依据被破坏 → 止盈全仓
```
