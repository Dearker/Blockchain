# BTC 量价分析：验证后可复用交易方法论

> 数据基础：Binance `BTCUSDT` 最近两年 K 线，验证区间 2024-06-28 至 2026-06-27。
> 适用对象：BTCUSDT 日线波段交易，4小时用于执行优化，1小时只用于观察细节；不构成投资建议。

## 1. 方法论定位

这套方法只保留《量价分析》中能被 BTC K 线复用并经两年样本检验后仍有执行价值的内容：成交量相对比较、实体与影线、确认与异常、支撑阻力突破、缩量回调、多时间跨度和交易计划。

“局内人吸筹/派筹”只作为解释语言，不作为证据语言。执行时统一改写为可观察表述：承接、卖压、突破确认、突破失败、缩量回调、放量异常。

## 2. 第一原则：量价信号必须落在位置里

同一根 K 线在上涨中段、上涨末端、下跌末端的含义不同。BTC 执行顺序固定为：

1. 周线判断背景：价格相对20周均线、是否处于大级别上升或下降环境。
2. 日线判断位置：趋势中段、震荡区间、突破边缘、上涨末端、下跌末端。
3. 单根 K 线判断形态：实体、上影线、下影线、收盘位置。
4. 成交量判断确认或异常：放量确认、缩量降级、低实体高量警示。
5. 4小时只用于优化触发和止损，不推翻日线计划。

## 3. 成交量定义

所有成交量都用相对值，不用绝对 BTC 数量：

| 类型 | 定义 | 用法 |
| --- | --- | --- |
| 低量 | 当前量 <= 20根均量的0.75-0.85倍 | 缩量回调、缩量反弹、缩量突破降级 |
| 正常量 | 当前量在20根均量的0.85-1.25倍 | 只做背景，不给强信号 |
| 高量 | 当前量 >= 20根均量的1.25-1.50倍 | 突破确认、长影线异常、承接/卖压 |
| 极高量 | 当前量处于过去180根的95%分位附近 | 潜在高峰信号，必须等待后续确认 |

阈值按周期微调：1小时噪声高，宁可把高量阈值提高；日线和4小时可使用1.25-1.50倍。

## 4. K 线判读规则

| 形态 | BTC 执行含义 | 处理 |
| --- | --- | --- |
| 高实体+高量 | 趋势推动较真实 | 顺主趋势可跟随，逆周线只观察 |
| 高实体+低量 | 推动缺乏参与 | 不追，等待下一根确认或回踩 |
| 低实体+高量 | 分歧、吸收或派发 | 单独不交易，结合长影线和位置加权 |
| 长上影+低实体+高量 | 上方卖压 | 上涨后减仓/收紧止损，突破位附近警惕失败 |
| 长下影+低实体+高量 | 恐慌释放或潜在承接 | 只触发观察，必须等重新站回关键位 |
| 缩量回调 | 原趋势可能未结束 | 持仓可忍受，不作为加仓充分条件 |
| 缩量突破 | 可信度不足 | 降级为观察，等待放量或回踩确认 |

两年样本中，日线“高实体+高量”后续顺向命中率为 54.39%，日线“高实体+低量”为 50.00%；这说明实体必须看量，但单根 K 线仍不足以替代趋势过滤。

## 5. 信号分层与状态机

VPA 的实盘问题不是“有没有信号”，而是“这个信号现在处于哪一层”。所有信号先分层，再决定动作：

| 层级 | 信号 | 动作 |
| --- | --- | --- |
| 硬触发 | 周线不逆向 + 日线放量突破 + 止损可控 | 可以进入交易计划 |
| 硬触发 | 上涨趋势缩量回调 + 4小时重新转强 | 可以进入交易计划 |
| 退出触发 | 上涨后长上影低实体高量、放量跌回突破区间、放量跌破50日线 | 先减仓或退出 |
| 观察信号 | 长下影低实体高量、低实体高量、单根高实体高量 | 只记录，不直接下单 |
| 禁用信号 | 缩量突破、周线逆向的日线突破、单根长下影抄底 | 不开新仓 |

执行状态机固定为六步：

1. 空仓观察：只标记周线背景和日线区间。
2. 候选信号：出现放量突破、缩量回调或长影高量。
3. 交易计划：写入入场触发、止损、第一目标和最大亏损。
4. 触发执行：只在收盘确认或4小时结构确认后执行。
5. 持仓管理：用缩量回调持仓，用放量失败或长上影高量保护利润。
6. 退出复盘：退出后记录信号层级、评分、亏损是否在计划内。

优先级永远是：风险和退出 > 周线背景 > 日线位置 > 成交量确认 > 4小时执行。低优先级信号不能推翻高优先级风险。

## 6. 入场规则

只做三类入场，其他信号都降级为观察。长下影高量本身不是入场，只有后续价格确认才可能升级为入场计划。

### A. 放量突破入场

条件：

1. 周线背景同向，或至少不是明显逆势。
2. 日线收盘突破20根高点或关键震荡区上沿。
3. 成交量 >= 20根均量的1.25倍。
4. 突破 K 线实体不能过小，收盘应靠近突破方向。
5. 止损可放在突破区间内侧或最近结构低点，单笔风险可控。

日线放量突破在周线同向时，本次样本命中率为 60.61%，平均结果 +1.44%，样本 33 个。若周线逆向，信号必须降级。

### B. 下跌末端二次确认入场

条件：

1. 前20根累计跌幅较大，价格接近前支撑或长期均线。
2. 出现长下影、低实体、高量，只记为观察信号，不立即买入。
3. 下一根或随后几根 K 线重新站回信号 K 线中位、前低或短均线。
4. 4小时形成更高低点，或放量收回短期压力位。
5. 止损放在长下影低点外侧，不允许跌破后继续补仓。

日线“长下影低实体高量/下跌后”样本命中率为 28.57%，平均结果 -3.36%。两年 BTC 样本不支持它作为直接买点，所以必须等待二次确认。

### C. 缩量回调后的趋势延续入场

条件：

1. 日线在50日/200日均线上方，50日均线走平或上行。
2. 回调阶段成交量低于20根均量，且未跌破关键结构。
3. 4小时重新放量收回短期压力位，或日线重新转强。
4. 止损放在回调低点下方。

日线“上涨趋势缩量回调”后续命中率为 56.90%，平均结果 +1.24%。它的核心用途是确认“回调不是趋势结束”。

## 7. 离场规则

离场优先于新增入场。出现以下任一情况，至少减仓：

1. 上涨20根以上后出现长上影、低实体、高量。
2. 放量突破后3-5根内无法延续，且跌回突破区间。
3. 日线跌破50日线，同时成交量高于20根均量。
4. 连续2-3次上攻都出现高量低实体或长上影。
5. 周线转弱，日线所有多头信号降级。

日线“长上影低实体高量/上涨后”样本命中率为 60.00%，平均结果 +4.81%。它更适合作为保护利润的警报，而不是机械做空信号。

## 8. 支撑阻力与突破流程

1. 用最近20-55根高低点标记震荡区间。
2. 区间内不预测方向，只记录每次靠近上沿/下沿时的量价反应。
3. 向上突破必须收盘站上区间，并放量；缩量突破不追。
4. 回踩原阻力时若缩量且不跌回区间，突破可信度提升。
5. 若突破后高量跌回区间，视为失败，先退出再观察。

日线20根通道放量突破命中率为 54.72%，缩量突破命中率为 12.50%。差异不代表放量突破必胜，只代表缩量突破必须降级。

## 9. 信号评分

每次入场前打分，低分不交易：

| 项目 | 分值 | 条件 |
| --- | --- | --- |
| 周线背景 | 2 | 周线在20周均线上方或突破方向同向 |
| 日线位置 | 2 | 位于震荡突破边缘、趋势健康回调或下跌末端二次确认区 |
| 成交量确认 | 2 | 放量突破、二次确认放量或回踩缩量 |
| K线结构 | 2 | 收盘位置、实体/影线与交易方向一致 |
| 风险收益比 | 1 | 第一目标至少1.5R |
| 4小时执行 | 1 | 4小时结构同向，未出现反向放量失败 |

执行阈值：

- 8-10分：允许标准仓位。
- 6-7分：只允许半仓或试探仓。
- 低于6分：不交易。
- 出现放量跌回突破区间、跌破结构止损、周线转弱时，评分自动失效。

## 10. 仓位管理

| 场景 | 仓位 |
| --- | --- |
| 周线同向 + 日线放量突破 + 回踩缩量成功 | 50%-100%计划仓位 |
| 下跌后长下影高量 + 二次价格确认 | 10%-30%试探仓 |
| 上涨趋势缩量回调后4小时转强 | 30%-70%计划仓位 |
| 周线逆向或日线缩量突破 | 0%-20%，默认观察 |
| 出现上涨后长上影高量 | 减仓30%-100%，视是否跌破结构 |

单笔风险建议控制在账户权益的0.5%-1.5%。BTC 波动大，不用“固定买多少 BTC”决定仓位，必须先用止损距离反推数量。

头寸计算：

| 步骤 | 公式或要求 |
| --- | --- |
| 账户最大亏损 | 账户权益 × 单笔风险比例 |
| 每 BTC 风险 | 入场价 - 止损价，做空则反向计算 |
| 可交易数量 | 账户最大亏损 / 每 BTC 风险 |
| 杠杆检查 | 强平价必须远离结构止损，否则降杠杆或放弃 |
| 总风险限制 | 同方向多笔仓位的合计计划亏损不超过账户权益的2%-3% |

若止损距离过宽导致仓位小到没有执行意义，不要把止损内移；正确动作是放弃交易。

## 11. 执行模板

| 字段 | 内容 |
| --- | --- |
| 周线背景 | 同向/中性/逆向 |
| 日线位置 | 突破、回调、支撑、上涨末端、下跌末端 |
| 信号层级 | 硬触发、退出触发、观察信号或禁用信号 |
| 量价信号 | 高实体高量、长下影高量、缩量回调、缩量突破等 |
| 信号评分 | 0-10分 |
| 入场触发 | 收盘突破、回踩确认、4小时转强 |
| 止损价 | 结构低点、突破区间内侧或信号K低点外侧 |
| 计划亏损 | 按账户权益百分比记录 |
| 第一目标 | 前高、区间等幅、2R或上方阻力 |
| 退出条件 | 长上影高量、跌回区间、跌破50日线、周线转弱 |
| 复盘字段 | 是否等确认、是否追缩量突破、是否按止损执行 |

## 12. 每日检查清单

- [ ] 周线是否支持当前方向？
- [ ] 日线在趋势中段、震荡区间、上涨末端还是下跌末端？
- [ ] 当前成交量相对20根均量是低、正常、高还是极高？
- [ ] 实体、上影线、下影线分别占全振幅多少？
- [ ] 当前信号属于硬触发、退出触发、观察信号还是禁用信号？
- [ ] 信号属于确认还是异常？
- [ ] 突破是否放量？回踩是否缩量？
- [ ] 是否存在长上影高量或长下影高量？
- [ ] 4小时是否同向，还是已经出现突破失败？
- [ ] 止损在哪里？若错了账户亏损是多少？
- [ ] 信号评分是否达到6分以上？

## 13. 禁用、降级与暂停规则

- 不把“主力正在吸筹/派筹”当作交易理由，只记录可见量价证据。
- 不在缩量突破时追单，除非随后放量确认或回踩成功。
- 不因单根长下影高量直接满仓抄底，必须等待价格收回。
- 不在周线逆向时使用日线信号做波段重仓。
- 不用普通 K 线替代 VAP、差量、累积差量；这些需要更细数据源。
- 不用新闻解释补强亏损交易，除非事前已把新闻事件纳入计划。

出现以下情况，暂停新开仓，只允许复盘和等待下一次日线结构：

- 连续3笔亏损，且至少2笔来自低于6分的交易。
- 最近5笔交易中有2笔以上违反“缩量突破不追”或“长下影不直接抄底”。
- 当前最大回撤达到预设月度亏损上限。
- 周线转弱，但仍想用日线信号做波段重仓。

## 14. 复盘与迭代规则

每20笔交易或每月复盘一次，只看四个问题：

1. 亏损是否主要来自低于6分的信号？
2. 是否追了缩量突破？
3. 是否在周线逆向时仍做了日线波段重仓？
4. 盈利单是否因单根普通阴线过早卖出，而不是等放量失败或结构破坏？

本次简化“日线VPA组合规则”总收益 0.63%、最大回撤 -15.83%、在场时间 10.27%；同期日线买入持有总收益 0.19%、最大回撤 -52.03%。这说明 VPA 的价值在于过滤和风控，而不是预测每根 K 线。

## 15. 一句话版本

先用周线决定能不能顺势交易，再用日线量价把信号分成硬触发、退出触发、观察或禁用，最后用4小时确认执行；放量突破才跟，缩量突破降级，长影高量只给警报，所有信号都必须服从位置、止损和仓位。
