# 自然交易理论——可复用交易方法论

---

## 一、分析框架

摒弃所有传统指标（MA/MACD/KDJ/BOLL/RSI），仅以**三要素**分析：

| 要素 | 工具 | 判定内容 |
|------|------|----------|
| **空间** | fib回撤 | 价格位置 |
| **时间** | fib趋势时间 | 波动发生的时间点 |
| **能量** | 量能分析 | 多空方向 |

**共振开仓规则**：至少两要素同向才可交易，仅一要素指向则旁观。

| 共振 | 仓位 |
|------|------|
| 三要素同向 | 10% |
| 两要素同向 | 5% |
| 仅一要素 | 不交易 |

---

## 二、空间：fib 回撤

### 画法

1. **定方向**：涨→0在下1在上，跌→0在上1在下。"1"=价格要去的方向，"0"=价格开始的方向
2. **定0.618**：让0.618尽可能触碰K柱最密集区域，0.618~1之间作为缓冲垫
3. **取段规则**：优先取最近的完整趋势段（一段上涨+一段回调）。大趋势另画一套fib，两套**方向一致时优先小级别**
4. **验证**：画好后观察K线是否高频触碰各级别点位，若不贴合则调整0.618位置

### 关键回撤位

| 回撤位 | 强度 | 操作含义 |
|--------|------|----------|
| **0.618** | 强引力 | 核心位——趋势延续确认/反转关键 |
| **0.382** | 强引力 | 与0.618对称，同等重要 |
| 0.5 | 次引力 | 过渡位，不作为独立依据 |
| 0.236 / 0.786 | 弱 | 辅助参考 |

### 交易规则

| 场景 | 条件 | 操作 |
|------|------|------|
| **趋势延续** | 0.618/0.382处企稳（≥2根K线未破） + 量能配合 | 顺势开仓 |
| **趋势反转** | 收盘价破0.618 + 后续K线不回落 + 量能放大 | 反向开仓 |
| **无效信号** | 0.5附近犹豫 / 0.618~0.786之间震荡 | 不操作 |
| **多层fib共振** | 大小级别fib关键位重合 | 信号升级（两共振→三共振） |

## 三、时间：fib 趋势时间

### 核心节奏

三拍子华尔兹——**长短长**：0→0.382=长，0.382→0.618=短，0.618→1=长

### 画法

1. 设**三个锚点**（第一、二锚点最关键），微调即"牵一发而动全身"
2. 让前面部分尽可能完美落在**有量能且波动较大的K柱**上
3. 适用周期：**1~2周**，不适合太长

### 三条规律

1. **0.618位置是波动高发区**
2. 竖线前后**两个单位时间**均有效
3. 波动方向需结合量能和fib回撤综合判断

### 关键限制

- **仅预测波动时间，不预测方向和点位**
- 不单独开仓，必须与空间/能量共振
- 未到时间节点 + 价格已到回撤位 → **等待**

## 四、能量：量能体系

### 4.1 基础判读

> **量在价先。量能放大的方向 = 真实方向。**

| 现象 | 操作指向 |
|------|----------|
| 放量上涨 | 看多 |
| 放量下跌 | 看空 |
| 放量后缩量 | 警惕转折 |
| 天量（极端放量） | 能量过载，准备反向 |
| 量价背离 | 假突破，不追 |

### 4.2 主次量能

| 级别 | 缩写 | 量化标准 | 作用 |
|------|------|----------|------|
| **主量能** | M | ≥ 近20根量柱均值的**2倍** | **决定趋势方向** |
| 次量能 | H | 均值的1~2倍 | 为主量能铺路巩固 |
| 次量能 | L | ≤ 均值 | 抵消对方，夯实趋势 |

运作逻辑：M定方向，H/L消耗对方、巩固大趋势。马太效应——强者越强，至力竭方被逆转。

**要点**：
- **单看一方量能不能判断拐点**——需同时观察多空双方
- 判断的是**波浪起伏**，不是直角转弯
- **0.618/0.382不是拐点位置**——到达不等于可反向开单
- 放慢交易速度（按周），更容易判断趋势

### 4.3 多周期分析

4H定方向（1~4周）→ 2H确认（3~7天）→ 1H找入场（1~3天）。**大周期优先。**

**周期冲突规则**：4H看多但1H看空 → 等1H转多再入场；4H看空则不做多。

### 4.4 量能精进

焦点：**单个K柱**中多空的绝对/相对强势。价值 = **提供更优的开平仓时机和点位**，而非判断走势方向。

| 等级 | 识别 | 处理 |
|------|------|------|
| **绝对强势** | 长实心柱体或长影线收线 | 重点参考——可判断短期走势与开平仓点 |
| **相对强势** | 己方影线远长于实心K柱（即使对方收线）；或实心K柱与影线旗鼓相当 | 必须纳入参考 |
| **均势** | 十字线 | 做好拐点准备 |
| **平庸** | 无明显特征 | 可忽略 |

- 多头绝对强势 = 长下影线或长实心阳柱收线
- 空头绝对强势 = 长上影线或长实心阴柱收线
- **必须结合三要素使用**，警惕频繁激进操作

## 五、交叉强化：引力与通道

### 5.1 引力

三要素在关键位产生**交叉强化效应**——价格被吸向引力点。

| 引力源 | 吸引力 |
|--------|--------|
| fib 0.618 / 0.382 | **强** |
| 量能放大点 | **强** |
| fib趋势时间点 | 中 |

**松紧带三阶段**：

| 阶段 | 现象 | 操作 |
|------|------|------|
| 吸引 | 价格被拉向引力点 | 不逆势 |
| 过载 | 冲过引力点 | 等待，不追 |
| 耗尽 | 过载后缩量平静，回弹 | **最佳开仓点**——沿回弹方向 |

### 5.2 Fib 通道

辅助指标，不判方向和点位，用于**认知大局**。

| 价格位置 | 缩量 | 放量 |
|----------|------|------|
| 通道上轨 | 大概率回落→做空 | 观望确认 |
| 通道下轨 | 大概率反弹→做多 | 观望确认 |
| 通道中运行 | 持仓不动 | 持仓不动 |

## 六、思维框架：绝对逻辑与随机博弈

### 绝对逻辑

可做出绝对判断的逻辑。判断涨跌时，**不堆砌证据，而寻找是否有遗漏的关键因素**——条件是否完备？是否真的"打开了一只盒子"？

- 交易中不存在绝对理性判断，但可以做到**绝对理性的开平仓**

### 随机博弈

市场大部分时间处于随机状态，不针对任何个体。意识到随机性，才能控制心态。

### 决策映射

| 状态 | 条件 | 应对 |
|------|------|------|
| **绝对逻辑区** | fib关键位 + 量能明确 + 时间节点到位 | 按规则开仓 |
| **随机博弈区** | 条件不完备 | **不交易** |

## 七、风控体系

### 7.1 仓位规则

| 信号强度 | 仓位 |
|----------|------|
| 三要素共振 | 10% |
| 两要素共振 | 5% |
| 仅一要素 | 不开仓 |

- ✅ 顺势加仓：盈利后回撤至关键位，总仓≤30%
- ❌ 逆势加仓、满仓操作：**严禁**
- 减仓：下一fib关键位 + 量能减弱 → 减半仓

### 7.2 止损

- **硬止损**：开仓时设定，1%~2%必止——**开单前先设止损**
- **逻辑止损**：开仓依据被破坏（如反向fib关键位突破+放量）

### 7.3 止盈

| 条件 | 操作 |
|------|------|
| 价格到达下一fib关键位 + 缩量 | 止盈半仓，剩余推止损到成本价 |
| 通道边缘 + 量能异常 | 止盈全仓 |
| 三要素任一反向 | 止盈全仓 |

**止盈目标位**：顺势单目标为**下一级fib关键位**——0.382入场→目标0.618；0.618入场→目标1.0（前高）或通道上轨。

## 八、不开仓场景清单

> 知道何时不开仓，比何时开仓更重要。

| 场景 | 原因 |
|------|------|
| 仅一要素指向，其余未确认 | 共振不足，随机博弈 |
| 价格在0.5附近 / 0.618~0.786之间 | 震荡区，无明确方向 |
| 4H主量能方向不明 | 大趋势未定 |
| 价格已到回撤位但时间节点未到 | 时机未到，耐心等待 |
| fib趋势时间到达但价格不在关键位 | 时间信号缺乏空间确认 |
| 量价背离 | 假突破风险 |

## 九、开仓决策流程

```
定方向（4H主次量能）
  M指向何方？ H/L是否夯实？  →  方向不明 → 不交易
      ↓ 方向明确
找点位（fib回撤 + 量能精进）
  在0.618/0.382附近？  不在0.618~0.786中间？
  K柱绝对/相对强势？  →  不在关键位/平庸K柱 → 等待
      ↓ 价格到位
确认时机（fib趋势时间）
  竖线附近？0.618高发区？  →  未到 → 等待
      ↓ 时机到达
共振确认（绝对逻辑检验）
  至少两要素同向？条件完备（非随机博弈区）？
  大小级别fib是否冲突？  →  否/冲突 → 不开仓
      ↓ 确认
执行开仓
  仓位：三共振10% / 两共振5%
  止损：同时设定硬止损1%~2%（开单前先设）
  止盈：目标下一fib关键位 + 缩量条件
      ↓ 持仓中
持仓管理
  趋势确认：价格按预期方向运行 + 量能持续 → 推止损到成本
  趋势失败：价格回落破开仓位 + 量能放大 → 逻辑止损离场
  通道中运行 → 持仓不动
  量能精进判K柱强弱 → 短周期调仓
  引力过载后缩量 → 加仓/减仓点
  三要素任一反向 → 止盈全仓
```
