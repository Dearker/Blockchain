## 2. 验证口径

《外汇超短线交易》的原版场景是 EUR/USD 的70笔交易图。为了避免把书中叙事直接套到BTC，本次将其拆成可检验代理：

- 20EMA：只验证其作为趋势过滤器的效果，不把它当支撑阻力。
- DD：顺势、近20EMA、连续2根小实体K线后突破。
- FB：强冲击后的第一次40%-75%回撤，再向原趋势突破。
- SB：回撤低点/高点被二次测试后，再向原趋势突破。
- BB/RB：20根区间突破，并区分是否出现“挤压”。
- 假突破：刺破区间边界后收回，随后反向突破。
- ARB：区间突破后回测边线，再向突破方向恢复。
- 结果口径：以入场K线收盘价为代理入场价，目标=1R，止损=1R，R由结构距离、ATR和最小价格波动共同确定；同一K线同时触及止盈止损时保守计为亏损。

## 6. 固定10点规则在BTC上的适配性

| 项目 | 数值 |
| --- | --- |
| 5m中位单K振幅 | 112.010 USDT |
| 中位振幅 / 10 USDT | 11.201 |
| 20根突破样本 | 24980 |
| 同一K线止盈止损顺序不可判定比例 | 63.03% |
| 保守口径胜率 | 15.53% |

解读：EUR/USD的10点不能机械换成BTC的10 USDT。BTC在5m内的常规波动已经显著大于10 USDT，普通K线又无法判断先触及止盈还是止损，因此固定10点规则在BTC上必须改为“结构止损 + 1R目标 + ATR下限”。

## 9. 最终取舍

适合进入BTC可复用方法论的内容：

- 保留“价格行为优先 + 20EMA过滤”的思想，但EMA只做背景，不做单独入场理由。
- 保留“双重压力”的核心：突破必须发生在对手盘可能止损、顺势盘可能入场的位置。
- 保留DD、SB、BB/RB、ARB等结构作为候选模板，但必须改写为可量化检查项，并通过高周期、空间和风险过滤后才可交易。
- 保留“错过不追、等待回撤再访问突破位”的纪律。
- 保留临界点技术，改为结构止损和逐步移动止损。
- 保留每笔最大风险模型，但BTC超短线建议从0.25%-1%开始，而不是默认2%。

需要剔除或降级的内容：

- 70-tick图不能由普通K线证明；若要原版复刻，需要逐笔成交数据。
- 固定10点止盈止损不适合BTC，必须改为结构R或ATR。
- 不验证市价单、一键执行、OCO、点差和滑点；这些属于交易系统和成交质量问题。
- “7种结构高胜率”不能被当作无条件结论；结构必须叠加趋势、蓄积、空间和风险收益比。
- 复利增长示例不能当收益承诺，BTC超短线要先控制连续亏损和手续费侵蚀。
